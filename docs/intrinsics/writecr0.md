# <a name="writecr0"></a>__writecr0
**Microsoft Specific**  
  
 Zapíše hodnota `Data` k CR0 registrace.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
void writecr0(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 [in] `Data`  
 Hodnota určená k zápisu do registru CR0.
