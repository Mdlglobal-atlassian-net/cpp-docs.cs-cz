# <a name="outdword"></a>__outdword
**Microsoft Specific**  
  
 Generuje `out` pokyn k odeslání doubleword `Data` z portu `Port`.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
void __outdword(   
   unsigned short Port,   
   unsigned long Data   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 [in] `Port`  
 Port pro odesílání dat na.  
  
 [in] `Data`  
 Doubleword k odeslání.
