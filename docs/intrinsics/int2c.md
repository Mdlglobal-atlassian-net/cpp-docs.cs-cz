# <a name="int2c"></a>__int2c

**Specifické pro Microsoft**

Generuje `int 2c` instrukce, která aktivuje `2c` přerušení.

## <a name="syntax"></a>Syntaxe

```
void __int2c(void);
```

## <a name="requirements"></a>Požadavky

|Vnitřní|Architektura|
|---------------|------------------|
|`__int2c`|x86, x64|
