# <a name="halt"></a>__halt
**Microsoft Specific**  
  
 Vašem mikroprocesoru a používané zastaví, dokud nenastane povoleno přerušení, nemaskovatelné přerušení (NMI) nebo provést obnovení.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
void __halt( void );  
```
