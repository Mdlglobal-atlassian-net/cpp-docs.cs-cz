# <a name="punctuation-and-special-characters"></a>Interpunkce a speciální znaky

Interpunkční znaménka a speciální znaky v sadě znaků jazyka C mají různá použití, od uspořádání textu programu k definování úkolů, které kompilátor nebo zkompilovaný program provede. Neurčují operaci, která má být provedena. Některé symboly interpunkce jsou také operátory (viz [operátory](../c-language/c-operators.md)). Kompilátor určí jejich použití z kontextu.

## <a name="syntax"></a>Syntaxe

`punctuator`: jedna z **() [] {} \* ,: =;   ... #**

Tyto znaky mají zvláštní významy v jazyce C. Jejich použití jsou popsána v celé této příručce. Křížové znaménko**#**() se může vyskytovat pouze v [direktivách předzpracování](../preprocessor/preprocessor-directives.md).
