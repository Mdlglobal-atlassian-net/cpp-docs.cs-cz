# <a name="signal-function-c"></a>signal – funkce (C)
**ANSI 4.7.1.1** sadu signály pro **signál** – funkce  
  
 První argument předaný **signál** musí mít jednu z symbolický konstanty popsané v *referenční dokumentace běhové knihovny* pro **signál** funkce. Informace v *referenční dokumentace běhové knihovny* také uvádí operační režim podpory pro každý signál. Konstanty jsou definovány také v SIGNAL.H.
