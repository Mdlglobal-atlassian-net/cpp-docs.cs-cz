# <a name="initialization"></a>Inicializace

„Inicializátor“ je hodnota nebo posloupnost hodnot, která má být přiřazena k deklarované proměnné. Proměnnou lze nastavit na počáteční hodnotu použitím inicializátoru na deklarátor v deklaraci proměnné. Hodnota nebo hodnoty inicializátoru jsou přiřazeny proměnné.

Následující části popisují, jak inicializovat proměnné [skalárních](../c-language/initializing-scalar-types.md), [agregačních](../c-language/initializing-aggregate-types.md)a [řetězcových](../c-language/initializing-strings.md) typů. „Skalární typy“ zahrnují všechny aritmetické typy a ukazatele. „Agregační typy“ zahrnují pole, struktury a sjednocení.
