# <a name="qualifiers-access-to-volatile-objects"></a>Kvalifikátory: přístup k nestálým objektům

**ANSI 3.5.5.3** co představuje přístup k objektu, který má kvalifikovaný typ volatile

Jakýkoli odkaz na kvalifikovaný typ volatile představuje přístup k programu.
