# <a name="character-sets"></a>Znakové sady

**3.1.3.4 ANSI** Mapování členů zdrojové znakové sady

Zdrojová znaková sada a spouštěcí znaková sada obsahuje znaky ASCII, které jsou uvedeny v následující tabulce. Řídicí sekvence jsou také zobrazeny v tabulce.

## <a name="escape-sequences"></a>Řídicí sekvence

|Řídicí sekvence|Znak|Hodnota ASCII|
|---------------------|---------------|-----------------|
|&#92;|Alarm/zvonek|7|
|&#92;b|Backspace|8|
|&#92;f|Informační kanál formuláře|12|
|&#92;n|Nového|10|
|&#92;r|Návrat na začátek řádku|13|
|&#92;t|Horizontální tabulátor|9|
|&#92;v|Vertikální tabulátor|11|
|&#92;"|Dvojité uvozovky|34|
|&#92;'|Jednoduché uvozovky|39|
|&#92;&#92;|Zpětné lomítko|92|
