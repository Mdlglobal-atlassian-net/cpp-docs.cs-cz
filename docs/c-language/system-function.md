**ANSI 4.10.4.5** obsah a režim spuštění řetězce, pomocí **systému** – funkce  
  
 **Systému** funkce provede příkaz interní operačního systému nebo. SOUBOR EXE, .COM (. CMD v systému Windows NT) nebo. A. BAT soubor z v rámci programu C spíše než z příkazového řádku.  
  
 Funkce system nalezne překladač příkazu, kterým je obvykle soubor CMD.EXE v operačním systému Windows NT nebo COMMAND.COM v systému Windows. Funkce system poté předá řetězec argumentu překladači příkazu.
