# <a name="remainders"></a>Připomenutí
**ANSI 3.3.5** znaménko zbývající na celočíselné dělení  
  
 Znaménko zbývající je stejný jako znaménko dělenec. Například  
  
```  
 50 / -6 == -8  
 50 % -6 ==  2  
-50 /  6 == -8  
-50 %  6 == -2  
```
