# <a name="remainders"></a>Připomenutí

**ANSI 3.3.5** znaménko zbývající na celočíselné dělení

Znaménko zbytek je stejný jako znaménko podíl. Například

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
