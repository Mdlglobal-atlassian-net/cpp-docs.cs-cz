# <a name="remainders"></a>Připomenutí

**3.3.5 ANSI** Znaménko zbytku pro dělení celého čísla

Znaménko zbytku je stejné jako znaménko dividendy. Například:

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
