# <a name="bits-per-character"></a>Bity na znak
**ANSI 2.2.4.2.1** počet bitů v znak  
  
 Počet bitů v znak je reprezentována manifestu konstanta **char_bit –**. OMEZENÍ. Soubor H definuje **char_bit –** jako 8.
