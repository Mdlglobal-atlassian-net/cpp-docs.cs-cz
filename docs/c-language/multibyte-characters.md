# <a name="multibyte-characters"></a>Vícebajtové znaky
**ANSI 2.2.1.2** posunutí stavy vícebajtové znaky  
  
 Vícebajtové znaky se používají v některých implementacích včetně jazyka Microsoft C k reprezentaci znaků cizího jazyka, které nejsou reprezentovány v základní znakové sadě. Jazyk Microsoft C však nepodporuje žádná kódování závislá na stavu. Proto nejsou k dispozici žádné stavy posunu. V tématu [vícebajtové a široké znaky](../c-language/multibyte-and-wide-characters.md) Další informace.
