# <a name="renaming-with-a-name-that-exists"></a>Přejmenování s názvem, který existuje.
**ANSI 4.9.4.2** vliv, pokud existuje soubor s novým názvem před volání **přejmenovat** – funkce  
  
 Pokud se pokusíte chcete přejmenovat soubor pomocí názvu, který již existuje, **přejmenovat** funkce selže a vrátí kód chyby.
