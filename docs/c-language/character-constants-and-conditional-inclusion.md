# <a name="character-constants-and-conditional-inclusion"></a>Konstanty a podmíněné zahrnutí znaků

**3.8.1 ANSI** Určuje, zda hodnota konstanty znaku s jedním znakem v konstantním výrazu, který řídí podmíněné zahrnutí, odpovídá hodnotě stejné znakové konstanty ve znakové sadě spuštění. Informace, zda taková znaková konstanta může mít zápornou hodnotu

Znaková sada použitá v příkazech preprocesoru je stejná jako znaková sada spuštění. Preprocesor rozpoznává záporné hodnoty znaků.
