# <a name="range-of-char-values"></a>Rozsah hodnot typu char
**ANSI 3.2.1.1** zda "plain" **char** má stejný rozsah hodnoty jako **podepsané char** nebo `unsigned char`  
  
 Všechny podepsané rozsahu hodnot znaků z -128 127 znaků. Všechny hodnoty znaků bez znaménka mají rozsah od 0 do 255.  
  
 Možnost kompilátoru /J změní výchozí hodnoty **podepsané** k `unsigned`.
