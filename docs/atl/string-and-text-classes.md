# <a name="string-and-text-classes"></a>Řetězcové a textové třídy

Tyto třídy poskytují podporu pro řetězce a textový řetězec převody.

- [Ca2aex –](../atl/reference/ca2aex-class.md) Tato třída se používá makra převodu řetězců CA2TEX CT2AEX a definice typedef CA2A.

- [Ca2caex –](../atl/reference/ca2caex-class.md) Tato třída se používá řetězec převodních maker CA2CTEX CT2CAEX a definice typedef CA2CA.

- [Ca2wex –](../atl/reference/ca2wex-class.md) tuto třídu používají makra převodu řetězců CA2TEX, CA2CTEX, CT2WEX a CT2CWEX a definice typedef CA2W.

- [Cw2aex –](../atl/reference/cw2aex-class.md) tuto třídu používají makra převodu řetězců CT2AEX, CW2TEX, CW2CTEX a CT2CAEX a definice typedef CW2A.

- [Cw2cwex –](../atl/reference/cw2cwex-class.md) Tato třída se používá makra převodu řetězců CW2CTEX CT2CWEX a definice typedef CW2CW.

- [Cw2wex –](../atl/reference/cw2wex-class.md) Tato třída se používá makra převodu řetězců CW2TEX CT2WEX a definice typedef CW2W.

- [CComBSTR](../atl/reference/ccombstr-class.md) Tato třída představuje obálku pro datových typů BSTR.

- [_U_stringorid –](../atl/reference/u-stringorid-class.md) tuto třídu adaptéru argument umožňuje buď názvy prostředků (`LPCTSTR`s) nebo ID (uvedený), které se mají předat funkci bez nutnosti volajícího k převodu na řetězec za použití makra MAKEINTRESOURCE ID prostředků.
