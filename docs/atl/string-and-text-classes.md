# <a name="string-and-text-classes"></a>Třídy Text a řetězce
Tyto třídy poskytují podporu pro řetězce a převody textový řetězec.  
  
-   [CA2AEX](../atl/reference/ca2aex-class.md) Tato třída se používá ve makra převodů řetězec `CA2TEX` a `CT2AEX`a typedef **CA2A**.  
  
-   [CA2CAEX](../atl/reference/ca2caex-class.md) Tato třída se používá ve makra převodů řetězec `CA2CTEX` a `CT2CAEX`a typedef **CA2CA**.  
  
-   [CA2WEX](../atl/reference/ca2wex-class.md) Tato třída se používá ve makra převodů řetězec `CA2TEX`, `CA2CTEX`, `CT2WEX`, a `CT2CWEX`a typedef **CA2W**.  
  
-   [CW2AEX](../atl/reference/cw2aex-class.md) Tato třída se používá ve makra převodů řetězec `CT2AEX`, `CW2TEX`, `CW2CTEX`, a `CT2CAEX`a typedef **CW2A**.  
  
-   [CW2CWEX](../atl/reference/cw2cwex-class.md) Tato třída se používá ve makra převodů řetězec `CW2CTEX` a `CT2CWEX`a typedef **CW2CW**.  
  
-   [CW2WEX](../atl/reference/cw2wex-class.md) Tato třída se používá ve makra převodů řetězec `CW2TEX` a `CT2WEX`a typedef `CW2W`.  
  
-   [CComBSTR](../atl/reference/ccombstr-class.md) Tato třída je obálka pro `BSTR`s.  
  
-   [_U_STRINGorID](../atl/reference/u-stringorid-class.md) Tato třída argument adaptér umožňuje buď názvy prostředků (`LPCTSTR`s) nebo ID prostředku (**Celé_číslo**s) mají být předány funkce bez nutnosti volající převést na řetězec pomocí ID **MAKEINTRESOURCE** makro.
