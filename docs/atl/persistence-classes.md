# <a name="persistence-classes"></a>Trvalost třídy
Následující třídy implementovat trvalost objektu:  
  
-   [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) umožňuje klientům pro načtení a uložit vlastnosti objektu do kontejneru objektů.  
  
-   [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) umožňuje klientům pro načtení a uložení objektu trvalé dat do datového proudu.  
  
-   [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) umožňuje klientům pro načtení a uložit do úložiště dat trvalého objektu.
