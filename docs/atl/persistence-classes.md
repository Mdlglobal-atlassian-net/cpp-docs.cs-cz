# <a name="persistence-classes"></a>Třídy trvalosti

Následující třídy implementaci trvalého objektu:

- [Ipersistpropertybagimpl –](../atl/reference/ipersistpropertybagimpl-class.md) umožňuje klientovi k načtení a uložení vlastností objektu do kontejneru objektů.

- [Ipersiststreaminitimpl –](../atl/reference/ipersiststreaminitimpl-class.md) umožňuje klientovi k načtení a uložení objektu trvalá data do datového proudu.

- [Ipersiststorageimpl –](../atl/reference/ipersiststorageimpl-class.md) umožňuje klientovi k načtení a uložení objektu trvalá data do úložiště.
