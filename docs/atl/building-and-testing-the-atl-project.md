# <a name="building-and-testing-the-atl-project"></a>Vytváření a testování projektu ATL

Jak je uvedeno v [vložení složeného ovládacího prvku](../atl/inserting-a-composite-control.md), jednu počáteční součást projektu je výchozí stránku HTML, který je hostitelem nové složeného ovládacího prvku. Po dokončení úprav složeného ovládacího prvku, klikněte na tlačítko **sestavit řešení** nebo **znovu sestavit řešení** z **sestavení** nabídky. Po úspěšném sestavení projektu, načtení stránky HTML, který je umístěný v kořenovém adresáři vašeho projektu složeného ovládacího prvku do aplikace Internet Explorer nebo jiný prohlížeč a otestovat funkci ovládacího prvku.

Můžete také otestovat složeného ovládacího prvku pomocí nástroje Test kontejneru nebo jakékoli jiné aplikace, který může hostovat ovládací prvek ActiveX. Zobrazit [testování vlastností a událostí pomocí testování kontejneru](../mfc/testing-properties-and-events-with-test-container.md) informace o tom, jak získat přístup ke kontejneru testů.
