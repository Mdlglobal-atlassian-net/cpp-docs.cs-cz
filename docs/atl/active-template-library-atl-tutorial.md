# <a name="active-template-library-atl-tutorial"></a>Tutoriál knihovny ATL (Active Template Library)

Knihovna ATL je navržené pro zjednodušení procesu vytvoření efektivní, flexibilní a jednoduché ovládací prvky. Tento kurz vás provede vytvořením ovládacího prvku ActiveX, ukázka mnoho základy ATL a modelu COM.

Podle tohoto kurzu se dozvíte, jak přidat ovládací prvek do projektu ATL, které kreslí kruh a plného mnohoúhelníku. Potom přidáte vlastnost označující počet stran mnohoúhelník bude mít a vytvořit kód pro vykreslování. pro aktualizaci ovládací prvek při změně vlastnosti. Ovládací prvek se pak zobrazí na webové stránce pomocí některé jazyk VBScript k němu reagovat na události.

Tento kurz je rozdělen do sedmi kroků. Každý krok je třeba provést v pořadí podle dalších krocích závisí na dříve dokončených úloh. Než začnete, by měl potvrzujete, že máte oprávnění požadovaná k registraci ovládacího prvku ActiveX na konkrétní počítač. Toto je obvykle pouze žádný problém, pokud používáte Visual Studio .NET přes připojení Terminálové služby.
