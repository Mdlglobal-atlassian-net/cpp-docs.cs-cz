# <a name="active-template-library-atl-tutorial"></a>Tutoriál knihovny ATL (Active Template Library)
ATL slouží ke zjednodušení procesu vytváření efektivních, flexibilní a lightweight ovládacích prvků. Tento kurz vás provede vytvoření ovládacího prvku ActiveX demonstraci mnoho základy ATL a COM.  
  
 Podle tohoto kurzu se dozvíte, jak přidání ovládacího prvku do projektu knihovny ATL, který se vykreslí kruh a vyplněné mnohoúhelníku. Potom přidáte vlastnost označující, kolik stranu mnohoúhelníku bude mít a vytvořit kreslení kód pro aktualizaci prvku při změně vlastnosti. Ovládací prvek se pak zobrazí na webové stránce pomocí některé VBScript ho reakce na události.  
  
 Tento kurz je rozdělené do sedmi kroků. Každý krok je třeba provést v pořadí podle dalších krocích závisí na dříve dokončené úlohy. Než začnete, měli potvrďte, zda máte oprávnění k registraci ovládacího prvku ActiveX na konkrétním počítači požadován. Toto je obvykle pouze o problém, pokud používáte Visual Studio .NET přes připojení Terminálové služby.
