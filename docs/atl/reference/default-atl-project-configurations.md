# <a name="default-atl-project-configurations"></a>Výchozí konfigurace projektu ATL
Průvodce projektem ATL ve výchozím nastavení vytvoří dvě konfigurace projektu:  
  
|Konfigurace|Znaková sada|Použití knihovny ATL|  
|-------------------|-------------------|----------------|  
|Vydaná verze|MBCS|DLL|  
|Ladit|MBCS|DLL|  
  
 **Znaková sada**, **použití knihovny ATL** a se dají změnit v **nastavení projektu** dialogového okna v části **Obecné** kartu. Můžete také přidat vlastní konfigurace pomocí nástroje Configuration Manager. Podrobnosti najdete v tématu [sestavení konfigurace](/visualstudio/ide/understanding-build-configurations).
