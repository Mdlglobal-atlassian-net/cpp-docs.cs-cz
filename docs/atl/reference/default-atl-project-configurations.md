# <a name="default-atl-project-configurations"></a>Výchozí konfigurace projektu knihovny ATL
Průvodce projektem ATL vytvoří dvě konfigurace projektu ve výchozím nastavení:  
  
|Konfigurace|Znakové sady|Použití knihovny ATL|  
|-------------------|-------------------|----------------|  
|Vydaná verze|MBCS|DLL|  
|Ladit|MBCS|DLL|  
  
 **Znaková sada**, **použití knihovny ATL** a se dají změnit v **nastavení projektu** dialogové okno pod **Obecné** kartě. Můžete také přidat vlastní konfiguraci pomocí nástroje Configuration Manager. Podrobnosti najdete v tématu [sestavení konfigurace](/visualstudio/ide/understanding-build-configurations).
