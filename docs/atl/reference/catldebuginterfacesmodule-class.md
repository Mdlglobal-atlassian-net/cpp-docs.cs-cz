# <a name="catldebuginterfacesmodule-class"></a>CAtlDebugInterfacesModule – třída

Tato třída poskytuje podporu pro rozhraní ladění.

## <a name="syntax"></a>Syntaxe

```cpp
class CAtlDebugInterfacesModule
```

## <a name="remarks"></a>Poznámky

`CAtlDebugInterfacesModule`poskytuje podporu požadovanou pro ladicí rozhraní. Je zahrnut v jakémkoli projektu, který definuje symbol [_ATL_DEBUG_QI](debugging-and-error-reporting-macros.md#_atl_debug_qi).

## <a name="requirements"></a>Požadavky

**Záhlaví:** atlbase. h
