# <a name="strings-atl-property-page-wizard"></a>Řetězce, Průvodce stránky vlastností knihovny ATL
Poskytuje text související s jeho stránku vlastností.  
  
 **Název**  
 Nastaví text, který se zobrazí na kartě stránku vlastností.  
  
 **Popisný řetězec**  
 Nastaví textový řetězec s popisem stránky. Tento řetězec lze zobrazit v dialogovém okně Vlastnosti list. Rámec vlastností může použít popis ve stavu řádku nebo v popisu tlačítka. Standardní rámec vlastností aktuálně nepoužívá tento řetězec.  
  
 **Soubor nápovědy**  
 Nastaví název souboru nápovědy, která popisuje, jak používat stránky vlastností. Tento název nesmí obsahovat cestu. Když uživatel stiskne **pomoci**, rámec otevře soubor nápovědy v adresáři s názvem v hodnotě klíče HelpDir v položkách registru stránky vlastností pod jeho CLSID.
