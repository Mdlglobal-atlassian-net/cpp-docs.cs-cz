# <a name="strings-atl-property-page-wizard"></a>Řetězce, Průvodce stránkou vlastností ATL

Obsahuje text přidružený k stránky vlastností.

- **Název**

   Nastaví text, který se zobrazí na kartě stránky vlastností.

- **Řetězec doc**

   Nastaví textový řetězec s popisem stránky. Tento řetězec lze zobrazit v dialogovém okně Vlastnosti list. Rámec vlastnosti použít popis v popisu stavu řádku nebo nástroje. Snímek standardní vlastnost právě nepoužívá tento řetězec.

- **Soubor nápovědy**

   Nastaví název souboru nápovědy, který popisuje způsob použití na stránce vlastností. Tento název by neměl obsahovat cestu. Když uživatel stiskne **pomáhají**, rámec otevře soubor nápovědy v adresáři s názvem v hodnotě klíče HelpDir v položkách registru stránky vlastností v rámci jeho identifikátor CLSID.
