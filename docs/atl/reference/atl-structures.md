# <a name="atl-structures"></a>Struktury knihovny ATL


Aktivní knihovna šablon zahrnuje následující struktury.  
  
## <a name="in-this-section"></a>V tomto oddílu  
 [ATL_DRAWINFO](../../atl/reference/atl-drawinfo-structure.md)  
 Obsahuje informace, které slouží pro vykreslení na různé cíle, jako jsou tiskárny, metafile nebo ovládací prvek ActiveX.  
  
 [_AtlCreateWndData](../../atl/reference/atlcreatewnddata-structure.md)  
 Obsahuje data instance třídy v oddílová kódu v ATL.  
  
 [_ATL_BASE_MODULE70](../../atl/reference/atl-base-module70-structure.md)  
 Použít žádný projekt, který používá ATL.  
  
 [_ATL_COM_MODULE70](../../atl/reference/atl-com-module70-structure.md)  
 Používaný kód související s COM v ATL.  
  
 [_ATL_FUNC_INFO](../../atl/reference/atl-func-info-structure.md)  
 Obsahuje informace o typu, které používají k popisu metody nebo vlastnosti na odesílajícím rozhraním.  
  
 [_ATL_MODULE70](../../atl/reference/atl-module70-structure.md)  
 Obsahuje data využívaná každých ATL modulu.  
  
 [_ATL_WIN_MODULE70](../../atl/reference/atl-win-module70-structure.md)  
 Používaný kód oddílová v ATL.
