# <a name="options-atl-property-page-wizard"></a>Možnosti Průvodce stránky vlastností knihovny ATL
Na této stránce průvodce můžete definovat vláken modelu a agregaci úroveň stránka vlastností, kterou vytváříte.  
  
 **Model vláken**  
 Určuje model vláken používá jeho stránku vlastností.  
  
 V tématu [nastavení modelu vláken pro projekt](../../atl/specifying-the-threading-model-for-a-project-atl.md) Další informace.  
  
|Možnost|Popis|  
|------------|-----------------|  
|`Single`|Stránka vlastností běží jenom v primární COM vlákno.|  
|**Objektu Apartment**|Stránka vlastností lze vytvořit v jakémkoliv jeden vlákně. Výchozí nastavení|  
  
 **Agregace**  
 Přidá podporu agregace pro stránku vlastností, kterou vytváříte. V tématu [agregace](../../atl/aggregation.md) Další informace.  
  
|Možnost|Popis|  
|------------|-----------------|  
|**Ano**|Vytvořte stránku vlastností, která se dají agregovat.|  
|**Ne**|Vytvořte stránku vlastností, která se nedá agregovat.|  
|**Pouze**|Vytvořte stránku vlastností, která může být vytvořená pouze prostřednictvím agregace.|
