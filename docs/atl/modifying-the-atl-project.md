# <a name="modifying-the-atl-project"></a>Úprava projektu ATL

V tomto okamžiku projektu složený ovládací prvek implementuje nezbytných objektů pro složeného ovládacího prvku. Dalším krokem je přidání všech ovládacích prvků, že bude obsahovat a zpracovat všechny potřebné události složeného ovládacího prvku.

Chcete-li přidat další ovládací prvky ActiveX nebo Windows, přidejte nový skript prostředků a potom použít editor dialogového okna. Další informace o přidávání ovládacích prvků (a související úlohy) v tématu [editoru dialogového okna](../windows/dialog-editor.md).

Zpracovat všechny potřebné události z ovládacích prvků ActiveX naleznete v tématu [přidávání funkcí do složeného ovládacího prvku](../atl/adding-functionality-to-the-composite-control.md).
