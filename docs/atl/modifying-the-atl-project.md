# <a name="modifying-the-atl-project"></a>Úprava projektu knihovny ATL
V tomto okamžiku projektu složeného ovládacího prvku implementuje nezbytných objektů pro vaše složeného ovládacího prvku. Dalším krokem je přidání všech ovládacích prvků, že bude obsahovat a zpracovat všechny potřebné události složeného ovládacího prvku.  
  
 Chcete-li přidat další ovládací prvky ActiveX nebo Windows, přidejte nový skript prostředků a pak použijte editoru dialogových oken. Další informace o přidání ovládacích prvků (a související úlohy) najdete v tématu [editoru dialogového okna](../windows/dialog-editor.md).  
  
 Pokud chcete zpracovat všechny potřebné události z ovládacích prvků ActiveX, přečtěte si téma [přidání funkce složeného ovládacího prvku](../atl/adding-functionality-to-the-composite-control.md).
