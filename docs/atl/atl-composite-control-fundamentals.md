# <a name="atl-composite-control-fundamentals"></a>ATL – principy vytváření složených prvků

Složený ovládací prvek je typu ovládacího prvku ActiveX, který může obsahovat (podobně jako do dialogového okna) jiné ovládací prvky ActiveX nebo ovládací prvky Windows. Jakmile je sestavená složeného ovládacího prvku, lze vložit všude, kde je možné hostovat ovládací prvek ActiveX.

Průvodce projektem ATL a **přidat třídu** dialogové okno automatizovat proces vytvoření a implementace projektu složených ovládacích prvků, podobně jako výsledek spuštění Průvodce aplikací k vytvoření aplikační rozhraní knihovny MFC. Proces vývoje se skládá z pěti kroků:

- [Vytvoření projektu ATL](../atl/reference/creating-an-atl-project.md)

- [Vložení složeného ovládacího prvku](../atl/inserting-a-composite-control.md)

- [Úprava projektu ATL](../atl/modifying-the-atl-project.md)

- [Přidání funkcí do složeného ovládacího prvku](../atl/adding-functionality-to-the-composite-control.md)

- [Vytváření a testování projektu ATL](../atl/building-and-testing-the-atl-project.md)
