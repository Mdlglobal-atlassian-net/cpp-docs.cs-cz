# <a name="inserting-a-composite-control"></a>Vkládání složeného ovládacího prvku
**Přidat třídu** dialogové okno umožňuje vložit objekt knihovny ATL do projektu. Přístup k toto dialogové okno kliknutím pravým tlačítkem myši na název projektu v Průzkumníku řešení, přejdete na příkaz **přidat**a pak levým na **přidat třídu**.  
  
 V **přidat třídu** dialogovém okně vyberte **ovládací prvek ATL**. Tato akce spustí [Průvodce ovládacím prvkem ATL](../atl/reference/atl-control-wizard.md). K vytvoření složeného ovládacího prvku, vyberte **možnosti** a klikněte **složeného ovládacího prvku** zaškrtávací políčko.  
  
 Vytvoří se výchozí stránku HTML pro zobrazení ovládacího prvku.
