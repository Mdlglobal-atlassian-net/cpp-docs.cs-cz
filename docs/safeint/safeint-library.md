# <a name="safeint-library"></a>SafeInt – knihovna

**SafeInt** je Přenosná knihovna, kterou lze použít s MSVC, RSZ nebo Clang, aby se zabránilo přetečení celého čísla, které by mohlo vést k tomu, když aplikace provádí matematické operace. Nejnovější verzi této knihovny najdete na adrese [https://github.com/dcleblanc/SafeInt](https://github.com/dcleblanc/SafeInt).
