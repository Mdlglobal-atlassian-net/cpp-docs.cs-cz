# <a name="jump-statements-c"></a>Jump – příkazy (C++)

Příkaz skoku jazyka C++ provádí okamžitý místní přenos řízení.

## <a name="syntax"></a>Syntaxe

```
break;
continue;
return [expression];
goto identifier;
```
