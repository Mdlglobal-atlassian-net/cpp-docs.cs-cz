# <a name="bstrtdetach"></a>_bstr_t::Detach
**Konkrétní Microsoft**  
  
 Vrátí `BSTR` zabalené službou `_bstr_t` a odpojí `BSTR` z `_bstr_t`.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
BSTR Detach( ) throw;  
  
```  
  
## <a name="return-value"></a>Návratová hodnota  
 `BSTR` zabalený pomocí `_bstr_t`.
