# <a name="__w64"></a>__w64

Toto klíčové slovo specifické pro společnost Microsoft je zastaralé. Ve verzích sady Visual Studio starších než Visual Studio 2013 to umožňuje označit proměnné, takže když kompilujete pomocí [/Wp64](../build/reference/wp64-detect-64-bit-portability-issues.md) , kompilátor bude hlásit všechna upozornění, která by byla ohlášena, pokud jste se pokoušeli kompilovat pomocí 64 kompilátoru.

## <a name="syntax"></a>Syntaxe

> *typ* **__w64** *identifikátoru*

### <a name="parameters"></a>Parametry

*type*<br/>
Jeden ze tří typů, který by mohl způsobit problémy v kódu z 32 až do 64 kompilátoru: **int**, **Long**nebo ukazatel.

*RID*<br/>
Identifikátor vytvářené proměnné.

## <a name="remarks"></a>Poznámky

> [!IMPORTANT]
>  Možnost kompilátoru [/Wp64](../build/reference/wp64-detect-64-bit-portability-issues.md) a klíčové slovo **__W64** jsou v aplikaci Visual Studio 2010 a Visual Studio 2013, které byly odebrány počínaje Visual Studio 2013, zastaralé. Použijete-li možnost kompilátoru `/Wp64` v příkazovém řádku, vyvolá kompilátor upozornění příkazového řádku D9002. Klíčové slovo **__w64** se tiše ignoruje. Místo použití této možnosti a klíčového slova k detekci 64 potíží s přenositelností použijte kompilátor C++ od Microsoftu, který se zaměřuje na 64 platformu. Další informace najdete v tématu [Konfigurace vizuálu C++ pro 64 cíle platformy x64](../build/configuring-programs-for-64-bit-visual-cpp.md).

Všechny definice typedef, které mají **__w64** , musí být 32 bitů na platformě x86 a 64 bitů na platformě x64.

Chcete-li zjistit problémy s přenositelností pomocí verzí C++ kompilátoru Microsoft staršího než Visual Studio 2010, klíčové slovo **__w64** by mělo být zadáno v jakémkoli definice typedef, které mění velikost mezi 32 bitovými a 64 bitovými platformami. Pro jakýkoliv takový typ **__w64** musí být uveden pouze na 32 definice typedef.

Z důvodu kompatibility s předchozími verzemi je **_w64** synonymem pro **__w64** , pokud je zadána možnost kompilátoru [/za \(Disable Language Extensions)](../build/reference/za-ze-disable-language-extensions.md) .

Klíčové slovo **__w64** je ignorováno, pokud kompilace nepoužívá `/Wp64`.

Další informace o převodu na 64 bitů naleznete v následujících tématech:

- [Parametry kompilátoru MSVC](../build/reference/compiler-options.md)

- [Přenos kódu 32-bit do 64 bitového kódu](../build/common-visual-cpp-64-bit-migration-issues.md)

- [Konfigurace Visual C++ pro 64bitové cíle x64](../build/configuring-programs-for-64-bit-visual-cpp.md)

## <a name="example"></a>Příklad

```cpp
// __w64.cpp
// compile with: /W3 /Wp64
typedef int Int_32;
#ifdef _WIN64
typedef __int64 Int_Native;
#else
typedef int __w64 Int_Native;
#endif

int main() {
   Int_32 i0 = 5;
   Int_Native i1 = 10;
   i0 = i1;   // C4244 64-bit int assigned to 32-bit int

   // char __w64 c;  error, cannot use __w64 on char
}
```
