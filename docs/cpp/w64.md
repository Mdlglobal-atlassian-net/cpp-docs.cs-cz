# <a name="w64"></a>__w64

**Specifické pro Microsoft** 

Toto klíčové slovo je zastaralý. Ve verzích sady Visual Studio starší než Visual Studio 2013, díky tomu můžete označit proměnné, takže pokud kompilujete s [/Wp64](../build/reference/wp64-detect-64-bit-portability-issues.md) kompilátor oznámí všechna upozornění, která by byla nahlášena při kompilaci s 64bitovým kompilátorem.  
  
## <a name="syntax"></a>Syntaxe  
  
> *typ* **__w64** *identifikátor*  
 
### <a name="parameters"></a>Parametry  

*Typ*  
 Jeden ze tří typů, které by mohly způsobit problémy v kódu při přenosu z 32-bit do 64bitového kompilátoru: **int**, **dlouhé**, nebo ukazatel.  
  
*identifikátor*  
Identifikátor vytvářené proměnné.  
  
## <a name="remarks"></a>Poznámky  
  
> [!IMPORTANT]
>  [/Wp64](../build/reference/wp64-detect-64-bit-portability-issues.md) – možnost kompilátoru a **__w64** – klíčové slovo se považují za zastaralé v sadě Visual Studio 2010 a Visual Studio 2013 a odebrat, spouští se v sadě Visual Studio 2013. Pokud používáte `/Wp64` – možnost kompilátoru na příkazovém řádku, kompilátor vyvolá [příkazového řádku upozornění D9002](http://msdn.microsoft.com/c58b405b-0f26-434e-b57f-4f05e1ca81e6). **__W64** – klíčové slovo je tiše ignorováno. Namísto použití této možnosti a klíčového slova ke zjišťování problémů přenositelnosti na 64 bitů je třeba použít kompilátor jazyka Visual C++, který cílí na 64bitovou platformu. Další informace najdete v tématu [konfigurovat Visual C++ pro 64bitové, x64 cíle](../build/configuring-programs-for-64-bit-visual-cpp.md).  
  
 Jakékoli definice typedef, která má **__w64** na x86 32 bity a 64 bitů na x64, musejí mít.  
  
 Detekce problémy s přenositelností pomocí verze starší než Visual Studio 2010, kompilátor Visual C++ **__w64** – klíčové slovo se musí nastavit u jakékoli definice TypeDef, která mění velikosti mezi 32bitovou a 64bitovou platformou. Pro takový typ **__w64** musí být uvedena pouze u 32bitové definice typedef.  
  
 **__W64** – klíčové slovo se ignoruje, pokud kompilace nepoužívá `/Wp64`.  
  
 Další informace o převodu na 64 bitů naleznete v následujících tématech:  
  
-   [Možnosti kompilátoru](../build/reference/compiler-options.md)  
  
-   [Portování 32bitového kódu do 64bitového kódu](../build/common-visual-cpp-64-bit-migration-issues.md)  
  
-   [Konfigurace Visual C++ pro 64bitové cíle x64](../build/configuring-programs-for-64-bit-visual-cpp.md)  
  
## <a name="example"></a>Příklad  
  
```cpp  
// __w64.cpp  
// compile with: /W3 /Wp64  
typedef int Int_32;  
#ifdef _WIN64  
typedef __int64 Int_Native;  
#else  
typedef int __w64 Int_Native;  
#endif  
  
int main() {  
   Int_32 i0 = 5;  
   Int_Native i1 = 10;  
   i0 = i1;   // C4244 64-bit int assigned to 32-bit int  
  
   // char __w64 c;  error, cannot use __w64 on char  
}  
```
