# <a name="expressions-c"></a>Výrazy (C++)

Tato část popisuje C++ výrazy. Výrazy jsou sekvence operátorů a operandů, které se používají k jednomu nebo více těmto účelům:

- Výpočet hodnoty z operandů.

- Určení objektů nebo funkcí.

- Generují se "vedlejší účinky". (Vedlejší účinky jsou jakékoli jiné akce než vyhodnocení výrazu – například změna hodnoty objektu.)

V C++nástroji mohou být operátory přetíženy a jejich význam může být definován uživatelem. Nicméně jejich priorita a počet operandů, které tyto operandy vybírají, nelze upravit. Tato část popisuje syntaxi a sémantiku operátorů, protože jsou dodávány s jazykem, nikoli přetížením.
