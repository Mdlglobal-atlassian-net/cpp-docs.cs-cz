# <a name="__m128i"></a>__m128i

**Specifické pro společnost Microsoft**

**__M128i** datový typ, který se má použít spolu s pokyny pro streaming SIMD Extensions 2 (SSE2), je definován v > \<emmintrin. h.

```cpp
// data_types__m128i.cpp
#include <emmintrin.h>
int main() {
   __m128i x;
}
```

## <a name="remarks"></a>Poznámky

Neměli byste přistupovat přímo k polím **__m128i** . Tyto typy však uvidíte v ladicím programu. Proměnná typu **__m128i** se mapuje na registry XMM [0-7].

Proměnné typu **_m128i** jsou automaticky zarovnány na hranice 16 bajtů.

> [!NOTE]
>  Použití proměnných typu **__m128i** způsobí, že kompilátor vygeneruje instrukci pro `movdqa` SSE2. Tato instrukce nezpůsobí chybu u procesorů Pentium III, ale způsobí selhání v tichém režimu, s případnými vedlejšími účinky, které způsobily jakékoli pokyny `movdqa` se přeloží na procesory Pentium III.

Datový typ **__m128i** není podporován na procesorech ARM.
