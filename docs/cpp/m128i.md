# <a name="m128i"></a>__m128i

## <a name="microsoft-specific"></a>Specifické pro Microsoft

**__M128i** datový typ, pro použití s vnitřní objekty pokyny Streaming SIMD Extensions 2 (SSE2), je definován v \<emmintrin.h >.

```cpp
// data_types__m128i.cpp
#include <emmintrin.h>
int main() {
   __m128i x;
}
```

## <a name="remarks"></a>Poznámky

Byste neměli přistupovat **__m128i** pole přímo. Tyto typy však uvidíte v ladicím programu. Proměnné typu **__m128i** mapuje na registry XMM [0-7].

Proměnné typu **_m128i** jsou automaticky zarovnány na hranice 16 bajtů.

> [!NOTE]
>  Použití proměnné typu **__m128i** způsobí, že kompilátor generuje SSE2 `movdqa` instrukce. Tento pokyn nezpůsobí chybu na procesorech Pentium III, ale povede tiché selhání s možné vedlejšími účinky způsobené libovolné pokyny `movdqa` překládá do na procesorech Pentium III.

**__M128i** datový typ není podporován na ARM procesorech.

**Specifické pro END Microsoft**
