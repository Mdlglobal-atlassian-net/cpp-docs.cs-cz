# <a name="indirection-on-array-types"></a>Deference na typech pole
Použití operátoru dereference (**\***) na *n*-rozměrný typ pole vrátí *n*-1 rozměrné pole. Pokud *n* 1, je vhodné skalární hodnota (nebo prvek pole).
