# <a name="virtual-specifier"></a>virtual – specifikátor

Klíčové slovo [Virtual](../cpp/virtual-cpp.md) lze použít pouze pro nestatické členské funkce třídy. To znamená, že vazba volání funkce je odložena až do spuštění. Další informace najdete v tématu [virtuální funkce](../cpp/virtual-functions.md).
