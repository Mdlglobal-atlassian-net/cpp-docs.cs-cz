# <a name="m64"></a>__m64
## <a name="microsoft-specific"></a>Specifické pro Microsoft  
 **__M64** datový typ je pro použití s MMX a 3DNow! vnitřní objekty a je definován v \<xmmintrin.h >.  
  
```cpp 
// data_types__m64.cpp  
#include <xmmintrin.h>  
int main()  
{  
   __m64 x;  
}  
```  
  
## <a name="remarks"></a>Poznámky  
 Byste neměli přistupovat **__m64** pole přímo. Tyto typy však uvidíte v ladicím programu. Proměnné typu **__m64** mapuje na registry MM [0-7].  
  
 Proměnné typu **výrazů _m64** jsou automaticky zarovnány na hranice 8 bajtů.  
  
 **__M64** datový typ není podporován na x64 procesory. Aplikace, které používají __m64 jako součást vnitřních objektů MMX __m64, musejí být přepsány, aby bylo možné použít ekvivalentní vnitřní objekty SSE a SSE2.  
  
**Specifické pro END Microsoft**
