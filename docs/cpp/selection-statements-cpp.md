# <a name="selection-statements-c"></a>Příkazy výběru (C++)

Příkazy C++ výběru, [Pokud](../cpp/if-else-statement-cpp.md) a [Switch](../cpp/switch-statement-cpp.md), poskytují způsob podmíněného provádění sekcí kódu.

Příkazy [__if_exists](../cpp/if-exists-statement.md) a [__if_not_exists](../cpp/if-not-exists-statement.md) umožňují podmíněně zahrnout kód v závislosti na existenci symbolu.

Syntaxi každého příkazu naleznete v jednotlivých tématech.
