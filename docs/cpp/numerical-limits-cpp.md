# <a name="numerical-limits-c"></a>Numerické limity (C++)

Dva standardní soubory k zahrnutí \<Limits. h > a \<float. h >, definují číselná omezení nebo minimální a maximální hodnoty, které může obsahovat proměnná daného typu. Tyto minimální a maximální hodnoty jsou zaručené přenosů do jakéhokoli C++ kompilátoru, který používá stejnou reprezentaci dat jako ANSI C. \<Limits. h > zahrnutí souboru definuje [číselné limity pro celočíselné typy](../cpp/integer-limits.md)a \<float. h > definuje [číselné limity pro plovoucí typy](../cpp/floating-limits.md).
