# <a name="virtual-c"></a>virtual (C++)

Klíčové slovo **Virtual** deklaruje virtuální funkci nebo virtuální základní třídu.

## <a name="syntax"></a>Syntaxe

```
virtual [type-specifiers] member-function-declarator
virtual [access-specifier] base-class-name
```

#### <a name="parameters"></a>Parametry

*Specifikátory typu*<br/>
Určuje návratový typ virtuální členské funkce.

*člen – funkce – deklarátor*<br/>
Deklaruje členskou funkci.

*přístup – specifikátor*<br/>
Definuje úroveň přístupu k základní třídě **Public**, **Protected** nebo **Private**. Může se vyskytovat před nebo za klíčovým slovem **Virtual** .

*název základní třídy*<br/>
Identifikuje dříve deklarovaný typ třídy.
