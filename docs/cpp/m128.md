# <a name="m128"></a>__m128

## <a name="microsoft-specific"></a>Specifické pro Microsoft

**__M128 data** typ, pro použití s Streaming SIMD Extensions a Streaming SIMD Extensions 2 pokyny vnitřních objektů, je definován v \<xmmintrin.h >.

```cpp
// data_types__m128.cpp
#include <xmmintrin.h>
int main() {
   __m128 x;
}
```

## <a name="remarks"></a>Poznámky

Byste neměli přistupovat **__m128** pole přímo. Tyto typy však uvidíte v ladicím programu. Proměnné typu **__m128** mapuje na registry XMM [0-7].

Proměnné typu **__m128** jsou automaticky zarovnány na hranice 16 bajtů.

**__M128** datový typ není podporován na ARM procesorech.

**Specifické pro END Microsoft**
