# <a name="date-and-time-database-support"></a>Datum a čas: Podpora databáze

Od verze 4.0 knihovna MFC databáze používá programovací [COleDateTime](../atl-mfc-shared/reference/coledatetime-class.md) a [coledatetimespan –](../atl-mfc-shared/reference/coledatetimespan-class.md) třídy, které představují data a času. Tyto třídy také použít ve službě Automation, jsou odvozeny z třídy [COleVariant](../mfc/reference/colevariant-class.md). Poskytnou lepší podporu pro správu data a času, než [CTime](../atl-mfc-shared/reference/ctime-class.md) a [ctimespan –](../atl-mfc-shared/reference/ctimespan-class.md).
