# <a name="c-in-visual-studio"></a>C++ v sadě Visual Studio

> [!NOTE]
> Tato dokumentace pro vývojáře platí pro Visual Studio 2015 a novější. Chcete-li zobrazit dokumentaci k upřednostňované verzi sady Visual Studio, použijte ovládací prvek pro výběr **verze.** Nachází se v horní části obsahu na této stránce.
>
> Pokud hledáte balíček visual c++ redistribuovatelné, takže můžete spustit program, přejděte na [Službu pro stažení softwaru společnosti Microsoft](https://www.microsoft.com/download/) a zadejte Visual **C++** do vyhledávacího pole.

Microsoft Visual C++, obvykle zkrácený na Visual C++ nebo MSVC, je název pro nástroje pro vývoj jazyka C++, C a sestavení, které jsou k dispozici jako součást sady Visual Studio v systému Windows. Tyto nástroje a knihovny umožňují vytvářet aplikace pro univerzální platformu Windows (UPW), nativní desktopové a serverové aplikace Windows, knihovny a aplikace napříč platformami, které běží ve Windows, Linuxu, Androidu a iOS, a také spravované aplikace a knihovny, které používají rozhraní .NET Framework. Pomocí visual c++ můžete psát cokoli od jednoduchých konzolových aplikací až po nejsofistikovanější a nejsložitější aplikace pro stolní počítače se systémem Windows, od ovladačů zařízení a komponent operačního systému až po hry napříč platformami pro mobilní zařízení a od nejmenších zařízení IoT až po vysoce výkonné výpočetní prostředky s více servery v cloudu Azure.

Visual Studio 2015, 2017 a 2019 lze nainstalovat vedle sebe. Pomocí Visual Studia 2019 (sada nástrojů kompilátoru v142) můžete upravovat a vytvářet programy pomocí sady nástrojů z Visual Studia 2015 (v140) a Visual Studia 2017 (v141).
