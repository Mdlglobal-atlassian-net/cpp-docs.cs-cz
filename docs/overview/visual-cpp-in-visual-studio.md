# <a name="c-in-visual-studio"></a>C++ v sadě Visual Studio

> [!NOTE]
> Tato dokumentace pro vývojáře se vztahuje na Visual Studio 2015 a novější. Chcete-li zobrazit dokumentaci k preferované verzi sady Visual Studio, použijte ovládací prvek selektor **verzí** . Nachází se v horní části obsahu na této stránce.
>
> Pokud hledáte Visual C++ Distribuovatelný balíček, abyste mohli spustit program, přejděte na [webu Microsoft Download Center](https://www.microsoft.com/download/) a do vyhledávacího pole zadejte **Visual C++** .

Microsoft Visual C++, obvykle zkrácením na Visual C++ nebo MSVC, je název pro vývojové nástroje jazyka C++, C a Assembly a knihovny dostupné jako součást sady Visual Studio ve Windows. Tyto nástroje a knihovny vám umožní vytvářet aplikace Univerzální platforma Windows (UWP), nativní desktopové a serverové aplikace Windows, knihovny pro různé platformy a aplikace, které běží na Windows, Linux, Androidu a v iOS, a taky spravované aplikace a knihovny, které používají .NET Framework. Visual C++ můžete použít k psaní jakýchkoli jednoduchých konzolových aplikací do nejdůležitějších a komplexních aplikací pro Windows Desktop, od ovladačů zařízení a součástí operačního systému po hry pro různé platformy pro mobilní zařízení a z nejmenších zařízení IoT až po vysoce výkonné výpočetní prostředí v cloudu Azure.

Sady Visual Studio 2015, 2017 a 2019 lze instalovat vedle sebe. Můžete použít sadu Visual Studio 2019 (V142 sady nástrojů kompilátoru) k úpravám a sestavování programů pomocí sady nástrojů ze sady Visual Studio 2015 (v140) a sady Visual Studio 2017 (v141).
