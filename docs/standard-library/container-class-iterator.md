# <a name="container-classiterator"></a>Třída kontejneru::iterator

> [!NOTE]
> Toto téma je v Microsoft C++ dokumentaci jako funkční příklad kontejnerů používané C++ standardní knihovny. Další informace najdete v tématu [kontejnery standardní knihovny C++](../standard-library/stl-containers.md).

Popisuje objekt, který může sloužit jako iterátoru řízené sekvence.

## <a name="syntax"></a>Syntaxe

```

typedef T5 iterator;
```

## <a name="remarks"></a>Poznámky

Je popsán jako synonymum pro neurčeného typu `T5`. Objekt typu `iterator` lze převést na objekt typu [const_iterator](../standard-library/container-class-const-iterator.md).
