# <a name="ltstackgt"></a>&lt;Zásobník&gt;

Definuje šablonu třídy zásobníku a dvě podpůrných šablon.

## <a name="requirements"></a>Požadavky

**Záhlaví:** \<zásobníku >

**Namespace:** std

> [!NOTE]
> \<Zásobníku > Knihovna také používá `#include <initializer_list>` příkazu.
