# <a name="ltinitializerlistgt"></a>&lt;initializer_list&gt;

Definuje třídu `initializer_list` šablony kontejneru a několik podpůrných šablon.

## <a name="syntax"></a>Syntaxe

```cpp
#include <initializer_list>
```

### <a name="classes"></a>Třídy

|Třída|Popis|
|-|-|
|[initializer_list](../standard-library/initializer-list-class.md)|Poskytuje přístup k poli prvků, ve kterých je každý člen zadaného typu.|
