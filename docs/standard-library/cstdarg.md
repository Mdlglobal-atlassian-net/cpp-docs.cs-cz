# <a name="ltcstdarggt"></a>&lt;cstdarg&gt;

Obsahuje hlavičku knihovny Standard C \<stdarg.h > a přidá názvy přidružené k `std` oboru názvů.

## <a name="syntax"></a>Syntaxe

```cpp
#include <cstdarg>

```

## <a name="remarks"></a>Poznámky

Včetně této hlavičky zajišťuje, že názvy deklarované s vnějším spojením v záhlaví knihovny Standard C jsou deklarovány v `std` oboru názvů.
