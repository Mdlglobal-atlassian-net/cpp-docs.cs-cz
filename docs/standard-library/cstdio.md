# <a name="ltcstdiogt"></a>&lt;cstdio&gt;

Obsahuje hlavičku \<standardní knihovny jazyka C stdio. h > a přidává přidružené názvy `std` do oboru názvů.

## <a name="syntax"></a>Syntaxe

```cpp
#include <cstdio>
```

## <a name="remarks"></a>Poznámky

Včetně této hlavičky zajišťuje, že názvy deklarované s vnějším propojením v hlavičce standardní knihovny jazyka C jsou deklarovány v `std` oboru názvů.
