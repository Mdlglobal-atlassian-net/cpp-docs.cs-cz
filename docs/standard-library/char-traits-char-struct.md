# <a name="chartraitsltchargt-struct"></a>char_traits –&lt;char&gt; – struktura

Struktura, která je specializace šablony struktura **char_traits –\<CharType >** pro element typu `char`.

## <a name="syntax"></a>Syntaxe

```cpp
template <>
struct char_traits<char>;
```

## <a name="remarks"></a>Poznámky

Specializace umožňuje struktura chcete využít výhod funkce knihovny, které pracují s objekty tohoto typu `char`.
