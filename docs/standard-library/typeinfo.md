# <a name="lttypeinfogt"></a>&lt;TypeInfo –&gt;
Zahrnují standardní hlavičku `<typeinfo>` definovat několik typů související s operátorem identifikace typu [typeid – operátor](../cpp/typeid-operator.md), které poskytuje informace o typech statické a dynamické.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
#include <typeinfo>  
  
```
