# <a name="ltcvtwbuffergt"></a>&lt;cvt/wbuffer &gt;

Hlavička `<cvt/wstring>` v předchozích verzích sady Visual Studio definovala [třídu wbuffer_convert](../standard-library/wbuffer-convert-class.md) šablony třídy v oboru názvů stdext:: CVT. Záhlaví je zachováno z důvodu zpětné kompatibility. Nový kód by měl používat verzi třídy, která je definovaná v [\<locale >](../standard-library/locale.md) v oboru názvů `std`

## <a name="syntax"></a>Syntaxe

```cpp
#include <cvt/wbuffer>
```
