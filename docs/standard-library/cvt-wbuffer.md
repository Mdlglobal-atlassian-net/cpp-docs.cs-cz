# <a name="ltcvtwbuffergt"></a>&lt;CVT/wbuffer –&gt;

Záhlaví `<cvt/wstring>` v předchozích verzích sady Visual Studio definované šablony třídy [wbuffer_convert – třída](../standard-library/wbuffer-convert-class.md) v oboru názvů stdext::cvt. Záhlaví bude zachována pro účely zpětné kompatibility. Nový kód by měl použít verzi třídu, která je definována v [ \<národní prostředí >](../standard-library/locale.md) v `std` obor názvů

## <a name="syntax"></a>Syntaxe

```cpp
#include <cvt/wbuffer>

```
