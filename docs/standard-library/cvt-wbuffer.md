# <a name="ltcvtwbuffergt"></a>&lt;CVT/wbuffer –&gt;

Záhlaví bude zachována pro účely zpětné kompatibility. Nový kód by měl použít verzi třídu, která je definována v [ \<národní prostředí >](../standard-library/locale.md) v `std` obor názvů
