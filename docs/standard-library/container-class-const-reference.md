# <a name="container-classconstreference"></a>Třída kontejneru::const_reference

> [!NOTE]
> Toto téma je v Microsoft C++ dokumentaci jako funkční příklad kontejnerů používané C++ standardní knihovny. Další informace najdete v tématu [kontejnery standardní knihovny C++](../standard-library/stl-containers.md).

Popisuje objekt, který může sloužit jako konstantní odkaz na prvek řízené sekvence.

## <a name="syntax"></a>Syntaxe

```

typedef T3 const_reference;
```

## <a name="remarks"></a>Poznámky

Je popsán jako synonymum pro neurčeného typu `T3` (obvykle `Alloc::const_reference`).
