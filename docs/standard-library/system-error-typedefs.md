##  <a name="generic_errno"></a>generic_errno  
 Typ, který reprezentuje výčet, který poskytuje symbolický názvy pro všechny makra kód chyby definované Posix v `<errno.h>`.  
  
```
typedef errc generic_error;
```  
  
### <a name="remarks"></a>Poznámky  
 Typ se jedná o synonymum [errc](../standard-library/system-error-enums.md#errc).
