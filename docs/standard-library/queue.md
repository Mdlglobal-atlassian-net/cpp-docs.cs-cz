# <a name="ltqueuegt"></a>&lt;queue &gt;

Definuje šablony tříd priority_queue a Queue a několik podpůrných šablon.

## <a name="requirements"></a>Požadavky

**Záhlaví:** \<queue >

**Obor názvů:** std

> [!NOTE]
> Knihovna \<queue > používá také příkaz `#include <initializer_list>`.
