# <a name="chartraitsltchar32tgt-struct"></a>char_traits –&lt;char32_t&gt; – struktura
Struktura, která je specializace šablony struktura **char_traits –\<CharType >** pro element typu `char32_t`.  
  
## <a name="syntax"></a>Syntaxe  
  
```
template <>  
struct char_traits<char32_t>;
```  
  
## <a name="remarks"></a>Poznámky  
 Specializace umožňuje struktura chcete využít výhod funkce knihovny, které pracují s objekty tohoto typu `char32_t`.
