# <a name="regex_traitsltchargt-class"></a>regex_traits &lt;char &gt; – třída

Specializace `regex_traits` pro **char**

## <a name="syntax"></a>Syntaxe

```cpp
template <>
class regex_traits<char>
```

## <a name="remarks"></a>Poznámky

Třída je explicitní specializace šablony třídy [regex_traits](../standard-library/regex-traits-class.md) pro prvky typu **char** (tak, aby mohla využívat funkce knihovny, které pracují s objekty tohoto typu).

## <a name="requirements"></a>Požadavky

**Záhlaví:** \<regex >

**Obor názvů:** std
