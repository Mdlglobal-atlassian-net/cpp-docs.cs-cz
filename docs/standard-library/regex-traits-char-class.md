# <a name="regextraitsltchargt-class"></a>regex_traits&lt;char&gt; Class
Specializace regex_traits pro char.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
template <>  
class regex_traits<char>  
```  
  
## <a name="remarks"></a>Poznámky  
 Třída je explicitní specializace šablon třídy [regex_traits](../standard-library/regex-traits-class.md) pro elementy typu `char` (tak, aby ji můžete využít výhod funkce knihovny, které pracují s objekty tohoto typu).  
  
## <a name="requirements"></a>Požadavky  
 **Záhlaví:** \<regex >  
  
 **Namespace:** – std
