# <a name="ltdequegt"></a>&lt;deque &gt;

Definuje šablonu třídy kontejneru deque a několik podpůrných šablon.

## <a name="requirements"></a>Požadavky

**Záhlaví**: \<deque >

> [!NOTE]
> Knihovna \<deque > používá také příkaz `#include <initializer_list>`.
