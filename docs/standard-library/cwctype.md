# <a name="ltcwctypegt"></a>&lt;cwctype&gt;

Obsahuje hlavičku \<standardní knihovny jazyka C wctype. h > a přidává přidružené názvy `std` do oboru názvů.

## <a name="syntax"></a>Syntaxe

```cpp
#include <cwctype>
```

## <a name="remarks"></a>Poznámky

Včetně této hlavičky zajišťuje, že názvy deklarované s vnějším propojením v hlavičce standardní knihovny jazyka C jsou deklarovány v `std` oboru názvů.

## <a name="constants"></a>Konstanty

```cpp
namespace std {
    using wint_t = see below ;
    using wctrans_t = see below ;
    using wctype_t = see below ;
}

#define WEOF see below
```
