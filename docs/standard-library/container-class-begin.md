# <a name="container-classbegin"></a>Třída kontejneru::begin

> [!NOTE]
> Toto téma se nachází v dokumentaci sady C++ Visual Studio jako nefunkční příklad kontejnerů použitých ve C++ standardní knihovně. Další informace najdete v tématu [ C++ standardní kontejnery knihovny](../standard-library/stl-containers.md).

Vrátí iterátor, který odkazuje na první prvek sekvence (nebo těsně za konec prázdné sekvence).

## <a name="syntax"></a>Syntaxe

```cpp
const_iterator begin() const;

iterator begin();
```
