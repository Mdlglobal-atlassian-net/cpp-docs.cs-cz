# <a name="container-classvaluetype"></a>Třída kontejneru::value_type

> [!NOTE]
> Toto téma se v dokumentaci k Visual C++ jako funkční příklad kontejnery použít ve standardní knihovně C++. Další informace najdete v tématu [kontejnery standardní knihovny C++](../standard-library/stl-containers.md).

Funguje synonymum pro parametr šablony **Ty**.

## <a name="syntax"></a>Syntaxe

```

typedef T4 value_type;
```

## <a name="remarks"></a>Poznámky

Je popsán sem jako synonymum pro neurčeného typu **T4** (obvykle **Alloc::value_type**).
