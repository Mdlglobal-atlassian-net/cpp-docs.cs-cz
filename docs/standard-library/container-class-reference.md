# <a name="container-classreference"></a>Třída kontejneru::reference

> [!NOTE]
> Toto téma se v dokumentaci k Visual C++ jako funkční příklad kontejnery použít ve standardní knihovně C++. Další informace najdete v tématu [kontejnery standardní knihovny C++](../standard-library/stl-containers.md).

Popisuje objekt, který může sloužit jako odkaz na element řízené sekvenci.

## <a name="syntax"></a>Syntaxe

```

typedef T2 reference;
```

## <a name="remarks"></a>Poznámky

Je popsán sem jako synonymum pro neurčeného typu **T2** (obvykle **Alloc::reference**). Objekt typu **odkaz** lze převést na objekt typu [const_reference –](../standard-library/container-class-const-reference.md).
