## <a name="swap"></a>adresu

Vyměňuje prvky dvou deques.

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>Parametry

*levý*\
Objekt typu `deque`.

*pravé*\
Objekt typu `deque`.
