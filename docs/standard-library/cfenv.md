# <a name="ltcfenvgt"></a>&lt;cfenv&gt;

Obsahuje hlavičku \<standardní knihovny jazyka C fenv. h > a přidává přidružené názvy `std` do oboru názvů.

## <a name="syntax"></a>Syntaxe

```cpp
#include <cfenv>
```

## <a name="remarks"></a>Poznámky

Včetně této hlavičky zajišťuje, že názvy deklarované s vnějším propojením v hlavičce standardní knihovny jazyka C jsou deklarovány v `std` oboru názvů.

## <a name="constants-and-types"></a>Konstanty a typy

```cpp

    using fenv_t = object type ;
```
