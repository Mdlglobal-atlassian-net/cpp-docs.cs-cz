# <a name="ltunordered_mapgt"></a>&lt;unordered_map &gt;

Definuje šablony tříd kontejneru [unordered_map](../standard-library/unordered-map-class.md) a [unordered_multimap](../standard-library/unordered-multimap-class.md) a jejich podpůrné šablony.

## <a name="syntax"></a>Syntaxe

```cpp
#include <unordered_map>
```

> [!NOTE]
> Knihovna \<unordered_map > používá také příkaz `#include <initializer_list>`.

### <a name="classes"></a>Třídy

|Třída|Popis|
|-|-|
|[unordered_map – třída](../standard-library/unordered-map-class.md)|Ukládá zatřiďovací tabulku pro dvojice {Key, mapované}.|
|[unordered_multimap – třída](../standard-library/unordered-multimap-class.md)|Ukládá zatřiďovací tabulku pro dvojice {Key, mapované}.|
