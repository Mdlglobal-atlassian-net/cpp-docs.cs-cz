# <a name="regex_traitsltwchar_tgt-class"></a>regex_traits &lt;wchar_t &gt; – třída

Specializace `regex_traits` pro **wchar_t**

## <a name="syntax"></a>Syntaxe

```cpp
template <>
class regex_traits<wchar_t>
```

## <a name="remarks"></a>Poznámky

Třída je explicitní specializace šablony třídy [regex_traits](../standard-library/regex-traits-class.md) pro prvky typu **wchar_t** (aby mohla využívat funkce knihovny, které pracují s objekty tohoto typu).

## <a name="requirements"></a>Požadavky

**Záhlaví:** \<regex >

**Obor názvů:** std
