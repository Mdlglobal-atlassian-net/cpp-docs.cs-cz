# <a name="ltforwardlistgt"></a>&lt;forward_list&gt;

Definuje forward_list – kontejner šablony třídy a několik podpůrných šablon.

## <a name="syntax"></a>Syntaxe

```cpp
#include <forward_list>
```

### <a name="operators"></a>Operátory

|Operátor|Popis|
|-|-|
|[operator==](../standard-library/forward-list-operators.md#op_eq_eq)|Testuje, zda objekt dopředu seznamu na levé straně operátoru roven objektu dopředné seznamu na pravé straně.|
|[operator!=](../standard-library/forward-list-operators.md#op_neq)|Testuje, zda je objekt dopředu seznamu na levé straně operátoru není roven objektu dopředné seznamu na pravé straně.|
|[Operator <](../standard-library/forward-list-operators.md#op_lt)|Testuje, zda objekt dopředu seznamu na levé straně operátoru menší než objekt dopředu seznamu na pravé straně.|
|[operator<=](../standard-library/forward-list-operators.md#op_lt_eq)|Testuje, zda je objekt seznamu vpřed na levé straně operátoru je menší než nebo roven objektu dopředné seznamu na pravé straně.|
|[operator>](../standard-library/forward-list-operators.md#op_gt)|Testuje, zda objekt dopředu seznamu na levé straně operátoru větší než objekt dopředu seznamu na pravé straně.|
|[operator>=](../standard-library/forward-list-operators.md#op_lt_eq)|Testuje, zda je objekt dopředu seznamu na levé straně operátoru větší než nebo roven objektu dopředné seznamu na pravé straně.|

### <a name="functions"></a>Funkce

|Funkce|Popis|
|-|-|
|[swap](../standard-library/forward-list-functions.md#swap)|Vymění prvky dvou seznamů vpřed.|

### <a name="classes"></a>Třídy

|Třída|Popis|
|-|-|
|[forward_list](../standard-library/forward-list-class.md)|Popisuje objekt, který řídí různé délky sekvence elementů. Sekvence se ukládá jako jednotlivě propojený seznam prvků, každá obsahuje člen typu `Type`.|
