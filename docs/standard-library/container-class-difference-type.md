# <a name="container-classdifferencetype"></a>Třída kontejneru::difference_type

> [!NOTE]
> Toto téma je v Microsoft C++ dokumentaci jako funkční příklad kontejnerů používané C++ standardní knihovny. Další informace najdete v tématu [kontejnery standardní knihovny C++](../standard-library/stl-containers.md).

Popisuje objekt, který může představovat rozdíl mezi adresami dva prvky řízené sekvence.

## <a name="syntax"></a>Syntaxe

```

typedef T1 difference_type;
```

## <a name="remarks"></a>Poznámky

Je popsán jako synonymum pro neurčeného typu `T1` (obvykle `Alloc::difference_type`).
