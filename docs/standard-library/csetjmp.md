## <a name="syntax"></a>Syntaxe

```cpp
#include <csetjmp>

```

Včetně tuto hlavičku zajistí, že jsou názvy deklarováno s použitím externí propojení v hlavičce knihovny standardní C deklarované v `std` oboru názvů.
