Definuje třídu šablony kontejneru `array` a několik podpůrných šablony.

## <a name="syntax"></a>Syntaxe

```cpp
#include <array>
```

|[tuple_size](../standard-library/tuple-size-class-tuple.md)|Zabalí velikost pole elementu.|
