# <a name="ltarraygt"></a>&lt;array &gt;

Definuje **pole** šablony třídy kontejneru a několik pomocných šablon.

## <a name="requirements"></a>Požadavky

**Záhlaví:** \<array >

**Obor názvů:** std

> [!NOTE]
> Knihovna \<array > používá také příkaz `#include <initializer_list>`.
