# <a name="container-classreverseiterator"></a>Třída kontejneru::reverse_iterator

> [!NOTE]
> Toto téma se v dokumentaci k Visual C++ jako funkční příklad kontejnery použít ve standardní knihovně C++. Další informace najdete v tématu [kontejnery standardní knihovny C++](../standard-library/stl-containers.md).

Popisuje objekt, který může sloužit jako reverzní iterator pro řízené sekvenci.

## <a name="syntax"></a>Syntaxe

```

typedef T7 reverse_iterator;
```

## <a name="remarks"></a>Poznámky

Je popsán sem jako synonymum pro neurčeného typu **T7** (obvykle `reverse_iterator` **\<** [iterator](../standard-library/container-class-iterator.md) **>**).
