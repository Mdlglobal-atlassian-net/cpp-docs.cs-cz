# <a name="container-classreverseiterator"></a>Třída kontejneru::reverse_iterator

> [!NOTE]
> Toto téma je v Microsoft C++ dokumentaci jako funkční příklad kontejnerů používané C++ standardní knihovny. Další informace najdete v tématu [kontejnery standardní knihovny C++](../standard-library/stl-containers.md).

Popisuje objekt, který může sloužit jako "reverse iterator" pro řízenou sekvenci.

## <a name="syntax"></a>Syntaxe

```

typedef T7 reverse_iterator;
```

## <a name="remarks"></a>Poznámky

Je popsán jako synonymum pro neurčeného typu `T7` (obvykle `reverse_iterator` **\<** [iterátoru](../standard-library/container-class-iterator.md)**>**).
