# <a name="chartraitsltchar16tgt-struct"></a>char_traits&lt;char16_t&gt; – struktura

Struktura, která je specializací šablony struktury **char_traits\<CharType >** na prvek typu `char16_t`.

## <a name="syntax"></a>Syntaxe

```cpp
template <>
struct char_traits<char16_t>;
```

## <a name="remarks"></a>Poznámky

Specializace umožňuje struktury, abyste mohli využívat funkce knihovny, které pracují s objekty typu `char16_t`.

## <a name="requirements"></a>Požadavky

**Záhlaví:** \<řetězec >

**Namespace:** std
