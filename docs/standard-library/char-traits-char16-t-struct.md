# <a name="chartraitsltchar16tgt-struct"></a>char_traits –&lt;char16_t&gt; – struktura

Struktura, která je specializace šablony struktura **char_traits –\<CharType >** pro element typu `char16_t`.

## <a name="syntax"></a>Syntaxe

```cpp

struct char_traits<char16_t>;
```

Specializace umožňuje struktura chcete využít výhod funkce knihovny, které pracují s objekty typu `char16_t`.
