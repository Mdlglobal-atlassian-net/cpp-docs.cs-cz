# <a name="using-ccomboboxex"></a>Používání atributu CComboBoxEx

Pole ovládacího prvku rozšířené pole se seznamem je rozšířením ovládacího prvku pole standardní pole se seznamem, který poskytuje nativní podporu pro obrázků položek. Tyto Image je možné do informací o stavu jednotlivých položek v poli se seznamem, jako je aktuálně vybraný a nevybrané položky. Aby obrázků položek snadno k dispozici, ovládací prvek podporuje image seznamu.

Pomocí tohoto ovládacího prvku nakonfigurovánu pole se seznamem bez nutnosti ručně kreslit položky grafiky.
