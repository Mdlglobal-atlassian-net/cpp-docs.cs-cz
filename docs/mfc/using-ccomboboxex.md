# <a name="using-ccomboboxex"></a>Používání atributu CComboBoxEx

Rozšířený ovládací prvek pole se seznamem je rozšíření standardního ovládacího prvku pole se seznamem, který poskytuje nativní podporu pro obrázky položek. Tyto obrázky lze použít k označení stavu jednotlivých položek v poli se seznamem, jako jsou aktuálně vybrané a nevybrané položky. Aby bylo možné k obrázkům položek snadno získat přístup, poskytuje ovládací prvek podporu seznamu obrázků.

Pomocí tohoto ovládacího prvku můžete poskytnout funkce pole se seznamem, aniž byste museli ručně kreslit grafiku položky.
