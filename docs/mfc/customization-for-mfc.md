# <a name="customization-for-mfc"></a>Přizpůsobení pro prostředí MFC
Toto téma uvádí Rady pro přizpůsobení aplikace knihovny MFC.  
  
## <a name="general-customizations"></a>Obecné přizpůsobení  
 Můžete uložit a načíst stav vaší aplikace do registru. Když povolíte tuto možnost, bude vaše aplikace načíst počátečního stavu z registru. Pokud změníte počáteční ukotvení rozložení pro vaši aplikaci, bude mít k vymazání dat registru pro vaši aplikaci. Data v registru, jinak se přepíše všechny změny provedené v původním rozložení.
