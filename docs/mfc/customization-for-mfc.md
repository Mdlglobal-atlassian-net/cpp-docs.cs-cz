# <a name="customization-for-mfc"></a>Přizpůsobení pro prostředí MFC

Toto téma uvádí Rady pro přizpůsobení aplikace MFC.

## <a name="general-customizations"></a>Přizpůsobení obecné

Můžete uložit a načíst stav vaší aplikace do registru. Když tuto možnost povolíte, aplikace se načtou počáteční stav z registru. Pokud změníte počáteční dokovací rozložení pro vaši aplikaci, budete muset vymazat data registru pro vaši aplikaci. Data v registru v opačném případě se přepíšou všechny změny, které jste provedli počáteční rozložení.
