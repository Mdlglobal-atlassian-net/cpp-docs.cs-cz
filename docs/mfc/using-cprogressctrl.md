# <a name="using-cprogressctrl"></a>Používání atributu CProgressCtrl

K indikaci průběhu zdlouhavé operace lze použít ovládací prvek průběh. Je to obdélník, který se postupně vyplní barvou zvýraznění systému jako průběh operace.

Ovládací prvek průběh je reprezentován v knihovně MFC podle třídy [atributu CProgressCtrl](../mfc/reference/cprogressctrl-class.md).

Při prvotním vytvoření ovládacího prvku průběhu určíte jeho velikost a polohu, nadřazené okno (obvykle dialogové okno) a ID. Pomocí parametru *dwStyle* můžete také určit různé styly oken pro ovládací prvek a styly pro jejich vyplňování.
