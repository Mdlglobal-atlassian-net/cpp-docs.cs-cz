# <a name="using-cprogressctrl"></a>Používání atributu CProgressCtrl

Použití ovládacího prvku průběhu označuje průběh déletrvající operace. Je obdélníku, který je postupně vyplněn barvou zvýraznění systému v průběhu operace.

Ovládací prvek průběh je reprezentován v prostředí MFC třídou [atributu CProgressCtrl](../mfc/reference/cprogressctrl-class.md).

Při počátečním vytvoření ovládacím prvkem průběh, zadejte jeho velikost a umístění, nadřazené okno (obvykle dialogového okna) a podle ID. S použitím *dwStyle* parametr, můžete také určit různé styly oken pro ovládací prvek a jak vyplní styly.
