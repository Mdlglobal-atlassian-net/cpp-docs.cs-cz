# <a name="using-cprogressctrl"></a>Používání atributu CProgressCtrl
Ovládací prvek Průběh můžete použít informace o průběhu časově náročná operace. Je obdélníku postupně, naplní se barva zvýraznění systému v průběhu operace.  
  
 Ovládací prvek průběh je reprezentován v prostředí MFC třídou [CProgressCtrl](../mfc/reference/cprogressctrl-class.md).  
  
 Když vytvoříte původně ovládací prvek průběh, zadejte jeho velikost a umístění, nadřazeného okna (obvykle dialogového okna) a ID. Pomocí *dwStyle* parametr, můžete také určit různé styly oken pro ovládací prvek a styly jak zaplňování.
