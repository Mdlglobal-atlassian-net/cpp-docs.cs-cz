# <a name="how-the-framework-searches-message-maps"></a>Jak framework prohledává mapy zpráv

Framework prohledává tabulku mapy zpráv pro shody s příchozí zprávy. Jakmile napíšete položku mapování zpráv pro každou zprávu, že chcete, aby třída pro zpracování a zápis odpovídající obslužné rutiny, volá framework váš obslužné rutiny automaticky. Následující témata popisují vyhledávání map zpráv:

- [Kde hledat mapy zpráv](../mfc/where-to-find-message-maps.md)

- [Mapy odvozených zpráv](../mfc/derived-message-maps.md)

- [Mapování oblasti zprávy, ID příkazů nebo ID ovládacích prvků na jednu obslužnou rutinu.](../mfc/handlers-for-message-map-ranges.md)
