# <a name="active-document-classes"></a>Třídy aktivních dokumentů

Aktivní dokumenty lze zobrazit buď v celém okně klienta webového prohlížeče, například v aplikaci Internet Explorer 5,5 nebo v aktivním kontejneru, jako je například systém Microsoft Office Binder, který podporuje aktivní dokumenty.

[CDocObjectServer](../mfc/reference/cdocobjectserver-class.md)<br/>
Mapuje rozhraní aktivního dokumentu a inicializuje a aktivuje objekt aktivního dokumentu.

[CDocObjectServerItem](../mfc/reference/cdocobjectserveritem-class.md)<br/>
Implementuje příkazy OLE serveru speciálně pro servery aktivních dokumentů.

[COleDocObjectItem](../mfc/reference/coledocobjectitem-class.md)<br/>
Implementuje zahrnutí aktivního dokumentu.
