# <a name="ole-mfc"></a>OLE (MFC)
Implementace funkce OLE v programu ovlivní vaše uživatelské rozhraní několika způsoby:  
  
-   Visual úpravy (místní aktivace) se zobrazí uživatelské rozhraní jiný program v vašeho programu windows a upraví nabídek vaší aplikace s položek z jiné aplikace.  
  
-   Přetažení umožňuje uživatelům přetažením objektů v rámci a mezi windows a i mezi aplikacemi.  
  
-   Snímače poskytuje vizuální upozornění na stav objektů v rámci úpravy s náhledem a přetažení.
