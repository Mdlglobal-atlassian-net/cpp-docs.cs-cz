# <a name="ole-mfc"></a>OLE (MFC)

Implementace funkce OLE v programu ovlivňuje vaše uživatelské rozhraní v několika ohledech:

- Vizuální úpravy (aktivace na místě) zobrazí uživatelské rozhraní z jiné aplikace v systému windows váš program a změní váš program nabídky s položkami z jiné aplikace.

- Přetažení umožňuje uživatelům přetažením objektů v rámci a mezi windows a dokonce i mezi aplikacemi.

- Snímače poskytuje vizuální upozornění na stav objektů v rámci vizuální úpravy a přetažení.
