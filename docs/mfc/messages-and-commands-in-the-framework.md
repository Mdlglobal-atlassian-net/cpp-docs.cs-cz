# <a name="messages-and-commands-in-the-framework"></a>Zprávy a příkazy v prostředí .NET Framework

Aplikace napsané pro Microsoft Windows jsou "zpráva řízené." V reakci na události, jako je například kliknutí myší, stisknutí kláves, okno pohybů plb typu a tak dále Windows a odesílá zprávy do okna správné. Rozhraní Framework aplikace zpracování zpráv Windows stejně jako jakékoli jiné aplikace pro Windows. Ale navíc poskytuje určitá vylepšení, které usnadňují zpracování zpráv snazší, jednodušší údržbu a lépe zapouzdřená rozhraní framework.
