# <a name="root-class-cobject"></a>Kořenová třída: CObject

Většina tříd v knihovně MFC (Microsoft Foundation Class) je odvozena z jedné základní třídy v kořenu hierarchie tříd. `CObject` poskytuje řadu užitečných funkcí pro všechny třídy, které jsou z nich odvozené, s velmi nízkou režií. Další informace o `CObject` a jeho funkcích naleznete v tématu [using CObject](../mfc/using-cobject.md).

[CObject](../mfc/reference/cobject-class.md)<br/>
Nejvyšší základní třída většiny tříd knihovny MFC. Podporuje serializaci dat a získání běhových informací o třídě.

[CRuntimeClass](../mfc/reference/cruntimeclass-structure.md)<br/>
Struktura sloužící k určení přesné třídy objektu v době běhu.
