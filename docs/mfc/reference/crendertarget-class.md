## <a name="crendertargetcreatecompatiblerendertarget"></a><a name="createcompatiblerendertarget"></a>CRenderTarget::CreateCompatibleRenderTarget

Vytvoří nový cíl vykreslení bitmapy pro použití při přechodovém výkresu mimo obrazovku, který je kompatibilní s aktuálním cílem vykreslení .

```
BOOL CreateCompatibleRenderTarget(
    CBitmapRenderTarget& bitmapTarget,
    CD2DSizeF sizeDesired = CD2DSizeF(0., 0.),
    CD2DSizeU sizePixelDesired = CD2DSizeU(0, 0),
    D2D1_PIXEL_FORMAT* desiredFormat = NULL,
    D2D1_COMPATIBLE_RENDER_TARGET_OPTIONS options = D2D1_COMPATIBLE_RENDER_TARGET_OPTIONS_NONE);
```

### <a name="parameters"></a>Parametry

*bitmaptarget*<br/>
Když tato metoda vrátí, obsahuje adresu ukazatele na nový cíl vykreslení bitmapy. Tento parametr se předává neinicializovaný.

*sizeDesired*<br/>
Požadovaná velikost nového cíle vykreslení v pixelech nezávislých na zařízení, pokud by se měla lišit od původního cíle vykreslení nebo NULL. Další informace naleznete v části Poznámky.

*sizePixelDesired*<br/>
Požadovaná velikost nového cíle vykreslení v obrazových bodech, pokud by se měla lišit od původního cíle vykreslení nebo NULL. Další informace naleznete v části Poznámky.

*desiredFormát*<br/>
Požadovaný formát obrazových bodů a alfa režim nového cíle vykreslení neboli NULL. Pokud je formát obrazových bodů nastaven na DXGI_FORMAT_UNKNOWN nebo pokud je tento parametr nulový, použije nový cíl vykreslení stejný formát obrazových bodů jako původní cíl vykreslení. Pokud je režim alfa D2D1_ALPHA_MODE_UNKNOWN nebo tento parametr je NULL, alfa režim nového cíle vykreslení výchozí D2D1_ALPHA_MODE_PREMULTIPLIED. Informace o podporovaných formátech obrazových bodů najdete v tématu Podporované formáty obrazových bodů a Alfa režimy.

*Možnosti*<br/>
Hodnota, která určuje, zda nový cíl vykreslení musí být kompatibilní s GDI.

### <a name="return-value"></a>Návratová hodnota

Pokud je metoda úspěšná, vrátí hodnotu TRUE. V opačném případě vrátí hodnotu NEPRAVDA.

## <a name="crendertargetdrawbitmap"></a><a name="drawbitmap"></a>CRenderTarget::DrawBitmap

Nakreslí formátovaný text popsaný zadaným objektem IDWriteTextLayout.

```cpp
void DrawBitmap(
    CD2DBitmap* pBitmap,
    const CD2DRectF& rectDest,
    float fOpacity = 1.0,
    D2D1_BITMAP_INTERPOLATION_MODE interpolationMode = D2D1_BITMAP_INTERPOLATION_MODE_LINEAR,
    const CD2DRectF* pRectSrc = NULL);
```

### <a name="parameters"></a>Parametry

*pBitmap*<br/>
Bitmapa k vykreslení.

*rectDest*<br/>
Velikost a umístění v pixelech nezávislých na zařízení v prostoru souřadnic cíle vykreslení oblasti, do které je bitmapa nakreslena. Pokud obdélník není dobře uspořádané, nic je nakreslena, ale cíl vykreslení nepřejde do chybového stavu.

*fOpacity*<br/>
Hodnota mezi 0.0f a 1.0f včetně, která určuje hodnotu krytí, která se má použít pro bitmapu; tato hodnota se vynásobí alfa hodnotami obsahu bitmapy.

*interpolationMode*<br/>
Režim interpolace, který se použije, pokud je bitmapa zmenšena nebo otočena operací kreslení.

*pRectSrc*<br/>
Velikost a umístění v obrazových bodech nezávislých na zařízení v souřadnicovém prostoru bitmapy oblasti v bitmapě k nakreslení.
