# <a name="adding-an-mfc-message-handler"></a>Přidání popisovače zpráv knihovny MFC

Pomocí Průvodce [třídou](mfc-class-wizard.md) nebo okna **Vlastnosti** v **zobrazení služby CLass** můžete přidat obslužnou rutinu zprávy (členovou funkci, která zpracovává zprávy systému Windows) do třídy a mapovat zprávy systému Windows na obslužnou rutinu zprávy. Můžete také přidat [obslužnou rutinu události pro libovolný ovládací prvek dialogového okna](../../windows/adding-event-handlers-for-dialog-box-controls.md).

Pomocí okna **Průvodce třídou** nebo **Vlastnosti** (v **zobrazení třídy)** k definování funkcí zpracování zpráv a událostí můžete automaticky aktualizovat tabulku odeslání zpráv (nebo mapu zpráv) a soubor záhlaví třídy.

> [!NOTE]
> Obslužnou rutinu zprávy můžete přidat do třídy ATL pomocí Průvodce třídou; některé výsledky se však mohou lišit. Další informace naleznete v tématu ATL [Přidání obslužné rutiny zprávy ATL](../../atl/adding-an-atl-message-handler.md).
