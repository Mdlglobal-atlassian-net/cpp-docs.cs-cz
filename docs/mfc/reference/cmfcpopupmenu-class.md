# <a name="cmfcpopupmenu-class"></a>Třída cmfcpopupmenu

Implementuje funkce rozbalovací nabídky systému Windows a rozšiřuje ji přidáním funkcí, jako jsou nabídky odtržení a popisky.
Další podrobnosti naleznete ve zdrojovém kódu umístěném ve složce **MFC\\knihovny\\VC src\\** instalace sady Visual Studio.

## <a name="syntax"></a>Syntaxe

```
class CMFCPopupMenu : public CMiniFrameWnd
```

## <a name="cmfcpopupmenucheckarea"></a><a name="checkarea"></a>CMFCPopupMenu::CheckArea

Určuje umístění bodu vzhledem k rozbalovací nabídce.

```
MENUAREA_TYPE CheckArea(const CPoint& ptScreen) const;
```

### <a name="parameters"></a>Parametry

*ptScreen*<br/>
[v] Bod, na souřadnicích obrazovky.

### <a name="return-value"></a>Návratová hodnota

Parametr MENUAREA_TYPE, který označuje, kde je bod relativní k rozbalovací nabídce.

### <a name="remarks"></a>Poznámky

Parametr MENUAREA_TYPE může mít některou z následujících hodnot.

- OUTSIDE - *ptScreen* je mimo rozbalovací menu.

- LOGO - *ptScreen* je nad oblastí loga.

- TEAROFF_CAPTION - *ptScreen* je nad odtrhávací titulek.

- SHADOW_BOTTOM - *ptScreen* je nad spodním stínem rozbalovací nabídky.

- SHADOW_RIGHT - *ptScreen* je nad pravým stínem rozbalovací nabídky.

- MENU - *ptScreen* je nad příkazem.
