Další podrobnosti naleznete ve zdrojovém kódu umístěném ve složce **MFC\\knihovny\\VC src\\** instalace sady Visual Studio.

class CPaneContainerManager : public CObject

### <a name="remarks"></a>Poznámky

Rozhraní automaticky vytvoří instance `CPaneContainerManager` objektů a vloží je buď do objektů [třídy CPaneDivider,](../../mfc/reference/cpanedivider-class.md) nebo do objektů [třídy CMultiPaneFrameWnd.](../../mfc/reference/cmultipaneframewnd-class.md)

Třída `CPaneContainerManager` ukládá ukazatel na kořen binárního stromu, který je sestaven z objektů [CPaneContainer.](../../mfc/reference/cpanecontainer-class.md)
