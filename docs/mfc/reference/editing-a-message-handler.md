# <a name="editing-a-message-handler"></a>Úpravy obslužné rutiny zpráv
Jakmile je definován popisovač zpráv, můžete přejít na definici členskou funkci Přidat nebo upravit kód.  
  
 Přejít k definici členské funkce z [editoru dialogového okna](../../windows/dialog-editor.md), dvakrát klikněte na ovládací prvek, pro kterou je obslužná rutina již definován. Přejde souboru, ve kterém je definovaný obslužné rutiny zpráv vybraný ovládací prvek.  
  
 Přejít k definici členské funkce z [zobrazení tříd](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), dvakrát klikněte na název funkce v zobrazení tříd.
