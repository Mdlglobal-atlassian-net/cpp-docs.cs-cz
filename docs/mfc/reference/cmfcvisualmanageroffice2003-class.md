[v] Určuje, zda bude vizuální správce používat nativní barvy systému Windows XP.

### <a name="remarks"></a>Poznámky

Pokud *je hodnota bDefaultWinXPColors* true, vizuální manažer použije nativní barvy systému Windows XP, například modrou, olivovou nebo stříbrnou. V opačném případě bude vizuální manažer `GetSysColor`používat barvy získané z aplikace . Vizuální manažer používá vizuální prvky, jako jsou COLOR_3DFACE, COLOR_3DSHADOW, COLOR_3DHIGHLIGHT, COLOR_3DDKSHADOW a COLOR_3DLIGHT.

Ve výchozím `CMFCVisualManagerOffice2003` nastavení objekt používá nativní barvy motivu systému Windows XP.
