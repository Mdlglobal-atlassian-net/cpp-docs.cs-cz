# <a name="cframewndex-class"></a>CFrameWndEx – třída

Implementuje funkce rozhraní SDI (Windows single document interface) (SDI) a poskytuje členy pro správu okna. Rozšiřuje [CFrameWnd](../../mfc/reference/cframewnd-class.md) třídy.

## <a name="syntax"></a>Syntaxe

```
class CFrameWndEx : public CFrameWnd
```
