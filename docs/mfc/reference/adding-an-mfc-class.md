# <a name="adding-an-mfc-class"></a>Přidání třídy knihovny MFC
Chcete-li přidat třídy odvozené od třídy knihovna Microsoft Foundation Class (MFC) do projektu, použijte **přidat třídu** příkaz k dispozici v [zobrazení tříd](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925). Zadejte název nové třídy, vyberte základní třídy a vyberte ID dialogových oken, ke kterému je přiřazeno (pokud existuje). Kód průvodce vytvoří soubor hlaviček a soubor implementace a přidá je do projektu.  
  
> [!NOTE]
>  MFC – třídy můžete přidat do aplikace ATL COM, pokud jste původně [vytvoření aplikace s podporou MFC](../../atl/reference/mfc-support-in-atl-projects.md). MFC – třídy můžete také přidat do Win32 projektů, které mají Podpora MFC.  
  
### <a name="to-add-an-mfc-class-to-your-project"></a>Přidání třídy knihovny MFC do projektu  
  
1.  Ze třídy zobrazení klikněte pravým tlačítkem na název projektu. Klikněte na tlačítko **přidat** a pak klikněte na **přidat třídu** otevřete [přidat třídu](../../ide/add-class-dialog-box.md) dialogové okno.  
  
2.  V podokně šablon vyberte **třídy knihovny MFC** a stiskněte klávesu **přidat** tlačítko.  
  
3.  Zadejte nastavení pro novou třídu v [Průvodce třídou MFC](../../mfc/reference/mfc-add-class-wizard.md) dialogové okno.  
  
4.  Klikněte na tlačítko **Dokončit** zavřete průvodce a zobrazení nové třídy v zobrazení tříd. Můžete také zobrazit soubory vytvořené průvodcem v **Průzkumníku řešení**.
