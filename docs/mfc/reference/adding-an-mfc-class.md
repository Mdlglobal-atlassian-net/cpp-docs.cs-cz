# <a name="adding-an-mfc-class"></a>Přidání třídy knihovny MFC

Chcete-li do projektu přidat třídy odvozené z tříd knihovny Třídy Microsoft Foundation (MFC), použijte tlačítko **Přidat třídu** v [Průvodci třídou](mfc-class-wizard.md). Zadejte název nové třídy, vyberte základní třídu a vyberte ID dialogového okna, ke kterému je přidružena (pokud existuje). Průvodce vytvoří soubor záhlaví a soubor implementace a přidá je do projektu.

> [!NOTE]
> Třídy knihovny MFC můžete přidat do aplikace knihovny ATL COM, pokud jste původně [vytvořili aplikaci s podporou knihovny MFC](../../atl/reference/mfc-support-in-atl-projects.md). Můžete také přidat třídy knihovny MFC do projektů Win32, které mají podporu knihovny MFC.

### <a name="to-add-an-mfc-class-to-your-project"></a>Přidání třídy knihovny MFC do projektu

V zobrazení třídy klikněte pravým tlačítkem myši na název projektu. Klikněte na **Průvodce třídami** a potom kliknutím na **Přidat třídu** otevřete dialogové okno [Přidat třídu knihovny MFC.](mfc-add-class-wizard.md) Pomocí **Průvodce třídou** přidejte obslužné rutiny zpráv nebo nové metody.
