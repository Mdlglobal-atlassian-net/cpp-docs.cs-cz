## <a name="remarks"></a>Poznámky  
 Pás karet panelů jsou logická seskupení souvisejících úloh, které vytvoříte v rámci kategorií pásu karet. Jaká je velikost změny pásu karet rozložení panelu automaticky upraví, aby tolik prvky zobrazení míře.  
  
 Můžete získat pásu karet panely, které je součástí kategorie pásu karet pomocí volání [CMFCRibbonCategory::GetPanel](../../mfc/reference/cmfcribboncategory-class.md#getpanel) metoda.

##  <a name="addtoolbar"></a>CMFCRibbonPanel::AddToolBar  
 Přidá panel nástrojů panel pásu karet.  
  
```  
CMFCRibbonButtonsGroup* AddToolBar(
UINT uiToolbarResID,  
UINT uiColdResID = 0,  
UINT uiHotResID = 0,  
UINT uiDisabledResID = 0);
```  
  
### <a name="parameters"></a>Parametry  
 [v]`uiToolbarResID`  
 Určuje ID prostředku panelu nástrojů přidat.  
  
 [v]`uiColdResID`  
 Určuje ID prostředku bitové kopie cold panelu nástrojů.  
  
 [v]`uiHotResID`  
 Určuje ID prostředku bitové kopie aktivního panelu nástrojů.  
  
 [v]`uiDisabledResID`  
 Určuje ID prostředku bitové kopie zakázané panelu nástrojů.  
  
### <a name="return-value"></a>Návratová hodnota  
 Volejte tuto metodu za účelem přidání panelu nástrojů na panelu pásu karet. Přidá se panelu nástrojů vedle pásu karet element přidal předchozího volání [CMFCRibbonPanel::Add](#add).  
  
### <a name="remarks"></a>Poznámky  
 Další informace o panely nástrojů, aktivní bitové kopie, cold bitové kopie a zakázané bitové kopie najdete v tématu [CMFCToolBar třída](../../mfc/reference/cmfctoolbar-class.md).

```  
void SetJustifyColumns(BOOL bSet = TRUE);
```  
  
### <a name="parameters"></a>Parametry  
 [v]`bSet`

### <a name="remarks"></a>Poznámky  
 Pokud je tato funkce povolena panelu pásu karet, šířek elementů pásu karet ve stejném sloupci upraveny tak, aby šířka elementu největší pásu karet ve stejném sloupci.
