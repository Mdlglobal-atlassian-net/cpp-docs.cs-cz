## <a name="remarks"></a>Poznámky  
 `CMFCTasksPane` Třída implementuje následující funkce:  
  
-   Je možné seskupit položky a každý seskupení položek může mít přidružené titulek.  
  
-   Seskupení položek můžete sbalené nebo rozšířit.  
  
-   Pro každou položku v podokně úloh lze přiřadit ikonu.  
  
-   Jednotlivé položky může být přidružen ID příkazu, který provede, když uživatel klikne na položku. Když dojde k kliknutím na možnost, `WM_COMMAND` vlastník ovládacího prvku podokno úloha odeslána zpráva.  
  
 Použít `CMFCTasksPane` řízení ve vaší aplikaci, postupujte takto:  
  
1.  Vložení `CMFCTasksPane` objekt do třídy hlavního rámce okna.  
  
2.  Při zpracování `WM_CREATE` zprávy, volejte `Create` metoda. Můžete použít běžné [ccontrolbar –](../../mfc/reference/ccontrolbar-class.md) stylů. Další informace naleznete v tématu `CControlBar::Create`.  
  
3.  Volání [CMFCTasksPane::AddGroup](#addgroup) metoda přidání různých skupin.  
  
4.  Volání [CMFCTasksPane::AddTask](#addtask), [CMFCTasksPane::AddLabel](#addlabel) nebo [CMFCTasksPane::AddMRUFilesList](#addmrufileslist) členské funkce k přidání nových položek (úlohy) ke každé skupině.  
  
5.  Volání [CMFCTasksPane::EnableGroupCollapse](#enablegroupcollapse) k určení, zda lze sbalit skupiny položek.

##  <a name="addgroup"></a>CMFCTasksPane::AddGroup  
 Přidá novou skupinu úloh do ovládacího prvku podokno úlohy.  
  
```  
int AddGroup(
    int nPageIdx,  
    LPCTSTR lpszGroupName,  
    BOOL bBottomLocation = FALSE,  
    BOOL bSpecial = FALSE,  
    HICON hIcon = NULL);

 
int AddGroup(
    LPCTSTR lpszGroupName,  
    BOOL bBottomLocation = FALSE,  
    BOOL bSpecial = FALSE,  
    HICON hIcon = NULL);
```  
  
### <a name="parameters"></a>Parametry  
 [v]`nPageIdx`  
 Určuje index počítaný od nuly stránky.  
  
 [v]`lpszGroupName`  
 Určuje název skupiny.  
  
 [v]`bBottomLocation`  
 `TRUE`Vytvořte skupinu v dolní části ovládacího prvku podokno úloha; v opačném `FALSE`.  
  
 [v]`bSpecial`  
 `TRUE`k označení této skupiny jako *speciální* skupiny; v opačném `FALSE`. Další informace o speciální skupiny, najdete v části poznámky `CMFCTasksPane`.  
  
 [v]`hIcon`  
 Určuje ikonu zobrazíte v záhlaví skupiny.

 Index založený na nule skupiny v seznamu interní skupin, které udržuje třídy.

 Volejte tuto metodu, vytvořte skupinu úloh a přidejte tuto skupinu do ovládacího prvku podokno úlohy.  
  
 Rozhraní framework zobrazí skupiny úloh v horní části ovládacího prvku podokno úlohy nebo v dolní části. Rozhraní framework můžete zobrazit jenom jedné skupiny v dolní části; Tato skupina je nutné přidat jako poslední.
