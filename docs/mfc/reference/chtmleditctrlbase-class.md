## <a name="chtmleditctrlbaseaddtoglyphtable"></a><a name="addtoglyphtable"></a>ChtmlEditCtrlBase::Addtoglyphtabletable

Přidá do tabulky glyfů položku, která určuje obrazy, které se mají zobrazit pro určité tagy v návrhovém režimu.

```
HRESULT AddToGlyphTable(
    LPCTSTR szTag,
    LPCTSTR szImgUrl,
    unsigned short nTagType,
    unsigned short nAlignment,
    unsigned short nPosInfo,
    unsigned short nDirection,
    unsigned int nImgWidth,
    unsigned int nImgHeight) const;
```

### <a name="parameters"></a>Parametry

*szTag*<br/>
Název značky (například "P" nebo "tabulka").

*szImgUrl*<br/>
Adresa URL obrázku.

*nTagType*<br/>
Typ značky: 0 znamená, že obrázek je určen pouze pro otevírací značku. 1 znamená, že obraz je určen pouze pro uzavírací značku. 2 znamená, že obraz je určen jak pro otevírací, tak pro zavírání značek. Jednotlivé značky, jako je br a komentář, musí být přidány s typem značky nastaveným na 0.

*nVyrovnání*<br/>
Zarovnání (pouze obdélníkové prvky): Tento parametr označuje, že obraz je pro prvek s atributem zarovnání. Vlevo = 0, střed = 1, vpravo = 2 a undefined = 3. Atributy vlevo, vpravo nebo na střed musí být explicitně nastaveny na elementu.

*nPosInfo*<br/>
Informace o umístění. Určuje, na jakou kaskádovou hodnotu umístění se glyf vztahuje, kde statické umístění = 0, absolutní umístění = 1, relativní umístění = 2 a všechny = 3. Toto pole umožňuje zadat jeden glyf pro značku, když není umístěna, a jiný glyf, který zobrazí kotevní bod, když je značka umístěna.

*nSměr*<br/>
Směr. Tento parametr určuje obrázek značky na základě pořadí čtení aktuálního jazyka. 0 určuje zleva doprava, 1 zprava doleva, 2 určuje shora dolů, 3 určuje zdola nahoru a 4 určuje vše. Toto pole obvykle nastavíte na 4.

*nImgWidth*<br/>
Šířka obrázku v obrazových bodech.

*nImgVýška*<br/>
Výška obrazu v obrazových bodech.

### <a name="return-value"></a>Návratová hodnota

Vrátí S_OK na úspěch nebo chybu HRESULT při selhání.

### <a name="remarks"></a>Poznámky

Další informace o parametrech naleznete v části Formát řetězce tabulky glyfů [v tématu Použití glyfů v části Úpravy glyfů](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa969614\(v=vs.85\)).
