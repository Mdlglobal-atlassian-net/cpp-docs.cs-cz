# <a name="cpanecontainer-class"></a>Třída CPaneContainer

Třída `CPaneContainer` je základní součástí dokovacího modelu implementovaného knihovnou MFC. Objekt této třídy ukládá ukazatele na dvě ukotvení podokna `CPaneContainer.` nebo dvě instance Je také ukládá ukazatel na oddělovač, který odděluje podokna (nebo kontejnery). Vnořením kontejnerů uvnitř kontejnerů může rozhraní vytvořit binární strom, který představuje komplexní rozložení ukotvení. Kořen binárního stromu je uložen v objektu [CPaneContainerManager.](../../mfc/reference/cpanecontainermanager-class.md)

Další podrobnosti naleznete ve zdrojovém kódu umístěném ve složce **MFC\\knihovny\\VC src\\** instalace sady Visual Studio.
