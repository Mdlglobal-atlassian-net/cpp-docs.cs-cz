# <a name="creating-windows"></a>Vytváření oken

Většina systému windows, které potřebujete rozhraní framework automaticky vytvoří v rámci programu. [Vytváření dokumentů/zobrazení](../mfc/document-view-creation.md) ukazuje způsob, jak vytvořit rozhraní okna rámce spojených s dokumenty a zobrazeními. Ale pro speciální účely lze vytvořit vlastní windows – včetně vlastní podřízených oken s rámečkem nebo zobrazení – kromě poskytnutých rozhraní systému windows.
