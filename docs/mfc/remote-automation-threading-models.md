# <a name="remote-automation-threading-models"></a>Modely vláken u vzdálené automatizace
Vzdálená automatizace podporuje single-threaded a modely apartment-threaded COM. Na 32bitové a 64bitové verze platformy, které jsou všechno, co podporuje Visual C++, model vláken musí být nastavena na "apartment." Další informace o tom a kde to udělat najdete v tématu [vytváření programy, použijte vzdálené automatizace](../mfc/creating-programs-that-use-remote-automation.md).
