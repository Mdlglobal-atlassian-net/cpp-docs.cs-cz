# <a name="setting-the-dialog-boxs-background-color"></a>Nastavení barvy pozadí dialogového okna

Můžete nastavit barvu pozadí dialogových oken, a to tak, že nakonfigurujete zprávy WM_CTLCOLOR pro okno dialogového okna. Barva, kterou nastavíte, se používá pouze pro zadané dialogové okno.

Příklad najdete v [blogu codexpert](https://codexpert.ro/blog/2013/03/13/painting-the-dialog-backround/) .
