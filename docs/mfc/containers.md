# <a name="containers"></a>Kontejnery

Aplikace typu kontejner je aplikace, která můžete začlenit do vlastní dokumenty vložené nebo propojené položky. Spravuje aplikace typu kontejner pro dokumenty musí mít k uložení a zobrazení součástí složeného dokumentu OLE, jakož i data vytvořená samotná aplikace. Aplikace typu kontejner musí také umožňují uživatelům vkládání nových položek nebo upravit existující položky.
