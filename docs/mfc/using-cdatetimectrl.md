# <a name="using-cdatetimectrl"></a>Používání atributu CDateTimeCtrl

Ovládací prvek výběr data a času[(CDateTimeCtrl](../mfc/reference/cdatetimectrl-class.md)) implementuje intuitivní a rozpoznatelnou metodu zadávání nebo výběru konkrétního data. Hlavní rozhraní ovládacího prvku je ve funkci podobné pole se seznamem. Pokud však uživatel ovládací prvek rozbalí, zobrazí se ovládací prvek kalendáře měsíce (ve výchozím nastavení), který uživateli umožní zadat určité datum. Je-li vybráno datum, ovládací prvek kalendář měsíce automaticky zmizí.

> [!NOTE]
> Chcete-li `CDateTimePicker` použít `CMonthCalCtrl` a třídy v projektu, musíte zahrnout AFXDTCTL. H, obvykle v projektu STDAFX. H soubor.
