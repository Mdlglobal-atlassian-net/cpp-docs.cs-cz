# <a name="using-ctooltipctrl"></a>Používání atributu CToolTipCtrl

[CToolTipCtrl](../mfc/reference/ctooltipctrl-class.md) třídy zapouzdřuje funkce ovládacího prvku nástroj tip, malého vyskakovacího okna, která zobrazuje jeden řádek textu popisujícího účel nástroje v aplikaci. Ve většině případů, povolí, jenom když uživatel umístí kurzor na nástroje a ponechá ho existuje pro přibližně polovinu sekundy je skrytý popisku tlačítka. Popis tlačítka zobrazen v blízkosti kurzoru a zmizí, když uživatel stiskne tlačítko myši nebo přesune kurzor mimo nástroj.
