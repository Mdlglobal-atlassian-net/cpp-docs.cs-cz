# <a name="using-ctooltipctrl"></a>Používání atributu CToolTipCtrl

Třída [CToolTipCtrl](../mfc/reference/ctooltipctrl-class.md) zapouzdřuje funkce ovládacího prvku popis tlačítka, což je malé překryvné okno, které zobrazuje jeden řádek textu popisujícího účel nástroje v aplikaci. Popis tlačítka je v tuto chvíli skrytý a zobrazuje se jenom v případě, že uživatel přesune kurzor na nástroj a ponechá ho na přibližně jednu polovinu sekundy. Pokud uživatel klikne na tlačítko myši nebo přesune kurzor mimo nástroj, zobrazí se popis tlačítka v blízkosti kurzoru a zmizí.
