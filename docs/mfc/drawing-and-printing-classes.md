# <a name="drawing-and-printing-classes"></a>Třídy pro vykreslování a tisk

Ve Windows se veškerý grafický výstup vykreslí ve virtuální oblasti pro vykreslování s názvem kontext zařízení (DC). Knihovna MFC poskytuje třídy pro zapouzdření různých typů řadičů domény a také zapouzdření pro nástroje pro kreslení systému Windows, jako jsou rastrové obrázky, štětce, palety a pera.
