# <a name="activex-controls"></a>ActiveX – ovládací prvky

V jazyce Visual C++ můžete vytvořit ovládací prvky ActiveX pomocí knihovny MFC ani ATL.

>[!IMPORTANT]
> ActiveX je starší technologie, která by neměla být používána při novém vývoji. Mnoho funkcí ovládacích prvků ActiveX lze provádět v jednodušší a mnohem bezpečnější způsob pomocí moderních technologií, jako je HTML5 a JavaScript, rozšíření moderní prohlížeče nebo WebAssembly moduly. Další informace najdete v tématu [přerušení z minulosti, část 2: zavření attachEvent ActiveX jazyka VBScript,...](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) a [nativní zasílání zpráv](/microsoft-edge/extensions/guides/native-messaging) a [rozšíření Microsoft Edge](/microsoft-edge/extensions) a [WebAssembly](https://webassembly.org/).
