# <a name="using-frame-windows"></a>Použití oken s rámečkem

Rozhraní MFC vytvoří dokument oken s rámečkem – a jejich zobrazení a dokumenty – jako součást jeho implementace nové a spuštění příkazů v nabídce Soubor. Protože většina práce oken s rámečkem rozhraní udělá za vás, hrát pouze malé roli při vytváření, používání a zničení otevřená okna. Můžete však explicitně vytvořit vlastní okna s rámečkem a podřízená okna pro zvláštní účely.
