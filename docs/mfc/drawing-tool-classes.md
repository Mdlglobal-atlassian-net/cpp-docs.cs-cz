# <a name="drawing-tool-classes"></a>Třídy nástrojů pro kreslení
Tyto třídy zapouzdřovat nástrojů pro kreslení, které se používají k vykreslení v kontextu zařízení.  
  
 [CGdiObject](../mfc/reference/cgdiobject-class.md)  
 Základní třída pro GDI nástrojů pro kreslení.  
  
 [CBrush](../mfc/reference/cbrush-class.md)  
 Zapouzdří GDI štětce, kterou je možné vybrat jako aktuální štětce v kontextu zařízení. Štětce slouží k naplnění vnitřek Přitahuje objekty.  
  
 [CPen](../mfc/reference/cpen-class.md)  
 Zapouzdří pera GDI, kterou je možné vybrat jako aktuální pero v kontextu zařízení. Pera se používají pro kreslení čar ohraničení objektů.  
  
 [CFont](../mfc/reference/cfont-class.md)  
 Zapouzdří GDI písmo, které lze vybrat jako aktuální písma v kontextu zařízení.  
  
 [CBitmap](../mfc/reference/cbitmap-class.md)  
 Zapouzdří poskytuje rozhraní pro manipulaci s bitovými mapami rastrový obrázek GDI.  
  
 [CPalette](../mfc/reference/cpalette-class.md)  
 Zapouzdří palety barev GDI pro použití jako rozhraní mezi aplikací a zařízení výstupní barva např. zobrazení.  
  
 [Crecttracker –](../mfc/reference/crecttracker-class.md)  
 Zobrazí a zpracovává uživatelské rozhraní pro změnu velikosti a přesunutí obdélníková objekty.
