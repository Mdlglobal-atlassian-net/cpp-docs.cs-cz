# <a name="using-ctabctrl"></a>Používání atributu CTabCtrl

"Ovládací prvek" TAB "je analogický k oddělovačům v poznámkovém bloku nebo k označeným složkám v souboru CAB. Použijte ovládací prvek karta reprezentovaný třídou [atributu CTabCtrl](../mfc/reference/ctabctrl-class.md), chcete-li zobrazit více stránek s informacemi nebo ovládacími prvky pro uživatele, jeden po druhém ve formátu, který navrhuje partnerský vztah nebo logickou relaci mezi každou stránkou.

Další informace o ovládacích prvcích karet naleznete v tématu [ovládací prvky karty](/windows/win32/Controls/tab-controls) v Windows SDK.
