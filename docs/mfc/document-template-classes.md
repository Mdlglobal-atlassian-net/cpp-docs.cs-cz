# <a name="document-template-classes"></a>Třídy šablony dokumentu

Objekty šablony dokumentu koordinují vytváření objektů, zobrazení a oken s rámečkem v případě, že je vytvořen nový dokument nebo zobrazení.

[CDocTemplate –](../mfc/reference/cdoctemplate-class.md)<br/>
Základní třída pro šablony dokumentů. Tuto třídu nikdy nebudete používat přímo; místo toho použijete jednu z dalších tříd šablony dokumentu odvozených z této třídy.

[CMultiDocTemplate](../mfc/reference/cmultidoctemplate-class.md)<br/>
Šablona pro dokumenty v rozhraní MDI (Multiple Document Interface). V aplikacích MDI může být otevřeno více dokumentů najednou.

[CSingleDocTemplate](../mfc/reference/csingledoctemplate-class.md)<br/>
Šablona pro dokumenty v rozhraní SDI (Single Document Interface). Aplikace SDI mají v jednom okamžiku otevřený pouze jeden dokument.

## <a name="related-class"></a>Související třídy

[CCreateContext](../mfc/reference/ccreatecontext-structure.md)<br/>
Struktura předaná šablonou dokumentu pro funkce vytváření oken, která koordinuje vytváření objektů dokumentů, zobrazení a oken s rámečkem.
