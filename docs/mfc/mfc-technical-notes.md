# <a name="mfc-technical-notes"></a>MFC – technické poznámky
Technická poznámka je napsán pro programátory v jazyce programátory dokumentu.  
  
 Každý Technická poznámka popisuje problém nebo funkce, která je nad rámec zbytek dokumentace knihovny MFC. Technické poznámky zadaný projeví žádosti o informace od uživatelů, stejně jako specializované informace, že vývojáři MFC nichž bude Pokročilí uživatelé mají.
