# <a name="tn050-mfcole-common-dialogs-mfcuix32"></a>TN050: Dialogová okna běžné knihovny MFC/OLE (MFCUIx32)

Tato poznámka původně popsané některé problémy a budoucnost grafického subsystému OLE běžných dialogových oken k dispozici a využívané prostředím MFC. Společná dialogová okna OLE jsou teď k dispozici jako součást integrované do systému (OLEDLG. Knihovny DLL) a jsou plně dokumentovány v článku v dokumentaci k produktu.
