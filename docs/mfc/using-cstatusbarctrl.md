# <a name="using-cstatusbarctrl"></a>Používání atributu CStatusBarCtrl

Můžete použít ovládací prvek panelu stavu ([CStatusBarCtrl](../mfc/reference/cstatusbarctrl-class.md)) Chcete-li vytvořit okno ovládacího prvku, který odráží různé druhy stavové informace o aplikaci. Stavové okno je možné rozdělit na oddíly, které zobrazují více než jeden typ informací.
