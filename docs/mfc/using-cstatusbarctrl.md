# <a name="using-cstatusbarctrl"></a>Používání atributu CStatusBarCtrl

Pomocí ovládacího prvku stavový řádek ([CStatusBarCtrl](../mfc/reference/cstatusbarctrl-class.md)) můžete vytvořit okno ovládacího prvku, které odráží různé druhy informací o stavu aplikace. Stavové okno lze rozdělit na části, které zobrazují více než jeden typ informací.
