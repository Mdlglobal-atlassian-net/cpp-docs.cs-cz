# <a name="automation-manager-mfc"></a>Správce automatizace (MFC)
AUTMGR32. EXE by měl zkopírovat do adresáře systému Windows každý počítač, který je hodláte zadejte vzdálené automatizace objekty. Je to obvykle C:\WINDOWS\SYSTEM32.  
  
 Pokud chcete povolit zpětná volání ze serveru a klienta, by tento spustitelný soubor také zkopírovat do adresáře systému každý klientský počítač.  
  
 Když je spuštěn Správce automatizace, zobrazí malém okně na serveru, který obsahuje informace o stručný stavu. Pokud chcete skrýt ho, podívejte se na článek Q138067 znalostní báze Microsoft Knowledge Base.
