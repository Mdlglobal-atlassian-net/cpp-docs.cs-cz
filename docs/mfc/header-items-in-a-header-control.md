# <a name="header-items-in-a-header-control"></a>Položky hlaviček v ovládacím prvku hlavička

Máte značné ovládat vzhled a chování položek záhlaví, které tvoří ovládacím prvkem záhlaví ([CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)). Každá položka záhlaví může mít řetězce, image s rastrovými obrázky, bitové kopie ze seznamu přidružených obrázků nebo s ním spojená hodnotu 32-bit definovaného aplikací. Řetězec, rastrový obrázek nebo obrázek se zobrazí v záhlaví položek.

Při jejich vytváření tím, že zavoláte můžete přizpůsobit vzhled a obsah nových položek [CHeaderCtrl::InsertItem](../mfc/reference/cheaderctrl-class.md#insertitem) nebo úpravou existující položky, se volání [CHeaderCtrl::GetItem](../mfc/reference/cheaderctrl-class.md#getitem) a [ CHeaderCtrl::SetItem](../mfc/reference/cheaderctrl-class.md#setitem).
