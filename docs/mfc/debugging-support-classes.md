# <a name="debugging-support-classes"></a>Ladění podpůrných tříd
Knihovna MFC poskytuje následující třídy pro ladění problémů s přidělováním dynamické paměti.  
  
 [CDumpContext](../mfc/reference/cdumpcontext-class.md)  
 Poskytuje cíl pro výpisy diagnostiky.  
  
 [CMemoryState](../mfc/reference/cmemorystate-structure.md)  
 Struktura, která poskytuje snímky využití paměti. Také se používá k porovnání snímky paměti dříve a později.
