# <a name="application-framework"></a>Aplikační framework

Jádrem knihovny Microsoft Foundation Class (MFC) je zapouzdření velké části rozhraní API systému Windows ve C++ formuláři. Třídy knihoven reprezentují okna, dialogová okna, kontexty zařízení, běžné objekty GDI, jako jsou štětce, ovládací prvky a další standardní položky Windows. Tyto třídy poskytují praktické C++ rozhraní členské funkce ke strukturám ve Windows, které jsou zapouzdřeny. Další informace o použití těchto tříd naleznete v tématu [témata o objektech oken](../mfc/window-objects.md).

Knihovna MFC ale také poskytuje vrstvu dalších funkcí aplikace postavených na C++ ZAPOUZDŘENÍ rozhraní API systému Windows. Tato vrstva je funkční aplikační architektura pro Windows, která poskytuje většinu běžných uživatelských rozhraní očekávajících programy pro Windows, včetně panelů nástrojů, stavových pruhů, tisku, náhledu tisku, podpory databází a podpory ActiveX. [Použití tříd pro psaní aplikací pro systém Windows](../mfc/using-the-classes-to-write-applications-for-windows.md) podrobněji vysvětluje rozhraní.
