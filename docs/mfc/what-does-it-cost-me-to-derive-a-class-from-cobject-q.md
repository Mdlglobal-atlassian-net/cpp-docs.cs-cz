# <a name="what-does-it-cost-me-to-derive-a-class-from-cobject"></a>Jaké jsou náklady na odvození třídy z objektu CObject?

Režie vyplývající z třídy [CObject](../mfc/reference/cobject-class.md) je minimální. Vaše odvozená třída dědí pouze čtyři virtuální funkce a jeden objekt [CRuntimeClass](../mfc/reference/cruntimeclass-structure.md) .
