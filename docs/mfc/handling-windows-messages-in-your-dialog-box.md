# <a name="handling-windows-messages-in-your-dialog-box"></a>Zpracování zpráv systému Windows ve vašem dialogovém okně

Dialogová okna jsou okna, která mohou zpracovávat zprávy systému Windows, pokud zadáváte příslušné funkce obslužných rutin. Když vytvoříte třídu dialogového okna pomocí [Průvodce třídou](reference/mfc-class-wizard.md), Průvodce přidá do třídy prázdnou mapu zprávy. Pomocí Průvodce namapujte libovolné zprávy nebo příkazy systému Windows, které má vaše třída zpracovávat.

Další informace najdete v tématu [mapování zpráv systému Windows na třídu dialogu](../mfc/mapping-windows-messages-to-your-class.md) .
