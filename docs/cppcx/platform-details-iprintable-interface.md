Toto rozhraní je určen pouze pro interní použití a není určena k použití pro vývoj.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
  
interface class IPrintable;  
```  
  
### <a name="remarks"></a>Poznámky
