# <a name="platformenum-class"></a>Platform::enum – třída

Hodnota třídy, která představuje sadou pojmenovaných konstant.

## <a name="syntax"></a>Syntaxe

```cpp
public class Enum
```

### <a name="members"></a>Členové

Enum class dědí z metody Equals() GetHashCode() a ToString() [Platform::Object – třída](../cppcx/platform-object-class.md).

### <a name="remarks"></a>Poznámky

Použití [veřejný výčet tříd](../extensions/enum-class-cpp-component-extensions.md) – klíčové slovo k vytvoření výčtů. Explicitně nepoužívejte typ Platform::Enum –. Další informace najdete v tématu [výčty](../cppcx/enums-c-cx.md).

### <a name="requirements"></a>Požadavky

**Minimální podporovaná klienta:** Windows 8

**Minimální podporovaná serveru:** Windows Server 2012

**Namespace:** Platforma

**Metadata:** platform.winmd
