# <a name="namespaces-reference-ccx"></a>Odkaz na obory názvů (C + +/ CX)
Články v této části dokumentace popisují obory názvů, které podporují kompilátor pro C + +/ CX.  
  
## <a name="compiler-supplied-namespaces"></a>Zadaný kompilátoru obory názvů  
 Aby se zjednodušila kódování programy, které cílí na prostředí Windows Runtime, s C + +/ CX kompilátoru a jeho podpůrné soubory hlaviček poskytují obory názvů, které definují řadu různých typů. Obory názvů definujte předdefinované číselnými typy; řetězce, pole a kolekce; Visual C++ výjimky, které představují prostředí Windows Runtime chyby; a vylepšení pro specifický jazyk, které standardní typy prostředí Windows Runtime.
