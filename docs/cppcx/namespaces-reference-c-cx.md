# <a name="namespaces-reference-ccx"></a>Odkaz na obory názvů (C++/CX)

Články v této části dokumentace popisují obory názvů, které podporují kompilátor pro C++/CX.

## <a name="compiler-supplied-namespaces"></a>Obory názvů dodaných kompilátorem

Pro zjednodušení kódování programů, které cílí na prostředí Windows Runtime, kompilátor C++/CX a jeho podpůrné hlavičkové soubory poskytují obory názvů, které definují široké spektrum typů. Obory názvů definují předdefinované číselné typy; řetězce, pole a kolekce; C++ výjimky, které reprezentují chyby prostředí Windows Runtime; a vylepšení pro standardní prostředí Windows Runtime pro konkrétní jazyk.
