# <a name="platformuintptr-value-class"></a>Platform::UIntPtr – hodnotová třída
Představuje nepodepsané ukazatele, jejíž aktuální velikost je vhodný pro platformu (32bitová nebo 64bitová verze).  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
public value struct UintPtr  
```  
  
### <a name="requirements"></a>Požadavky  
 **Minimální podporovaná klienta:** Windows 8  
  
 **Minimální podporovaná serveru:** systému Windows Server 2012  
  
 **Namespace:** platformy  
  
 **Metadata:** platform.winmd
