# <a name="platformdetailsheapentryhandler-delegate"></a>Delegát Platform::details::HeapEntryHandler
Tento delegát je určen pouze pro interní použití a není určena k použití pro vývoj.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
  
delegate HeapEntryHandler;  
```  
  
### <a name="remarks"></a>Poznámky  
 Tato třída je určena výhradně pro úplnost, protože může být prověřovány s prohlížeči.
