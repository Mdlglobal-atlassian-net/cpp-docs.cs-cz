# <a name="c-features-deprecated-in-visual-studio-2019-preview"></a>C++Funkce zastaralé ve verzi preview. 2019 Visual Studio

## <a name="support-for-windows-xp-development"></a>Podpora pro vývoj pro Windows XP

Visual Studio 2019 později nezahrnuje podporu pro vytváření kódu pro systém Windows XP s využitím sady nástrojů v142. Podpora Windows XP vývoj pomocí sady nástrojů verze 141, dodávané v sadě Visual Studio 2017 je dostupné jako volitelná součást v instalačním programu sady Visual Studio. Informace o tom, jak nainstalovat sadu nástrojů verze 141 Windows XP platformy najdete v tématu [konfigurace aplikací pro Windows XP](../build/configuring-programs-for-windows-xp.md).
