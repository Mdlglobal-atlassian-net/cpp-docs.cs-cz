# <a name="operator-low32"></a>LOW32 – operátor

Vrátí dolní 32 (počet bitů) *výrazu*. Výrazy MASM jsou 64 bitové hodnoty.

## <a name="syntax"></a>Syntaxe

> *Výraz* low32 –
