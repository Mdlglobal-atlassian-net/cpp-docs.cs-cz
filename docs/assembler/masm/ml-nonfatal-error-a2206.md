# <a name="ml-nonfatal-error-a2206"></a>Méně závažná chyba nástroje ML A2206
**Chybí operátor ve výrazu**  
  
 Výraz nelze vyhodnotit, protože neobsahuje operátor. Tato chybová zpráva může být také vedlejším účinkem předchozí chybě programu.  
  
 Následující řádek vygeneruje se tato chybová zpráva:  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
