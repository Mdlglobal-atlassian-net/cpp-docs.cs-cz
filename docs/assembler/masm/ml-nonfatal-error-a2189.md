# <a name="ml-nonfatal-error-a2189"></a>Méně závažná chyba nástroje ML A2189

**Neplatná kombinace s zarovnáním segmentů**

Zarovnání určené direktivou **align** nebo `EVEN` bylo větší než aktuální zarovnání segmentu, jak je uvedeno v direktivě **segmentu** .
