# <a name="operator-ne"></a>operátor NE

Vrátí hodnotu true (-1), pokud *Výraz1* není rovna *Výraz2*, nebo vrátí false (0), pokud ano.

## <a name="syntax"></a>Syntaxe

> *Výraz1* **ne** *Výraz2*
