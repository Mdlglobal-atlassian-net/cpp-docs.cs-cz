# <a name="operator-ne"></a>operátor NE

Vrátí hodnotu true (-1) Pokud *expression1* nerovná *expression2*, nebo vrátí hodnotu false (0), pokud už existoval.

## <a name="syntax"></a>Syntaxe

> Ne. expression2 Expression1
