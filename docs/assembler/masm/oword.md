# <a name="oword"></a>OWORD

Používá se jako specifikátor typu, když je vyžadován 16bitový datový typ.

## <a name="syntax"></a>Syntaxe

> ⟦*Name*⟧ **OWORD**  *inicializátor* ⟦ __,__ *inicializátor* ... ⟧
