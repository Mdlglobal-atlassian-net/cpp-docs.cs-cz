# <a name="ml-nonfatal-error-a2044"></a>Méně závažná chyba nástroje ML A2044

**Neplatný znak v souboru**

Zdrojový soubor obsahuje znak mimo komentář, řetězce nebo literál, který nebyl rozpoznán jako operátor nebo jiné právní znak.
