# <a name="ml-nonfatal-error-a2044"></a>Méně závažná chyba nástroje ML A2044

**neplatný znak v souboru**

Zdrojový soubor obsahoval znak mimo komentář, řetězec nebo literál, který nebyl rozpoznán jako operátor nebo jiný právní znak.
