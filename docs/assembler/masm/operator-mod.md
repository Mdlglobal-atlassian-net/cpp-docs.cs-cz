# <a name="operator-mod"></a>MOD – operátor
Vrací celočíselnou hodnotu zbytek (modulo) při dělení *expression1* podle *Výraz2*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 MOD expression2  
```
