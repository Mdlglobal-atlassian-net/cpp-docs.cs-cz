# <a name="version"></a>Verze \@

Hlavní a podverze MASMy hlášené na příkazovém řádku jako jedno číslo (textové makro). Například MASM verze 14.23.28107.0 vytvoří "1423".

## <a name="syntax"></a>Syntaxe

> **Verze \@**
