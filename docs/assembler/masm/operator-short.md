# <a name="operator-short"></a>SHORT – operátor

Nastaví typ *popisku* na short. Všechny skoky na *návěští* musí být krátké (v rozsahu od-128 do + 127 bajtů z instrukce skoku na *návěští*).

## <a name="syntax"></a>Syntaxe

> **Krátký** *popisek*
