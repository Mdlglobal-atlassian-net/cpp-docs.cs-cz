# <a name="operator-short"></a>SHORT – operátor

Sets the type of *label* to short. All jumps to *label* must be short (within the range -128 to +127 bytes from the jump instruction to *label*).

## <a name="syntax"></a>Syntaxe

> **SHORT** *label*
