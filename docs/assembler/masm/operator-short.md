# <a name="operator-short"></a>SHORT – operátor
Nastaví typ *popisek* na krátké. Všechny přeskočí *popisek* musí být krátký (v rozsahu -128 bajtů + 127 z přechod pokyn k *popisek*).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
SHORT label  
```
