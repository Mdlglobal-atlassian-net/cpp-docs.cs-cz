# <a name="data"></a>@data
Název skupiny výchozí data. Vyhodnotí jako DGROUP pro všechny modely, s výjimkou **ploché**. Vyhodnotí jako **ploché** pod **ploché** modelu paměti (text makro).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
@data  
  
```
