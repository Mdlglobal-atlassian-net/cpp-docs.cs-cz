# <a name="data"></a>data \@

Název výchozí datové skupiny. Vyhodnotí jako DGROUP pro všechny modely s výjimkou **plochých**. Vyhodnotí se jako **ploché** v modelu **ploché** paměti (textové makro).

## <a name="syntax"></a>Syntaxe

> **data \@**
