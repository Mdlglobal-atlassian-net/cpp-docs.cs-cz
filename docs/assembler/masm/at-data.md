# <a name="data"></a>@data

Název výchozí skupinu dat. Vyhodnotí jako DGROUP pro všechny modely s výjimkou **PLOCHÝ**. Vyhodnotí jako **PLOCHÝ** pod **PLOCHÝ** paměťový model (text makra).

## <a name="syntax"></a>Syntaxe

> @data
