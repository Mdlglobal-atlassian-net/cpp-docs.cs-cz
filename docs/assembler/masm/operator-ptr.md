# <a name="operator-ptr"></a>PTR – operátor

První operátor vynutí, aby se *výraz* zpracoval se zadaným *typem*. Druhý operátor určuje ukazatel na *typ*.

## <a name="syntax"></a>Syntaxe

> *typ* **PTR** – *výraz*
>
> \[*vzdálenost*] *typ* PTR
