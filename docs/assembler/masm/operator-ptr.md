# <a name="operator-ptr"></a>PTR – operátor
První vynutí operátor *výraz* považován za s zadaný `type`. Druhý operátor určuje ukazatel na `type`.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
