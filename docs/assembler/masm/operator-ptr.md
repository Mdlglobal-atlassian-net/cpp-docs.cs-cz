# <a name="operator-ptr"></a>PTR – operátor

První operátor sil *výraz* zacházeno jako s danou *typ*. Druhý operátor určuje ukazatel na *typ*.

## <a name="syntax"></a>Syntaxe

> *typ* PTR *výraz*
>
> \[*vzdálenost*] PTR *typu*
