# <a name="qword"></a>QWORD

Přidělí a případně inicializuje 8 bajtů úložiště pro každý *inicializátor*. Lze také použít jako specifikátor typu kdekoli je typ právní. Hodnota **QWORD** je synonymem [DQ](dq.md).

## <a name="syntax"></a>Syntaxe

> ⟦*Name*⟧ **QWORD** *inicializátor* ⟦ __,__ *inicializátor* ... ⟧
