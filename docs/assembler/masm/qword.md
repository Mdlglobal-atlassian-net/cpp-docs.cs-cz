# <a name="qword"></a>QWORD
Přiděluje a volitelně inicializuje 8 bajtů úložiště pro každou `initializer`. Můžete také použít jako specifikátor typu kdekoli je právní typu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
