# <a name="fword"></a>FWORD
Přiděluje a volitelně inicializuje 6 bajtů úložiště pro každou `initializer`.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
[[name]] FWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Poznámky  
 Můžete také použít jako specifikátor typu kdekoli je právní typu.
