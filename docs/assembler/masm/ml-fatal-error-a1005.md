# <a name="ml-fatal-error-a1005"></a>Závažná chyba nástroje ML A1005

**limit assembleru: úplná tabulka názvů parametr – makro**

Příliš mnoho parametrů, místní hodnoty nebo popisky – makro byly definovány pro makra. V tabulce Název makra došlo není další místo.

Definování kratší nebo méně názvy, nebo odeberte nepotřebné makra.
