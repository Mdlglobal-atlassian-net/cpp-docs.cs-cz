# <a name="ml-fatal-error-a1005"></a>Závažná chyba nástroje ML A1005

**limit assembleru: tabulka názvů parametrů makra je plná.**

Pro makro bylo definováno příliš mnoho parametrů, místních hodnot nebo popisků makra. V tabulce názvů maker není více místa.

Definujte kratší nebo míň názvů nebo odeberte nepotřebná makra.
