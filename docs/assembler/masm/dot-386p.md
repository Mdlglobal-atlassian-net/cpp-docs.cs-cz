# <a name="386p"></a>.386P

Umožňuje sestavení všechny pokyny (včetně privileged) 80386 procesoru; Zakáže sestavení instrukcí zavedena v novější procesorů systémech.

## <a name="syntax"></a>Syntaxe

> .386P

## <a name="remarks"></a>Poznámky

Taky umožňuje 80387 pokyny.
