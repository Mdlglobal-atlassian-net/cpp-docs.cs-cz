# <a name="386p"></a>.386P
Umožňuje sestavení všechny pokyny (včetně privileged) pro 80386 procesoru; Zakáže sestavení pokyny, které se poprvé objevila v novější procesory.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
.386P  
  
```  
  
## <a name="remarks"></a>Poznámky  
 Taky umožňuje 80387 pokyny.
