# <a name="ml-nonfatal-error-a2019"></a>Méně závažná chyba nástroje ML A2019

**operandem musí být typ záznamu nebo pole.**

Operand následující po operátoru [Width](operator-width.md) nebo [maskování](operator-mask.md) není platný.

Operátor [Width](operator-width.md) převezme operand, který je názvem pole nebo záznamu. Operátor [maskování](operator-mask.md) převezme operand, který je názvem pole nebo typu záznamu.
