# <a name="continue"></a>.CONTINUE
Generuje kód na začátku [. Při](../../assembler/masm/dot-while.md) nebo [. OPAKUJTE](../../assembler/masm/dot-repeat.md) blokovat Pokud `condition` hodnotu true.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
.CONTINUE [[.IF condition]]  
```
