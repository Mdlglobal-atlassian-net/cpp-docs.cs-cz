# <a name="operator-eq"></a>operátor EQ
Vrátí hodnotu true (-1) Pokud *expression1* rovná *Výraz2*, nebo vrátí hodnotu false (0), pokud neexistuje.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 EQ expression2  
```
