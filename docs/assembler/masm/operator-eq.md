# <a name="operator-eq"></a>operátor EQ

Vrátí hodnotu true (-1) Pokud *expression1* rovná *expression2*, nebo pokud tomu tak není, vrátí hodnotu false (0).

## <a name="syntax"></a>Syntaxe

> Expression1 EQ expression2
