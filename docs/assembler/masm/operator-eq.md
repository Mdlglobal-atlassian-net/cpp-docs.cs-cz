# <a name="operator-eq"></a>operátor EQ

Vrátí hodnotu true (-1), je-li *Výraz1* Equals *Výraz2*, nebo vrátí hodnotu false (0), pokud ne.

## <a name="syntax"></a>Syntaxe

> *Výraz1* **EQ** *Výraz2*
