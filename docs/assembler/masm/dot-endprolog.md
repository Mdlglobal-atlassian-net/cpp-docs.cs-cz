# <a name="endprolog"></a>.ENDPROLOG
Označuje konec prologu deklarace.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
.ENDPROLOG  
```  
  
## <a name="remarks"></a>Poznámky  
 Jedná se o chybu k používání některé z deklarace prologu mimo oblast mezi [PROC](../../assembler/masm/proc.md) RÁMEČKEM a. ENDPROLOG.  
  
 Další informace najdete v tématu [MASM pro x64 (ml64.exe)](../../assembler/masm/masm-for-x64-ml64-exe.md).
