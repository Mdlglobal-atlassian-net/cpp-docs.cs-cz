Deklaruje typ struktury, která má zadané *deklarace Field*. Každé pole musí být platná definice dat. Stejné jako [STRUC](struc.md).

> *název* **struktury** ⟦*Zarovnání*⟧ ⟦__,__ **nejedinečné**⟧ \
> *deklarace pole*\
> *name* **konec** názvu

## <a name="remarks"></a>Poznámky

Argument *Name* musí být stejný v příkazu pro otevření a zavření.
