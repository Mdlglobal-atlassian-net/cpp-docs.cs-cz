# <a name="struct-masm"></a>STRUCT (MASM)
Deklaruje typu Struktura s zadaný *fielddeclarations*. Každé pole musí obsahovat definici platná data. Stejné jako [struc –](../../assembler/masm/struc.md).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
