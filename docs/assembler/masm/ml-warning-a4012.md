# <a name="ml-warning-a4012"></a>Upozornění nástroje ML A4012

**informace o číslech řádků pro segment bez třídy ' CODE '**

V segmentu, který neměl název třídy, končí znakem "CODE", byly pokyny. Assembler negeneroval informace CodeView pro tyto pokyny.

CodeView nemůže zpracovat moduly s kódem ve segmentech s názvy tříd, které nekončí "CODE".
