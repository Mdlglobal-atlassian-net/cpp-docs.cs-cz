# <a name="operator-"></a>|| – operátor

Logický operátor OR. Používáno pouze v rámci [. Pokud](dot-if.md), [. WHILe](dot-while.md)nebo [. OPAKUJe](dot-repeat.md) bloky a vyhodnocují v době běhu, nikoli v době sestavení.

## <a name="syntax"></a>Syntaxe

> *výraz1* **||** *Výraz2*
