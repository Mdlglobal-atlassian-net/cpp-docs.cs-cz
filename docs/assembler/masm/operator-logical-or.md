# <a name="operator-"></a>|| – operátor

Logický operátor OR. Používáno pouze v rámci [. Pokud](../../assembler/masm/dot-if.md), [. WHILe](../../assembler/masm/dot-while.md)nebo [. OPAKUJe](../../assembler/masm/dot-repeat.md) bloky a vyhodnocují v době běhu, nikoli v době sestavení.

## <a name="syntax"></a>Syntaxe

> *výraz1* **||** *Výraz2*
