# <a name="operator-"></a>|| – operátor

Logické `OR`. Použít pouze v rámci [. Pokud](../../assembler/masm/dot-if.md), [. ZATÍMCO](../../assembler/masm/dot-while.md), nebo [. OPAKUJTE](../../assembler/masm/dot-repeat.md) a ostatní porty blokuje vyhodnocen v době běhu, ne v okamžiku sestavení.

## <a name="syntax"></a>Syntaxe
