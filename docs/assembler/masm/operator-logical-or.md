# <a name="operator-"></a>|| – operátor
Logické `OR`. Použít jenom v [. Pokud](../../assembler/masm/dot-if.md), [. Při](../../assembler/masm/dot-while.md), nebo [. OPAKUJTE](../../assembler/masm/dot-repeat.md) blokuje a vyhodnocovány v době běhu, nikoli v době sestavení.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 || expression2  
```
