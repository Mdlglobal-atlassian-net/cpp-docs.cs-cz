# <a name="for-masm"></a>FOR (MASM)
Označí blok, který se bude opakovat jednou pro každou `argument`, s aktuálním `argument` nahrazení `parameter` pro každý opakování.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
   FOR   
   parameter [[:REQ | :=default]] , <argument [[, argument]]...>  
statements  
ENDM  
```
