# <a name="for-masm"></a>FOR (MASM)

Označí blok, který se bude opakovat jednou pro každý *argument*, s aktuálním *argument* nahrazení *parametr* pro každého opakování.

## <a name="syntax"></a>Syntaxe

> PRO *parametr* [[: no |: =*výchozí*]], \< *argument* [[, *argument*]]... ><br/>
> *Příkazy*<br/>
> ENDM

## <a name="remarks"></a>Poznámky

Stejné jako [IRP](../../assembler/masm/irp.md).
