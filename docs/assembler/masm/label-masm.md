# <a name="label-masm"></a>Popisek (MASM)

Vytvoří nový popisek tak, že přiřadíte hodnotu aktuální umístění čítače a daný *typ* k *název*.

## <a name="syntax"></a>Syntaxe

> *název* popisek *typu*
>
> *název* popisek \[NEAR | POKUD | PROC] PTR \[ *typ*]
