# <a name="label-masm"></a>Popisek (MASM)
Vytvoří nový štítek přiřazením aktuální hodnotu čítače umístění a daný `type` k *název*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
