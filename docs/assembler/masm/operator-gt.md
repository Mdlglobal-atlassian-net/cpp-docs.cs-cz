# <a name="operator-gt"></a>operátor GT
Vrátí hodnotu true (-1) Pokud *expression1* je větší než *Výraz2*, nebo pokud není, vrátí hodnotu false (0).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 GT expression2  
```
