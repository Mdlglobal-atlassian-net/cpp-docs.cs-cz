# <a name="operator-gt"></a>operátor GT

Vrátí hodnotu true (-1), je-li *Výraz1* větší než *Výraz2*, nebo vrátí hodnotu false (0), pokud není.

## <a name="syntax"></a>Syntaxe

> *Výraz1* **gt** *Výraz2*
