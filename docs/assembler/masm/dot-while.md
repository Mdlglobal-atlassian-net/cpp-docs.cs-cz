# <a name="while"></a>.WHILE
Generuje kód, který spouští blok `statements` při `condition` zůstává hodnotu true.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
   .WHILE condition  
statements  
.ENDW  
```
