# <a name="ml-nonfatal-error-a2060"></a>Méně závažná chyba nástroje ML A2060

**vynucená Chyba: řetězce se neshodují**

Direktiva podmíněné chyby [. ERRDIF](../../assembler/masm/dot-errdif.md) nebo **. ERRDIFI** se použil k vygenerování této chyby.
