# <a name="dd"></a>DD

Přidělí a volitelně inicializuje dvojité slovo (4 bajty) úložiště pro každý *inicializátor*. **DD** je synonymum [typu DWORD](dword.md).

## <a name="syntax"></a>Syntaxe

> ⟦*Name*⟧ **DD** *inicializátor* ⟦ __,__ *inicializátor* ... ⟧

## <a name="remarks"></a>Poznámky

Lze také použít jako specifikátor typu kdekoli je typ právní.
