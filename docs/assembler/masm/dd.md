# <a name="dd"></a>DD
Přiděluje a volitelně inicializuje dvojitou slova (4 bajtů) úložiště pro každou `initializer`. `DD` je synonymum z [DWORD](../../assembler/masm/dword.md).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
[[name]] DD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>Poznámky  
 Můžete také použít jako specifikátor typu kdekoli je právní typu.
