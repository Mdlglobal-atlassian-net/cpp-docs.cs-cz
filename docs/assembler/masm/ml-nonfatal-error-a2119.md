# <a name="ml-nonfatal-error-a2119"></a>Méně závažná chyba nástroje ML A2119

**je nutné zadat typ jazyka.**

Definici procedury nebo prototypu nebyl udělen typ jazyka.

Typ jazyka musí být deklarován v každé definici procedury nebo prototypu, pokud není zadán výchozí typ jazyka. Výchozí typ jazyka je nastaven pomocí [. ](../../assembler/masm/dot-model.md)Direktiva modelu **, parametr lang**nebo parametry příkazového řádku ml **/GC** nebo **/GD**.
