# <a name="sbyte-masm"></a>SBYTE (MASM)
Přiděluje a volitelně inicializuje podepsaný bajt úložiště pro každou `initializer`. Můžete také použít jako specifikátor typu kdekoli je právní typu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name SBYTEinitializer [[, initializer]]...  
```
