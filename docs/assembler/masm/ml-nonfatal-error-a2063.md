# <a name="ml-nonfatal-error-a2063"></a>Méně závažná chyba nástroje ML A2063

**lze zarovnat pouze k mocnině 2: výraz**

Výraz zadaný pomocí direktivy [align](../../assembler/masm/align-masm.md) byl neplatný.

Výraz **align** musí být mocninou 2 mezi 2 a 256 a musí být menší nebo roven zarovnání aktuálního segmentu, struktury nebo sjednocení.
