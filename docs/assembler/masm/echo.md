# <a name="echo"></a>ECHO

Zobrazí *zpráva* do standardního výstupního zařízení (ve výchozím nastavení na obrazovce).

## <a name="syntax"></a>Syntaxe

> ECHO *zprávy*

## <a name="remarks"></a>Poznámky

Stejné jako [% OUT](../../assembler/masm/percent-out.md).
