# <a name="operator-this"></a>THIS – operátor
Vrací hodnotu zadaného operand z `type` jejichž posun a segment hodnoty jsou stejná jako aktuální hodnota čítače umístění.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
THIS type  
```
