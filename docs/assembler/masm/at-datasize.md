# <a name="datasize"></a>@DataSize
0 pro **velmi malé**, **malé**, **střední**, a **ploché** modelů, 1 pro **COMPACT** a  **VELKÉ** modely a 2 pro **HUGE** modelu (číselný označení rovnosti).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
@DataSize  
  
```
