# <a name="sqword"></a>SQWORD
Přiděluje a volitelně inicializuje 8 podepsaný bajtů úložiště pro každou `initializer`. Můžete také použít jako specifikátor typu kdekoli je právní typu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name SQWORD initializer [[, initializer]]...  
```
