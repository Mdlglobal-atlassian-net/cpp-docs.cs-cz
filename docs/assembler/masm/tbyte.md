# <a name="tbyte"></a>TBYTE

Přiděluje a volitelně inicializuje 10 bajtů úložiště pro každou `initializer`. Slouží také jako specifikátor typu kdekoli typ je platný.

## <a name="syntax"></a>Syntaxe
