# <a name="tbyte"></a>TBYTE
Přiděluje a volitelně inicializuje 10 bajtů úložiště pro každou `initializer`. Můžete také použít jako specifikátor typu kdekoli je právní typu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
[[name]] TBYTEinitializer [[, initializer]]...  
```
