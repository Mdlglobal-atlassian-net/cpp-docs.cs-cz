# <a name="real8"></a>REAL8
Přiděluje a volitelně inicializuje číslo s plovoucí desetinnou čárkou dvojitou přesností (8 bajtů) pro každou *inicializátoru*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name REAL8 initializer [[, initializer]]...  
```
