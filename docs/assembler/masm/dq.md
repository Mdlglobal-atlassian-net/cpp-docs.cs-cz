# <a name="dq"></a>DQ

Přiděluje a volitelně inicializuje 8 bajtů úložiště pro každou `initializer`. Můžete také použít jako specifikátor typu kdekoli typ je platný. `DQ` je synonymum pro [QWORD](../../assembler/masm/qword.md).

## <a name="syntax"></a>Syntaxe

> [[název]] Inicializátor DQ [[, inicializátor]]...
