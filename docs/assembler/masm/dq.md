# <a name="dq"></a>DQ
Přiděluje a volitelně inicializuje 8 bajtů úložiště pro každou `initializer`. Můžete také použít jako specifikátor typu kdekoli je právní typu. `DQ` je synonymum z [QWORD](../../assembler/masm/qword.md).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
[[name]] DQ initializer [[, initializer]]...  
```
