# <a name="dword"></a>DWORD

Přidělí a volitelně inicializuje dvojité slovo (4 bajty) úložiště pro každý *inicializátor*. **Hodnota DWORD** je synonymem [DD](dd.md).

## <a name="syntax"></a>Syntaxe

> ⟦ *⟧* *inicializátoru* **DWORD** ⟦ __,__ *inicializátor* ... ⟧

## <a name="remarks"></a>Poznámky

Lze také použít jako specifikátor typu kdekoli je typ právní.
