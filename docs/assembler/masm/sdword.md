# <a name="sdword"></a>SDWORD
Přiděluje a volitelně inicializuje podepsaný slovo double (4 bajtů) úložiště pro každou `initializer`. Můžete také použít jako specifikátor typu kdekoli je právní typu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
