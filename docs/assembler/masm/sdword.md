# <a name="sdword"></a>SDWORD

Přiděluje a volitelně inicializuje podepsaný double word (4 bajtů) úložiště pro každou `initializer`. Můžete také použít jako specifikátor typu kdekoli typ je platný.

## <a name="syntax"></a>Syntaxe

> Název SDWORD inicializátor [[, inicializátor]]...
