# <a name="textequ"></a>TEXTEQU
Přiřadí *textitem* k *název*. *Textitem* může být řetězcový literál konstanta před sebou `%`, nebo řetězec vrácený makro funkce.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name TEXTEQU [[textitem]]  
```
