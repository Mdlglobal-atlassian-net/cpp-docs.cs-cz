# <a name="textequ"></a>TEXTEQU

Přiřadí *textitem* k *názvu*. *Textitem* může být řetězcový literál, konstanta předchází `%`, nebo řetězec vrácený funkcí makra.

## <a name="syntax"></a>Syntaxe

> *název* **TEXTEQU** ⟦*textitem*⟧
