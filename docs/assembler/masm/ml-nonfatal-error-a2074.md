# <a name="ml-nonfatal-error-a2074"></a>Méně závažná chyba nástroje ML A2074

**Nelze získat přístup k popisku prostřednictvím segmentových registrů**

Byl proveden pokus o přístup k popisku prostřednictvím registru segment, který se předpokládá, že jeho segmentu nebo skupiny.
