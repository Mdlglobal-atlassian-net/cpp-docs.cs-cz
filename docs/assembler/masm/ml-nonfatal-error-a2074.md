# <a name="ml-nonfatal-error-a2074"></a>Méně závažná chyba nástroje ML A2074
**Nelze získat přístup k popisku prostřednictvím segment registry**  
  
 Byl proveden pokus o přístup k štítek prostřednictvím registru segment, který nebyl předpokládá, že její segmentu nebo skupiny.
