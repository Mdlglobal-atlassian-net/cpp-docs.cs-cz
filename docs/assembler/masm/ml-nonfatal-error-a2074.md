# <a name="ml-nonfatal-error-a2074"></a>Méně závažná chyba nástroje ML A2074

**Nelze získat přístup k popisku prostřednictvím registrů segmentů.**

Došlo k pokusu o přístup k popisku prostřednictvím registru segmentu, který se nepředpokládalo na jeho segment nebo skupinu.
