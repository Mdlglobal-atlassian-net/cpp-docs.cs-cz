# <a name="errdef"></a>.ERRDEF
Vygeneruje chybu, pokud *název* je dříve definovaný štítek, proměnné nebo symbol.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
.ERRDEF name [[, message]]  
```
