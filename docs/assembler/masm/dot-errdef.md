# <a name="errdef"></a>.ERRDEF

Vygeneruje chybu, pokud je *název* dříve definovaný popisek, proměnná nebo symbol.

## <a name="syntax"></a>Syntaxe

> **. ERRDEF** *název* ⟦ __,__ *zpráva*⟧
