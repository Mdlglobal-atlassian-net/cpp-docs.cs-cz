# <a name="typedef"></a>TYPEDEF

Definuje nový typ nebo se **mu** říká *název*, který je ekvivalentní *typu* nebo *protoDefinition*.

## <a name="syntax"></a>Syntaxe

> *název* *typu* typedef | **proto** *protoDefinition*
