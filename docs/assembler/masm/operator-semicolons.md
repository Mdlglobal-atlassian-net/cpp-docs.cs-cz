# <a name="operator-"></a>;; – operátor

Zpracovává *text* jako komentář makro, které se zobrazí pouze v definici makra. Výpis nezobrazuje *text* kde rozbalení makra je.

## <a name="syntax"></a>Syntaxe

> ;; *text*
