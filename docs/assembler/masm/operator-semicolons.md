# <a name="operator-"></a>;; – operátor
Zpracovává *text* jako komentář ve makro, který se zobrazí jenom v definici makra. Seznam nezobrazuje *text* kde je rozbalený makro.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
;;  
text  
  
```
