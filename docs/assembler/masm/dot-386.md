# <a name="386"></a>.386
Umožňuje sestavení neoprávněným pokyny 80386 procesoru; Zakáže sestavení pokyny, které se poprvé objevila v novější procesory.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
.386  
  
```  
  
## <a name="remarks"></a>Poznámky  
 Taky umožňuje 80387 pokyny.
