# <a name="386-32-bit-masm"></a>.386 (32-bit MASM)

Umožňuje sestavení neprivilegovaných instrukcí pro procesor 80386. zakáže sestavení instrukcí představených s novějšími procesory. (jenom 32-bitová MASM.)

## <a name="syntax"></a>Syntaxe

> **.386**

## <a name="remarks"></a>Poznámky

Také povoluje 80387 instrukcí.
