# <a name="386"></a>.386

Umožňuje sestavení neoprávněným pokyny, jak 80386 procesoru; Zakáže sestavení instrukcí zavedena v novější procesorů systémech.

## <a name="syntax"></a>Syntaxe

> .386

## <a name="remarks"></a>Poznámky

Taky umožňuje 80387 pokyny.
