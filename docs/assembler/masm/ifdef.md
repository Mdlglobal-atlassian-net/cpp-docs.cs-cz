# <a name="ifdef"></a>IFDEF

Uděluje sestavení, když *název* je dříve definovaný popisek, proměnné nebo symbol.

## <a name="syntax"></a>Syntaxe

> IFDEF název

## <a name="remarks"></a>Poznámky

Zobrazit [IF](../../assembler/masm/if-masm.md) pro úplnou syntaxi.
