# <a name="ml-fatal-error-a1000"></a>Závažná chyba nástroje ML A1000

**soubor nelze otevřít: název souboru**

Assembler nemohl otevřít zdrojový, zahrnutý nebo výstupní soubor.

Příčinou může být jedna z následujících příčin:

- Soubor neexistuje.

- Soubor je používán jiným procesem.

- Název souboru není platný.

- Soubor, který je jen pro čtení, má název výstupního souboru již existuje.

- Aktuální jednotka je plná.

- Aktuální adresář je kořen a je plný.

- Do zařízení se nedá zapisovat.

- Jednotka není připravena.
