# <a name="exitm"></a>EXITM

Rozbalení aktuálního bloku opakovat nebo makro se ukončí a zahájí sestavení dalšímu příkazu mimo blok.

## <a name="syntax"></a>Syntaxe

> EXITM [[textitem]]

## <a name="remarks"></a>Poznámky

Ve funkci – makro *textitem* je vrácená hodnota.
