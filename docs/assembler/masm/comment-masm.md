# <a name="comment-masm"></a>COMMENT (MASM)

Zpracuje všechny *text* mezi nebo na stejném řádku jako oddělovače jako komentář.

## <a name="syntax"></a>Syntaxe

> Oddělovač KOMENTÁŘE [[text]]<br/>
> [[text]]<br/>
> Oddělovač [[textu]] [[text]]
