# <a name="comment"></a>COMMENT

Zpracovává veškerý *text* mezi nebo na stejném řádku jako oddělovače jako komentář.

## <a name="syntax"></a>Syntaxe

> *Oddělovač* komentářů ⟦*text*⟧ \
> ⟦*text*⟧ \
> ⟦*text*⟧ *oddělovač* ⟦*text*⟧
