# <a name="nolistif"></a>.NOLISTIF

## <a name="syntax"></a>Syntaxe

> **.NOLISTIF**

## <a name="remarks"></a>Poznámky

Toto nastavení je výchozí. Stejné jako [. SFCOND](../../assembler/masm/dot-sfcond.md).
