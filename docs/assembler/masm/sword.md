# <a name="sword"></a>SWORD
Přiděluje a volitelně inicializuje podepsaný slova (2 bajtů) úložiště pro každou `initializer`. Můžete také použít jako specifikátor typu kdekoli je právní typu.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name SWORD initializer [[, initializer]]...  
```
