# <a name="sword"></a>SWORD

Přiděluje a volitelně inicializuje podepsaný word (2 bajtů) úložiště pro každou `initializer`. Slouží také jako specifikátor typu kdekoli typ je platný.

## <a name="syntax"></a>Syntaxe

> Název SWORD inicializátor [[, inicializátor]]...
