# <a name="catstr"></a>CATSTR
Zřetězí textu položky.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## <a name="remarks"></a>Poznámky  
 Každá položka text může být řetězcový literál konstanta před sebou `%`, nebo řetězec vrácený makro funkce.
