# <a name="catstr"></a>CATSTR

Zřetězí textové položky.

## <a name="syntax"></a>Syntaxe

> Pojmenujte CATSTR [[textitem1 [[, textitem2]]...]]

## <a name="remarks"></a>Poznámky

Každá položka text může být řetězcový literál, konstanta předcházet párový příkaz `%`, nebo řetězec vrácený funkcí makra.
