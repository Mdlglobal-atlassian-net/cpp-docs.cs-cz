# <a name="instr"></a>@InStr
Makro funkce, která vyhledá první výskyt *řetězec2* v *řetězec1*, počáteční v *pozice* v rámci *řetězec1*. Pokud *pozice* nezobrazí, hledání začíná na začátku *řetězec1*. Vrátí celé číslo pozice nebo 0, pokud *řetězec2* nebyl nalezen.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
@InStr( [[position]], string1, string2 )  
```
