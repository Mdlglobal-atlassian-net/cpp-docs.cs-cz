# <a name="instr"></a>@InStr

Funkce makra, která vyhledá první výskyt *řetězec2* v *řetězec1*, začínající na *pozice* v rámci *řetězec1*. Pokud *pozice* nezobrazí, hledání začne na začátku *řetězec1*. Vrátí celé číslo pozice nebo 0, pokud *řetězec2* nebyl nalezen.

## <a name="syntax"></a>Syntaxe

> @InStr([[funkce]], řetězec1 řetězec2)
