# <a name="byte-masm"></a>BYTE (MASM)
Přiděluje a volitelně inicializuje bajt úložiště pro každou `initializer`.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
[[name]] BYTEinitializer [[, initializer]] ...  
```  
  
## <a name="remarks"></a>Poznámky  
 Můžete také použít jako specifikátor typu kdekoli je právní typu.
