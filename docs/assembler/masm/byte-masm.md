# <a name="byte-masm"></a>BYTE (MASM)

Přiděluje a volitelně inicializuje bajtů úložiště pro každou `initializer`.

## <a name="syntax"></a>Syntaxe

> [[název]] BYTEinitializer [[, inicializátor]]...

## <a name="remarks"></a>Poznámky

Slouží také jako specifikátor typu kdekoli typ je platný.
