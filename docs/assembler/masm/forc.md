# <a name="forc"></a>FORC
Označí blok, který se bude opakovat jednou pro každý znak v `string`, s aktuální znak nahrazení `parameter` pro každý opakování.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```  
  
## <a name="remarks"></a>Poznámky  
 Stejné jako [irpc –](../../assembler/masm/irpc.md).
