# <a name="forc"></a>FORC

Označí blok, který se opakuje jednou pro každý znak v *řetězec*, s aktuální znak nahrazení *parametr* pro každého opakování.

## <a name="syntax"></a>Syntaxe

> FORC *parametr*, \< *řetězec*><br/>
> *Příkazy*<br/>
> ENDM

## <a name="remarks"></a>Poznámky

Stejné jako [IRPC](../../assembler/masm/irpc.md).
