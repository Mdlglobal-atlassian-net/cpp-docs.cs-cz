# <a name="operator-highword"></a>operátor HIGHWORD
Vrátí vysoké 16 bity nízkou 32 bity *výraz*. Výrazy MASM jsou 64bitové hodnoty.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
HIGHWORD expression  
```
