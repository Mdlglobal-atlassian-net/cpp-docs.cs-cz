# <a name="operator-highword"></a>operátor HIGHWORD

Vrátí maximum 16 bitů s nízkou 32 verzemi *výrazu*. Výrazy MASM jsou 64 bitové hodnoty.

## <a name="syntax"></a>Syntaxe

> *Výraz* HIGHWORD
