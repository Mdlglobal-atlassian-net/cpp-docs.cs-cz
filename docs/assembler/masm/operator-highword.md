# <a name="operator-highword"></a>operátor HIGHWORD

Vrátí vysokou 16 bitů s nízkou 32 bitů *výraz*. Výrazy MASM jsou hodnoty 64-bit.

## <a name="syntax"></a>Syntaxe

> Výraz HIGHWORD
