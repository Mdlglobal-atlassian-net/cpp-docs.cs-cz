# <a name="operator-ge"></a>operátor GE

Vrátí hodnotu true (-1) Pokud *expression1* je větší než nebo rovna hodnotě *expression2*, nebo pokud není, vrátí hodnotu false (0).

## <a name="syntax"></a>Syntaxe

> Expression1 GE expression2
