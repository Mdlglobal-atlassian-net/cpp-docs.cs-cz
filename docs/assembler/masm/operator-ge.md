# <a name="operator-ge"></a>operátor GE

Vrátí hodnotu true (-1), pokud je *Výraz1* větší než nebo rovno hodnotě *Výraz2*, nebo vrátí hodnotu false (0), pokud není.

## <a name="syntax"></a>Syntaxe

> *Výraz1* **GE** *Výraz2*
