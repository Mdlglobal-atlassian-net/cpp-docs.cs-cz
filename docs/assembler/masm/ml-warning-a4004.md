# <a name="ml-warning-a4004"></a>Upozornění nástroje ML A4004
**nelze předpokládat CS**  
  
 Byl proveden pokus o předpokládat, že hodnotu registru CS. CS je vždycky nastavený na aktuální segment nebo skupinu.
