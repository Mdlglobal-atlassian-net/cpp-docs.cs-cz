# <a name="ml-warning-a4004"></a>Upozornění nástroje ML A4004

**CS nejde předpokládat.**

Došlo k pokusu o zavolání hodnoty pro registraci CS. CS se vždycky nastavuje na aktuální segment nebo skupinu.
