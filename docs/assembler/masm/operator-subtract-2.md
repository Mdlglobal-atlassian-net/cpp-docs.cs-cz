# <a name="operator--"></a>\- – operátor

Vrátí *Výraz1* minus *Výraz2*. Druhý operátor vrátí znaménko *výrazu*.

## <a name="syntax"></a>Syntaxe

> *výraz1* **-** *Výraz2*
>
> *výraz*-
