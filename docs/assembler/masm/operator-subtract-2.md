# <a name="operator--"></a>Operator-

Vrátí *expression1* minus *expression2*. Druhý operátor obrátí znaménko *výraz*.

## <a name="syntax"></a>Syntaxe

> *Expression1* - *expression2*
>
> -*Výraz*
