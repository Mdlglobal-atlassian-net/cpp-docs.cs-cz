# <a name="word"></a>WORD

Přiděluje a volitelně inicializuje slova (2 bajtů) úložiště pro každou `initializer`. Slouží také jako specifikátor typu kdekoli typ je platný.

## <a name="syntax"></a>Syntaxe

> [[*název*]] slovo *inicializátor* [[, *inicializátor*]]...
