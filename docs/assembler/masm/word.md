# <a name="word"></a>WORD

Přidělí a případně inicializuje slovo (2 bajty) úložiště pro každý *inicializátor*. Lze také použít jako specifikátor typu kdekoli je typ právní. [DW](dw.md) je synonymem **slova**.

## <a name="syntax"></a>Syntaxe

> ⟦*název*⟧ *inicializátoru* slov ⟦ __,__ *inicializátor* ... ⟧
