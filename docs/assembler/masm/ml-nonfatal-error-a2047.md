# <a name="ml-nonfatal-error-a2047"></a>Méně závažná chyba nástroje ML A2047

**prázdný řetězec (null)**

Řetězec, který se skládá z dvojice uvozovek a neobsahuje žádné znaky v rámci.

Aby byl řetězec platný, musí obsahovat 1-255 znaků.
