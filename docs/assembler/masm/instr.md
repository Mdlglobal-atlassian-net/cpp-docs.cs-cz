# <a name="instr"></a>INSTR
Najde první výskyt *textitem2* v *textitem1*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name  
 INSTR [[position,]] textitem1, textitem2  
```  
  
## <a name="remarks"></a>Poznámky  
 Počáteční *pozice* je volitelný. Každá položka text může být řetězcový literál konstanta před sebou `%`, nebo řetězec vrácený makro funkce.
