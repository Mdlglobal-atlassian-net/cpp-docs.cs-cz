# <a name="instr"></a>INSTR

Vyhledá první výskyt *textitem2* v *textitem1*.

## <a name="syntax"></a>Syntaxe

> *name* INSTR [[*position*,]] *textitem1*, *textitem2*

## <a name="remarks"></a>Poznámky

Počáteční *pozice* je volitelný. Každá položka text může být řetězcový literál, konstanta předcházet párový příkaz `%`, nebo řetězec vrácený funkcí makra.
