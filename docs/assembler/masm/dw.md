# <a name="dw"></a>DW

Přiděluje a volitelně inicializuje slova (2 bajtů) úložiště pro každou `initializer`. Slouží také jako specifikátor typu kdekoli typ je platný. `DW` je synonymum pro [slovo](../../assembler/masm/word.md).

## <a name="syntax"></a>Syntaxe

> [[název]] Inicializátor datového skladu [[, inicializátor]]...
