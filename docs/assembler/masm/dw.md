# <a name="dw"></a>DW

Přidělí a případně inicializuje slovo (2 bajty) úložiště pro každý *inicializátor*. Lze také použít jako specifikátor typu kdekoli je typ právní. **DW** je synonymem [slova](word.md).

## <a name="syntax"></a>Syntaxe

> ⟦*Name*⟧ **–** *inicializátor* ⟦ __,__ *inicializátor* ... ⟧
