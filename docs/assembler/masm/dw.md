# <a name="dw"></a>DW
Přiděluje a volitelně inicializuje slova (2 bajtů) úložiště pro každou `initializer`. Můžete také použít jako specifikátor typu kdekoli je právní typu. `DW` je synonymum z [WORD](../../assembler/masm/word.md).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
