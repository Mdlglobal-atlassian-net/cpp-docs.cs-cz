# <a name="dt"></a>DT

Přidělí a případně inicializuje 10 bajtů úložiště pro každý *inicializátor*. Lze také použít jako specifikátor typu kdekoli je typ právní. **DT** je synonymum pro [TBYTE](tbyte.md).

## <a name="syntax"></a>Syntaxe

> ⟦*Name*⟧ **DT** *inicializátor* ⟦ __,__ *inicializátor* ... ⟧
