# <a name=""></a>\@\@:

Definuje popisek kód rozpoznatelných pouze mezi *label1* a *label2*, kde *label1* se buď počáteční kód nebo předchozí `@@:` popisek a *label2* je buď konce kódu nebo další `@@:` popisek. Zobrazit [ \@B](../../assembler/masm/at-b.md) a [ \@F](../../assembler/masm/at-f.md).

## <a name="syntax"></a>Syntaxe

> **\@\@:**
