# <a name="while"></a>WHILE

Opakuje sestavení *příkazů* bloku, pokud *constantExpression* zůstává true.

## <a name="syntax"></a>Syntaxe

> *ConstantExpression*\
> \ *příkazů*
> **ENDM**
