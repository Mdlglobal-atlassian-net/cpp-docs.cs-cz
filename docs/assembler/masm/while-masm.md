# <a name="while-masm"></a>WHILE (MASM)
Opakuje sestavení bloku `statements` tak dlouho, dokud *výraz* zůstává hodnotu true.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
   WHILE expression  
statements  
ENDM  
```
