# <a name="operator-high32"></a>HIGH32 – operátor

Vrátí vysoké 32y bitů *výrazu*. Výrazy MASM jsou 64 bitové hodnoty.

## <a name="syntax"></a>Syntaxe

> *Výraz* high32 –
