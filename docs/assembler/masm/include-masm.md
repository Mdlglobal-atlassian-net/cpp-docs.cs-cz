# <a name="include-masm"></a>INCLUDE (MASM)
Vložení zdrojového kódu ze zdrojového souboru poskytují *filename* do aktuální zdrojový soubor během sestavení.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
INCLUDE filename  
```  
  
## <a name="remarks"></a>Poznámky  
 *Filename* musí být uzavřené v lomených závorkách, pokud obsahuje zpětné lomítko, středník, větší – než symbol, méně – než symbol, jednoduché uvozovky nebo uvozovky.
