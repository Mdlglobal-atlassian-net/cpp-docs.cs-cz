# <a name="const-32-bit-masm"></a>. CONSt (32-bit MASM)

Při použití s nástrojem [. MODEL](dot-model.md)spustí konstantní datový segment (s názvem segmentu **const**).

## <a name="syntax"></a>Syntaxe

> **. MX**\
> ⟦ *segmentItem* ⟧...

## <a name="remarks"></a>Poznámky

Tento segment má atribut jen pro čtení.
