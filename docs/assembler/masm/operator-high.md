# <a name="operator-high"></a>operátor HIGH

Vrátí horních 8 bitů s nízkým počtem bitů *výrazu*. Výrazy MASM jsou 64 bitové hodnoty.

## <a name="syntax"></a>Syntaxe

> *Výraz* High
