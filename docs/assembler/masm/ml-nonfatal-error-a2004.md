# <a name="ml-nonfatal-error-a2004"></a>Méně závažná chyba nástroje ML A2004

**konflikt typu symbolu: identifikátor**

Direktiva [EXTERNDEF](externdef.md) nebo [Label](label-masm.md) byla použita pro proměnnou, symbol, datovou strukturu nebo popisek, který byl definován ve stejném modulu, ale s jiným typem.
