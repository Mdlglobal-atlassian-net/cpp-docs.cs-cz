# <a name="operator-"></a>: – operátor

Přepíše výchozí segment *výraz* s *segmentu*. *Segmentu* může být registrace segmentu, název skupiny, název segmentu nebo výraz segmentu. *Výraz* musí být konstanta.

## <a name="syntax"></a>Syntaxe

> *segment* : *výraz*
