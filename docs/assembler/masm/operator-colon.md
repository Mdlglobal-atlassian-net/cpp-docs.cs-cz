# <a name="operator-"></a>: – operátor

Přepíše výchozí segment *výrazu* *segmentem*. *Segmentem* může být registr segmentu, název skupiny, název segmentu nebo výraz segmentu. *Výraz* musí být konstanta.

## <a name="syntax"></a>Syntaxe

> *segment* **:** *výraz*
