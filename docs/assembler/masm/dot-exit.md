# <a name="exit-32-bit-masm"></a>. EXIT (32-bit MASM)

Generuje ukončovací kód. (jenom 32-bitová MASM.)

## <a name="syntax"></a>Syntaxe

> **. UKONČIT** ⟦*výrazu*⟧

## <a name="remarks"></a>Poznámky

Vrátí volitelný *výraz* do prostředí shell.
