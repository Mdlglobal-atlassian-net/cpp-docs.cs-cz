# <a name="includelib-masm"></a>INCLUDELIB (MASM)
Informuje o tom, který aktuální modul musí být propojena s linkeru *NázevKnihovny*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
INCLUDELIB libraryname  
```  
  
## <a name="remarks"></a>Poznámky  
 *NázevKnihovny* musí být uzavřené v lomených závorkách, pokud obsahuje zpětné lomítko, středník, větší – než symbol, méně – než symbol, jednoduché uvozovky nebo uvozovky.
