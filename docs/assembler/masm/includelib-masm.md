# <a name="includelib-masm"></a>INCLUDELIB (MASM)

Informuje o tom, který aktuální modul by měly být propojeny s linker *NázevKnihovny*.

## <a name="syntax"></a>Syntaxe

> Includelib – NázevKnihovny

## <a name="remarks"></a>Poznámky

*NázevKnihovny* musí být uzavřen do lomených závorek obsahuje zpětné lomítko, středník, větší-než symbol, méně – než symbol, jednoduché uvozovky nebo znak uvozovek.
