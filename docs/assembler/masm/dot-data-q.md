# <a name="data"></a>.DATA?

(jenom 32-bitová MASM.) Při použití s nástrojem [. MODEL](dot-model.md), spustí poblíž datového segmentu pro neinicializovaná data (název segmentu _BSS).

## <a name="syntax"></a>Syntaxe

> **. DATA?** \
> ⟦ *segmentItem* ⟧...
