# <a name="ymmword"></a>YMMWORD
Použít pro operandy multimédií 256 bitů s pokyny Intel Advanced Vector Extensions (AVX).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
YMMWORD  
```  
  
## <a name="remarks"></a>Poznámky  
 `YMMWORD`reprezentuje stejného typu jako `__m256` pro vnitřní AVX funkce.
