# <a name="real10"></a>REAL10

Přidělí a volitelně inicializuje číslo s plovoucí desetinnou čárkou pro každý *inicializátor*.

## <a name="syntax"></a>Syntaxe

> *název* **REAL10** *inicializátoru* ⟦ __,__ *inicializátor* ... ⟧
