# <a name="real10"></a>REAL10

Přiděluje a volitelně inicializuje číslo s plovoucí desetinnou čárkou 10 bajtů pro každou *inicializátor*.

## <a name="syntax"></a>Syntaxe

> Název REAL10 inicializátor [[, inicializátor]]...
