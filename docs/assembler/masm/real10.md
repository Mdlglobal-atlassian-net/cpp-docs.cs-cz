# <a name="real10"></a>REAL10
Přiděluje a volitelně inicializuje číslo s plovoucí desetinnou čárkou 10 bajtů pro každý *inicializátoru*.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
name REAL10 initializer [[, initializer]]...  
```
