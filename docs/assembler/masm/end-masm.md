# <a name="end"></a>END

Označí konec modulu a volitelně nastaví vstupní bod programu na *procId*.

## <a name="syntax"></a>Syntaxe

> **Konec** ⟦*procId*⟧

## <a name="remarks"></a>Poznámky

Argument *procId* je platný pouze v 32 bitové MASM.
