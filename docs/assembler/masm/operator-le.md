# <a name="operator-le"></a>operátor LE
Vrátí hodnotu true (-1) Pokud *expression1* je menší než nebo rovno *Výraz2*, nebo pokud není, vrátí hodnotu false (0).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 LE expression2  
```
