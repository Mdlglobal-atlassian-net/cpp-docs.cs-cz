# <a name="operator-le"></a>operátor LE

Vrátí hodnotu true (-1), pokud *Výraz1* je menší nebo rovna hodnotě *Výraz2*, nebo vrátí hodnotu false (0), pokud není.

## <a name="syntax"></a>Syntaxe

> *Výraz1* **Le** *Výraz2*
