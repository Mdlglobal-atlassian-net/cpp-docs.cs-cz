# <a name="operator-lt"></a>operátor LT
Vrátí hodnotu true (-1) Pokud *expression1* je menší než *Výraz2*, nebo pokud není, vrátí hodnotu false (0).  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
expression1 LT expression2  
```
