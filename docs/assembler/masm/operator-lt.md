# <a name="operator-lt"></a>operátor LT

Vrátí hodnotu true (-1) Pokud *expression1* je menší než *expression2*, nebo pokud není, vrátí hodnotu false (0).

## <a name="syntax"></a>Syntaxe

> Expression1 LT expression2
