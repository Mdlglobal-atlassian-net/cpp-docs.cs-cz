# <a name="operator-lt"></a>operátor LT

Vrátí hodnotu true (-1), je-li *Výraz1* menší než *Výraz2*, nebo vrátí hodnotu false (0), pokud není.

## <a name="syntax"></a>Syntaxe

> *Výraz1* **lt** *Výraz2*
