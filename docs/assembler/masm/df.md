# <a name="df"></a>DF

Přidělí a případně inicializuje 6 bajtů úložiště pro každý *inicializátor*. **DF** je synonymum pro [FWORD](fword.md).

## <a name="syntax"></a>Syntaxe

> ⟦*Name*⟧ **DF** *inicializátor* ⟦ __,__ *inicializátor* ... ⟧

## <a name="remarks"></a>Poznámky

Lze také použít jako specifikátor typu kdekoli je typ právní.
