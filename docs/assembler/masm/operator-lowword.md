# <a name="operator-lowword"></a>operátor LOWWORD

Vrátí dolní počet 16 bitů *výrazu*. Výrazy MASM jsou 64 bitové hodnoty.

## <a name="syntax"></a>Syntaxe

> *Výraz* LOWWORD
