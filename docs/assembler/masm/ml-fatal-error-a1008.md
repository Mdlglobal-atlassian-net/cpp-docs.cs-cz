# <a name="ml-fatal-error-a1008"></a>Závažná chyba nástroje ML A1008
**vnoření neodpovídající – makro**  
  
 Buď makro nebyl ukončen před koncem souboru nebo ukončující – direktiva [endm –](../../assembler/masm/endm.md) byl nalezen mimo blok makro.  
  
 Jednou z příčin této chyby je vynechání tečky před [. OPAKUJTE](../../assembler/masm/dot-repeat.md) nebo [. Při](../../assembler/masm/dot-while.md).
