# <a name="ml-fatal-error-a1008"></a>Závažná chyba nástroje ML A1008

**nespárované vnořování maker**

Buď nebylo ukončeno makro před koncem souboru, nebo byla ukončena direktiva [ENDM](endm.md) mimo blok makra.

Jednou z příčin této chyby je vynechání tečky před [. Opakujte](dot-repeat.md) nebo [. WHILe](dot-while.md).
