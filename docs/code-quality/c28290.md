# <a name="c28290"></a>C28290

> Warning C28290: Poznámka pro funkci obsahuje více externích typů, než je skutečný počet parametrů.

Nástroj Analýza kódu oznamuje toto upozornění, pokud Poznámka pro funkci obsahuje více externích typů, než je skutečný počet parametrů.
