# <a name="c28275"></a>C28275

> Warning C28275: parametr pro \_\_\_ hodnoty makra má hodnotu null.

Toto upozornění znamená, že došlo k vnitřní chybě v souboru modelu, nikoli v analyzovaném kódu. Funkce *macroValue* byla volána bez parametru.
