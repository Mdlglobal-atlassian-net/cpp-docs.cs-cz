# <a name="c28211"></a>C28211

> Upozornění C28211: pro SAL_context se očekává statický název kontextu.

Toto upozornění signalizuje, že operand `_Static_context_` anotaci musí být název kontextu definovaného nástrojem. Toto upozornění nikdy nesmí nastat při běžném použití.
