# <a name="c28224"></a>C28224

> Upozornění C28224: Poznámka vyžaduje parametry.

Toto upozornění znamená, že pojmenovaná Poznámka se používá bez parametrů a je vyžadován alespoň jeden parametr. To by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává při běžném použití.
