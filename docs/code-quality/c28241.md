# <a name="c28241"></a>C28241

> Upozornění C28241: Poznámka pro funkci v parametru nebyla rozpoznána.

Byl použit nerozpoznaný název poznámky. To by nemělo být možné, pokud jsou pro poznámky používána standardní makra. Toto upozornění se neočekává v typickém použití.
