# <a name="c28022"></a>C28022

> Upozornění C28022: třídy funkcí \<classlist1 > této funkce se neshodují s classlist2 >ou ve \<třídě typedef použité k jejímu definování.

Toto upozornění se oznamuje v případě, že poznámky obsahují chybu. Definice typedef i funkce mají `_Function_class_` poznámky, ale neodpovídají. Pokud se obě použijí, musí se shodovat.
