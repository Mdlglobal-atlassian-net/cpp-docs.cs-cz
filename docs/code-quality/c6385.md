# <a name="c6385"></a>C6385

> Upozornění C6385: neplatná data: přístup *do vyrovnávací paměti – název*, čitelná velikost je *size1* bajtů, ale mohou být přečteny *size2* bajty: řádky: *x*, *y*

Čitelný rozsah vyrovnávací paměti může být menší než index, který se používá ke čtení. Pokusy o čtení dat mimo platný rozsah vede k přetečení vyrovnávací paměti.

## <a name="example"></a>Příklad

Následující kód vygeneruje toto upozornění:

```cpp
void f(unsigned int i)
{
   char a[20];
   char j;
   if (i <= 20)  // C6385
   {
      j = a[i];
   }
}
```

Chcete-li toto upozornění opravit, použijte následující kód:

```cpp
void f(unsigned int i)
{
   char a[20];
   char j;
   if (i < 20)  // Okay
   {
      j = a[i];
   }
}
```
