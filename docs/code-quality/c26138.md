# <a name="c26138"></a>C26138

> Upozornění C26138: pozastaví korutinu a drží zámek \<zámek >.

Upozornění C26138 varuje, když je korutina pozastavena a drží zámek. Obecně nemůžeme zjistit, jak dlouho bude korutina zůstat v pozastaveném stavu, takže tento model může mít za následek delší kritické oddíly, než se očekávalo.

## <a name="example"></a>Příklad

Následující kód vygeneruje C26138.

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}

generator<int> mutex_acquiring_generator_report_once() {
  global_m.lock();
  ++var;
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  co_yield 1;                           // @expected(26138), global_m is hold while yielding.
  global_m.unlock();
}
```

## <a name="example"></a>Příklad

Následující kód bude tato upozornění opravovat.

```cpp

#include <experimental/generator>
#include <future>
#include <mutex>

using namespace std::experimental;

std::mutex global_m;
_Guarded_by_(global_m) int var = 0;

generator<int> mutex_acquiring_generator2() {
  {
    global_m.lock();
    ++var;
    global_m.unlock();
  }
  co_yield 1;                           // no 26138, global_m is already released above.
}
```
