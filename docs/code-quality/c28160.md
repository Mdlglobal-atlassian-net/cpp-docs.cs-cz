# <a name="c28160"></a>C28160

> Upozornění C28160: chybová Poznámka

Toto upozornění je oznámeno, když byla zjištěna `__drv_error` Poznámka. Tato poznámka se používá k označení postupů kódování, které by měly být opraveny, a lze ji použít s `__drv_when` anotaci k označení specifických kombinací parametrů.
