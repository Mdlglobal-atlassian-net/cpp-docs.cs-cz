# <a name="c28300"></a>C28300

> Warning C28300: < parameter_name >: operandy výrazu nekompatibilních typů pro operátor < operator_name >

Toto upozornění aktivuje anotaci SAL, která obsahuje výraz obsahující nekompatibilní typy.

## <a name="example"></a>Příklad

```cpp
union MyUnion
{
  int length;
  //...
};

// Oops, int and MyUnion are not compatible with the + operator.
void f(_In_reads_(10 + value) int *buffer, MyUnion value)
{
  for(int i = 0 ; i < (10 + value.length); i++)
  {
    //...
  }
}
```

V předchozím příkladu nezapomněl vývojář získat přístup k příslušné členské proměnné. V jiných případech může být nutné chybu opravit explicitním přetypováním.

```cpp
void f(_In_reads_(10 + value.length) int *buffer, MyUnion value)
{
  for(int i = 0 ; i < (10 + value.length); i++)
  {
    //...
  }
}
```
