# <a name="c28351"></a>C28351

> Warning C28351: anotace \<anotace > popisuje, kde v podmínce nelze použít dynamickou hodnotu (proměnnou).

Toto upozornění je oznámeno při použití poznámky, kde je kontrolován typ CC++ /.

Výraz ve `_When_` by měl být vyhodnocen jako konstanta. `_When_` se ignoruje.
