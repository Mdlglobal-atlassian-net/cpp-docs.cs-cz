# <a name="c26498-use_constexpr_for_functioncall"></a>C26498 USE_CONSTEXPR_FOR_FUNCTIONCALL

Toto volání funkce% Function% může použít `constexpr`, pokud je požadováno vyhodnocení v době kompilace. Viz [ C++ základní pokyny con. 5](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-constexpr).
