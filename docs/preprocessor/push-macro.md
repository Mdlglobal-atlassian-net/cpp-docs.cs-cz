# <a name="push_macro-pragma"></a>push_macro – direktiva pragma

Uloží hodnotu makra *název makra* v horní části zásobníku pro toto makro.

## <a name="syntax"></a>Syntaxe

> **#pragma push_macro ("** _název makra_ **")**

## <a name="remarks"></a>Poznámky

Hodnotu pro *název makra* můžete načíst pomocí `pop_macro`.

Ukázku najdete v tématu [pop_macro](../preprocessor/pop-macro.md) .
