# <a name="autosearch"></a>auto_search
**Konkrétní C++**  
  
 Určuje, že v případě, kdy je na knihovnu typů odkazováno direktivou #import a sama odkazuje na jinou knihovnu typů, může kompilátor zavést implicitní direktivu #import na tuto jinou knihovnu typů.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
auto_search  
```  
  
## <a name="remarks"></a>Poznámky  
 Konkrétní END C++
