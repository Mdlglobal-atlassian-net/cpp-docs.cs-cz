# <a name="null-directive"></a>Direktiva s hodnotou Null
Jednotné přihlašování číslo je null direktivy preprocesoru (**#**) samostatně na řádek. Nemá žádný vliv.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
#  
  
```
