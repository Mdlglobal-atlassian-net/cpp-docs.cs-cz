# <a name="null-directive"></a>Direktiva s hodnotou Null
Direktiva preprocesoru null je jeden znak čísla (**#**) samostatně na řádek. Nemá žádný vliv.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
#  
```
