# <a name="null-directive"></a>Direktiva s hodnotou Null

Direktiva preprocesoru null je jediné číslo znaku ( **#** ) samostatně na řádku. Nemá žádný vliv.

## <a name="syntax"></a>Syntaxe

> **#**
