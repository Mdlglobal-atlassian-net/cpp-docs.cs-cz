# <a name="omp-pragma"></a>omp – direktiva pragma

Přijímá jednu nebo více direktiv OpenMP spolu se všemi volitelnými klauzulemi direktivy.

## <a name="syntax"></a>Syntaxe

> **#pragma omp** *direktiva*
