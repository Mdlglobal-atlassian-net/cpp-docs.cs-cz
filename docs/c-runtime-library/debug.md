# <a name="_debug"></a>_DEBUG

Kompilátor definuje `_DEBUG`, když zadáte možnost/MTd nebo/MDd. Tyto možnosti určují ladicí verze knihovny run-time jazyka C.

Další informace naleznete v tématu [techniky ladění CRT](/visualstudio/debugger/crt-debugging-techniques).
