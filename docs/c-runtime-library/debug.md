Kompilátor definuje `_DEBUG` Pokud zadáte možnost /MTd nebo /MDd. Tyto možnosti určují ladicí verze běhové knihovny jazyka C.  
  
 Další informace najdete v tématu [techniky ladění CRT](/visualstudio/debugger/crt-debugging-techniques).
