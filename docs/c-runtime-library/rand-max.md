# <a name="randmax"></a>RAND_MAX

## <a name="syntax"></a>Syntaxe

```

#include <stdlib.h>

```

## <a name="remarks"></a>Poznámky

Konstanta `RAND_MAX` je maximální hodnota, která může být vrácen `rand` funkce. `RAND_MAX` je definován jako hodnota 0x7fff.
