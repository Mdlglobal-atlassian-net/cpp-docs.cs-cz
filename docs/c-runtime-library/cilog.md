# <a name="cilog"></a>_CIlog
Vypočítá přirozený logaritmus nejvyšší hodnotu v zásobníku.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
void __cdecl _CIlog();  
```  
  
## <a name="remarks"></a>Poznámky  
 Tato verze `log` funkce má specializované konvence volání, která funguje s technologií kompilátoru. Ji urychluje spuštění, protože kopie brání generován a pomáhá s přidělení registru.  
  
 Výsledná hodnota se posune do horní části zásobníku.  
  
## <a name="requirements"></a>Požadavky  
 **Platforma:** x86
