# <a name="windows-runtime-unsupported-crt-functions"></a>Nepodporované funkce CRT prostředí Windows Runtime

Mnoho C Runtime (CRT) rozhraní API nelze použít v aplikacích pro univerzální platformu Windows (UPW), které jsou spouštěny v modulu Windows Runtime. Tyto aplikace jsou vytvořeny pomocí příznaku kompilátoru/Zw. Seznam nepodporovaných funkcí CRT naleznete v tématu [CRT funkce nejsou podporovány v aplikacích pro univerzální platformu Windows](../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md).

Všechna rozhraní API CRT jsou popsána v [abecední odkazy funkcí](../c-runtime-library/reference/crt-alphabetical-function-reference.md) část dokumentace.
