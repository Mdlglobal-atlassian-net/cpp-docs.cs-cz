# <a name="windows-runtime-unsupported-crt-functions"></a>Nepodporované funkce CRT prostředí Windows Runtime

Mnoho rozhraní API běhového běhu (CRT) se nedá použít v aplikacích Univerzální platforma Windows (UWP), které se spouštějí v prostředí Windows Runtime. Tyto aplikace jsou sestaveny pomocí příznaku kompilátoru/ZW. Seznam nepodporovaných funkcí CRT naleznete [v tématu funkce CRT nejsou v aplikacích Univerzální platforma Windows podporovány](../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md).

Všechna rozhraní API CRT jsou popsána v části [Referenční příručka k abecedním funkcím](../c-runtime-library/reference/crt-alphabetical-function-reference.md) v dokumentaci.
