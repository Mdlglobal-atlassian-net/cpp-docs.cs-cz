# <a name="windows-runtime-unsupported-crt-functions"></a>Nepodporované funkce CRT prostředí Windows Runtime

Mnoho C Runtime (CRT) rozhraní API nelze použít v aplikacích pro univerzální platformu Windows (UWP), které jsou spuštěny v prostředí Windows Runtime. Tyto aplikace jsou vytvořeny pomocí příznaku /ZW kompilátoru. Seznam nepodporované funkce CRT najdete v tématu [CRT – funkce není podporována v aplikacích pro univerzální platformu Windows](../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md).

Všechna rozhraní API CRT jsou popsané v [abecedně řazená referenční dokumentace funkce](../c-runtime-library/reference/crt-alphabetical-function-reference.md) části dokumentace.
