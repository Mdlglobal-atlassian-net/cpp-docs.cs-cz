# <a name="windows-runtime-unsupported-crt-functions"></a>Nepodporované funkce CRT prostředí Windows Runtime
Mnoho C Runtime (CRT) rozhraní API nelze použít v aplikacích pro univerzální platformu Windows (UWP), které jsou spuštěny v prostředí Windows Runtime. Tyto aplikace jsou vytvořeny pomocí příznaku /ZW kompilátoru. Seznam nepodporované funkce CRT najdete v tématu [funkcí CRT nepodporuje /ZW](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx).  
  
 Všechna rozhraní API CRT jsou popsané v [abecedně řazená referenční dokumentace funkce](../c-runtime-library/reference/crt-alphabetical-function-reference.md) části dokumentace.
