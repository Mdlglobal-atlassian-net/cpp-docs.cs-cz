# <a name="nolock-functions"></a>Funkce jazyka _nolock

Toto jsou funkce, které nebudete provádět žádné uzamčení. Jsou určeny pro uživatele, kteří potřebují maximální výkon. Další informace najdete v tématu [výkon vícevláknových knihoven](../c-runtime-library/multithreaded-libraries-performance.md).

_Nolock – funkce použijte jenom v případě, že váš program je skutečně s jedním vláknem nebo pokud už existoval vlastní uzamčení.
