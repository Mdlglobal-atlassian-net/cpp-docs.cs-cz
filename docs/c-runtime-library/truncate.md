# <a name="truncate"></a>_TRUNCATE
Určuje chování zkracování řetězců.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
#include <stdlib.h>  
```

## <a name="example"></a>Příklad  
  
```  
// crt_truncate.c  
#include <stdlib.h>  
#include <errno.h>  
  
int main()  
{  
   char src[] = "1234567890";  
   char dst[5];  
   errno_t err = strncpy_s(dst, _countof(dst), src, _TRUNCATE);  
   if ( err == STRUNCATE )  
      printf( "truncation occurred!\n" );  
   printf( "'%s'\n", dst );  
}  
```  
  
```Output  
truncation occurred!  
'1234'  
```
