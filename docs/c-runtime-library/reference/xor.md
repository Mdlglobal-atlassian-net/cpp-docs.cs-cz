# <a name="xor"></a>xor

Alternativa k operátoru ^.

## <a name="syntax"></a>Syntaxe

```C

#define xor ^
```

## <a name="remarks"></a>Poznámky

Makro provede operátor ^.

## <a name="example"></a>Příklad

```cpp
// iso646_xor.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 3, b = 2, result;

   result= a ^ b;
   cout << result << endl;

   result= a xor_eq b;
   cout << result << endl;
}
```

```Output
1
1
```
