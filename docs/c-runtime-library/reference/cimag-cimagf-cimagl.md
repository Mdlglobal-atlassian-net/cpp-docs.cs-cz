# <a name="cimag-cimagf-cimagl"></a>cimag, cimagf, cimagl
Načte pomyslná součástí komplexního čísla.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
double cimag(   
   _Dcomplex z   
);  
float cimag(   
   _Fcomplex z   
);  // C++  
long double cimag(   
  _Lcomplex z   
);  // C++  
float cimagf(   
   _Fcomplex z   
);  
long double cimagl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `z`  
 Komplexní čísla.  
  
## <a name="return-value"></a>Návratová hodnota  
 Pomyslná součástí `z`.  
  
## <a name="remarks"></a>Poznámky  
 Protože C++ umožňuje, aby přetížení, můžete volat přetížení `cimag` trvají `_Fcomplex` nebo `_Lcomplex` hodnoty a vrátí `float` nebo `long double` hodnoty. V programu C `cimag` má vždy `_Dcomplex` hodnotu a vrátí `double` hodnotu.
