# <a name="ismbbkana-ismbbkanal"></a>_ismbbkana, _ismbbkana_l
Testů pro katakana symbol a je specifická pro znaková stránka 932.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
int _ismbbkana(  
   unsigned int c   
);  
int _ismbbkana_l(  
   unsigned int c,  
   _locale_t locale  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `c`  
 Celé číslo má být testována.  
  
 `locale`  
 Národní prostředí použít.  
  
## <a name="return-value"></a>Návratová hodnota  
 `_ismbbkana` vrátí nenulovou hodnotu, pokud na celé číslo `c` je katakana symbol nebo 0, pokud není. `_ismbbkana` používá aktuální národní prostředí informace znak závislých na národním prostředí. `_ismbbkana_l` se shoduje s tím rozdílem, že používá předaný objekt národního prostředí. Další informace najdete v tématu [národního prostředí](../../c-runtime-library/locale.md).
