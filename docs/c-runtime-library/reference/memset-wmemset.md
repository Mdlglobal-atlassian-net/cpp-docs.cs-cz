# <a name="memset-wmemset"></a>memset, wmemset
Nastaví vyrovnávací paměti je zadaný znak.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
      void *memset(  
   void *dest,  
   int c,  
   size_t count   
);  
wchar_t *wmemset(  
   wchar_t *dest,  
   wchar_t c,  
   size_t count  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 *cíle*  
 Ukazatel na cílový.  
  
 `c`  
 Znak, který má nastavit.  
  
 *počet*  
 Počet znaků.  
  
## <a name="return-value"></a>Návratová hodnota  
 Hodnota `dest`.  
  
## <a name="remarks"></a>Poznámky  
 Nastaví první `count` znaků `dest` znaku `c`.  
  
 **Poznámka k zabezpečení** Ujistěte se, že cílová vyrovnávací paměť je dostatek místa pro nejméně `count` znaků. Další informace najdete v tématu [zabraňující způsobí přetečení vyrovnávací paměti](http://msdn.microsoft.com/library/windows/desktop/ms717795).

|-------------|---------------------|  
|`memset`|\<Memory.h > nebo \<string.h >|

 Další informace o kompatibilitě, najdete v části [kompatibility](../../c-runtime-library/compatibility.md) v úvodu.

 Všechny verze [běhové knihovny jazyka C](../../c-runtime-library/crt-library-features.md).  
  
## <a name="example"></a>Příklad  
  
```  
// crt_memset.c  
/* This program uses memset to  
 * set the first four chars of buffer to "*".  
 */  
  
#include <memory.h>  
#include <stdio.h>  
  
int main( void )  
{  
   char buffer[] = "This is a test of the memset function";  
  
   printf( "Before: %s\n", buffer );  
   memset( buffer, '*', 4 );  
   printf( "After:  %s\n", buffer );  
}  
```  
  
## <a name="output"></a>Výstup  
  
```  
Before: This is a test of the memset function  
After:  **** is a test of the memset function  
```  
  
 Tady je příklad použití wmemset –:  
  
```  
// crt_wmemset.c  
/* This program uses memset to  
 * set the first four chars of buffer to "*".  
 */  
  
#include <wchar.h>  
#include <stdio.h>  
  
int main( void )  
{  
   wchar_t buffer[] = L"This is a test of the wmemset function";  
  
   wprintf( L"Before: %s\n", buffer );  
   wmemset( buffer, '*', 4 );  
   wprintf( L"After:  %s\n", buffer );  
}  
```  
  
## <a name="output"></a>Výstup  
  
```  
Before: This is a test of the wmemset function  
After:  **** is a test of the wmemset function  
```
