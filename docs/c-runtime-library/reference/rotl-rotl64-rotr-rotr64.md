# <a name="_rotl-_rotl64-_rotr-_rotr64"></a>_rotl, _rotl64, _rotr, _rotr64

Otočí bity doleva ( **_rotl**) nebo Right ( **_rotr**).

## <a name="syntax"></a>Syntaxe

```C

unsigned int _rotl(
   unsigned int value,
   int shift
);
unsigned __int64 _rotl64(
   unsigned __int64 value,
   int shift
);
unsigned int _rotr(
   unsigned int value,
   int shift
);
unsigned __int64 _rotr64(
   unsigned __int64 value,
   int shift
);
```

### <a name="parameters"></a>Parametry

*value*<br/>
Hodnota, která se má otočit

*posouvá*<br/>
Počet bitů, které se mají posunout

## <a name="return-value"></a>Návratová hodnota

Otočená hodnota. Nevrátila se žádná chybová zpráva.

## <a name="remarks"></a>Poznámky

Funkce **_rotl** a **_rotr** otočí *hodnotu* bez znaménka pomocí *SHIFT* BITS. **_rotl** otočí hodnotu vlevo. **_rotr** otočí hodnotu vpravo. Obě funkce zabalí bity za jeden konec *hodnoty* na druhý koncový objekt.

## <a name="example"></a>Příklad

```C
// crt_rot.c
/* This program shifts values to rotate an integer.
*/

#include <stdlib.h>
#include <stdio.h>

int main( void )
{
   unsigned val = 0x0fd93;
   __int64 val2 = 0x0101010101010101;

   printf( "0x%4.4x rotated left three times is 0x%4.4x\n",
           val, _rotl( val, 3 ) );
   printf( "0x%4.4x rotated right four times is 0x%4.4x\n",
           val, _rotr( val, 4 ) );

   printf( "%I64x rotated left three times is %I64x\n",
           val2, _rotl64( val2, 3 ) );
   printf( "%I64x rotated right four times is %I64x\n",
           val2, _rotr64( val2, 4 ) );
}
```

### <a name="output"></a>Výstup

```Output
0xfd93 rotated left three times is 0x7ec98
0xfd93 rotated right four times is 0x30000fd9
101010101010101 rotated left three times is 808080808080808
101010101010101 rotated right four times is 1010101010101010
```
