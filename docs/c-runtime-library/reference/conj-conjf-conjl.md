# <a name="conj-conjf-conjl"></a>conj conjf, conjl

Načte sdružené komplexního čísla.

## <a name="syntax"></a>Syntaxe

```C
_Dcomplex conj(
   _Dcomplex z
);
_Fcomplex conj(
   _Fcomplex z
);  // C++ only
_Lcomplex conj(
   _Lcomplex z
);  // C++ only
_Fcomplex conjf(
   _Fcomplex z
);
_Lcomplex conjl(
   _Lcomplex z
);
```

### <a name="parameters"></a>Parametry

*z*<br/>
Komplexní čísla.

## <a name="return-value"></a>Návratová hodnota

Sdružené *z*.  Výsledek má stejné reálné a imaginární části jako *z*, ale s opačným znaménkem.

## <a name="remarks"></a>Poznámky

Protože C++ umožňuje přetížení, můžete volat přetížení **conj** , která používají a vrací **_Fcomplex** a **_Lcomplex** hodnoty. V programu jazyka C **conj** vždy převezme a vrátí **_Dcomplex** hodnotu.
