# <a name="environmental-constants"></a>Konstanty prostředí

## <a name="syntax"></a>Syntaxe

```
#include <stdlib.h>
```

## <a name="remarks"></a>Poznámky

Tato konstanta definuje prostředí pro řetězce.

|Konstanta|Význam|
|--------------|-------------|
|`_MAX_ENV`|Maximální velikost řetězce prostředí řetězce.|
