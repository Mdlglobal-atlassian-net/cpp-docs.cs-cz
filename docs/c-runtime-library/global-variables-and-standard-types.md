# <a name="global-variables-and-standard-types"></a>Globální proměnné a standardní typy

Knihovna run-time společnosti Microsoft obsahuje definice [globálních proměnných](../c-runtime-library/global-variables.md), [řídicích příznaků](../c-runtime-library/control-flags.md)a [standardních typů](../c-runtime-library/standard-types.md) používaných rutinami knihovny. K těmto proměnným, příznakům a typům přistupujete jejich deklarováním v programu nebo zahrnutím příslušných hlavičkových souborů.
