# <a name="tznamemax"></a>TZNAME_MAX

**Zastaralé**. Maximální přípustnou délku řetězce pro název proměnné časového pásma. Toto makro bylo definováno v \<limits.h > v sadě Visual Studio 2012 a dřívějších verzích. Není definována v sadě Visual Studio 2013 a novějších verzích. Chcete-li získat délka povolená pro uložení názvu aktuálního časového pásma, použijte [_get_tzname –](../c-runtime-library/reference/get-tzname.md).

## <a name="syntax"></a>Syntaxe

```
#include <limits.h>
```
