# <a name="windows-platforms-crt"></a>Platformy systému Windows (CRT)

Knihovny run-time jazyka C pro Visual Studio podporují aktuální verze Windows a Windows serveru, Windows 8, Windows Server 2012, Windows 7, Windows Server 2008 a Windows Vista a volitelně podporují Windows XP Service Pack 3 (SP3) pro x86, Windows XP Service Pack 2 (SP2) pro x64 a Windows Server 2003 Service Pack 2 (SP2) pro x86 i x64. Všechny tyto operační systémy podporují rozhraní Windows Desktop API (Win32) a poskytují podporu kódování Unicode. Kromě toho může jakákoli aplikace Win32 použít vícebajtovou znakovou sadu (MBCS).

> [!NOTE]
> Výchozí instalace **desktopového vývoje s C++**  úlohou v rámci sady Visual Studio 2017 neobsahuje podporu pro vývoj pro Windows XP a Windows Server 2003. Chcete-li povolit sadu nástrojů platformy Windows XP, je nutné nainstalovat volitelnou součást **Windows XP optioned Support pro C++**  .
