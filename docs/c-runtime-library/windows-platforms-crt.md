# <a name="windows-platforms-crt"></a>Platformy systému Windows (CRT)
Běhové knihovny jazyka C pro [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] podporu [!INCLUDE[win8](../build/reference/includes/win8_md.md)], [!INCLUDE[winserver8](../build/reference/includes/winserver8_md.md)], [!INCLUDE[win7](../build/includes/win7_md.md)], [!INCLUDE[winsvr08](../build/reference/includes/winsvr08_md.md)], Windows Vista, [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 3 (SP3) pro x86, [!INCLUDE[winxp](../build/includes/winxp_md.md)] Service Pack 2 (SP2) pro x64, a [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] Aktualizaci Service Pack 2 (SP2) pro x86 a x64. Všechny tyto operační systémy podporují Windows aplikace programovací rozhraní (API) a podporují kódování Unicode. Všechny aplikace Win32 kromě toho můžete použít vícebajtové znakové sady (MBCS).  
  
> [!NOTE]
>  [!INCLUDE[vs_dev11_long](../build/includes/vs_dev11_long_md.md)]nepodporuje [!INCLUDE[winxp](../build/includes/winxp_md.md)] a [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)] vývoj. Je nutné nainstalovat [!INCLUDE[vs2012_upd01](../c-runtime-library/includes/vs2012_upd01_md.md)] přidání podpory pro [!INCLUDE[winxp](../build/includes/winxp_md.md)] a [!INCLUDE[WinXPSvr](../build/includes/winxpsvr_md.md)].
