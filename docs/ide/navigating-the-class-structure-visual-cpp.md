# <a name="navigating-the-class-structure-visual-c"></a>Navigace strukturou třídy (Visual C++)
Struktura třídy projektů, včetně funkcí, dědičnosti a buď pomocí rozhraní můžete přejít [zobrazení tříd](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925) nebo [Prohlížeč objektů](http://msdn.microsoft.com/en-us/f89acfc5-1152-413d-9f56-3dc16e3f0470).  
  
 Můžete přejít na deklaraci funkce dvojím kliknutím na název funkce v zobrazení tříd. Dvojím kliknutím na název třídy v zobrazení tříd, můžete přejít na definici třídy.
