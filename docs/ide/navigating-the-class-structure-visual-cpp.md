# <a name="navigating-the-class-structure-visual-c"></a>Navigace strukturou třídy (Visual C++)

Třída strukturu projektu, včetně funkcí, dědičnosti a buď pomocí rozhraní se můžete dostat **zobrazení tříd** nebo **prohlížeče objektů**. Další informace najdete v tématu [zobrazení struktury kódu](/visualstudio/ide/viewing-the-structure-of-code).

Poklikáním na název funkce v zobrazení tříd můžete přejít k deklaraci funkce. Poklikáním na název třídy v zobrazení tříd můžete přejít k definici třídy.
