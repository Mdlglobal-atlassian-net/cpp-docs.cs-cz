# <a name="navigate-the-class-structure"></a>Navigace strukturou třídy

Struktura třídy projektů můžete Navigovat pomocí buď **zobrazení tříd** nebo **prohlížeče objektů**. Struktura třídy projektů může zahrnovat funkce, dědičnost a rozhraní. Další informace najdete v tématu [zobrazení struktury kódu](/visualstudio/ide/viewing-the-structure-of-code).

Chcete-li přejít k deklaraci funkce, dvakrát klikněte na název funkce v zobrazení tříd. Přejít k definici třídy, dvakrát klikněte na název třídy v zobrazení tříd.
