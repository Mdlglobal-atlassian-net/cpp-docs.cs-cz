# <a name="building-external-projects"></a>Sestavení externích projektů
Externí projekt je projektu Visual C++, který používá souboru pravidel nebo jiných zařízení, které jsou mimo (cizí nebo mimo ni) vývojového prostředí jazyka Visual C++.  
  
 Pokud máte externí projekt (například projektem souboru pravidel), který chcete vytvořit ve vývojovém prostředí Visual C++, vytvoření projektu souboru pravidel a zadat vašeho projektu sestavení příkaz a výstup v Průvodci aplikací souboru pravidel. Další informace najdete v tématu [vytvoření projektu souboru pravidel](../ide/creating-a-makefile-project.md).  
  
 Všimněte si, že Visual C++ již nepodporuje možnost Exportovat soubor pravidel pro aktivní projekt z vývojového prostředí. Použití [příkazového řádku DEVENV](/visualstudio/ide/reference/devenv-command-line-switches) k sestavení projektů sady Visual Studio na příkazovém řádku.
