# <a name="advanced-manifest-tool-configuration-properties-ltprojectnamegt-property-pages-dialog-box"></a>Pokročilé, Nástroj Manifest, vlastnosti konfigurace &lt;Projectname&gt; dialogové okno stránky vlastností
Pomocí tohoto dialogového okna Rozšířené možnosti pro [Mt.exe](http://msdn.microsoft.com/library/aa375649).  
  
 Pro přístup k toto dialogové okno stránky vlastností, otevřete stránku vlastností projektu nebo seznam vlastnosti. Rozbalte **Nástroj Manifest** pod uzlem **vlastnosti konfigurace**a potom vyberte **Upřesnit**.  
  
## <a name="uielement-list"></a>Seznam prvků uživatelského rozhraní  
 **Aktualizace hodnoty hash souboru**  
 Používá k určení, že nástroj manifestu bude výpočetní hodnoty hash souborů zadaný v hashupdate `<file>` prvky a poté aktualizace hash atributy s vypočtenou hodnotou.  
  
 **Cesta hledání hodnoty hash souboru aktualizace**  
 Určuje cestu pro soubory, které jsou odkazované ve vyhledávání `<file>` elementy. Tato volba používá také hashupdate.
