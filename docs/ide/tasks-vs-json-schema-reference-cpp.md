# <a name="tasksvsjson-c"></a>Tasks.vs.JSON (C++)

A `tasks.vs.json` soubor lze přidat do projektu otevřít složku definovat všechny libovolné úlohy, a poté vyvolat z **Průzkumníka řešení** kontextové nabídky. Projekty CMake obvykle tento soubor nepoužívejte vzhledem k tomu, že všechny příkazy sestavení jsou uvedeny v `CMakeLists.txt`. Pro systémy sestavení než CMake jde, ve kterém můžete zadat sestavení příkazům a vyvolání skriptů sestavení. Obecné informace o používání tasks.vs.json najdete v tématu [přizpůsobení sestavení a ladění úlohy pro vývoj "Otevřít složku"](/visualstudio/ide/customize-build-and-debug-tasks-in-visual-studio).
