# <a name="xml-data-generator-tool-property-page"></a>Stránka vlastností nástroje generátoru dat XML
**Nástroje generátoru dat XML** stránka vlastností je k dispozici při přidání datové sady do projektu.  
  
 **Nástroje generátoru dat XML** vlastnost stránka obsahuje následující vlastnosti:  
  
 **Výstupní soubor**  
 Určuje název výstupního souboru použít.  
  
 **Potlačit úvodní nápis při spouštění**  
 Potlačí zobrazení zpráv banner a informace o spuštění.  
  
 **Jazyk vygenerovaný proxy server**  
 Určuje, zda se pro vydávání spravovaného kódu.  
  
 Informace o tom, jak získat přístup **nástroje generátoru dat XML** stránce vlastností, najdete v části [práce s vlastnostmi projektu](../ide/working-with-project-properties.md).  
  
 Informace o tom, jak programově přístup ke členům tohoto objektu najdete v tématu <xref:Microsoft.VisualStudio.VCProjectEngine.VCXMLDataGeneratorTool>
