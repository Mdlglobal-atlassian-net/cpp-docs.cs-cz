# <a name="makefile-project-wizard"></a>Průvodce projektem souboru pravidel
Pomocí tohoto průvodce k vytvoření příkazového řádku pro modul sestavení provést. Jakmile vytvoříte projekt, můžete zobrazit a upravit vlastnosti projektu v projektu [stránka vlastností](../ide/working-with-project-properties.md).  
  
## <a name="overview"></a>Přehled  
 Tato stránka průvodce popisuje aktuální nastavení projektu pro projekt, který vytváříte. Ve výchozím nastavení, je nastavení projektu sestavení obou ladění a vydání konfigurace *ProjName*.exe.  
  
 Chcete-li změnit toto výchozí nastavení, klikněte na tlačítko [nastavení aplikace](../ide/application-settings-makefile-project-wizard.md) v levém sloupci průvodce a proveďte požadované změny.
