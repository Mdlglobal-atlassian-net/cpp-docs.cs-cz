# <a name="makefile-project-wizard"></a>Průvodce projektem souboru pravidel

Tohoto průvodce použijte k vytvoření příkazový řádek, který stroj sestavení ke spuštění. Po vytvoření projektu můžete zobrazit a upravit vlastnosti projektu v projektu [stránku vlastností](../ide/working-with-project-properties.md).

## <a name="overview"></a>Přehled

Tato stránka průvodce popisuje aktuální nastavení projektu pro projekt, který vytváříte. Ve výchozím nastavení, je nastavení projektu k sestavení i ladění a vydání konfigurace *název_projektu*.exe.

Chcete-li změnit výchozí nastavení, klikněte na tlačítko [nastavení aplikace](../ide/application-settings-makefile-project-wizard.md) kartu v levém sloupci průvodce a proveďte požadované změny.
