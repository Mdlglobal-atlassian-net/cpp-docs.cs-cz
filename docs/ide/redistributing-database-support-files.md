# <a name="redistributing-database-support-files"></a>Redistribuce pomocných souborů databáze
Můžete znovu distribuovat podporované soubory pro objektů DAO (Data Access) a pro databázové technologie v sadě Microsoft Data Access SDK.  
  
## <a name="installing-dao-support-files"></a>Instalace podporovaných souborů rozhraní DAO  
 Pokud chcete získat nejnovější verzi rozhraní DAO, viz [článek 239114: Jak získat nejnovější aktualizaci service pack pro databázový stroj Microsoft Jet 4.0](http://go.microsoft.com/fwlink/p/?linkid=198014) na webu Microsoft Support.  
  
## <a name="installing-microsoft-data-access-sdk-support-files"></a>Instalace Microsoft Data Access SDK podporovaných souborů  
 K instalaci podpory pro rozhraní ODBC, OLE DB, objekty ADO (ActiveX Data OBJECTS) a Remote Data Services (RDS) použijte Mdac_typ.exe. Mdac_typ.exe je umístěn ve... Složka \WCU\MDAC28\ na instalačním médiu nástroje Visual Studio. Můžete také stáhnout z Mdac_typ.exe [Microsoft Download Center](http://go.microsoft.com/fwlink/p/?linkid=198015) webu.  
  
 Další informace najdete v [MSDN](http://go.microsoft.com/fwlink/p/?linkid=198016) webu, vyhledejte "Redistribuce MDAC 2.8 SP1". Používáte-li instalační projekty Visual Studio k nasazení vaší aplikace, najdete v článku [nasazení a závislostí](https://msdn.microsoft.com/49e9b84d-bd6a-4388-b9ac-46ea79cf0733).
