# <a name="redistributing-database-support-files"></a>Redistribuce pomocných souborů databáze
Pro objekty DAO (Data Access) a pro databázové technologie v sadě Microsoft Data Access SDK je možné znovu distribuovat soubory podpory.  
  
## <a name="installing-dao-support-files"></a>Instalace souborů podpory rozhraní DAO  
 Chcete-li získat nejnovější verzi rozhraní DAO, přečtěte si téma [článku 239114: Jak získat nejnovější aktualizaci service pack pro databázový stroj Microsoft Jet 4.0](http://go.microsoft.com/fwlink/p/?linkid=198014) na webu Microsoft Support.  
  
## <a name="installing-microsoft-data-access-sdk-support-files"></a>Instalace podpory souborů Microsoft datové sady SDK  
 Mdac_typ.exe použijte k instalaci podpory pro rozhraní ODBC, OLE DB, objektů ADO (ActiveX Data) a Remote Data Services (RDS). Mdac_typ.exe se nachází v... Složka \WCU\MDAC28\ na instalačním médiu nástroje Visual Studio. Můžete také stáhnout Mdac_typ.exe z [Microsoft Download Center](http://go.microsoft.com/fwlink/p/?linkid=198015) webu.  
  
 Další informace najdete na [MSDN](http://go.microsoft.com/fwlink/p/?linkid=198016) webu, vyhledejte "Redistribuce MDAC 2.8 SP1". Pokud používáte projekty instalace Visual Studio pro nasazení aplikace, najdete v části [nasazení a závislostí](http://msdn.microsoft.com/en-us/49e9b84d-bd6a-4388-b9ac-46ea79cf0733).
