# <a name="adding-a-generic-c-class"></a>Přidání generické třídy jazyka C++

Můžete přidat obecnou C++ třídu pomocí **zobrazení tříd**. Generické třídy jazyka C++ je třída, která definujete nebo, která je odvozena z třídy, které definujete.

### <a name="to-add-a-generic-c-class-to-a-project"></a>Přidání generické třídy jazyka C++ do projektu

1. V **zobrazení tříd**, klikněte pravým tlačítkem na projekt, ke kterému chcete přidat novou třídu, klikněte na tlačítko **přidat**a potom klikněte na tlačítko **třídy**.

1. V [přidat třídu](../ide/add-class-dialog-box.md) kliknutím na dialogové okno, v podokně šablony **třídy C++**. Klikněte na tlačítko **přidat** zobrazíte [Průvodci obecnými třídami C++](../ide/generic-cpp-class-wizard.md).

1. V průvodci zadejte název třídy a potom definovat nastavení nebo přijměte výchozí hodnoty.

1. Chcete-li ukončit průvodce a zobrazit nové generické třídy jazyka C++ v projektu, klikněte na tlačítko **Dokončit**.
