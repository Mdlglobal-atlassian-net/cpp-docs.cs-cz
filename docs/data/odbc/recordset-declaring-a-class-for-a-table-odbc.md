# <a name="recordset-declaring-a-class-for-a-table-odbc"></a>Sada záznamů: Deklarování třídy pro tabulku (ODBC)

> [!NOTE]
> Průvodce příjemcem knihovny MFC rozhraní ODBC není dostupný v aplikaci Visual Studio 2019 a novějším. Příjemce můžete přesto vytvořit ručně.

Toto téma se vztahuje na třídy knihovny MFC rozhraní ODBC.

Nejběžnější třída sady záznamů otevře jednu tabulku. Chcete-li deklarovat třídu sady záznamů pro jednu tabulku, použijte [Průvodce příjemcem knihovny MFC rozhraní ODBC](../../mfc/reference/adding-an-mfc-odbc-consumer.md) z **Přidat třídu** a zvolte každý požadovaný sloupec pojmenováním odpovídajícího datového člena pole sady záznamů.

Mezi další použití pro sady záznamů patří:

- Spojení dvou nebo více tabulek.

- Obsahuje výsledky předdefinovaného dotazu.
