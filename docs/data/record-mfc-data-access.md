# <a name="record-mfc-data-access"></a>Záznam (přístup k datům MFC)

Záznam je kolekce dat o jedné entitě, jako je například účet nebo zákazník, uložený v tabulce (řádek tabulky). Záznam se skládá ze skupiny souvislých sloupců (někdy označovaných jako pole), které obsahují data různých typů. Sada záznamů vybraných ze zdroje dat, která se často nazývá sada výsledků v databázových výrazech, se nazývá sada záznamů v knihovně MFC. Další informace naleznete v tématu [Sada záznamů (ODBC)](../data/odbc/recordset-odbc.md).
