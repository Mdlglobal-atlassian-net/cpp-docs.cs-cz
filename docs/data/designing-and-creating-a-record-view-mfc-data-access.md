# <a name="designing-and-creating-a-record-view--mfc-data-access"></a>Návrh a vytvoření zobrazení záznamu (přístup k datům MFC)

Můžete vytvořit třídu zobrazení záznamu pomocí [Průvodce aplikací knihovny MFC](../mfc/reference/database-support-mfc-application-wizard.md). Použijete-li Průvodce aplikací, vytvoří třídu zobrazení záznamu a prostředek šablony dialogového okna (bez ovládacích prvků). Chcete-li přidat ovládací C++ prvky do prostředku šablony dialogového okna, je nutné použít Editor dialogového okna Visual. Na druhé straně, pokud použijete **Přidat třídu**, musíte nejprve vytvořit prostředek šablony dialogového okna v editoru dialogového okna a pak vytvořit třídu zobrazení záznamu.

#### <a name="to-create-your-record-view-with-the-mfc-application-wizard"></a>Vytvoření zobrazení záznamu pomocí Průvodce aplikací knihovny MFC

1. Viz [Podpora databáze, Průvodce aplikací MFC](../mfc/reference/database-support-mfc-application-wizard.md).

#### <a name="to-design-your-form"></a>Návrh formuláře

1. Viz [Editor dialogových oken](../windows/dialog-editor.md).

#### <a name="to-create-your-record-view-class"></a>Vytvoření třídy zobrazení záznamu

1. Viz [Přidání příjemce knihovny MFC rozhraní ODBC](../mfc/reference/adding-an-mfc-odbc-consumer.md).
