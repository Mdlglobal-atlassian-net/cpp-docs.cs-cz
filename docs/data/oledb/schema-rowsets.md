# <a name="schema-rowsets"></a>Sady řádků schématu

Schéma (nebo metadata) je jakékoli informace o struktuře nebo organizaci databáze, jako jsou informace o poskytovateli, sadě řádků, tabulce, sloupcích nebo jiných informacích, z nichž se nejedná o skutečný obsah databáze. OLE DB umožňuje získat takové informace bez otevření sady řádků pomocí sad řádků schématu.

Informace o tom, jak používat sady řádků schématu, naleznete v tématu [Získávání metadat pomocí sad řádků schématu](../../data/oledb/obtaining-metadata-with-schema-rowsets.md).
