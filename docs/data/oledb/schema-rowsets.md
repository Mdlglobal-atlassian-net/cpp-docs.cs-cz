# <a name="schema-rowsets"></a>Sady řádků schématu

Schéma (nebo metadat) se žádné informace o společnosti, například informace o poskytovatele, sady řádků, tabulky, sloupce, nebo struktura databáze nebo jiné informace kromě skutečný obsah databáze. OLE DB umožňuje získat tyto informace bez nutnosti otevřít v sadě řádků pomocí sad řádků schématu.  
  
Informace o tom, jak pomocí sad řádků schématu najdete v tématu [získávání metadat pomocí sad řádků schématu](../../data/oledb/obtaining-metadata-with-schema-rowsets.md).
