# <a name="creating-an-ole-db-consumer"></a>Vytvoření příjemce technologie OLE DB

::: moniker range="vs-2019"

Průvodce příjemcem OLE DB ATL není v aplikaci Visual Studio 2019 a novějších k dispozici. Tuto funkci můžete přesto přidat ručně. Další informace najdete v tématu [Vytvoření příjemce bez použití Průvodce](creating-a-consumer-without-using-a-wizard.md).

::: moniker-end

::: moniker range="<=vs-2017"

Můžete vytvořit OLE DB jako příjemce šablon s nebo bez **průvodce OLE DB příjemce ATL**, i když použití Průvodce je doporučenou metodou. Příjemce může pracovat v libovolné aplikaci, která podporuje ATL.

Tato část obsahuje následující témata:

- [Vytvoření příjemce OLE DB pomocí průvodce](../../data/oledb/creating-an-ole-db-consumer-using-a-wizard.md)

- [Vytvoření příjemce OLE DB bez použití Průvodce](../../data/oledb/creating-a-consumer-without-using-a-wizard.md)

::: moniker-end
