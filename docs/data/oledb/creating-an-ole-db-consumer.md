# <a name="creating-an-ole-db-consumer"></a>Vytvoření příjemce technologie OLE DB

::: moniker range="vs-2019"

Průvodce spotřebitele ATL OLE DB není k dispozici v aplikaci Visual Studio 2019 a novějším. Funkce můžete přesto přidat ručně. Další informace najdete v tématu [vytvoření příjemce bez použití průvodce](creating-a-consumer-without-using-a-wizard.md).

::: moniker-end

::: moniker range="<=vs-2017"

Spotřebitel šablony technologie OLE DB můžete vytvořit s nebo bez něj **průvodce příjemcem ATL OLE DB**, když pomocí Průvodce je doporučená metoda. Příjemce může pracovat v libovolné aplikaci, která podporuje ATL.

Tato část obsahuje následující témata:

- [Vytvoření příjemce OLE DB pomocí průvodce](../../data/oledb/creating-an-ole-db-consumer-using-a-wizard.md)

- [Vytvoření příjemce technologie OLE DB bez použití Průvodce](../../data/oledb/creating-a-consumer-without-using-a-wizard.md)

::: moniker-end
