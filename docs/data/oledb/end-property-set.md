# <a name="endpropertyset"></a>END_PROPERTY_SET
Označuje konec sadu vlastností.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
END_PROPERTY_SET(guid)  
  
```  
  
#### <a name="parameters"></a>Parametry  
 `guid`  
 [v] Vlastnost identifikátor GUID.
