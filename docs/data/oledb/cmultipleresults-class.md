# <a name="cmultipleresults-class"></a>CMultipleResults – třída

Pokud chcete, aby příkaz zpracovával více sad výsledků, použijte `CMultipleResults` pro `CCommand` argument šablony *TMultiple*.

## <a name="syntax"></a>Syntaxe

```cpp
class CMultipleResults
```

## <a name="remarks"></a>Poznámky

Pro zpracování více sad výsledků musí [CCommand](../../data/oledb/ccommand-class.md) dědit z této třídy.

## <a name="requirements"></a>Požadavky

**Záhlaví:** atldbcli
