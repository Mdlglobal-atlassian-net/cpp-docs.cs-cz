# <a name="endcolumnmap"></a>END_COLUMN_MAP
Označuje konec položek mapování sloupců.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## <a name="remarks"></a>Poznámky  
 Používá se s přistupujícím objektem jeden pro sadu řádků. `BEGIN_COLUMN_MAP` Makro byla dokončena s `END_COLUMN_MAP` makro.
