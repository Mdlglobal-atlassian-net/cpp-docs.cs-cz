# <a name="beginparammap"></a>BEGIN_PARAM_MAP
Označuje začátek položek mapování parametru.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
BEGIN_PARAM_MAP(x)  
  
```  
  
#### <a name="parameters"></a>Parametry  
 *x*  
 [v] Název třídy záznamu uživatele.  
  
## <a name="remarks"></a>Poznámky  
 Parametry jsou používány [příkazy](https://msdn.microsoft.com/en-us/library/ms724608.aspx).  
  
## <a name="example"></a>Příklad  
 Podívejte se příklad [BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md) makro.
