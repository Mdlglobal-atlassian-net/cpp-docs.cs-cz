# <a name="cviewtableusage-cviewtableinfo"></a>CViewTableUsage, CViewTableInfo
Call – třída definice typedef **CViewTableUsage** k implementaci jeho – třída parametru **CViewTableInfo**.  
  
## <a name="remarks"></a>Poznámky  
 V tématu [třídy sady řádků schématu a definiční třídy typů](../../data/oledb/schema-rowset-classes-and-typedef-classes.md) pro další informace o použití definiční třídy typů.  
  
 Tato třída identifikuje zobrazenou tabulky, definované v katalogu, které jsou dostupné pro daného uživatele.  
  
 Následující tabulka uvádí členy třídy dat a jejich odpovídající OLE DB sloupce. V tématu [VIEW_TABLE_USAGE řádků](https://msdn.microsoft.com/en-us/library/ms719727.aspx) v *referenční příručka programátora technologie OLE DB* Další informace o schématu a sloupců.

|m_szName|VIEW_NAME|
