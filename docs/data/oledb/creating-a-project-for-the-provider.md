# <a name="creating-a-project-for-the-provider"></a>Vytvoření projektu pro zprostředkovatele
#### <a name="to-create-a-project-in-which-the-ole-db-provider-will-reside"></a>Vytvoření projektu, ve kterém se bude nacházet zprostředkovatel OLE DB  
  
1.  Z **soubor** nabídky, klikněte na tlačítko `New`a potom klikněte na **projektu**.  
  
     **Nový projekt** zobrazí se dialogové okno.  
  
2.  V podokně typy projektů klikněte na složku projekty Visual C++. V podokně šablon klikněte na tlačítko **projektu knihovny ATL**.  
  
3.  V **název** pole, zadejte název projektu a pak klikněte na tlačítko **OK**.  
  
     Zobrazí se průvodce ATL projektu.  
  
4.  V Průvodci projektu knihovny ATL zvolte **dynamická knihovna (DLL)** pro **typ serveru**.  
  
5.  Klikněte na tlačítko **Dokončit**.
