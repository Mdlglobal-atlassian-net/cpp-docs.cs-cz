# <a name="crowsetmovefirst"></a>CRowset::MoveFirst
Posune kurzor na počáteční pozici a načte počáteční řádek.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp
HRESULT MoveFirst() throw();  
  
```  
  
## <a name="return-value"></a>Návratová hodnota  
 Standardní `HRESULT`.
