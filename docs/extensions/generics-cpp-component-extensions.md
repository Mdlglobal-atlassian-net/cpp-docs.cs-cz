# <a name="generics--ccli-and-ccx"></a>Obecné typy (C++/CLI a C++/CX)

Obecné typy jsou parametrizované typy a metody. V této části zjistíte, které obecné funkce jsou podporovány prostředí Windows Runtimem i modulem CLR (Common Language Runtime) a které jsou podporovány pouze modulem CLR (Common Language Runtime). Přečtěte si také, jak vytvářet vlastní obecné metody a typy v C++/CLI a jak používat obecné typy vytvořené v jazyce .NET Framework v C++/CLI. A konečně, Tato část poskytuje porovnání obecných typů a C++ šablon.
