# <a name="15-normative-references"></a>1.5 Normativní odkazy
-   ISO/IEC 9899:1999 *informace o technologii - programovací jazyky - C*. Toto rozhraní API OpenMP specifikace označuje ISO/IEC 9899:1999 jako C99.  
  
-   ISO/IEC 9899:1990 *informace o technologii - programovací jazyky - C*. Toto rozhraní API OpenMP specifikace označuje ISO/IEC 9899:1990 jako C90.  
  
-   ISO/IEC 14882:1998 *informace o technologii - programovací jazyky - C++*. Toto rozhraní API OpenMP specifikace označuje ISO/IEC 14882:1998 jako C++.  
  
 Kde tato specifikace rozhraní API OpenMP odkazuje C, se odkazuje na základní jazyk podporovaný implementace.
