# <a name="resource-compiler-error-rc2103"></a>Chyba kompilátoru prostředků RC2103

Neočekávaný konec souboru v řetězcovém literálu

Byl nalezen konec souboru před koncem řetězce. Řetězci pravděpodobně chybí uzavírací dvojité uvozovky ( **"** ).
