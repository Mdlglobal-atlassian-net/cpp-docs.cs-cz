# <a name="expression-evaluator-error-cxx0066"></a>Chyba při vyhodnocování výrazu CXX0066

statický člen není přítomen.

Statický člen třídy se nepovedlo najít nebo není definovaný. Tato chyba může být způsobena členem statické třídy, která je deklarována, ale není definována, nebo je definována a odkazována pouze v modulech, které neobsahují informace o ladění.

Tato chyba je shodná s CAN0066.
