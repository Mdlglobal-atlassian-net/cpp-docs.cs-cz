# <a name="expression-evaluator-error-cxx0032"></a>Chyba při vyhodnocování výrazu CXX0032
dělení 0  
  
 Výraz obsahuje dělitel nulové hodnoty, který je neplatný. Tato dělitel může být literál hodnotu nula, nebo může být výraz, který se vyhodnotí na hodnotu nula.  
  
 Tato chyba je stejný jako CAN0032.
