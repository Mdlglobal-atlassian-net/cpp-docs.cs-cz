# <a name="resource-compiler-fatal-error-rc1047"></a>Závažná chyba kompilátoru prostředků RC1047

"příliš mnoho možnost volby,"řetězec""

Příliš mnohokrát byl zadán dané volby. Daný řetězec je argument pro možnost, která způsobila chybu.
