# <a name="resource-compiler-fatal-error-rc1047"></a>Závažná chyba kompilátoru prostředků RC1047
"příliš mnoho možnost možnosti, 'řetězec'"  
  
 Danou možnost byla zadána příliš mnohokrát. Daný řetězec je argument pro možnost, která způsobila chybu.
