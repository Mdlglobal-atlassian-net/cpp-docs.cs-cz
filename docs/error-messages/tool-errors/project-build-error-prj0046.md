# <a name="project-build-error-prj0046"></a>Chyba sestavení projektu PRJ0046

Nebylo možné vytvořit příkazový řádek, protože zadaný byl prázdný.

Pro konfiguraci souboru pravidel byl zadán prázdný příkazový řádek; příkazový řádek je povinný.
