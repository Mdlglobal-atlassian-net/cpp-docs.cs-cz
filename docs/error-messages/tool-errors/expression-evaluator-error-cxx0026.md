# <a name="expression-evaluator-error-cxx0026"></a>Chyba při vyhodnocování výrazu CXX0026

Chybný řetězec formátu

Řetězec formátu nebyl správně zadán. Podívejte se na syntaxi výrazu.

Tato chyba je shodná s CAN0026.

Další informace naleznete v tématu [specifikátory formátu v C++ ](/visualstudio/debugger/format-specifiers-in-cpp).
