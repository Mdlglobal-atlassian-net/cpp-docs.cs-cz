# <a name="resource-compiler-fatal-error-rc1004"></a>Závažná chyba kompilátoru prostředků RC1004

Neočekávaný konec souboru nalezen

Tuto chybu může způsobovat chybějící znak odřádkování a návrat na začátek znaky na posledním řádku z textového souboru.
