# <a name="project-build-error-prj0044"></a>Chyba sestavení projektu PRJ0044

Vlastnost Další závislosti pro pravidlo vlastního buildu Rule přiřazené souboru ' file ' je neplatná. Vlastnost obsahovala hodnotu ' String ', která je vyhodnocena na hodnotu ' value '.

Vlastnost **Další závislosti** vyhodnocená na prázdný řetězec nebo na řetězec, který obsahuje neplatné znaky (libovolný znak, který nemůže být v názvu souboru nebo adresáře). Vlastní pravidla sestavení potřebují výstup akce sestavení.

Další informace naleznete v tématu [určení vlastních nástrojů sestavení](../../build/specifying-custom-build-tools.md).
