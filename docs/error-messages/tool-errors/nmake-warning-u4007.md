# <a name="nmake-warning-u4007"></a>Upozornění nástroje NMAKE U4007

název souboru filename je příliš dlouhý; zkrácení na 8,3

Základní název daného souboru má více než osm znaků nebo přípona obsahuje více než tři znaky. NMAKE zkrátil název na bázi osmi znaků a příponu se třemi znaky.

Pokud systém souborů podporuje dlouhé názvy souborů, uzavřete název do dvojitých uvozovek ( **"** ).
