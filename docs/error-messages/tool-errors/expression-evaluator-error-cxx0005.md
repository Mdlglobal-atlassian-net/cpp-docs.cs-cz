# <a name="expression-evaluator-error-cxx0005"></a>Chyba při vyhodnocování výrazu CXX0005

operátor není podporován

Ve výrazu byl zadán nepodporovaný operátor jazyka C. Napište ekvivalentní výraz pomocí podporovaných operátorů jazyka C.

Tato chyba je shodná s CAN0005.
