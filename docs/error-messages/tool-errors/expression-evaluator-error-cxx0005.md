# <a name="expression-evaluator-error-cxx0005"></a>Chyba při vyhodnocování výrazu CXX0005

operátor není podporován

Ve výrazu byl zadán nepodporovaný operátor C. Zápis ekvivalentní výraz, který používá podporovaný operátory jazyka C.

Tato chyba se shoduje s CAN0005.
