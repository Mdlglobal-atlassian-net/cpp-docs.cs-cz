# <a name="expression-evaluator-error-cxx0012"></a>Chyba při vyhodnocování výrazu CXX0012

Chybí levá složená závorka.

Výraz obsahuje nevyváženou složenou závorku. Znovu zadejte výraz s vyváženými složenými závorkami.

Tato chyba je shodná s CAN0012.
