# <a name="resource-compiler-error-rc2017"></a>Chyba kompilátoru prostředků RC2017

Neplatná řídicí sekvence

Řídicí sekvence se zobrazila tam, kde nebyla očekávána.

Řídicí sekvence – zpětné lomítko ( **\\** ) následované číslem nebo písmenem, se může vyskytovat pouze ve znaku nebo v řetězcové konstantě.
