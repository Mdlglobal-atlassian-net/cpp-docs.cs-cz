# <a name="command-line-error-d8036"></a>Chyba příkazového řádku D8036

'/Option ' není povoleno s více zdrojovými soubory

Tyto možnosti kompilátoru nelze použít s více zdrojovými soubory:

- Seznam názvových souborů sestavení (/FA)

- Přejmenovat soubor objektu (/FO)

- Vytvoření informací o prohlížeči zdrojového kódu bez místních proměnných (/FR)

- Vytváření informací o zdrojovém prohlížeči, které zahrnují místní proměnné (/FR)
