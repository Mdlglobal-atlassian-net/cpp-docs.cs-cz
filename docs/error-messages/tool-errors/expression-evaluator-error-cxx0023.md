# <a name="expression-evaluator-error-cxx0023"></a>Chyba při vyhodnocování výrazu CXX0023

Chybný základ číselné soustavy

Chyba při vyhodnocování výrazů C určeném nerozpozná. Pouze desítkové, hexadecimální a interpretace osmičková radixes jsou platné.

Tato chyba se shoduje s CAN0023.
