# <a name="expression-evaluator-error-cxx0023"></a>Chyba při vyhodnocování výrazu CXX0023

Chybná základová

Vyhodnocovací filtr výrazů jazyka C nerozpoznává zadanou základovou číselnou soustavou. Platné jsou jenom hodnoty Decimal, šestnáctkové a osmičkové radixes.

Tato chyba je shodná s CAN0023.
