# <a name="expression-evaluator-error-cxx0023"></a>Chyba při vyhodnocování výrazu CXX0023
základ chybný –  
  
 Vyhodnocovací filtr výrazů C nerozpoznal základ – zadán. Platné jsou pouze decimal, hexadecimální a osmičková radixes.  
  
 Tato chyba je stejný jako CAN0023.
