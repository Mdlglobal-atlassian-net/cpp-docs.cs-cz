# <a name="command-line-warning-d9040"></a>Upozornění příkazového řádku D9040

ignoruje se možnost/Analyze; V této edici kompilátoru nejsou dostupná upozornění analýzy kódu.

Možnost příkazového řádku **/analyze** není k dispozici ve všech edicích sady Visual Studio. Chcete-li toto upozornění vyřešit, buď přepněte na podporovanou verzi sady Visual Studio, nebo odeberte možnost příkazového řádku.
