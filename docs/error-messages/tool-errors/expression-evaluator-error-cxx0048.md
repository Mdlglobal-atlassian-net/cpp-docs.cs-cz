# <a name="expression-evaluator-error-cxx0048"></a>Chyba při vyhodnocování výrazu CXX0048

volací sekvence není podporována.

Funkce zadaná ve výrazu používá sekvenci volání, která není vyhodnocovacím filtrem výrazu C podporována. Tuto funkci nelze volat ve výrazu okno Kukátko.

Tato chyba je shodná s CAN0048.
