# <a name="linker-tools-warning-lnk4199"></a>Upozornění linkerů LNK4199

/DELAYLOAD: Název_souboru_DLL se ignoroval; z Název_souboru_DLL se nenašly žádné importy.

Linker ignoruje `dllname`, protože nepotřebuje žádnou z funkcí, které `dllname` export.
