# <a name="project-build-error-prj0019"></a>Chyba sestavení projektu PRJ0019

Nástroj vrátil kód chyby z

Úroveň chyby byla nenulová pro vlastní krok sestavení nebo událost sestavení.

PROJEKTU PRJ0019 se zobrazí také v případě, že nástroj vrátil kód chyby, ale není k dispozici žádná chybová zpráva. K tomu může dojít například při přesměrování výstupu MIDL na NUL.

Další informace najdete v tématu [řešení potíží s vlastními kroky sestavení a události sestavení](../../build/troubleshooting-build-customizations.md) .

K této chybě může dojít také v případě, že pracujete jako člen skupiny Users a je zapotřebí přístup pro správu. Další informace najdete v tématu [spuštění jako člen skupiny uživatelů](../../security/running-as-a-member-of-the-users-group.md).
