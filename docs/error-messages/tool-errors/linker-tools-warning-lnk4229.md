# <a name="linker-tools-warning-lnk4229"></a>Upozornění linkerů LNK4229

Neplatná direktiva /directive nalezen. Ignorovat

Direktivu předány prostřednictvím [komentář](../../preprocessor/comment-c-cpp.md) – Direktiva pragma nebyla platná. Propojovací program ignoruje `/directive`.
