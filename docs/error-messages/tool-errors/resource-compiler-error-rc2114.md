# <a name="resource-compiler-error-rc2114"></a>Chyba kompilátoru prostředků RC2114
Název třídy očekávané ovládacího prvku  
  
 `class` Pole příkazu ovládací PRVEK v **dialogové okno** příkaz musí mít jednu z následujících typů: tlačítko, COMBOBOX, upravit, LISTBOX, SCROLLBAR, statické, nebo definované uživatelem. Ujistěte se, že třída je napsán správně.
