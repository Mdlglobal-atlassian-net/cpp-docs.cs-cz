# <a name="expression-evaluator-error-cxx0051"></a>Chyba při vyhodnocování výrazu CXX0051

více než jeden přetížené symbolu určenému v zarážku

Chyba při vyhodnocování výrazu nelze zjistit, které více než jeden přetížené symbolu pro použití jako zarážku.

Tato chyba se shoduje s CAN0051.
