# <a name="expression-evaluator-error-cxx0051"></a>Chyba při vyhodnocování výrazu CXX0051

v zarážce byl zadán více než jeden přetížený symbol.

Vyhodnocovací filtr výrazů nemůže určit, který z více než jednoho přetíženého symbolu použít jako zarážku.

Tato chyba je shodná s CAN0051.
