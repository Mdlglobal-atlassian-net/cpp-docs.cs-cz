# <a name="expression-evaluator-error-cxx0051"></a>Chyba při vyhodnocování výrazu CXX0051
více než jeden přetížené symbol zadaný v zarážek  
  
 Vyhodnocovací filtr výrazů se nepodařilo určit, která z více než jeden přetížené symbol, který má používat jako zarážky.  
  
 Tato chyba je stejný jako CAN0051.
