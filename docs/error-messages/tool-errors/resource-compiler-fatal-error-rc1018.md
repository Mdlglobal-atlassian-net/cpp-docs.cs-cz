# <a name="resource-compiler-fatal-error-rc1018"></a>Závažná chyba kompilátoru prostředků RC1018

Neočekávaný #elif

Direktiva `#elif` se neobjevila v rámci `#if`, **#ifdef**nebo konstrukce **#ifndef** .

Ujistěte se, že je před tímto příkazem k dispozici `#if`, **#ifdef**nebo příkaz **#ifndef** .
