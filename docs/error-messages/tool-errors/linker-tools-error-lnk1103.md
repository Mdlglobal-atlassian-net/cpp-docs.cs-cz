# <a name="linker-tools-error-lnk1103"></a>Chyba linkerů LNK1103
informace o ladění poškozený; znovu zkompiluje modulu  
  
 K této chybě může dojít, protože kompilace byl ukončen před platný objekt soubor byl vytvořen.  
  
 Znovu zkompiluje soubor daného objektu. Pokud nutnosti rekompilace problém nevyřeší, Všimněte si v případech chyba, zkuste izolovat daný problém a vytvořit reprodukovatelnou testovacího případu. Pak najdete v souboru nápovědy technické podpory nebo části technické podpory v jednom z vaší příručky.
