# <a name="expression-evaluator-error-cxx0021"></a>Chyba při vyhodnocování výrazu CXX0021
Struktura nebo použít jako skalární sjednocení  
  
 Struktura nebo sjednocení byla použita ve výrazu, ale byl zadán žádný element.  
  
 Při manipulaci s struktura nebo union proměnnou, může se název proměnné samostatně, bez kvalifikátor pole. Pokud struktury nebo sjednocení se používá ve výrazu, musí být kvalifikovaný pomocí prvku specifické potřeby.  
  
 Zadejte element, jehož hodnota má být použita ve výrazu.  
  
 Tato chyba je stejný jako CAN0021.
