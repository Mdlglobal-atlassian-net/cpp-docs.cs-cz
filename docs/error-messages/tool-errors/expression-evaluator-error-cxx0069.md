# <a name="expression-evaluator-error-cxx0069"></a>Chyba při vyhodnocování výrazu CXX0069

Proměnná vyžaduje rámec zásobníku.

Vyhodnocovací filtr výrazů nemůže vyhodnotit proměnnou, protože k ní nedochází v rámci bloku zásobníku. To může být způsobeno proměnnými deklarovanými jako součást vložené funkce.
