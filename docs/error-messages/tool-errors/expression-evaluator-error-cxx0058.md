# <a name="expression-evaluator-error-cxx0058"></a>Chyba při vyhodnocování výrazu CXX0058

přetížený operátor nebyl nalezen.

Typ třídy byl zadán jako levý operand ve výrazu, ale pro třídu byl definován přetížený operátor.

Tato chyba je shodná s CAN0058.
