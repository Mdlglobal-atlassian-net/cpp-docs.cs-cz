# <a name="expression-evaluator-error-cxx0058"></a>Chyba při vyhodnocování výrazu CXX0058
přetížené operátor nebyl nalezen  
  
 Typ třídy byl zadán jako levý operand ve výrazu, ale nebyla pro třídu definovaný přetížené operátor.  
  
 Tato chyba je stejný jako CAN0058.
