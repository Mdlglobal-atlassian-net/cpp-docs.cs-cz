# <a name="expression-evaluator-error-cxx0058"></a>Chyba při vyhodnocování výrazu CXX0058

přetížený operátor nebyl nalezen

Byl zadán typ třídy jako levý operand ve výrazu, ale nebyl definován přetíženého operátoru pro třídu.

Tato chyba se shoduje s CAN0058.
