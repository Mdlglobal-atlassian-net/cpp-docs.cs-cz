# <a name="linker-tools-error-lnk1240"></a>Chyba linkerů LNK1240

Nepovedlo se zkompilovat obsah IDL.

Linker vytvořil MIDL pro zkompilování vloženého IDL, ale došlo k problému. Zkontroluje chyby zadané v MIDL.
