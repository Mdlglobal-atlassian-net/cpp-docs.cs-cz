# <a name="linker-tools-error-lnk1240"></a>Chyba linkerů LNK1240
IDL obsah kompilace se nezdařila.  
  
 Linkeru vytvořený MIDL zkompilovat embedded IDL, ale došlo k potížím. Podívejte se na chyby určeného MIDL.
