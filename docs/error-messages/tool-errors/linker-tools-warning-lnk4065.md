# <a name="linker-tools-warning-lnk4065"></a>Upozornění linkerů LNK4065

'function' nelze seřadit; Ignorovat

Danou funkci nebyl zkompilován jako zabalené funkce. Znovu zkompilovat pomocí [/Gy](../../build/reference/gy-enable-function-level-linking.md).
