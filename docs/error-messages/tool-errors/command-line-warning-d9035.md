# <a name="command-line-warning-d9035"></a>Upozornění příkazového řádku D9035

"možnost" je zastaralá a bude v budoucí verzi odebrána

Jste určili možnost kompilátoru, která bude v budoucí verzi kompilátoru jazyka Visual C++ odebrána. Toto upozornění je následována upozornění [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) při navrhované náhrada za *možnost*.

Zadaná možnost stále funguje, ale měli byste aktualizovat konfiguraci sestavení. V důsledku toho projekt je větší pravděpodobnost pokračovat v sestavení, když upgradujete kompilátor.
