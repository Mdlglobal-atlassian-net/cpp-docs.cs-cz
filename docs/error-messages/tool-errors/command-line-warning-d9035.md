# <a name="command-line-warning-d9035"></a>Upozornění příkazového řádku D9035
možnost "možnost" se už nepoužívá a bude v budoucí verzi odebrána  
  
 Zadali jste možnost kompilátoru, která bude v budoucí verzi Visual C++ compiler odebrána. Toto upozornění je následován upozornění [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) Pokud je navržené náhradou za *možnost*.  
  
 Zadaná možnost pořád funguje, ale by měl aktualizovat konfiguraci sestavení. V důsledku toho je pravděpodobnější nadále sestavení při upgradu kompilátor projektu.
