# <a name="expression-evaluator-error-cxx0038"></a>Chyba při vyhodnocování výrazu CXX0038

počet argumentů funkce nebo Neshoda typu

Zadané volání funkce se neshoduje s prototypem funkce.

Znovu zadejte volání se správným počtem argumentů. V případě potřeby zadejte přetypování každého argumentu tak, aby odpovídal prototypu.

Tato chyba je shodná s CAN0038.
