# <a name="expression-evaluator-error-cxx0028"></a>Chyba při vyhodnocování výrazu CXX0028

Nejedná se o prvek struct/Union.

Byl zadán výraz ve formátu "struct. Member" nebo "pStruct-> member", ale \<člen > není element struktury.

Výraz nesmí být správně v závorkách.

Tato chyba je shodná s CAN0028.
