# <a name="resource-compiler-error-rc2167"></a>Chyba kompilátoru prostředků RC2167
neznámé pole VERSIONINFO; POČÁTEČNÍ nebo byla očekávána čárka  
  
 Neznámé pole byl nalezen v **PEVNÉ** součástí **VERSIONINFO** struktury deklarace. A **VERSIONINFO** pole musí být hodnota DWORD oddělených čárkou.
