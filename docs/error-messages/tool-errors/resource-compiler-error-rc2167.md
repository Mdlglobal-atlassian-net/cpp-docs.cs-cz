# <a name="resource-compiler-error-rc2167"></a>Chyba kompilátoru prostředků RC2167

neznámé pole VERSIONINFO; BEGIN nebo byla očekávána čárka

Byla nalezena nerozpoznaná pole v **FIXED** součástí **VERSIONINFO** deklaraci struktury. A **VERSIONINFO** pole musí být hodnota DWORD oddělené čárkou.
