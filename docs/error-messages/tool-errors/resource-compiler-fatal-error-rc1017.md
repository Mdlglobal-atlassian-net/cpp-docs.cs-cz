# <a name="resource-compiler-fatal-error-rc1017"></a>Závažná chyba kompilátoru prostředků RC1017

Neplatný celočíselný konstantní výraz

Výraz v `#if` direktiva neexistuje nebo není vyhodnocen na konstantu.
