# <a name="expression-evaluator-error-cxx0054"></a>Chyba při vyhodnocování výrazu CXX0054

Registrace do výrazu zarážky není platná.

Ve výrazu zarážky nelze použít registr.

Tato chyba je shodná s CAN0054.
