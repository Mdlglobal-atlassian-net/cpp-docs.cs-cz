# <a name="resource-compiler-fatal-error-rc1116"></a>Závažná chyba kompilátoru prostředků RC1116

RC – ukončení po chybách preprocesoru

Kompilátor prostředků se zastavil kvůli jiným chybám. Opravte ostatní chyby a zkompilujte je znovu.
