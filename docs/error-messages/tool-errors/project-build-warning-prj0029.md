# <a name="project-build-warning-prj0029"></a>Upozornění sestavení projektu PRJ0029
Vlastnost 'Výstupy' pro úrovni projektu vlastní krok sestavení není nastavená. Vlastní krok sestavení bude přeskočen.  
  
 Vlastní krok sestavení nebyla spuštěna, protože nebyl zadán žádný výstup.  
  
 Pokud chcete tuto chybu vyřešit, proveďte následující:  
  
-   Vyloučení vlastního kroku sestavení ze sestavení.  
  
-   Přidáte výstup.  
  
-   Odstraňte obsah příkaz krok vlastní sestavení.
