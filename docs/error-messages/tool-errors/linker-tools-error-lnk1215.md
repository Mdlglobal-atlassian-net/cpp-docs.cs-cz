# <a name="linker-tools-error-lnk1215"></a>Chyba linkerů LNK1215
metadata operace se nezdařila (HRESULT): Chyba  
  
 Linkeru obdržel chybu z modulu runtime .NET při pokusu provést aktualizaci metadat prostřednictvím modulu runtime rozhraní .NET.  
  
 `HRESULT` je HRESULT z metody modul runtime rozhraní .NET. `error` je. NET zadaný text.  
  
 Pravděpodobně máte neodpovídající linkeru a modulu runtime .NET; znovu nainstalujte Visual C++.
