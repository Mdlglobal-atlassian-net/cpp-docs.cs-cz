# <a name="linker-tools-error-lnk1169"></a>Chyba linkerů LNK1169
jeden nebo více násobení definované symboly nalezen  
  
 Sestavení se nezdařilo z důvodu několik definic jeden nebo více symbolů. Tato chyba je před chyba [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md).  
  
 [/FORCE](../../build/reference/force-force-file-output.md) nebo **/FORCE:MULTIPLE** možnost přepsání této chybě.
