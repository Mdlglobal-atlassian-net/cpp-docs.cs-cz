# <a name="resource-compiler-error-rc2165"></a>Chyba kompilátoru prostředků RC2165

v příkazu DLGINCLUDE se nenašel řetězec.

Příkaz neurčil platný soubor include.

Příkaz **DLGINCLUDE** musí používat následující syntaxi:

**DLGINCLUDE "** <em>filename</em> **. h"**
