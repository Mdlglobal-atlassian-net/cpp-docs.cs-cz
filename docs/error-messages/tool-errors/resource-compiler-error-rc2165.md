# <a name="resource-compiler-error-rc2165"></a>Chyba kompilátoru prostředků RC2165
nebyl nalezen v příkazu DLGINCLUDE řetězec  
  
 Příkaz nezadala platnou vloženého souboru.  
  
 **DLGINCLUDE** příkaz musíte použít následující syntaxi:  
  
 **DLGINCLUDE "*filename*.h"**
