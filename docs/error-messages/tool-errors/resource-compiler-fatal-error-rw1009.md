# <a name="resource-compiler-fatal-error-rw1009"></a>Závažná chyba kompilátoru prostředků RW1009
Chyba při vytváření název prostředku  
  
 Nepodařilo se vytvořit .res zadaný soubor. Ujistěte se, že není vytváří na disku jen pro čtení. Použijte /V a zjistěte, zda je soubor vytvořen.
