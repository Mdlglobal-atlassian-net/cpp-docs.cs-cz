# <a name="command-line-warning-d9036"></a>Upozornění příkazového řádku D9036

> "*option\_2*" místo "*Option\_1*"

## <a name="remarks"></a>Poznámky

Upozornění D9036 je pokračování [příkazového řádku upozornění D9035](../../error-messages/tool-errors/command-line-warning-d9035.md). Pokud je upřednostňována jedna možnost kompilátoru namísto zastaralé možnosti, je uvedena zde. Odkazy na Další informace naleznete v tématu [zastaralé a odebrané možnosti kompilátoru](../../build/reference/compiler-options-listed-by-category.md#deprecated-and-removed-compiler-options).
