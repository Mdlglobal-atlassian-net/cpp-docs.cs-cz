# <a name="command-line-warning-d9036"></a>Upozornění příkazového řádku D9036

> "*možnost\_2*"namísto z"*možnost\_1*.

## <a name="remarks"></a>Poznámky

Upozornění D9036 je pokračováním [příkazového řádku D9035 upozornění](../../error-messages/tool-errors/command-line-warning-d9035.md). Pokud jeden – možnost kompilátoru je upřednostňována namísto zastaralé možnost, je uvedený tady. Odkazy na další informace naleznete v části [zastaralé funkce a možnosti kompilátoru odebrané](../../build/reference/compiler-options-listed-by-category.md#deprecated-and-removed-compiler-options).
