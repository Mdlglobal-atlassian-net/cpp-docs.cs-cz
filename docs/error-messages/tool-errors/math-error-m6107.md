# <a name="math-error-m6107"></a>Chyba matematické operace M6107
unemulated  
  
 Byl proveden pokus o spuštění koprocesor instrukce, který je neplatný nebo není podporovaný emulátorem.  
  
 Program se ukončí s ukončovacím kódem 135.
