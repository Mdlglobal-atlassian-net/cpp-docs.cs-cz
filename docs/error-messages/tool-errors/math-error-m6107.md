# <a name="math-error-m6107"></a>Chyba matematické operace M6107

unemulated

Došlo k pokusu o spuštění instrukce koprocesoru, která je neplatný nebo není podporována v emulátoru.

Program se ukončí s ukončovacím kódem 135.
