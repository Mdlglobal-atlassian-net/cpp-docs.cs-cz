# <a name="linker-tools-warning-lnk4039"></a>Upozornění linkerů LNK4039
oddíl s možností /SECTION zadaný název neexistuje.  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md) nebo [nástroje EDITBIN](../../build/reference/editbin-reference.md) nelze najít oddíl s názvem `name` ve vstupním souboru.  
  
 Chcete-li najdete v částech v souboru, spusťte DUMPBIN s [/HEADERS](../../build/reference/headers.md) možnost.
