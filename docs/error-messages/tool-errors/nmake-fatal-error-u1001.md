# <a name="nmake-fatal-error-u1001"></a>Závažná chyba nástroje NMAKE U1001
Chyba syntaxe: neplatný znak 'znak' – makro  
  
 Daného znaku se zobrazí v makru, ale není písmenem, číslicí nebo podtržítkem.  
  
 Tato chyba může být způsobeno chybí dvojtečka v rozšíření makra:  
  
```  
syntax error : illegal character '=' in macro  
```
