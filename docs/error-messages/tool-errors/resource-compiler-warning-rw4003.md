# <a name="resource-compiler-warning-rw4003"></a>Upozornění kompilátoru prostředků RW4003

Klávesa SHIFT nebo CONTROL použitá bez VIRTKEY

V prostředku tabulky akcelerátorů vyžaduje aplikace SHIFT nebo CONTROL VIRTKEY. Vzhledem k tomu, že klávesy SHIFT a CONTROL jsou označeny jako bity příznaku v akcelerátoru typu VIRTKEY, nemohou existovat nezávisle na VIRTKEY.
