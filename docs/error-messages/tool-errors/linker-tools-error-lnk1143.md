# <a name="linker-tools-error-lnk1143"></a>Chyba linkerů LNK1143

soubor je neplatný nebo poškozený: není k dispozici žádný symbol pro číslo oddílu COMDAT.

Tato chyba může být způsobena poškozením souboru objektu.

Znovu sestavte soubor.
