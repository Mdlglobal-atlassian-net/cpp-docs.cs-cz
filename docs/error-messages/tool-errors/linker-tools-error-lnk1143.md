# <a name="linker-tools-error-lnk1143"></a>Chyba linkerů LNK1143
Neplatný nebo poškozený soubor: žádné symbol pro číslo sekvence COMDAT oddílu  
  
 Tato chyba může být způsobena Pokud objekt soubor je poškozený.  
  
 Vytvoření nového souboru.
