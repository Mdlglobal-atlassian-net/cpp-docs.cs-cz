# <a name="linker-tools-error-lnk1143"></a>Chyba linkerů LNK1143

soubor je neplatný nebo poškozený: žádný symbol pro číslo oddílu COMDAT.

Tato chyba může nastat, pokud objekt soubor je poškozený.

Znovu sestavte soubor.
