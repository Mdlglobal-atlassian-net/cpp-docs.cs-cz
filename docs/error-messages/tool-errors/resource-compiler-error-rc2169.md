# <a name="resource-compiler-error-rc2169"></a>Chyba kompilátoru prostředků RC2169
Název souboru souboru prostředků není ve formátu 2.03  
  
 Zadaný prostředek použít starší než verze 2.03 formátu. Soubor prostředků převeďte nebo znovu vytvoří pomocí formátu pro verze 3,00 nebo novější.
