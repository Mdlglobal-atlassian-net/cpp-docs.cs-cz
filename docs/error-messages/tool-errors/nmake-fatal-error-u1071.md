# <a name="nmake-fatal-error-u1071"></a>Závažná chyba nástroje NMAKE U1071

cyklus ve stromu závislostí pro cíl TargetName

Ve stromové struktuře závislosti pro daný cíl existuje cyklická závislost. Daný cíl je závislý na jednom z závislých prvků daného cíle. Cyklické závislosti jsou neplatné.
