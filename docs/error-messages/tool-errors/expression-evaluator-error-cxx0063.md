# <a name="expression-evaluator-error-cxx0063"></a>Chyba při vyhodnocování výrazu CXX0063

přetíženého operátoru -> není podporována

Výraz použit operátor přístup člena přetížené třídy (**->**).

Tato chyba se shoduje s CAN0063.
