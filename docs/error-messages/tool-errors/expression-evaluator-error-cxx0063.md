# <a name="expression-evaluator-error-cxx0063"></a>Chyba při vyhodnocování výrazu CXX0063
přetížené operátor -> není podporována  
  
 Výraz používá operátor přístupu členů přetížené – třída (**->**).  
  
 Tato chyba je stejný jako CAN0063.
