# <a name="project-build-error-prj0028"></a>Chyba sestavení projektu PRJ0028
Dočasný soubor 'file' obsahuje obsah kódování Unicode, které nebylo možné přeložit na uživatele ANSI znakovou stránku.  
  
 Byla zadána hodnota s [/MIDL (zadejte možnosti příkazového řádku MIDL)](../../build/reference/midl-specify-midl-command-line-options.md) – možnost linkeru který nebylo možné přeložit podle znakové stránky systému.  
  
 Znaková stránka používá při zadání příkazu MIDL (vstupní kódové stránky) musí být stejný jako systémová znaková stránka.
