# <a name="resource-compiler-fatal-error-rc1105"></a>Závažná chyba kompilátoru prostředků RC1105

Neplatný přepínač, možnost: moc velký počet přepínačů/d

Pomocí možnosti definovat symboly (/d) bylo definováno příliš mnoho symbolů. Definujte některé symboly ve zdroji a znovu zkompilujte.
