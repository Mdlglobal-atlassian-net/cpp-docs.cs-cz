# <a name="resource-compiler-fatal-error-rc1022"></a>Závažná chyba kompilátoru prostředků RC1022
Očekávaný '#endif.  
  
 `#if`, **#Ifdef**, nebo **#ifndef** – direktiva nebyl byla ukončena s `#endif` – direktiva.  
  
 Ujistěte se, že je `#if`, **#ifdef**, nebo **#ifndef** v platnosti než tohoto prohlášení.
