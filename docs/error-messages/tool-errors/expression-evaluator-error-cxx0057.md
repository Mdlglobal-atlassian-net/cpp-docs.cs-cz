# <a name="expression-evaluator-error-cxx0057"></a>Chyba při vyhodnocování výrazu CXX0057

žádný kód na řádku číslo

Nevygeneroval se žádný kód pro zadaný počet řádků. Nelze použít jako zarážku.

Tato chyba se shoduje s CAN0057.
