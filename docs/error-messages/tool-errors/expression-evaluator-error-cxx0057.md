# <a name="expression-evaluator-error-cxx0057"></a>Chyba při vyhodnocování výrazu CXX0057

žádný kód na řádku číslo

Pro zadané číslo řádku nebyl vygenerován žádný kód. Nelze jej použít jako zarážku.

Tato chyba je shodná s CAN0057.
