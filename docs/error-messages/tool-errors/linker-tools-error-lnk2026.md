# <a name="linker-tools-error-lnk2026"></a>Chyba linkerů LNK2026

modul není pro bitovou kopii SAFESEH

[/ SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) byla zadána, ale modul nebyl kompatibilní s funkcí bezpečného zpracování výjimek. Pokud chcete použít tento modul se **/SAFESEH**, pak je potřeba znovu zkompilovat modulu.
