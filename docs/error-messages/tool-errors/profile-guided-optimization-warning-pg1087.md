# <a name="profile-guided-optimization-warning-pg1087"></a>Upozornění optimalizace na základě profilu PG1087

Soubor PGC souboru je neplatný.

Soubor. pgc je poškozený, pravděpodobně z důvodu abnormálního konce relace profilování.  Spusťte znovu scénář profilace a normálně ho ukončete.
