# <a name="profile-guided-optimization-warning-pg1087"></a>Upozornění optimalizace na základě profilu PG1087

Soubor PGC 'file' není platný.

Soubor .pgc je poškozen, pravděpodobně z důvodu abnormální ukončení relace profilování.  Znovu spustit scénář profilace a jej měla ukončit normálně.
