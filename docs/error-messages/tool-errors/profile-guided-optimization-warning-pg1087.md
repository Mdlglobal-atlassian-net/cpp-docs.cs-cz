# <a name="profile-guided-optimization-warning-pg1087"></a>Upozornění optimalizace na základě profilu PG1087
Soubor PGC 'file' není platný.  
  
 .Pgc soubor je poškozen, pravděpodobně z důvodu neobvyklé mohli relace profilování.  Znovu spustit profilování scénář a ukončete normálně.
