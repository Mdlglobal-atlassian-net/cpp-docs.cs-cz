# <a name="expression-evaluator-error-cxx0040"></a>Chyba při vyhodnocování výrazu CXX0040

funkce vyžaduje implicitní převod.

Vyhodnocovací filtr výrazů jazyka C nepodporuje implicitní převody zahrnující volání konstruktoru.

Tato chyba je shodná s CAN0040.
