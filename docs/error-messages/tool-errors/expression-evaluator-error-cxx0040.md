# <a name="expression-evaluator-error-cxx0040"></a>Chyba při vyhodnocování výrazu CXX0040

implicitní převod vyžaduje – funkce

Chyba při vyhodnocování výrazů C nepodporuje implicitní převody zahrnující volá konstruktor.

Tato chyba se shoduje s CAN0040.
