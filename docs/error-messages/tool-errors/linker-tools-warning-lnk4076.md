# <a name="linker-tools-warning-lnk4076"></a>Upozornění linkerů LNK4076
Neplatný přírůstkové stavový soubor 'název souboru'; nonincrementally propojení  
  
 Odkaz nemůže zapisovat do souboru stavu přírůstkové (.ilk). Buď `filename` je poškozený nebo se nejedná o databázi serveru linking přírůstkové. Odeberte soubor a znovu připojit.
