# <a name="bscmake-error-bk1517"></a>Chyba nástroje BSCMAKE BK1517

zdrojový soubor pro sbrfile kompilovaný s/Yc a/Yu

Soubor. sbr odkazuje sám na sebe. Po kompilaci s/Yc. se pravděpodobně znovu zkompiluje pomocí/Yu. Obnovte možnost kompilátoru pro zdrojový soubor na/Yc a pak vyberte znovu **sestavit** pro generování nových souborů. sbr. Nekompilovat zdrojový soubor pomocí/Yu.
