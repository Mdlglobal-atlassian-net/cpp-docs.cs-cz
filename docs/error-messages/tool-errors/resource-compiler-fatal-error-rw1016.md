# <a name="resource-compiler-fatal-error-rw1016"></a>Závažná chyba kompilátoru prostředků RW1016

Ukončuje se po chyb předprocesoru RC

Nástroj Resource Compiler se zastavilo kvůli jiné chyby. Odstraňte ostatní chyby a znovu zkompilovat.
