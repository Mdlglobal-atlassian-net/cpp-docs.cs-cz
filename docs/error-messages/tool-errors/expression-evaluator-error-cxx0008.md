# <a name="expression-evaluator-error-cxx0008"></a>Chyba při vyhodnocování výrazu CXX0008

**Chybí "na konci řetězce**

Dvojité uvozovky očekává na konci řetězce literálu nebyl nalezen. Znovu zadejte výraz uzavření řetězcový literál v uvozovkách.

Tato chyba se shoduje s CAN0008.
