# <a name="expression-evaluator-error-cxx0008"></a>Chyba při vyhodnocování výrazu CXX0008
**Chybí "na konci řetězce**  
  
 Dvojité uvozovky, měl být na konci řetězce literálu nebyl nalezen. Znovu zadejte výraz obklopuje řetězcový literál v uvozovkách.  
  
 Tato chyba je stejný jako CAN0008.
