# <a name="nmake-fatal-error-u1097"></a>Závažná chyba nástroje NMAKE U1097
Syntaxe částí názvu souboru vyžaduje závislé  
  
 Aktuální závislost nemá explicitní závislé nebo implicitní závislé. Syntaxe částí názvu souboru, který používá znakem procent (`%`) specifikátor, představuje součásti z první závislých aktuální cíle.
