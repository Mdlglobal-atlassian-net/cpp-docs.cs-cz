# <a name="resource-compiler-error-rc2175"></a>Chyba kompilátoru prostředků RC2175
Název souboru souboru prostředků není ve formátu 3,00  
  
 Zadaný prostředek použít starší než verze 3.00 formátu. Soubor prostředků převeďte nebo znovu vytvoří pomocí formátu pro verze 3,00 nebo novější.
