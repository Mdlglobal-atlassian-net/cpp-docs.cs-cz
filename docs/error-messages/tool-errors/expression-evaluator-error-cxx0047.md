# <a name="expression-evaluator-error-cxx0047"></a>Chyba při vyhodnocování výrazu CXX0047
Seznam argumentů neodpovídá funkce  
  
 Výraz volal funkci se seznamu skutečný parametr, který neodpovídá seznamu formální parametr všechny funkce se stejným názvem definované v programu.  
  
 Přetížené funkce lze volat pouze v případě, že dojde přesný parametr nebo shodu, která nevyžaduje vytváření objektu.  
  
 Tato chyba je stejný jako CAN0047.
