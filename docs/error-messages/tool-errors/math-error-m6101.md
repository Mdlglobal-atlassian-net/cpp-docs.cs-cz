# <a name="math-error-m6101"></a>Chyba matematické operace M6101

neplatné

Neplatná operace.

Tato chyba může být způsobena tím, že je operand NaN (nejedná se o číslo) nebo nekonečno.

Program se ukončí s ukončovacím kódem 129.
