# <a name="math-error-m6101"></a>Chyba matematické operace M6101

neplatné

Neplatná operace.

Tato chyba může nastat, pokud operand je typu NaN (není číslo) nebo nekonečno.

Program se ukončí s ukončovacím kódem 129.
