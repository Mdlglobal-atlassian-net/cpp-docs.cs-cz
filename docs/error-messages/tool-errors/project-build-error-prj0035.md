# <a name="project-build-error-prj0035"></a>Chyba sestavení projektu PRJ0035
Soubor XML 'file' obsahuje obsah kódování Unicode, které nebylo možné přeložit na uživatele ANSI znakovou stránku.  
  
 ***UNICODE obsah souboru***  
  
 ***soubor*** je soubor XML, který je vytvořen jako příkazový řádek pro nástroj nasazení webu.  
  
 Systém projektu nalezen znaky kódování Unicode v některé vlastnosti na stránce vlastností nasazení webu, který nelze správně přeložit na ANSI.  
  
 Řešením této chyby je aktualizovat obsah vlastnost použít ANSI nebo na počítač nainstalujte znakovou stránku a nastavte jej jako výchozí systémové nastavení.
