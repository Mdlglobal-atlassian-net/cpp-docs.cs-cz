# <a name="resource-compiler-error-rc2015"></a>Chyba kompilátoru prostředků RC2015

příliš mnoho znaků v konstantě

Znaková konstanta obsahuje více než dva znaky.

Konstanty znaků jsou omezené na jeden znak (standardní znakové konstanty) nebo dva znaky (long znakové konstanty).

Všimněte si, že řídicí sekvence (například složka \t pro kartu) je převedena na jeden znak.
