# <a name="resource-compiler-error-rc2015"></a>Chyba kompilátoru prostředků RC2015

příliš mnoho znaků v konstantě

Znaková konstanta obsahovala více než dva znaky.

Konstanty znaků jsou omezené na jeden znak (standardní znakové konstanty) nebo dva znaky (dlouhé znakové konstanty).

Všimněte si, že řídicí sekvence (například \t pro tabulátor) je převedena na jeden znak.
