# <a name="resource-compiler-error-rc2015"></a>Chyba kompilátoru prostředků RC2015
příliš mnoho znaků v konstanta  
  
 Konstanta znaků obsahuje více než dva znaky.  
  
 Konstanty znaků je omezen na jeden znak (konstanty standardní znaků) nebo dva znaky (konstanty dlouho znaků).  
  
 Všimněte si, že v řídicí sekvenci (například \t karta) je převést na jednoho znaku.
