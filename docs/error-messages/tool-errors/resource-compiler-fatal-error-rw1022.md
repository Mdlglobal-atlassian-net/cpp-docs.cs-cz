# <a name="resource-compiler-fatal-error-rw1022"></a>Závažná chyba kompilátoru prostředků RW1022

**Vstup/výstup – chyba zápisu do souboru**

Kompilátor prostředků nemohl zapisovat do souboru.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Opravu provedete kontrolou následujících možných příčin.

1. Nedostatek místa na disku. Volné místo musí být alespoň dvojnásobně velikost spustitelného souboru, který vytváříte.

1. Svazek je jen pro čtení.

1. Chybný sektor.

1. Narušení sdílení.
