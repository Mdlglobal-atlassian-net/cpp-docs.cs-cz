# <a name="resource-compiler-fatal-error-rc1020"></a>Závažná chyba kompilátoru prostředků RC1020

Neočekávaný #endif

Direktiva `#endif` se zobrazila bez odpovídajícího `#if`, **#ifdef**nebo direktivy **#ifndef** .

Ujistěte se, že existuje odpovídající `#endif` pro každý `#if`, **#ifdef**a příkaz **#ifndef** .
