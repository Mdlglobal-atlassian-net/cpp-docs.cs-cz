# <a name="math-error-m6110"></a>Chyba matematické operace M6110

přetečení zásobníku

Výraz s plovoucí desetinnou čárkou způsobil přetečení zásobníku s plovoucí desetinnou čárkou.

Výjimky s plovoucí desetinnou čárkou přetečení jsou zachyceny až do limitu sedmi úrovní, který je obvykle podporován koprocesorem 8087/287/387.

Program se ukončí s ukončovacím kódem 138.
