# <a name="linker-tools-error-lnk1248"></a>Chyba linkerů LNK1248

> velikost obrázku ('*Size*') překračuje maximální povolenou velikost (80000000)

Linker zjistil, že velikost výstupního souboru bude přesáhnout největší možnou velikost pro bitovou kopii 32 bitového programu. Můžete chtít vytvořit program do více knihoven DLL.
