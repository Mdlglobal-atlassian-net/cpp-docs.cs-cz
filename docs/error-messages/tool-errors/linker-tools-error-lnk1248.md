# <a name="linker-tools-error-lnk1248"></a>Chyba linkerů LNK1248

> velikost obrázku ('*velikost*') přesahuje maximální povolenou velikost (80000000)

Linkeru určit, že velikost výstupního souboru překročí maximální možné velikosti pro bitovou kopii 32bitový program. Můžete nastavit, aby váš program do více knihovny DLL.
