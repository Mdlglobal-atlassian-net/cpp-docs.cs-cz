# <a name="linker-tools-error-lnk1248"></a>Chyba linkerů LNK1248
velikost bitové kopie (velikost) přesahuje maximální povolenou velikost (80000000)  
  
 Linkeru určit, že velikost výstupního souboru překročí maximální možné velikosti pro image pořízenou prostřednictvím programu. Můžete nastavit, aby váš program do více knihovny DLL. Další informace o spustitelného souboru programu (PE), najdete v článku "Partnerský vztah uvnitř PE: A prohlídka z Win32 přenosné spustitelný formát souboru" v [knihovny MSDN](http://go.microsoft.com/fwlink/?linkid=556).
