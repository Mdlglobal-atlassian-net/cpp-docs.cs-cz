# <a name="linker-tools-error-lnk1248"></a>Chyba linkerů LNK1248

> velikost obrázku ("*velikost*") překračuje maximální povolenou velikost (80000000)

Propojovací program určit, že velikost výstupního souboru překročí maximální možné velikosti pro 32bitový program image. Můžete chtít vytvořit více knihoven DLL váš program.
