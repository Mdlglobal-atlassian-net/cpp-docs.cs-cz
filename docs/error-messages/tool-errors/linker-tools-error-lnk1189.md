# <a name="linker-tools-error-lnk1189"></a>Chyba linkerů LNK1189

LIBTOOMANYMEMBERS:: Library překročil limit počtu objektů.

Překročil se limit počtu 65535 objektů nebo členů v knihovně.

Chcete-li tento problém vyřešit, rozdělte knihovnu do dvou nebo více knihoven s menším počtem objektů nebo členů.
