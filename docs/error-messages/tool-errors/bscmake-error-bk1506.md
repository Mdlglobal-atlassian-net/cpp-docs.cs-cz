# <a name="bscmake-error-bk1506"></a>Chyba nástroje BSCMAKE BK1506
Nelze otevřít soubor filename [: Důvod]  
  
 BscMake – nelze otevřít soubor.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Chcete-li vyřešit kontrolou následující možné příčiny  
  
1.  Soubor uzamčen jiným procesem. Pokud `reason` uvádí **bylo odepřeno oprávnění**, soubor je pravděpodobně používán v prohlížeči. Zavřete okno Procházet a opakujte sestavení.  
  
2.  Celého disku.  
  
3.  Chyba hardwaru.  
  
4.  Zadaný výstupní soubor má stejný název jako existující podadresáři.
