# <a name="expression-evaluator-error-cxx0072"></a>Chyba při vyhodnocování výrazu CXX0072
Chyba: typ informace chybí nebo je neznámý  
  
 Souboru .pch získat propojený není v nebo kód obsahuje odkaz na typ, který je v modulu není kompilovat s /Zi.  
  
 Tato chyba je stejný jako CAN0067.
