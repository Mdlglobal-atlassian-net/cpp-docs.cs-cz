# <a name="project-build-error-prj0047"></a>Chyba sestavení projektu PRJ0047

Nepodařilo se obnovit pozastavený proces.  Sestavení se nezdařilo.

Ve vývojovém prostředí došlo k chybě.

Chcete-li tuto chybu vyřešit, zavřete a znovu načtěte projekt. V případě potřeby ukončete a restartujte Visual Studio.
