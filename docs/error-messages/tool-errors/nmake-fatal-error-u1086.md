# <a name="nmake-fatal-error-u1086"></a>Závažná chyba nástroje NMAKE U1086
**Odvozená pravidla nemůže mít závislé objekty**  
  
 Dvojtečkou (**:**) v odvození pravidlo musí být následováno jednu z těchto:  
  
-   Znak nového řádku  
  
-   Středník (**;**), který může následovat příkaz  
  
-   Počet přihlášení (**#**), který může následovat komentář
