# <a name="nmake-fatal-error-u1086"></a>Závažná chyba nástroje NMAKE U1086

**odvozené pravidlo nemůže mít závislé položky**

Dvojtečka (**:**) v odvození pravidlo musí být následováno jednu z těchto:

- Znak nového řádku

- Středník (**;**), který může být následován příkaz

- Znak (**#**), který může být následován komentář
