# <a name="nmake-fatal-error-u1086"></a>Závažná chyba nástroje NMAKE U1086

**odvozené pravidlo nemůže mít závislé položky.**

Dvojtečka ( **:** ) v pravidle odvození musí následovat jednou z následujících:

- Znak nového řádku

- Středník ( **;** ), který může následovat po příkazu

- Znak čísla ( **#** ), na který může následovat komentář
