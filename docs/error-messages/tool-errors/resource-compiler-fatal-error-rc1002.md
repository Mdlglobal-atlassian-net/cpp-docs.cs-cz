# <a name="resource-compiler-fatal-error-rc1002"></a>Závažná chyba kompilátoru prostředků RC1002

nedostatek prostoru v haldě

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Oprava pomocí následujících možných řešení

1. Zvětšete místo v souboru Windows swap-File. Další informace o zvětšení místa pro odkládací soubory najdete v tématu virtuální paměť v nápovědě k Windows.

1. Rozdělte aktuální soubor na menší soubory a zkompilujte je samostatně.

1. Odeberte ostatní programy nebo ovladače běžící v systému.
