# <a name="expression-evaluator-error-cxx0049"></a>Chyba při vyhodnocování výrazu CXX0049

zastaralé OMF – zkuste prosím znovu připojit program.

Program použil starý OMF (formát modulu objektů).

Znovu propojte program pomocí aktuální verze linkeru.

Tato chyba je shodná s CAN0049.
