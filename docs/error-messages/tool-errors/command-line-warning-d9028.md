# <a name="command-line-warning-d9028"></a>Upozornění příkazového řádku D9028
minimální opětovné sestavení selhání, návrat k normální sestavení  
  
 Soubor projektu IDB je poškozen. Odstraňte soubor a znovu vytvořit.  
  
 Další informace o minimální znovu sestaví a souboru IDB, najdete v článku [/Gm – možnost kompilátoru](../../build/reference/gm-enable-minimal-rebuild.md).
