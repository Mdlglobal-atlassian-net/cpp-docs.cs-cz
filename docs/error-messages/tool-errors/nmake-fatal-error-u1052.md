# <a name="nmake-fatal-error-u1052"></a>Závažná chyba nástroje NMAKE U1052

> soubor*filename*se nenašel.

NMAKE nemůže najít zadaný soubor s jedním z následujících způsobů:

- **/F** – možnost

- **! ZAHRNUTÍ** direktivy předzpracování

- Specifikátor na znaménko ( **\@** ) pro soubor odezvy

Ověřte, zda soubor existuje a zda je název souboru zadán správně.
