# <a name="nmake-fatal-error-u1052"></a>Závažná chyba nástroje NMAKE U1052

> Soubor '*filename*' nebyl nalezen

NMAKE nelze nalézt zadaný soubor s jedním z následujících akcí:

- **/F** možnost

- **! ZAHRNOUT** direktiva předzpracování

- Zavináč (**\@**) specifikátor pro soubor odpovědí

Zkontrolujte, zda soubor existuje a název souboru je napsán správně.
