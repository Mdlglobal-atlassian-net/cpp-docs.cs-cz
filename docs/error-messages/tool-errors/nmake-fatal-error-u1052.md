# <a name="nmake-fatal-error-u1052"></a>Závažná chyba nástroje NMAKE U1052
Soubor nebyl nalezen filename  
  
 NMAKE Nepodařilo se najít zadaný soubor s jedním z následujících akcí:  
  
-   **/F** možnost  
  
-   **! ZAHRNOUT** předběžného zpracování – direktiva  
  
-   Znaku zavináče (**@**) – specifikátor pro soubor odezvy  
  
 Zkontrolujte, zda soubor existuje a zda je název souboru je napsán správně.
