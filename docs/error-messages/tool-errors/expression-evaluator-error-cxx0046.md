# <a name="expression-evaluator-error-cxx0046"></a>Chyba při vyhodnocování výrazu CXX0046

seznam argumentů vyžadovaný pro členskou funkci

Výraz volal členskou funkci, ale neurčil žádné skutečné parametry.

Tato chyba je shodná s CAN0046.
