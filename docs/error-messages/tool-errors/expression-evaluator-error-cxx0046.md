# <a name="expression-evaluator-error-cxx0046"></a>Chyba při vyhodnocování výrazu CXX0046

Seznam argumentů, vyžaduje se pro členské funkce

Výraz volat členské funkce, ale nejsou specifikovány žádné skutečných parametrů.

Tato chyba se shoduje s CAN0046.
