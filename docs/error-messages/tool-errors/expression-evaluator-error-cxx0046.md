# <a name="expression-evaluator-error-cxx0046"></a>Chyba při vyhodnocování výrazu CXX0046
Seznam argumentů, které jsou potřebné pro členské funkce  
  
 Výraz volali členské funkce, ale nezadali žádné skutečné parametry.  
  
 Tato chyba je stejný jako CAN0046.
