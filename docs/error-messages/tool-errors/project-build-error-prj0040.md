# <a name="project-build-error-prj0040"></a>Chyba sestavení projektu PRJ0040

Vnitřní chyba při sestavování. Nelze pokračovat. Načtěte prosím znovu projekt a zkuste to znovu.

Systém projektu nemohl zpracovat váš projekt.

Zkuste znovu načíst projekt.
