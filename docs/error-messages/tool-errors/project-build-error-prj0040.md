# <a name="project-build-error-prj0040"></a>Chyba sestavení projektu PRJ0040

Vnitřní chyba při sestavování. Nelze pokračovat. Načtěte projekt znovu a zkuste to znovu.

Systém projektu se nepodařilo zpracovat váš projekt.

Zkuste znovu načíst projekt.
