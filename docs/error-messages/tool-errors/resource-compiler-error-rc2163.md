# <a name="resource-compiler-error-rc2163"></a>Chyba kompilátoru prostředků RC2163

Typ akcelerátoru vyžaduje [ASCII nebo VIRTKEY]

`type` Pole **AKCELERÁTORY** příkazu musí obsahovat hodnotu ASCII nebo VIRTKEY.
