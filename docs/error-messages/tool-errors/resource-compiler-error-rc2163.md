# <a name="resource-compiler-error-rc2163"></a>Chyba kompilátoru prostředků RC2163

vyžaduje se typ akcelerátoru [ASCII nebo VIRTKEY].

Pole `type` v příkazu **akcelerátory** musí obsahovat hodnotu ASCII nebo VIRTKEY.
