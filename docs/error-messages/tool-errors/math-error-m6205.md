# <a name="math-error-m6205"></a>Chyba matematické operace M6205
'function': _tloss – chyba  
  
 Celkový počet ztrátu násobek (přesnost) došlo k chybě.  
  
 Tato chyba může být způsobeno tím, že velmi velký počet jako operand sin, cos nebo tan, protože operand musí být omezeny na číslo v rozsahu od 0, 2 * pí.
