# <a name="resource-compiler-fatal-error-rc1021"></a>Závažná chyba kompilátoru prostředků RC1021

Neplatný příkaz preprocesoru "řetězec"

Znaky následující znak čísla (**#**) netvoří platné direktiva preprocesoru.

Znak čísla nelze použít jako první znak identifikátoru.
