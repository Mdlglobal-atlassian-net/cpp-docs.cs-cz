# <a name="resource-compiler-error-rc2111"></a>Chyba kompilátoru prostředků RC2111

Neplatný ovládací prvek typu

Každý příkaz ovládacího PRVKU v **dialogové okno** příkazu musí být jedna z následujících akcí: 3STATE, AUTO3, AUTOCHECK, AUTORADIO, BEDIT, ZAŠKRTÁVACÍ POLÍČKO, POLE SE SEZNAMEM, OVLÁDACÍ PRVEK, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, IKONY, IEDIT, LISTBOX, LTEXT, PUSHBOX, PUSHBUTTON, OVLÁDACÍHO PRVKU RADIOBUTTON, RTEXT, POSUVNÍK, USERBUTTON.

Ujistěte se, že se že tyto řídicí příkazy jsou zadány správně.
