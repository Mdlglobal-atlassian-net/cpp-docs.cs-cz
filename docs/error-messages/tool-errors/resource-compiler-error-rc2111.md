# <a name="resource-compiler-error-rc2111"></a>Chyba kompilátoru prostředků RC2111

Neplatný typ ovládacího prvku

Každý příkaz ovládacího prvku v příkazu **dialogového okna** musí být jeden z následujících: 3STATE, AUTO3, AutoCheck, Autoradio, BEDIT, CheckBox, ComboBox, Control, CTEXT, DEFPUSHBUTTON, EditText, skupinový, HEdit, Icon, IEDIT, ListBox, LTEXT, PUSHBOX, (pushbutton), RadioButton, RText, SCROLLBAR, UserButton.

Ujistěte se, že jsou tyto příkazy ovládacích prvků zadány správně.
