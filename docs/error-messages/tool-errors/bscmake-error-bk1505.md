# <a name="bscmake-error-bk1505"></a>Chyba nástroje BSCMAKE BK1505

Nelze číst z soubor 'filename' [: z důvodu]

BscMake – nelze načíst ze souboru.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Chcete-li vyřešit tak, že zkontrolujete následující možné příčiny

1. Došlo k poškození souboru.

1. Soubor nemá dostatek místa na disku nebo přerušení při vytvoření souboru .sbr kompilátoru zkrácení.
