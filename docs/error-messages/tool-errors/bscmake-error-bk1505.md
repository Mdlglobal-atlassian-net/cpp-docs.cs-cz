# <a name="bscmake-error-bk1505"></a>Chyba nástroje BSCMAKE BK1505
Nelze číst ze souboru 'název souboru' [: Důvod]  
  
 BscMake – nelze číst ze souboru.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Chcete-li vyřešit kontrolou následující možné příčiny  
  
1.  Došlo k poškození souboru.  
  
2.  Soubor zkrácení kompilátoru dostatek místa na disku nebo se při vytváření souboru .sbr přerušena.
