# <a name="resource-compiler-fatal-error-rc1121"></a>Závažná chyba kompilátoru prostředků RC1121

Vstupně-výstupní chyba čtení souboru

Kompilátor prostředků nedokázal přečíst soubor. Ověřte, zda je jednotka obsahující soubor k dispozici a zda je soubor platný.
