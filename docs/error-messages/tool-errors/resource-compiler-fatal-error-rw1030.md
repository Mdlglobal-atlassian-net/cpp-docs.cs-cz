# <a name="resource-compiler-fatal-error-rw1030"></a>Závažná chyba kompilátoru prostředků RW1030

Výstupní chyba

Tato chyba může být způsobena tím, že kompilátor prostředků nemohl vytvořit cílový soubor. Ujistěte se, že je na disku dostatek místa a že máte oprávnění k zápisu do tohoto svazku.
