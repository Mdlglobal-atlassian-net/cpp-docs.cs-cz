# <a name="resource-compiler-fatal-error-rw1030"></a>Závažná chyba kompilátoru prostředků RW1030

Chyba výstupu

Tato chyba může nastat, pokud kompilátor prostředků nebyl schopen vytvořit cílový soubor. Ujistěte se, že je dostatek místa na disku a zda máte oprávnění k zápisu na svazku.
