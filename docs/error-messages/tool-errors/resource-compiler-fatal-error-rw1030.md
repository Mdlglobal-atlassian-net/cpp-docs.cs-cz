# <a name="resource-compiler-fatal-error-rw1030"></a>Závažná chyba kompilátoru prostředků RW1030
Chyby výstupu  
  
 Tato chyba může být způsobena, pokud kompilátor prostředků se nepodařilo vytvořit cílový soubor. Ujistěte se, zda je dostatek místa na disku a zda máte oprávnění k zápisu na svazku.
