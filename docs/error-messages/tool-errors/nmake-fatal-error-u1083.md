# <a name="nmake-fatal-error-u1083"></a>Závažná chyba nástroje NMAKE U1083
Makro cíl "target" zasahuje do nic  
  
 Daný cíl je k vyvolání makra, která nebyla definována nebo má hodnotu null. NMAKE nemůže zpracovat cílovou hodnotu null.
