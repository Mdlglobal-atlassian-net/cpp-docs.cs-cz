# <a name="expression-evaluator-error-cxx0013"></a>Chyba při vyhodnocování výrazu CXX0013

chybějící operátor

Ve výrazu byl očekáván operátor, který však nebyl nalezen. Podívejte se na syntaxi výrazu.

Tato chyba je shodná s CAN0013.
