# <a name="expression-evaluator-error-cxx0013"></a>Chyba při vyhodnocování výrazu CXX0013
chybějící – operátor  
  
 Operátor byl očekáván výraz ale nebyl nalezen. Zkontrolujte syntaxi výrazu.  
  
 Tato chyba je stejný jako CAN0013.
