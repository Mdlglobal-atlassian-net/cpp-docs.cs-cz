# <a name="expression-evaluator-error-cxx0045"></a>Chyba při vyhodnocování výrazu CXX0045

Nejedná se o funkci

Byl zadán seznam argumentů pro symbol v programu, který není názvem funkce.

## <a name="example"></a>Příklad

```
queue( alpha, beta )
```

Když `queue` není funkce.

Tato chyba je shodná s CAN0045.
