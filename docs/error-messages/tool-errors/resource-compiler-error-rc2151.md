# <a name="resource-compiler-error-rc2151"></a>Chyba kompilátoru prostředků RC2151

Řetězcové konstanty nejde znovu použít.

V příkazu typu **String** se používá stejná hodnota dvakrát. Ujistěte se, že nemísíte překrývající se desítkové a hexadecimální hodnoty.

Každé ID v **řetězcích řetězců** musí být jedinečné. Pro maximální efektivitu používejte souvislé konstanty, které začínají na násobcích 16.
