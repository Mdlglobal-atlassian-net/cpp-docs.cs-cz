# <a name="resource-compiler-error-rc2151"></a>Chyba kompilátoru prostředků RC2151
nelze znovu použít řetězcové konstanty  
  
 Používáte stejnou hodnotu dvakrát v **STRINGTABLE** příkaz. Ujistěte se, že nejsou kombinování překrývání desetinné hodnoty, šestnáctkové hodnoty.  
  
 V každé ID **STRINGTABLE** musí být jedinečný. Pro maximální efektivity použití souvislý konstanty, spusťte v několika 16.
