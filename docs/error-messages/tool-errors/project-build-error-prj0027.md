# <a name="project-build-error-prj0027"></a>Chyba sestavení projektu PRJ0027

Obsah zprávy protokolu Unicode obsahuje obsah, který nebylo možné přeložit na znakovou stránku ANSI uživatele.

Toto upozornění se obvykle zobrazí ve spojení s chybami při vytváření souborů dávky nebo odpovědí.

Řešením této chyby je aktualizovat obsah protokolu sestavení tak, aby používal standard ANSI nebo nainstalovat znakovou stránku do počítače a nastavit jej jako výchozí systém.
