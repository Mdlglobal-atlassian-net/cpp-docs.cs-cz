# <a name="project-build-error-prj0023"></a>Chyba sestavení projektu PRJ0023
Nástroj 'nástroj' Neznámá vlastnost obsahuje neplatný název souboru 'file'.  
  
 Název souboru ***soubor***zadaný pro nástroj, ***nástroj***, byl neplatný.  
  
 Jste už použili makru neznámá nebo neplatná.  
  
 V tématu [nastavení vlastností projektu Visual C++](../../ide/working-with-project-properties.md) informace o tom, jak získat přístup k vlastnosti projektu.
