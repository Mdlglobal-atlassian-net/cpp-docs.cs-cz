# <a name="project-build-error-prj0023"></a>Chyba sestavení projektu PRJ0023

> Nástroj "*nástroj*", Neznámá vlastnost obsahuje neplatný název souboru"*souboru*".

Název souboru *souboru* zadaný pro nástroj *nástroj* byl neplatný.

Možná jste použili neznámému nebo neplatnému – makro.

Zobrazit [nastavení vlastností projektu Visual C++](../../ide/working-with-project-properties.md) informace o tom, jak přistupovat k vlastnosti projektu.
