# <a name="project-build-error-prj0015"></a>Chyba sestavení projektu PRJ0015
Váš systém chybí zařízení hodnotu NULL. Nepovedlo se nám spustíte sestavení.  
  
 Tato chyba může být způsobeno nedostatek systémových prostředků. Ukončete některé aplikace nebo restartovat.  
  
 Může být také způsoben, pokud nemáte dostatečná oprávnění pro zařízení NULL v počítači.
