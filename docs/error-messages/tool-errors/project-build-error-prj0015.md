# <a name="project-build-error-prj0015"></a>Chyba sestavení projektu PRJ0015

Ze systému chybí zařízení NULL. Nepovedlo se nám ke spuštění sestavení.

Tato chyba může být způsobeno nedostatek systémových prostředků. Ukončete některé aplikace nebo po restartování.

To může také tehdy, když na počítači nemáte dostatečná oprávnění k zařízení NULL.
