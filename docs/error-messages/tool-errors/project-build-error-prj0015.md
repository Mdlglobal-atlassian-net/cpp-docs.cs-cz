# <a name="project-build-error-prj0015"></a>Chyba sestavení projektu PRJ0015

V systému chybí zařízení s hodnotou NULL. Nepovedlo se nám spustit sestavení.

Tato chyba může být způsobena nedostatkem systémových prostředků. Ukončete některé aplikace nebo restartujte počítač.

Může to být také způsobeno tím, že nemáte dostatečná oprávnění k zařízení s hodnotou NULL v počítači.
