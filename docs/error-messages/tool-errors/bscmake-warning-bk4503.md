# <a name="bscmake-warning-bk4503"></a>Upozornění nástroje BSCMAKE BK4503
méně závažné chyby v. Název souboru souboru SBR ignorovat  
  
 Soubor .sbr obsahoval chybu, která není zastavení sestavení, ale bude výsledný soubor .bsc může být nesprávný. Chcete-li tento problém vyřešit, znovu zkompiluje vytvořit nové .sbr.
