# <a name="bscmake-warning-bk4503"></a>Upozornění nástroje BSCMAKE BK4503

méně důležitá Chyba v. Název souboru souboru SBR ignorovat

Soubor .sbr obsahovala chybu, která není zastavit sestavení, ale výsledný soubor .bsc může být nesprávný. Chcete-li vyřešit tento problém, znovu zkompilujte vytvořit nové .sbr.
