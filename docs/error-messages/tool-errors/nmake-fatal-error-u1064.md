# <a name="nmake-fatal-error-u1064"></a>Závažná chyba nástroje NMAKE U1064

Nenašel se soubor MAKEFILE a není zadaný žádný cíl.

Příkazový řádek NMAKE neurčil soubor pravidel nebo cíl a aktuální adresář neobsahoval soubor s názvem MAKEFILE.

NMAKE vyžaduje buď soubor pravidel nebo cíl příkazového řádku (nebo obojí). Chcete-li zpřístupnit soubor pravidel NMAKE, buď zadejte možnost/F, nebo umístěte soubor s názvem MAKEFILE do aktuálního adresáře. NMAKE může vytvořit cíl příkazového řádku pomocí odvozeného pravidla, pokud není zadaný soubor pravidel.
