# <a name="nmake-fatal-error-u1007"></a>Závažná chyba nástroje NMAKE U1007

v názvu není povolená dvojitá uvozovka.

Zadaný cílový název nebo název souboru obsahovaly dvojitou uvozovku ( **"** ).

Dvojité uvozovky mohou obklopit název souboru, ale nemohou být obsaženy v rámci něj.
