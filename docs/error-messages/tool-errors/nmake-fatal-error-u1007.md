# <a name="nmake-fatal-error-u1007"></a>Závažná chyba nástroje NMAKE U1007

dvojitá uvozovka není ve jméně povolená

Zadaný cílový název nebo název souboru obsaženy dvojité uvozovky (**"**).

Dvojité uvozovky ohraničíte název souboru, ale nemůže být jsou v něm obsaženy.
