# <a name="expression-evaluator-error-cxx0010"></a>Chyba při vyhodnocování výrazu CXX0010

chybějící levá hranatá závorka

Výraz obsahuje nevyvážené hranaté závorky. Znovu zadejte výraz s vyrovnanými hranatými závorkami.

Tato chyba je shodná s CAN0010.
