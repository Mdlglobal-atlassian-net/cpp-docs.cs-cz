# <a name="bscmake-error-bk1509"></a>Chyba nástroje BSCMAKE BK1509

nedostatek prostoru v haldě

BSCMAKE nemá dostatek paměti, včetně virtuální paměti.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Oprava pomocí následujících možných řešení

1. Uvolněte místo na disku.

1. Zvětšete velikost odkládacího souboru.

1. Zvětšete velikost stránkovacího souboru systému Windows.

1. Zmenšení BSCMAKE paměti vyžaduje použití/EI nebo/ES k vyloučení některých vstupních souborů nebo/em k vyloučení těla makra.
