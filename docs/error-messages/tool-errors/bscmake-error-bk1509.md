# <a name="bscmake-error-bk1509"></a>Chyba nástroje BSCMAKE BK1509

nedostatek prostoru v haldě

BscMake – není dostatek paměti, včetně virtuální paměti.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Chcete-li vyřešit pomocí následujících možná řešení

1. Uvolněte místo na disku.

1. Zvětšete velikost stránkovacího souboru.

1. Zvětšete velikost stránkovacího souboru Windows.

1. Snížit velikost paměti, které vyžaduje BSCMAKE pomocí /Ei nebo zadat /Es odstranit některé soubory nebo /Em k odstranění těla makra.
