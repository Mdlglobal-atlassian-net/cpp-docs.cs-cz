# <a name="cvtres-warning-cvt4001"></a>Upozornění CVTRES CVT4001

> není zadán; typ počítače předpokládá, že *typu*

CVTRES specifikace počítač nebyl nalezen. Předpokládá, že typ daného počítače. Pokud se výchozí hodnota je nesprávná, znovu spusťte CVTRES pomocí možnosti/Machine.
