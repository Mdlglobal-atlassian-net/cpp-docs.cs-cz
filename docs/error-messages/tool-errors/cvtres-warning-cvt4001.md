# <a name="cvtres-warning-cvt4001"></a>Upozornění CVTRES CVT4001
**typ počítače není určen; předpokládá, že**   
 ***Typ***  
  
 CVTRES nenalezl specifikaci počítače. Předpokládá, že typ daného počítače. Pokud výchozí hodnota je nesprávná, znovu spusťte CVTRES pomocí možnosti /MACHINE.
