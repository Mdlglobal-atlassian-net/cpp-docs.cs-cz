# <a name="cvtres-warning-cvt4001"></a>Upozornění CVTRES CVT4001

> typ počítače není zadaný. předpokládaný *typ*

CVTRES nenašel specifikace počítače. Byl předpokládána daný typ počítače. Pokud je výchozí hodnota nesprávná, znovu spusťte CVTRES pomocí možnosti/MACHINE.
