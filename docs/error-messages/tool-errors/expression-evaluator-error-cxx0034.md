# <a name="expression-evaluator-error-cxx0034"></a>Chyba při vyhodnocování výrazu CXX0034

**typy nekompatibilní s operátorem**

Zadané typy operandů nejsou pro operaci platné.

Například ukazatel nemůže být vynásoben žádnou hodnotou.

Možná budete muset napsat operandy přetypování na typ kompatibilní s operátorem.
