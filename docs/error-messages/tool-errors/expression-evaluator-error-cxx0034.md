# <a name="expression-evaluator-error-cxx0034"></a>Chyba při vyhodnocování výrazu CXX0034

**typy, které jsou kompatibilní s – operátor**

Typy operandů zadaný nejsou povoleny pro operaci.

Například nelze ukazatel vynásobené žádnou hodnotu.

Je třeba zadat operandy typu kompatibilní s operátorem přetypování.
