# <a name="expression-evaluator-error-cxx0034"></a>Chyba při vyhodnocování výrazu CXX0034
**typy, které jsou kompatibilní s – operátor**  
  
 Zadané typy operand nejsou právní pro operaci.  
  
 Například nelze ukazatel vynásobeny žádnou hodnotu.  
  
 Je třeba zadat přetypovat na typ, který je kompatibilní s operátorem operandy.
