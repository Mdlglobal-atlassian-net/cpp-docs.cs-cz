# <a name="project-build-error-prj0005"></a>Chyba sestavení projektu PRJ0005

Nelze vytvořit dočasný soubor v adresáři Directory.

Volání pro vytvoření dočasného souboru se nezdařilo. Důvody k selhání zahrnují:

- Došlo k nedostatku názvů dočasných souborů.

- Dočasný adresář je jen pro čtení.

- Neexistuje žádná dočasná složka ani proměnná prostředí TMP.

- V počítači není volné místo na disku.
