# <a name="project-build-error-prj0014"></a>Chyba sestavení projektu PRJ0014

Objekt úlohy použitý k řízení vytvářených procesů se nezdařil.  Sestavení nemůže pokračovat.

Ve vývojovém prostředí došlo k chybě.

Chcete-li tuto chybu vyřešit, zavřete a znovu načtěte projekt. V případě potřeby ukončete a restartujte Visual Studio.
