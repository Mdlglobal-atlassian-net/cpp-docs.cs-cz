# <a name="linker-tools-error-lnk1332"></a>Chyba linkerů LNK1332

zjistil se\<počet > typů prostředí Windows Runtime importovaných v jednom modulu a definovaný v jiném modulu.

Při vytvoření aktuálního cíle linker zjistil <`count`> prostředí Windows Runtime typy, z nichž každá je importována v jednom modulu a také definována v jiném modulu.

- Opravte všechny chyby LINKERŮ LNK2039 v sestavení podle návrhu v chybové zprávě.
