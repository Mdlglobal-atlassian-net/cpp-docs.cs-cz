# <a name="expression-evaluator-error-cxx0044"></a>Chyba při vyhodnocování výrazu CXX0044

Použití ukazatele _based (void) vyžaduje: > operator.

Ukazatel na základě `void` nelze použít přímo. Je nutné vytvořit kompletní ukazatel pomocí operátoru **: >** .

Tato chyba je shodná s CAN0044.
