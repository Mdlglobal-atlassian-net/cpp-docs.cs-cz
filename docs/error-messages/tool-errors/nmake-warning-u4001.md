# <a name="nmake-warning-u4001"></a>Upozornění nástroje NMAKE U4001
Nelze vyvolat příkazový soubor pouze z příkazového řádku  
  
 Soubor příkazů, který lze vyvolat pomocí znaku zavináče (**@**) specifikátor, nemůže obsahovat specifikaci pro jiný příkaz soubor. Takové vnoření není povoleno. Specifikace byl ignorován.
