# <a name="nmake-warning-u4001"></a>Upozornění nástroje NMAKE U4001

> soubor příkazů se dá vyvolat jenom z příkazového řádku.

Soubor příkazů, který je vyvolán specifikátorem na znaménko ( **\@** ), nemůže obsahovat specifikace pro jiný soubor příkazů. Tato vnořování není povolená. Specifikace byla ignorována.
