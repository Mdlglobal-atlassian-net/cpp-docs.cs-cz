# <a name="nmake-warning-u4001"></a>Upozornění nástroje NMAKE U4001

> soubor příkazů lze volat pouze z příkazového řádku

Soubor příkazů, které se vyvolá podle zavináč (**\@**) specifikátor, nemůže obsahovat specifikaci pro jiný soubor příkazů. Takové vnoření není povoleno. Specifikace byl ignorován.
