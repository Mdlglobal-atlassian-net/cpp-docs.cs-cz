# <a name="linker-tools-error-lnk1113"></a>Chyba linkerů LNK1113

Neplatný typ typu počítače

Typ počítače zadaný v záhlaví objektu není platný.

Tato chyba může být způsobena poškozením souboru. Znovu Sestavte objekt.
