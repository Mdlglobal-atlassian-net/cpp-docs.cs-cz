# <a name="expression-evaluator-error-cxx0004"></a>Chyba při vyhodnocování výrazu CXX0004

Chyba syntaxe

Syntaxe výrazu je nesprávná.

Znovu zadejte výraz se správnou syntaxí.

Tato chyba je shodná s CAN0004.
