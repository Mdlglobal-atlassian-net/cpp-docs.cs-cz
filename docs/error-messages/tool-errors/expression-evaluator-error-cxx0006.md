# <a name="expression-evaluator-error-cxx0006"></a>Chyba při vyhodnocování výrazu CXX0006

Chybí levá závorka.

Ve výrazu byly nalezeny nevyvážené závorky. Znovu zadejte výraz s vyrovnanými závorkami.

Tato chyba je shodná s CAN0006.
