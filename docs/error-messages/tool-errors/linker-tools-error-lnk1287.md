# <a name="linker-tools-error-lnk1287"></a>Chyba linkerů LNK1287

Neplatný spravovaný vstupní bod funkce

Vstupní bod není platný pro spravované image. Návratový typ funkce spravovaný vstupní bod lze pouze `void` nebo `int`. Typ parametru funkce spravovaný vstupní bod lze pouze `void` nebo `String []`.
