# <a name="linker-tools-error-lnk1287"></a>Chyba linkerů LNK1287
funkce neplatný spravované vstupního bodu  
  
 Vstupní bod není platný pro spravované bitovou kopii. Návratový typ funkce spravované vstupního bodu lze pouze `void` nebo `int`. Typ parametru funkce spravované vstupního bodu lze pouze `void` nebo `String []`.
