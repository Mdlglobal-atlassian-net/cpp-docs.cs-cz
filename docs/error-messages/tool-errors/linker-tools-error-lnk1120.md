# <a name="linker-tools-error-lnk1120"></a>Chyba linkerů LNK1120
*číslo* nepřeložené externích typů  
  
Chyba LNK1120 udává počet (*číslo*) nerozpoznané externí symbol chyb pro tuto operaci odkaz. Většina nepřeložené samostatně tak, že vznikly chyby externí symbol [chyba Linkerů LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md) a [chyba Linkerů LNK2019](../../error-messages/tool-errors/linker-tools-error-lnk2019.md), který předcházet tato chybová zpráva, jednou pro každou nerozpoznané externí symbol došlo k chybě.  
  
Odstranění této chyby, opravte všechny ostatní nerozpoznané externí chyby nebo jiné linkeru chyby, které předcházet ve výstupu sestavení. Tato chyba není hlášena, když zůstanou žádné nerozpoznané externí chyby.
