# <a name="linker-tools-error-lnk1120"></a>Chyba linkerů LNK1120

> *číslo* nerozpoznaných externích typů

Chyba LNK1120 udává počet (*číslo*) nerozpoznaný externí symbol chyb pro tuto operaci odkaz. Většina nevyřešené externí symbol chyby označené samostatně tak, že [chyba Linkerů LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md) a [chyba Linkerů LNK2019](../../error-messages/tool-errors/linker-tools-error-lnk2019.md), který předcházet tato chybová zpráva, jednou pro každý nevyřešené externí symbol došlo k chybě.

Chcete-li vyřešit tuto chybu, opravte všechny ostatní nevyřešené externí chyby nebo jiné chyby, které jej předcházejí ve výstupu sestavení. Tato chyba není hlášena, když žádné nevyřešené externí chyby zůstanou.
