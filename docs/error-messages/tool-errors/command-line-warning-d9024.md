# <a name="command-line-warning-d9024"></a>Upozornění příkazového řádku D9024

Neznámý typ zdrojového souboru filename, předpokládá se soubor objektu.

Rozšíření zadaného souboru nebylo rozpoznáno. Předpokládá se, že soubor je soubor objektu a byl předán linkeru.

Jsou rozpoznaná následující rozšíření:

- . c (zdrojový soubor C)

- . příponu CXX (C++ zdrojový soubor)

- . cpp (C++ zdrojový soubor)

- . obj (soubor objektu)

- . lib (soubor knihovny)

- . def (soubor definice modulu)

- . exp (linker exportuje soubor vytvořený ODKAZem/LIB)
