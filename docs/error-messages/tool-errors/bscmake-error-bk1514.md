# <a name="bscmake-error-bk1514"></a>Chyba nástroje BSCMAKE BK1514
všechny. Soubory SBR zkrácen, žádný se nenašel v názvu souboru  
  
 Žádné soubory .sbr zadané pro aktualizaci byly součástí k původní procházení (.bsc) souboru. Pro vyhledání názvů .sbr soubory, které se této chyby, přečtěte si [nástroje BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md) upozornění, která uveďte před ním.  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Chcete-li vyřešit kontrolou následující možné příčiny  
  
1.  Nesprávný název souboru zadaný pro .sbr nebo .bsc.  
  
2.  Poškozená .bsc soubor požadován BSCMAKE znovu sestavit.
