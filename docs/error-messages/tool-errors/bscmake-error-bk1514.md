# <a name="bscmake-error-bk1514"></a>Chyba nástroje BSCMAKE BK1514

všechny. Soubory SBR zkráceno, žádná se nenašla v názvu souboru

Žádné soubory .sbr zadané pro aktualizaci byly součástí původní soubor procházení (.bsc) informace. Názvy soubory .sbr vedoucí k této chybě najdete v článku [nástroje BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md) upozornění, které jej předcházejí.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Chcete-li vyřešit tak, že zkontrolujete následující možné příčiny

1. Chybný název souboru zadaný pro .sbr nebo .bsc.

1. Poškozená .bsc soubor požadován BSCMAKE její opětovné sestavení.
