# <a name="expression-evaluator-error-cxx0020"></a>Chyba při vyhodnocování výrazu CXX0020
Typy operandů chybný pro tuto operaci  
  
 Operátor bylo použito pro výraz s neplatným typem pro tento operátor.  
  
 Například není platná pro trvat adresu registrace nebo dolní index pole s výraz s plovoucí desetinnou čárkou.  
  
 Tato chyba je stejný jako CAN0020.
