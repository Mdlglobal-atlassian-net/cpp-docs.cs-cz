# <a name="resource-compiler-error-rc2170"></a>Chyba kompilátoru prostředků RC2170

Název souboru rastrového obrázku není ve formátu 3.00

Rastrové obrázky ve formátu Windows verze 2.x nelze použít v souborech prostředků verze 3.x.

Chcete-li vyřešit tento problém, ho překreslit nebo převést do formátu 3.x.
