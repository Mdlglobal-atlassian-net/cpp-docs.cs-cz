# <a name="resource-compiler-error-rc2170"></a>Chyba kompilátoru prostředků RC2170

název souboru rastrového obrázku není ve formátu 3,00.

Rastrové obrázky používající formát Windows verze 2. x nelze použít ve zdrojových souborech verze 3. x.

Chcete-li tento problém vyřešit, překreslete nebo převeďte rastrový obrázek na formát 3. x.
