# <a name="resource-compiler-error-rc2170"></a>Chyba kompilátoru prostředků RC2170
Název souboru soubor rastrového obrázku není ve formátu 3,00  
  
 Rastrové obrázky v systému Windows verze 2.x formátu nelze použít ve verzi 3.x zdrojové soubory.  
  
 Chcete-li tento problém vyřešit, ho překreslit nebo převést na formát 3.x bitové mapy.
