# <a name="project-build-error-prj0022"></a>Chyba sestavení projektu PRJ0022

> Neznámý nástroj, vlastnost '*Property*' obsahuje neplatný název souboru '*File*'.

*Soubor* s názvem souboru zadaný ve *vlastnosti* Property byl neplatný.

Možná jste použili neznámé nebo neplatné makro.

Informace o tom, jak získat přístup k vlastnostem projektu, naleznete v tématu [Nastavení vlastností projektu Visual C++ ](../../build/working-with-project-properties.md) .
