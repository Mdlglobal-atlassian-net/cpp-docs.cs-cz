# <a name="linker-tools-warning-lnk4096"></a>Upozornění linkerů LNK4096

Hodnota/BASE "Number" není platná pro systémy Windows 95 a Windows 98; Image se možná nespustí.

Základní adresa, kterou jste zadali, není platná. Spustitelné soubory Windows 95 a Windows 98 musí mít základní adresu větší než 0x400000. Další informace o základních adresách naleznete v tématu [/Base](../../build/reference/base-base-address.md) – možnost linkeru.
