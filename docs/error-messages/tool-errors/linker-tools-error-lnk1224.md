# <a name="linker-tools-error-lnk1224"></a>Chyba linkerů LNK1224
Neplatný základní adresa  
  
 Zadali jste neplatnou základní adresu pro bitovou kopii. Základní adresy musí být zarovnána 64KB (poslední čtyři hexadecimálních číslic musí být nula) a základní image vejít hodnotu 32-bit podepsaný držitelem nebo bez znaménka.
