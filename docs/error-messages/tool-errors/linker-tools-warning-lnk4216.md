# <a name="linker-tools-warning-lnk4216"></a>Upozornění linkerů LNK4216

Exportoval se vstupní bod vstupu

Vstupní bod byl exportován z knihovny DLL. Vstupní bod knihovny DLL není nutné exportovat.
