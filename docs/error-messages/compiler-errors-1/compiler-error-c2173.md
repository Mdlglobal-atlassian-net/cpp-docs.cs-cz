# <a name="compiler-error-c2173"></a>Chyba kompilátoru C2173

'function': skutečný parametr není ukazatel: Parametr Číslo1, číslo2 seznam parametrů

Parametr `number1` předaný do seznamu parametrů `number2` není ukazatel. Funkce očekává ukazatel.
