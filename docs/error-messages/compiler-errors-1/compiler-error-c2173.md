# <a name="compiler-error-c2173"></a>Chyba kompilátoru C2173

' function ': skutečný parametr není ukazatel: parametr Číslo1, seznam parametrů je číslo2

Parametr `number1` předán do seznamu parametrů `number2` není ukazatel. Funkce očekává ukazatel.
