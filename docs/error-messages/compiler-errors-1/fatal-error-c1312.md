# <a name="fatal-error-c1312"></a>Závažná chyba C1312

Funkce má příliš mnoho podmíněných větví. Zjednodušte nebo refaktorujte zdrojový kód.

Kód je příliš složitý, aby mohl kompilátor zpracovat bez vychodu paměti zásobníku.  Zjednodušte svůj kód.
