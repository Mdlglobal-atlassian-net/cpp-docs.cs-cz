# <a name="compiler-error-c2185"></a>Chyba kompilátoru C2185

identifikátor: neplatné přidělení na základě

Proměnná registru nebo automatická (místní) proměnná je deklarována `__based`. `__based`lze deklarovat pouze globální proměnné.
