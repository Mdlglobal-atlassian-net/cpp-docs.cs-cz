# <a name="compiler-error-c2185"></a>Chyba kompilátoru C2185

'identifier': neplatné přidělování based

Je deklarován na proměnné registru nebo automaticky (místní) proměnnou `__based`. Lze deklarovat pouze globální proměnné `__based`.
