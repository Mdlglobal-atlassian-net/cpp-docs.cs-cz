# <a name="fatal-error-c1075"></a>Závažná chyba C1075

levý token se neshodoval na konci souboru.

Kompilátor očekával *token* , který se shoduje předtím, než dosáhl konce souboru.

Tato chyba může být způsobena neodpovídající hranatou závorkou, závorkou nebo jiným spárovaným znakem.
