# <a name="fatal-error-c1075"></a>Závažná chyba C1075

levé token byl bezkonkurenční na konci souboru

Kompilátor byly očekávány párové *token* předtím, než bylo dosaženo konce souboru.

Tato chyba může být způsobeno nesprávné umístění závorky, složené závorky, nebo jiné spárované znak.
