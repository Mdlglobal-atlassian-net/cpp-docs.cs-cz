# <a name="fatal-error-c1075"></a>Závažná chyba C1075
levé token neodpovídající na konci souboru  
  
 Kompilátor očekává odpovídající *tokenu* předtím, než bylo dosaženo konce souboru.  
  
 Tato chyba může být způsobeno neodpovídající závorky, složené závorce, nebo jiné spárovat znak.
