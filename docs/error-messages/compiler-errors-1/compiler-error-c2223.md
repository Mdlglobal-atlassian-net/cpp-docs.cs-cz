# <a name="compiler-error-c2223"></a>Chyba kompilátoru C2223

Levá část-> identifikátoru musí ukazovat na strukturu/sjednocení.

Operand vlevo od `->` není ukazatel na třídu, strukturu nebo sjednocení.

Tato chyba může být způsobena levým operandem, který je nedefinovanou proměnnou (proto typ `int`).
