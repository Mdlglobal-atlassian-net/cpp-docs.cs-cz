# <a name="compiler-error-c2241"></a>Chyba kompilátoru C2241

'identifier': člen má omezený přístup

Kód se pokusí o přístup ke členu soukromé nebo chráněné.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Chcete-li vyřešit pomocí následujících možná řešení

1. Změníte úroveň přístupu členu.

1. Deklarace člena `friend` přístup k funkci.
