# <a name="compiler-error-c2241"></a>Chyba kompilátoru C2241

identifikátor: přístup ke členu je omezený.

Kód se pokusí o přístup k soukromému nebo chráněnému členu.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Oprava pomocí následujících možných řešení

1. Změňte úroveň přístupu člena.

1. Deklarujte člen a `friend` funkce přístupu.
