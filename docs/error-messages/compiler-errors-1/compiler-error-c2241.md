# <a name="compiler-error-c2241"></a>C2241 chyby kompilátoru
"identifikátor": je omezený přístup ke členu  
  
 Kód pokusí o přístup člena soukromé nebo chráněné.  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>Chcete-li odstranit pomocí následující možná řešení  
  
1.  Změníte úroveň přístupu tohoto člena.  
  
2.  Deklarace člena `friend` přístupem funkce.
