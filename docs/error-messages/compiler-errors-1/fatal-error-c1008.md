# <a name="fatal-error-c1008"></a>Závažná chyba C1008
nebyl zadán vstupní soubor  
  
 Kompilátor nebyl poskytnut zdrojového souboru jazyka C nebo C++ zkompilovat. Zkontrolujte **CL** příkazového řádku a proměnnou prostředí pro specifikace název souboru.
