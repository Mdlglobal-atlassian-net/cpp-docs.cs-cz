# <a name="fatal-error-c1121"></a>Závažná chyba C1121
volání rozhraní CryptoAPI se nezdařilo  
  
 Kompilátor k volání rozhraní CryptoAPI a volání se nezdařilo. Znovu nainstalujte Visual Studio a které by mohly mít přeinstalovat operační systém.
