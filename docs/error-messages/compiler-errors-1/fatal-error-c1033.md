# <a name="fatal-error-c1033"></a>Závažná chyba C1033

Nelze otevřít soubor pdb databáze programu

Tuto chybu může způsobovat chyba disku, dočasné zámek vytvořené antivirový program, předchozí instanci ladicího programu, který nebyl vypnout plně, nebo procesy mspdbsrv.exe paralelní sestavení, které pokus o přístup ke stejnému souboru, mezi další možné příčiny.
