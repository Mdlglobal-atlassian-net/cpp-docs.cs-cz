# <a name="compiler-error-c2114"></a>Chyba kompilátoru C2114

'operator': ukazatel nalevo; musí být celočíselná hodnota vpravo

Levý operand `operator` byl ukazatel, takže pravý operand musí být celočíselná hodnota.
