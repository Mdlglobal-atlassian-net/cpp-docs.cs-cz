# <a name="fatal-error-c1073"></a>Závažná chyba C1073

Došlo k vnitřní chybě týkající se přírůstkové kompilace (soubor kompilátoru 'filename', číslo řádku)

Znovu zkompilujte soubor bez použití přírůstková kompilace.
