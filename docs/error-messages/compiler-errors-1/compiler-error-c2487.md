# <a name="compiler-error-c2487"></a>Chyba kompilátoru C2487

identifikátor: člen třídy rozhraní DLL nemůže být deklarovaný s rozhraním dll.

Můžete deklarovat celou třídu nebo určité členy třídy rozhraní bez knihoven DLL s rozhraním DLL. Třídu s rozhraním DLL nelze deklarovat a poté deklarovat člena této třídy s rozhraním DLL.
