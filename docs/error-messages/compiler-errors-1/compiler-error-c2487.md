# <a name="compiler-error-c2487"></a>Chyba kompilátoru C2487

'identifier': člen interface class pro dll nemůže být deklarovaný s rozhraním dll

Je možné deklarovat celé jedné třídy nebo některé členy třídy rozhraní – knihovny DLL, s rozhraním DLL. Nelze deklarovat třídu s rozhraním DLL a potom deklarovat člena této třídy s rozhraním DLL.
