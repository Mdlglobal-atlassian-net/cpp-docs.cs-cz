# <a name="compiler-error-c2487"></a>C2487 chyby kompilátoru
"identifikátor": člena třídy knihovny dll rozhraní nemusí deklarovat s knihovny dll rozhraní  
  
 Je možné deklarovat celou třídu nebo určité členy třídy rozhraní – knihovny DLL, knihovny DLL rozhraní. Nelze deklarovat třídu rozhraní knihovny DLL a potom deklarovat členem třídy knihovny DLL rozhraní.
