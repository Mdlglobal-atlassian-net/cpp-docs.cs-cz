# <a name="fatal-error-c1014"></a>Závažná chyba C1014

moc velký počet vložených souborů: Hloubka = úroveň

Vnořování direktiv `#include` je moc hluboko. Vnořené direktivy můžou zahrnovat otevřené soubory. Zdrojový soubor, který obsahuje direktivu, se počítá jako jeden soubor.
