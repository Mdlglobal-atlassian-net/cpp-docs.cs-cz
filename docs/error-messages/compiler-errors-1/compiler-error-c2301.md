# <a name="compiler-error-c2301"></a>C2301 chyby kompilátoru
nalevo od ' -> ~ identifikátor ' musí odkazovat na třída, struktura/sjednocení  
  
 Výraz, který se nalevo od `->` operátor není vyhodnocen ukazatel na třídy, struktury nebo union.
