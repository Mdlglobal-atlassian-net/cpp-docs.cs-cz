# <a name="compiler-error-c2160"></a>Chyba kompilátoru C2160

"##" nemůže být na začátku definice makra

Definice makra začalo se operátor vložení tokenu (##).

Následující ukázka generuje C2160:

```
// C2160.cpp
// compile with: /c
#define mac(a,b) #a   // OK
#define mac(a,b) ##a   // C2160
```
