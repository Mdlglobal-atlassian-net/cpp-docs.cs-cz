# <a name="fatal-error-c1057"></a>Závažná chyba C1057

Neočekávaný konec souboru v rozšíření makra

Kompilátor dosáhl konce zdrojového souboru při shromažďování argumentů volání makra, pravděpodobně v důsledku chybějící pravé závorky ve volání makra.
