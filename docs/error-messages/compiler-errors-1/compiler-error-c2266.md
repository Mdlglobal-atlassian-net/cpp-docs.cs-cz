# <a name="compiler-error-c2266"></a>Chyba kompilátoru C2266

' identifier ': odkaz na nekonstantní vázané pole je neplatný

Odkaz je deklarován pro pole s nekonstantní vazbou. Pole musí mít konstantní meze.
