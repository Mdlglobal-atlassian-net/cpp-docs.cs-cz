# <a name="compiler-error-c2022"></a>Chyba kompilátoru C2022

'number': moc velké pro znak

Osmičkové číslo po zpětné lomítko (\\) ve znaku nebo řetězcová konstanta je příliš dlouhý pro reprezentaci znaků.
