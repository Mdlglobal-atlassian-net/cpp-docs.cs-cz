# <a name="compiler-error-c2022"></a>Chyba kompilátoru C2022

' Number ': moc velké pro znak

Osmičkové číslo za zpětným lomítkem (\\) ve znaku nebo v řetězcové konstantě je příliš velké, aby představovalo znak.
