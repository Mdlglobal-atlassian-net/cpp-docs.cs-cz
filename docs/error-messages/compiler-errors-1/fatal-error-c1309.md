# <a name="fatal-error-c1309"></a>Závažná chyba C1309

Neshodné verze C2. DLL a PGODB\<verze >. DLL

Sada nástrojů, kterou používáte k sestavování a používání [optimalizací](../../build/profile-guided-optimizations.md) na základě profilu, obsahuje neodpovídající součásti.  Pokud tuto chybu nemůžete ručně vyřešit, přeinstalujte vizuál C++.
