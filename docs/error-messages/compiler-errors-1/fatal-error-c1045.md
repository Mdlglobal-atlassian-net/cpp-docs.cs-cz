# <a name="fatal-error-c1045"></a>Závažná chyba C1045
omezení kompilátoru: Specifikace propojení vnořeny příliš hluboko  
  
 Vnořené externals překročila omezení kompilátoru. Vnořené externals jsou povoleny s externí propojení typu, například `extern` "C++". Snižte počet vnořených externals, chcete-li vyřešit chyby.
