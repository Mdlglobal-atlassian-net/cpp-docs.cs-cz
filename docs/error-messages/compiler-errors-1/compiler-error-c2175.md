# <a name="compiler-error-c2175"></a>C2175 chyby kompilátoru
"národního prostředí": Neplatný národní prostředí  
  
 Zadaný národní prostředí není platný. V tématu [řetězce země nebo oblast a jazyk](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) v *referenční dokumentace běhové knihovny* pro podporované národní prostředí.
