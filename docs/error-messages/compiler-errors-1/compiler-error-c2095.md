# <a name="compiler-error-c2095"></a>C2095 chyby kompilátoru
'function': skutečný parametr obsahuje typ void: parametr "číslo"  
  
 Parametr předaný funkci je typ `void`, což není povolené. Použijte ukazatel na void ( `void *`) místo.  
  
 `number` Určuje, které parametr `void`.
