# <a name="fatal-error-c1903"></a>Závažná chyba C1903

nejde obnovit z přechozích chyb; kompilace se zastavuje

Kompilátor nalezeno příliš mnoho chyb, abyste mohli pokračovat. Opravte chyby a znovu zkompilovat.
