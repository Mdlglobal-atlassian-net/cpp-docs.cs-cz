# <a name="compiler-error-c2124"></a>Chyba kompilátoru C2124

dělení nulou nebo dělení nulou se zbytkem

Konstantní výraz je nula jmenovatel. Chcete-li vyřešit chybu, ne dělení nulou.

Následující ukázka generuje C2124:

```
// C2124.cpp
int main() {
  int i = 1 / 0;   // C2124  do not divide by zero
  int i2 = 12 / 2;   // OK
}
```
