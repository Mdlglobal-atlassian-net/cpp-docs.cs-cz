# <a name="compiler-error-c2124"></a>C2124 chyby kompilátoru
dělení nebo mod nulou  
  
 Konstantní výraz má nulové jmenovatel. Chcete-li vyřešit chyby, není dělení nulou.  
  
 Následující ukázka generuje C2124:  
  
```  
// C2124.cpp  
int main() {  
  int i = 1 / 0;   // C2124  do not divide by zero  
  int i2 = 12 / 2;   // OK  
}  
```
