# <a name="compiler-error-c2411"></a>Chyba kompilátoru C2411

' identifier ': neplatný člen struktury/sjednocení v ' Context '

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Opravu provedete kontrolou následujících možných příčin.

1. Identifikátor není členem viditelné struktury nebo sjednocení v tomto kontextu.

1. Identifikátor není členem struktury nebo sjednocení zadaný pomocí operátoru výběru členů (.).
