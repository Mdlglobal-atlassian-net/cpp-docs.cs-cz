# <a name="compiler-error-c2411"></a>C2411 chyby kompilátoru
"identifikátor": Neplatný člen struktura/sjednocení v "context"  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>Chcete-li vyřešit kontrolou následující možné příčiny  
  
1.  Identifikátor není členem skupiny viditelné struktury nebo sjednocení v tomto kontextu.  
  
2.  Identifikátor není členem struktury nebo zadaným výběr členů (.) operátor union.
