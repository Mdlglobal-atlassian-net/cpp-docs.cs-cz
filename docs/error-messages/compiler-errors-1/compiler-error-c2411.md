# <a name="compiler-error-c2411"></a>Chyba kompilátoru C2411

'identifier': Neplatný člen struktura/sjednocení v "context"

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Chcete-li vyřešit tak, že zkontrolujete následující možné příčiny

1. Identifikátor není členem viditelné struktury nebo sjednocení v tomto kontextu.

1. Identifikátor není členem struktury nebo sjednocení zadané s operátorem výběru členů (.).
