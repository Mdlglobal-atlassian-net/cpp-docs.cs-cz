# <a name="compiler-error-c2155"></a>Chyba kompilátoru C2155

?: Neplatný levý operand, očekával se aritmetický typ nebo typ ukazatele.

Výraz na levé straně `?` nelze porovnat s hodnotou nula. Je nutné použít aritmetický výraz nebo výraz ukazatele, který lze porovnat s nulou.
