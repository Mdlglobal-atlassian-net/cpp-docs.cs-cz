# <a name="compiler-error-c2465"></a>C2465 chyby kompilátoru
nelze definovat anonymního typu v závorkách  
  
 Výraz v závorkách je definován anonymní struktury, sjednocení nebo výčtového typu. To je neplatný v jazyce C++, protože definice se smysl v oboru funkce.
