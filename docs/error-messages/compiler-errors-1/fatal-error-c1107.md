# <a name="fatal-error-c1107"></a>Závažná chyba C1107

Nepovedlo se najít sestavení File: Zadejte prosím cestu pro vyhledávání sestavení pomocí/AI nebo nastavte proměnnou prostředí LIBPATH.

Soubor metadat byl předán do direktivy [#using](../../preprocessor/hash-using-directive-cpp.md) , kterou kompilátor nemohl najít.

LIBPATH, která je popsána v tématu pro `#using`a možnost kompilátoru [/AI](../../build/reference/ai-specify-metadata-directories.md) umožňuje zadat adresáře, ve kterých bude kompilátor hledat odkazované soubory metadat.
