# <a name="fatal-error-c1035"></a>Závažná chyba C1035
výraz je příliš složitý; zjednodušení výraz  
  
 Kompilátor nelze generovat kód pro složitý výraz. Rozdělení výraz do jednodušší výrazy a znovu zkompilovat.
