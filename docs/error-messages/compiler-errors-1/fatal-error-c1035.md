# <a name="fatal-error-c1035"></a>Závažná chyba C1035

výraz je příliš složitý; zjednodušit výraz

Kompilátor nelze generovat kód pro složitý výraz. Rozdělit výraz, který jednodušší výrazy a znovu zkompilovat.
