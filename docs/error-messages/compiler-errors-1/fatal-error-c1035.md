# <a name="fatal-error-c1035"></a>Závažná chyba C1035

výraz je moc složitý; zjednodušit výraz

Kompilátorem se nepodařilo vygenerovat kód pro složitý výraz. Rozdělit výraz na jednodušší výrazy a znovu kompilovat.
