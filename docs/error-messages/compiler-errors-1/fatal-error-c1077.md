# <a name="fatal-error-c1077"></a>Závažná chyba C1077
omezení kompilátoru: nelze mít více než číslo možnosti příkazového řádku  
  
 Počet možností příkazového řádku překračuje vnitřní limit.  
  
 Může být příliš mnoho symboly definovaný s [/D](../../build/reference/d-preprocessor-definitions.md). (Umístit definice v záhlaví souboru místo.)
