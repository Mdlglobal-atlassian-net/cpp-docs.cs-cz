# <a name="fatal-error-c1077"></a>Závažná chyba C1077

limit kompilátoru: nelze mít více než číslo možnosti příkazového řádku

Počet parametrů příkazového řádku překračuje vnitřní limit.

Může být příliš mnoho symbolů definovaných s [/D](../../build/reference/d-preprocessor-definitions.md). (Umístit definice v hlavičkovém souboru místo toho.)
