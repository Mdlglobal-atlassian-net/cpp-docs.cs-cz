# <a name="compiler-error-c2325"></a>Chyba kompilátoru C2325

'type': Neočekávaný typ napravo od "name"

Je provedeno volání destruktoru nesprávného typu.

Následující ukázka generuje C2325:

```
// C2325.cpp
// compile with: /c
class A {};
typedef A* pA_t;
void f() {
    A** ppa = new A *;
    ppa->~A*;   // C2325

   pA_t *ppa2 = new pA_t;
   ppa2->~pA_t();   // OK
}
```
