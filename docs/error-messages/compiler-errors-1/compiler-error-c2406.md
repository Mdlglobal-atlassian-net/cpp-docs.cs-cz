# <a name="compiler-error-c2406"></a>Chyba kompilátoru C2406

' identifier ': název není definován v kontextu '

Pomocí operátoru `SIZE`, `LENGTH`nebo výběru členů (.) se používá nedefinovaný identifikátor.
