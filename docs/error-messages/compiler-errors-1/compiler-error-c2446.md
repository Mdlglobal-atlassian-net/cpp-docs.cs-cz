# <a name="compiler-error-c2446"></a>Chyba kompilátoru C2446

'operator': převod z 'type1' na 'type2'

Kompilátor nelze převést `type1` k `type2`. Převod nemusí mít smysl, protože porušuje sémantiku jazyka C/C++.
