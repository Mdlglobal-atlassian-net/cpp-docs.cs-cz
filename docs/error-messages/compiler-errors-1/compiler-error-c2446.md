# <a name="compiler-error-c2446"></a>C2446 chyby kompilátoru
'operátor': žádný převod z 'type1' do 'type2'.  
  
 Nelze převést do kompilátoru `type1` k `type2`. Převod nemusí mít smysl, protože narušuje sémantiku C/C++.
