# <a name="compiler-error-c2446"></a>Chyba kompilátoru C2446

' operator ': žádný převod z ' typ1 ' na ' typ2 '

Kompilátor nemůže převést `type1` na `type2`. Převod nemusí být smysl, protože je v rozporu s sémantikouC++ jazyka C nebo.
