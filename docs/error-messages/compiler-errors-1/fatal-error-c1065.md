# <a name="fatal-error-c1065"></a>Závažná chyba C1065

limit kompilátoru: došly značky

Zdrojový soubor obsahuje více než 65,523 třídy, struktury, sjednocení, obory názvů nebo výčty.

Soubor rozdělte na menší soubory.
