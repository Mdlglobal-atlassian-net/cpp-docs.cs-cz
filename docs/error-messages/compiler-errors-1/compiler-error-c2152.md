# <a name="compiler-error-c2152"></a>C2152 chyby kompilátoru
"identifikátor": ukazatelé na funkce s různé atributy  
  
 Ukazatel na funkci s jeden konvence volání (`__cdecl`, `__stdcall`, nebo `__fastcall`) je přiřazena k ukazatel na funkci s jinou konvence volání.
