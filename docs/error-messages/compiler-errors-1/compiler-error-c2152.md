# <a name="compiler-error-c2152"></a>Compiler Error C2152

'identifier': ukazatele na funkce s rozdílnými atributy

Ukazatel na funkci s jeden konvence volání (`__cdecl`, `__stdcall`, nebo `__fastcall`) je přiřadit ukazateli na funkci s jinou konvence volání.
