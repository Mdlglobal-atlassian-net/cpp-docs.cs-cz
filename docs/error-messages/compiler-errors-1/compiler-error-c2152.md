# <a name="compiler-error-c2152"></a>Chyba kompilátoru C2152

' identifier ': ukazatele na funkce s různými atributy

Ukazatel na funkci s jednou konvencí volání (`__cdecl`, `__stdcall`nebo `__fastcall`) je přiřazen k ukazateli na funkci s jinou konvencí volání.
