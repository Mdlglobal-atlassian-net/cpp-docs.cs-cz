# <a name="compiler-error-c2318"></a>Chyba kompilátoru C2318

k této obslužné rutině catch není přidružen žádný blok try.

Je definována obslužná rutina `catch`, ale před ní není `try` blok.

Následující ukázka generuje C2318:

```cpp
// C2318.cpp
// compile with: /EHsc
#include <eh.h>
int main() {
   // no try block
   catch( int ) {}   // C2318
}
```

Možné řešení:

```cpp
// C2318b.cpp
// compile with: /EHsc
#include <eh.h>
int main() {
   try{}
   catch( int ) {}
}
```
