# <a name="compiler-error-c2206"></a>C2206 chyby kompilátoru
'function': typedef nelze použít pro definice funkce  
  
 A `typedef` se používá k definování typu funkce.  
  
 Následující ukázka generuje C2206:  
  
```  
// C2206.cpp  
typedef int functyp();  
typedef int MyInt;  
functyp func1 {};   // C2206  
int main() {  
   MyInt i = 0;   // OK  
}  
```
