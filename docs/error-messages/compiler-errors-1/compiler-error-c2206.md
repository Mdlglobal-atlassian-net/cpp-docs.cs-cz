# <a name="compiler-error-c2206"></a>Chyba kompilátoru C2206

'function': typedef nejde používat pro definici funkce

A `typedef` slouží k definování typu funkce.

Následující ukázka generuje C2206:

```
// C2206.cpp
typedef int functyp();
typedef int MyInt;
functyp func1 {};   // C2206
int main() {
   MyInt i = 0;   // OK
}
```
