# <a name="fatal-error-c1206"></a>Závažná chyba C1206
Data na appdomain nepodporuje verzi modulu runtime nainstalován  
  
 Některé funkce, například podle dat aplikací domény, jsou podporovány pouze pomocí modulu common language runtime, který podporuje funkci.  
  
 C1206 označuje, že není v počítači nainstalován nejnovější verzi modulu runtime. Nainstalujte běžné verzi modulu runtime jazyka, která je určena pro použití s vaší kompilátoru.  
  
 V tématu [appdomain](../../cpp/appdomain.md) Další informace.
