# <a name="fatal-error-c1051"></a>Závažná chyba C1051

soubor databáze programu, ' pdbfile ', má zastaralý formát, odstraní jej a znovu zkompiluje

Kompilátor nemůže aktualizovat soubor databáze programu, který má starší číslo verze. Odstraňte soubor a znovu zkompilujte program pomocí **/Zi** nebo **/Zi**. Další informace najdete v tématu [/Z7,/Zi,/Zi (formát ladicích informací)](../../build/reference/z7-zi-zi-debug-information-format.md) .
