# <a name="fatal-error-c1196"></a>Závažná chyba C1196

' identifier ': identifikátor nalezen v knihovně typů ' typelib ' není platný C++ identifikátor

Jeden z identifikátorů v knihovně typů není platný C++ identifikátor. Knihovna typů není k dispozici pro použití s [#import](../../preprocessor/hash-import-directive-cpp.md).
