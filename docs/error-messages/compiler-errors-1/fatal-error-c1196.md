# <a name="fatal-error-c1196"></a>Závažná chyba C1196
"identifikátor": identifikátor knihovny typů "typelib" není platný identifikátor C++  
  
 Jedna identifikátory v knihovně typu není platný identifikátor C++. Knihovny typů není k dispozici pro použití s [#import](../../preprocessor/hash-import-directive-cpp.md).
