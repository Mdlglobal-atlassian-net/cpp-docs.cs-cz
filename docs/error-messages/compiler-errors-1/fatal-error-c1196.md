# <a name="fatal-error-c1196"></a>Závažná chyba C1196

'identifier': identifikátor v knihovně typů 'knihovny typů' není platný identifikátor C++

Jeden z identifikátorů v knihovna typů není platný identifikátor C++. Knihovna typů není k dispozici pro použití s [#import](../../preprocessor/hash-import-directive-cpp.md).
