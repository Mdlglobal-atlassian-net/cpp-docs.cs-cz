# <a name="compiler-error-c2028"></a>Chyba kompilátoru C2028

Člen struktury/sjednocení musí být uvnitř struktury/sjednocení.

Členy struktury nebo sjednocení musí být deklarována v rámci struktury nebo sjednocení.
