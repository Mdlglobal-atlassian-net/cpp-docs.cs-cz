# <a name="fatal-error-c1207"></a>Závažná chyba C1207

Spravované šablony nejsou podporované nainstalovanou verzí modulu runtime.

K C1207 dojde, když máte kompilátor pro aktuální vydání, ale modul CLR (Common Language Runtime) z předchozí verze.

Některé funkce kompilátoru nemusí fungovat v předchozí verzi doby běhu.

Chcete-li vyřešit C1207, nainstalujte verzi modulu Common Language Runtime, která je určena pro použití s vaším kompilátorem.
