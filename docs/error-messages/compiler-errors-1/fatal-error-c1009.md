# <a name="fatal-error-c1009"></a>Závažná chyba C1009

limit kompilátoru: makra jsou vnořená moc hluboko.

Kompilátor se pokusil rozbalovat příliš mnoho maker současně. Kompilátor má limit 256 úrovní vnořených maker. Rozdělit vnořená makra do jednodušších maker.
