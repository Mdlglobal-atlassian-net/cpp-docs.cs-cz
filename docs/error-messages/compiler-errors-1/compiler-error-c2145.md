# <a name="compiler-error-c2145"></a>C2145 chyby kompilátoru
Chyba syntaxe: chybějící token před identifikátor  
  
 Kompilátor očekává `token` a místo toho nalezen identifikátor.  
  
 Tato chyba může být způsobeno chybějící středníkem po poslední deklarace v bloku.
