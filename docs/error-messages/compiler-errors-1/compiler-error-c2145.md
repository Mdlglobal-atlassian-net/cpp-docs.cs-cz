# <a name="compiler-error-c2145"></a>Chyba kompilátoru C2145

Chyba syntaxe: chybí token před identifikátorem

Kompilátor očekává `token` a místo toho našel se identifikátor.

Tato chyba může být způsobeno chybějící středníkem za poslední deklaraci v bloku.
