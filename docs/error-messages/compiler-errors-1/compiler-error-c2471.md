# <a name="compiler-error-c2471"></a>Chyba kompilátoru C2471

Nelze aktualizovat databázi programu 'file'

Kompilátor nemůže zapisovat do souboru databáze.

### <a name="to-fix-the-problem"></a>Chcete-li vyřešit tento problém

- Ujistěte se, že disk obsahuje dostatek volného místa.

- Ujistěte se, že soubor není jen pro čtení.

- Ujistěte se, že neexistují žádná narušení sdílení.
