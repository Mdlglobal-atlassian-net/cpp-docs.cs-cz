# <a name="compiler-error-c2471"></a>Chyba kompilátoru C2471

nelze aktualizovat databázi programu ' file '

Kompilátor nemůže zapisovat do databázového souboru.

### <a name="to-fix-the-problem"></a>Řešení problému

- Ujistěte se, že na disku je dostatek volného místa.

- Ujistěte se, že soubor není jen pro čtení.

- Ujistěte se, že nedochází k narušení sdílení.
