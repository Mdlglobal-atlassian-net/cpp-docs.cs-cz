# <a name="compiler-error-c2488"></a>Chyba kompilátoru C2488

'identifier': 'naked' jde použít jedině pro definice funkce bez členů

[Naked](../../cpp/naked-cpp.md) atribut byla použita k deklaraci funkce.

Následující ukázka generuje C2488:

```
// C2488.cpp
// compile with: /c
// processor: x86
__declspec( naked ) void func();   // C2488  declaration, not definition
__declspec( naked ) void i;   // C2488  i is not a function

__declspec( naked ) void func() {}   // OK
```
