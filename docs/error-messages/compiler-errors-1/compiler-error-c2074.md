# <a name="compiler-error-c2074"></a>C2074 chyby kompilátoru
"identifikátor": 'klíč třídy' Inicializace musí složené závorky  
  
 Neexistují žádné složené závorky kolem zadané třídy, struktury nebo union inicializátor.
