# <a name="compiler-error-c2074"></a>Chyba kompilátoru C2074

'identifier': "klíč třídy" Inicializace potřebuje složených závorek

Nebyly žádné složené závorky kolem dané třídy, struktury nebo sjednocení inicializátor.
