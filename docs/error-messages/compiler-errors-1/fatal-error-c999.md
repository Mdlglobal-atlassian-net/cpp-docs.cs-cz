# <a name="fatal-error-c999"></a>Závažná chyba C999
Neznámá zpráva Zvolte prosím příkaz se na technickou podporu v nabídce Nápověda Visual C++, nebo otevře soubor nápovědy se na technickou podporu pro další informace  
  
 Tato chyba obvykle znamená, že používáte smíšené soubory z různých verzích kompilátor, nebo kompilátoru instalaci je poškozený. Použití **programy a funkce** apletu ovládacího panelu opravy a nové instalace produktu.
