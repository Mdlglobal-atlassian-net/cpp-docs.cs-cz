# <a name="compiler-error-c2106"></a>C2106 chyby kompilátoru
'operátor': levý operand musí být hodnota l  
  
 Operátor musí mít hodnotu l jako levý operand.  
  
 Následující ukázka generuje C2106:  
  
```  
// C2106.cpp  
int main() {  
   int a;  
   1 = a;   // C2106  
   a = 1;   // OK  
}  
```
