# <a name="fatal-error-c1603"></a>Závažná chyba C1603

cíl větve vloženého sestavení mimo rozsah o 'number' bajtů

Vypočítané vzdálenost mezi JCXZ nebo JECXZ instrukce a zadané cílové popisek byl větší než 128 bajtů. Aktualizujte svůj kód tak, aby popisek je blíž ke podle pokynů.
