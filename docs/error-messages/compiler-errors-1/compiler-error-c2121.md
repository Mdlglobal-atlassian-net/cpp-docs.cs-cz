# <a name="compiler-error-c2121"></a>Chyba kompilátoru C2121

#: neplatný znak: může být výsledkem rozšíření makra.

Neplatný znak # byl pravděpodobně vložen nesprávným makrem, které používá operátor vložení tokenu (# #) namísto operátoru převodu (#).
