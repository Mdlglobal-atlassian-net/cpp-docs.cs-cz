# <a name="compiler-error-c2121"></a>Chyba kompilátoru C2121

'#': neplatný znak: může být výsledkem rozšíření makra

Znak neplatný # může vložen nesprávný – makro, která používá operátor vložení tokenu (##) namísto operátoru převodu na řetězec (#).
