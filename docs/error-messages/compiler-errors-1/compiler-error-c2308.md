# <a name="compiler-error-c2308"></a>Chyba kompilátoru C2308

zřetězení neodpovídajících řetězců

Pro zřetězení byly zadány řetězce znaků pro velké i jiné znaky. Nemůžete zřetězit řetězec znaků a řetězec znaků, který není v rozsahu.
