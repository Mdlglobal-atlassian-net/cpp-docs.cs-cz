# <a name="compiler-error-c2222"></a>Chyba kompilátoru C2222

Neočekávaný typ Type: očekávala se základní (Base) třída nebo člen.

Seznam inicializátorů může inicializovat pouze základní třídy nebo členy typu. Chcete-li tuto chybu opravit, ověřte, zda jsou inicializovány pouze základní třídy nebo členy typu v seznamu inicializátorů.
