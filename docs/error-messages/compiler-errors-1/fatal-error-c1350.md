# <a name="fatal-error-c1350"></a>Závažná chyba C1350
Chyba při načítání knihovny dll, knihovny dll': knihovna dll nebyla nalezena  
  
 Nebyla nalezena knihovnu DLL, kterou podporuje pokus o operaci.  
  
 To znamená problém s instalací a nainstalujte knihovnu DLL z disku CD produktu.
