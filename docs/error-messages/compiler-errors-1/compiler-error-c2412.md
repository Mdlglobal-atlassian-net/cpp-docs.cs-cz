# <a name="compiler-error-c2412"></a>Chyba kompilátoru C2412

"štítek": malá a velká písmena Návěstí bylo předefinováno

Popisek je definován více než jednou v aktuální funkci. Změňte pravopis popisek a jeho odkazy.
