# <a name="compiler-error-c2412"></a>Chyba kompilátoru C2412

' label ': Předefinovaná jmenovka bez rozlišení velkých a malých písmen

Popisek je v aktuální funkci definovaný víc než jednou. Změňte pravopis popisku a jeho odkazů.
