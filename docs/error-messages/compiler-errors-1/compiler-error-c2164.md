# <a name="compiler-error-c2164"></a>Chyba kompilátoru C2164

'function': není deklarovaná vnitřní funkce

`intrinsic` – Direktiva pragma používá nedeklarovaný – funkce (dochází pouze s **/Oi**). Nebo jeden z vnitřní funkce kompilátoru byl použit bez zahrnutí souboru hlaviček.

Následující ukázka generuje C2164:

```
// C2164.c
// compile with: /c
// processor: x86
// Uncomment the following line to resolve.
// #include "xmmintrin.h"
void b(float *p) {
   _mm_load_ss(p);   // C2164
}
```
