# <a name="fatal-error-c1053"></a>Závažná chyba C1053

\<> identifikátoru: funkce je moc velká.

Funkce je příliš velká pro zkompilování.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Oprava pomocí následujících možných řešení

1. Zkuste kompilovat bez optimalizace.

1. Rozdělení funkce na menší funkce.

1. Omezte volání na vložené funkce.
