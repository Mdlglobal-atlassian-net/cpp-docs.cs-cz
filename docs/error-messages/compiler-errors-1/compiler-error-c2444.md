# <a name="compiler-error-c2444"></a>Chyba kompilátoru C2444

' identifier ': používal se prototyp ANSI, našlo se: Type, očekávalo se: {nebo;

Prototypem funkce je následován typ.

Tato chyba může být způsobena chybějící středníkem nebo složenou závorkou.
