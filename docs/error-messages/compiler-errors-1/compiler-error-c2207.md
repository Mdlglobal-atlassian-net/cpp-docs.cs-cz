# <a name="compiler-error-c2207"></a>Chyba kompilátoru C2207

member: člen šablony třídy nemůže získat typ funkce.

`member` šablony třídy byly dříve analyzovány jako nestatický datový člen. Nelze ji předefinovat jako členskou funkci.
