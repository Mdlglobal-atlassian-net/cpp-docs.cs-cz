# <a name="compiler-error-c2219"></a>Chyba kompilátoru C2219

Chyba syntaxe: kvalifikátor typu musí být za hvězdičkou (*).

Kvalifikátor typu (`const` nebo `volatile`) se zobrazí tam, kde není povolený.
