# <a name="compiler-error-c2219"></a>Chyba kompilátoru C2219

Chyba syntaxe: kvalifikátor typu musí následovat po "*"

Kvalifikátor typu (`const` nebo `volatile`) se zobrazí, pokud není povolená.
