# <a name="fatal-error-c1351"></a>Závažná chyba C1351
Chyba při načítání knihovny dll, knihovny dll': nekompatibilní verze  
  
 Byl nalezen nesprávná verze knihovny DLL.  
  
 To znamená problém s instalací a nainstalujte knihovnu DLL z disku CD produktu.
