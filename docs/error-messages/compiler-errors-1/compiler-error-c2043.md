# <a name="compiler-error-c2043"></a>Chyba kompilátoru C2043

Neplatné přerušení

A [přerušení](../../cpp/break-statement-cpp.md) je platný pouze v rámci `do`, `for`, `while`, nebo `switch` příkazu.
