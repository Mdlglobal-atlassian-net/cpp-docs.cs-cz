# <a name="fatal-error-c1046"></a>Závažná chyba C1046

limit kompilátoru: struktura je vnořená moc hluboko.

Struktura, sjednocení nebo třída překročila omezení vnořování, což je 15 úrovní. Přepište definici pro snížení úrovně vnořování. Rozdělení struktury, sjednocení nebo třídy na dvě nebo více částí pomocí `typedef` pro definování jedné nebo více vnořených struktur.
