# <a name="fatal-error-c1082"></a>Závažná chyba C1082
Nelze zavřít soubor typ souboru: 'file': zpráva  
  
 Pokud zpráva říká "chybného souboru číslo", soubor může mít byla zavření v popředí při kompilování na pozadí.
