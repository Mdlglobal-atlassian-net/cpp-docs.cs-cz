# <a name="fatal-error-c1023"></a>Závažná chyba C1023

'file': došlo k neočekávané chybě se zkuste znovu sestavit soubor pch

C1023 mohlo být způsobeno jedním z několika problémy, řešení, do kterého se opětovné sestavení předkompilovaného souboru hlaviček.

Pokud se pokusíte použít soubor předkompilované hlavičky v počítači než na kterém byla vytvořena, může být způsobeno C1023.
