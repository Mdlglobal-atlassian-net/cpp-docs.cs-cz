# <a name="fatal-error-c1023"></a>Závažná chyba C1023
'file': Neočekávaná chyba s pch, zkuste znovu sestavit pch  
  
 C1023 může být způsobeno jedním z několika problémy, řešení, ke kterému je znovu vytvořit předkompilovaný hlavičkový soubor.  
  
 Pokud se pokusíte použít předkompilovaný hlavičkový soubor na počítači, na kterém byl vytvořený, může být způsobeno C1023.
