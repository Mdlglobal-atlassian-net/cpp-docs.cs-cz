# <a name="fatal-error-c1023"></a>Závažná chyba C1023

' file ': došlo k neočekávané chybě souboru PCH, zkuste soubor PCH znovu sestavit.

C1023 může být způsoben jedním z několika problémů, což je řešení, do kterého je znovu sestaven soubor předkompilované hlavičky.

C1023 může být způsobena tím, že se pokusíte použít předkompilovaný hlavičkový soubor na jiném počítači, než na kterém byl sestaven.
