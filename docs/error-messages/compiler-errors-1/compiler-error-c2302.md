# <a name="compiler-error-c2302"></a>Chyba kompilátoru C2302

Levá strana ". ~ identifikátor ' musí být typu Třída/struktura/sjednocení

Výraz na levé straně operátoru tečka (.) není třída, struktura nebo sjednocení.
