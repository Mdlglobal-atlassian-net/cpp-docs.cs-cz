# <a name="compiler-error-c2302"></a>Chyba kompilátoru C2302

Levá část '. ~ identifier ' musí mít typ třída/struktura/sjednocení.

Výraz nalevo od operátoru period (.) není třída, struktura nebo sjednocení.
