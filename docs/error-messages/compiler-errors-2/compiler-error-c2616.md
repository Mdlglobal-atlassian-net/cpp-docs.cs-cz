# <a name="compiler-error-c2616"></a>Chyba kompilátoru C2616

' conversion ': Nelze implicitně převést typ ' typ1 ' na ' typ2 ', který není const

Odkaz se nedá inicializovat z jiné než hodnoty lvalue.

Jedná se o chybu kompatibility ANSI ([/za](../../build/reference/za-ze-disable-language-extensions.md)) a upozornění v části rozšíření společnosti Microsoft ( **/ze**).
