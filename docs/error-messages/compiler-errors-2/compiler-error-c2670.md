# <a name="compiler-error-c2670"></a>C2670 chyby kompilátoru
"identifikátor": funkce šablony nelze převést z typu "typ" Parametr číslo  
  
 Parametr nelze převést na požadovaný typ.  
  
 Tato chyba může být stanovena, pokud vytvoříte explicitní převod.
