# <a name="compiler-error-c2532"></a>Chyba kompilátoru C2532

'identifier': Neplatný modifikátor pro odkaz

Odkaz byl změněn. Odkazy, nelze změnit pro odkazování na jiný objekt. Místo toho použijte ukazatel.
