# <a name="compiler-error-c2532"></a>Chyba kompilátoru C2532

' identifier ': neplatný modifikátor pro referenci

Odkaz byl změněn. Odkazy nelze upravovat, aby odkazovaly na jiný objekt. Místo toho použijte ukazatel.
