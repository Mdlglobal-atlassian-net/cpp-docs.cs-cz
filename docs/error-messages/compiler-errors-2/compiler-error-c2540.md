# <a name="compiler-error-c2540"></a>C2540 chyby kompilátoru
není konstantní výraz jako pole vázán  
  
 Pole musí mít konstanta vázána.  
  
 Následující ukázka generuje C2540:  
  
```  
// C2540.cpp  
void func(int n, int pC[]) {  
   int i = ((int [n])pC)[1];   // C2540  
}  
  
void func2(int n, int pC[]) {  
   int i = (pC)[1];   // OK  
}  
  
int main() {  
   int pC[100];  
   func(100, pC);  
   func2(100, pC);  
}  
```
