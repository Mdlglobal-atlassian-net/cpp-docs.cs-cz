# <a name="compiler-error-c2877"></a>C2877 chyby kompilátoru
"značkou" není přístupný ze 'class'.  
  
 Všichni členové odvozené ze základní třídy musí být přístupné v odvozené třídě.  
  
 Následující ukázka generuje C2877:  
  
```  
// C2877.cpp  
// compile with: /c  
class A {  
private:  
   int a;  
};  
  
class B : public A {  
   using A::a;   // C2877  
};  
```
