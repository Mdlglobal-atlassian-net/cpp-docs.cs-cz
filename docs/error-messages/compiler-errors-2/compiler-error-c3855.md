# <a name="compiler-error-c3855"></a>Chyba kompilátoru C3855

' class ': parametr typu ' para ' je nekompatibilní s deklarací

Kompilátor nalezl Netypový šablonu nebo obecné parametry s různými názvy. K této chybě může dojít, pokud zadaný parametr šablony v definici specializace šablony není kompatibilní s jeho deklarací.

Následující ukázka generuje C3855:

```cpp
// C3855.cpp
template <int N>
struct C {
   void f();
};

template <char N>
void C<N>::f() {}   // C3855
```

Možné řešení:

```cpp
// C3855b.cpp
// compile with: /c
template <int N>
struct C {
   void f();
};

template <int N>
void C<N>::f() {}
```

C3855 může také nastat při použití generických typů:

```cpp
// C3855c.cpp
// compile with: /clr
generic <class T>
ref struct GC1 {
   generic <class U>
   ref struct GC2;
};

generic <class T>
generic <class U>
generic <class V>
ref struct GC1<T>::GC2 { };   // C3855
```

Možné řešení:

```cpp
// C3855d.cpp
// compile with: /clr /c
generic <class T>
ref struct GC1 {
   generic <class U>
   ref struct GC2;
};

generic <class T>
generic <class U>
ref struct GC1<T>::GC2 { };
```
