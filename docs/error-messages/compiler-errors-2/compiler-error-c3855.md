# <a name="compiler-error-c3855"></a>Chyba kompilátoru C3855

'class': parametr typu 'param' není kompatibilní s deklarací

Kompilátor najít šablony bez typu nebo obecné parametry s různými názvy. Tato situace může nastat, pokud zadaná šablona parametr v definici specializace šablony je nekompatibilní s deklarací.

Následující ukázka generuje C3855:

```
// C3855.cpp
template <int N>
struct C {
   void f();
};

template <char N>
void C<N>::f() {}   // C3855
```

Možná řešení:

```
// C3855b.cpp
// compile with: /c
template <int N>
struct C {
   void f();
};

template <int N>
void C<N>::f() {}
```

C3855 může dojít také při použití obecných typů:

```
// C3855c.cpp
// compile with: /clr
generic <class T>
ref struct GC1 {
   generic <class U>
   ref struct GC2;
};

generic <class T>
generic <class U>
generic <class V>
ref struct GC1<T>::GC2 { };   // C3855
```

Možná řešení:

```
// C3855d.cpp
// compile with: /clr /c
generic <class T>
ref struct GC1 {
   generic <class U>
   ref struct GC2;
};

generic <class T>
generic <class U>
ref struct GC1<T>::GC2 { };
```
