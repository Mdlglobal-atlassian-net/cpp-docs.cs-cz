# <a name="compiler-error-c3832"></a>Chyba kompilátoru C3832

' knihovna typů ': Knihovna typů vypadá jako v případě, že byla sestavena pro 32-bitů ukazatelů; Změňte prosím kvalifikátor ptrsize.

Explicitní informace poskytnuté atributem `ptrsize` direktivy [#import](../../preprocessor/hash-import-directive-cpp.md) nesouhlasily s tím, co kompilátor našel v knihovně typů.
