# <a name="compiler-error-c3832"></a>Chyba kompilátoru C3832

'knihovny typů': Knihovna typů vypadá, jako kdyby byla vytvořená pro 32bitové ukazatele; Změňte prosím kvalifikátor ptrsize.

Explicitní informace poskytnuté s `ptrsize` atribut [#import](../../preprocessor/hash-import-directive-cpp.md) – direktiva nedošlo ke shodě s co kompilátor našel v knihovně typů.
