# <a name="compiler-error-c2948"></a>Chyba kompilátoru C2948

explicitní vytváření instancí; specifikátor třídy úložiště není pro specializaci povolený specifikátor

Nelze použít specifikátory třídy úložiště (například `extern`) ve specializaci šablony třídy, která byla dříve explicitně vytvořena instance.
