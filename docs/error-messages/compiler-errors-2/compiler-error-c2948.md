# <a name="compiler-error-c2948"></a>C2948 chyby kompilátoru
Explicitní vytvoření instance; specifikátor třídy úložiště není povolený v specializace specifikátor  
  
 Nemůžete použít specifikátory třídy úložiště (například `extern`) v specializace šablony třídy, která byla dříve explicitně vytvořena instance.
