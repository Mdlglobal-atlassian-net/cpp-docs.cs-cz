# <a name="compiler-error-c3356"></a>Chyba kompilátoru C3356

'attribute': nejde volat atribut multicast s plně kvalifikovaným názvem

Atribut, který zpracovává více než jeden proces, například kompilátor a knihovny ATL poskytovatele byl nesprávně zadán.
