# <a name="compiler-error-c3371"></a>Chyba kompilátoru C3371

' idl_module ': tady je povolená jenom vlastnost Name.

[idl_module](../../windows/idl-module.md) použití přímo v deklaraci funkce nemůže mít jiné parametry než Name.

Následující ukázka generuje C3371:

```cpp
// C3371.cpp
[idl_module(name="Name", dllname="Some.dll")];
[idl_module(name="Name", helpstring="Some help")]   // C3371
int f1();
// try
// [idl_module(name="Name")]
// int f1();

int main()
{
}
```
