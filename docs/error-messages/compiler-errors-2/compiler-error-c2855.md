# <a name="compiler-error-c2855"></a>Chyba kompilátoru C2855

parametr příkazového řádku Optional je nekonzistentní s předkompilovanou hlavičkou.

Zadaná možnost příkazového řádku se liší od možnosti použité k vytvoření předkompilované hlavičky. Znovu vygenerujte předkompilovanou hlavičku pomocí dané možnosti příkazového řádku.
