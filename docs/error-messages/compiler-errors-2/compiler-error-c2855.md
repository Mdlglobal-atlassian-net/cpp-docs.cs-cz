# <a name="compiler-error-c2855"></a>Chyba kompilátoru C2855

možnost příkazového řádku "možnost" konzistentní s předkompilovanou hlavičkou

Daný parametr příkazového řádku se liší od možnost použít k vytvoření předkompilovaných hlaviček. Znovu vygenerovat předkompilované hlavičky s danou možnost příkazového řádku.
