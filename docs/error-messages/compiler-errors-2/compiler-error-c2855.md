# <a name="compiler-error-c2855"></a>C2855 chyby kompilátoru
možnost příkazového řádku "možnost" konzistentní s předkompilovaných hlaviček  
  
 Danou možnost příkazového řádku se liší od možnost použít k vytvoření předkompilovaných hlaviček. Znovu vygenerujte předkompilovaných hlaviček s danou možností příkazového řádku.
