# <a name="compiler-error-c2551"></a>Chyba kompilátoru C2551

typ void * potřebuje explicitní přetypování.

Pomocí implicitního převodu se přiřadí ukazatel `void` k neanulovánmu ukazateli. Je nutné použít explicitní přetypování.
