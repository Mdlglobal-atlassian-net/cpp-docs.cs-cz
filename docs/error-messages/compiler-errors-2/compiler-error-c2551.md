# <a name="compiler-error-c2551"></a>C2551 chyby kompilátoru
' void *' typ musí explicitní přetypování  
  
 A `void` ukazatel přiřadí k nonvoid ukazatel implicitní převod. Je nutné použít explicitní přetypování.
