# <a name="compiler-error-c2551"></a>Chyba kompilátoru C2551

"void *" typ vyžaduje explicitní přetypování.

A `void` přiřadí ukazatel na ukazatel nonvoid implicitní převod. Je nutné použít explicitní přetypování.
