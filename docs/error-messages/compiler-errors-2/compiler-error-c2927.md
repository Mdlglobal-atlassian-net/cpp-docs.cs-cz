# <a name="compiler-error-c2927"></a>C2927 chyby kompilátoru
'function': šablonu funkce musí být volána s alespoň jedním argumentem  
  
 Nelze volat funkci šablony bez argumentů. Typ šablony argumenty, které určuje, jaká verze funkce pro generování.
