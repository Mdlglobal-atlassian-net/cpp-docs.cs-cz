# <a name="compiler-error-c2656"></a>Chyba kompilátoru C2656

' function ': funkce není povolená jako bitové pole

Funkce je deklarována jako člen bitového pole.

Tato chyba může být způsobena chybnou syntaxí v seznamu inicializátorů konstruktoru.
