# <a name="compiler-error-c2704"></a>C2704 chyby kompilátoru
"identifikátor": __va_start vnitřní povoleny pouze v vararg  
  
 `__va_start` Vnitřní se používá v deklaraci pro funkci s pevný počet argumentů.
