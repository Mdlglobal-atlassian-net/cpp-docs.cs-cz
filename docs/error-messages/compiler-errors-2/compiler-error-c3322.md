# <a name="compiler-error-c3322"></a>Chyba kompilátoru C3322

' Property ': není platná vlastnost pro atribut ' Attribute '

Atributu byla předána neplatná nebo nerozpoznaná vlastnost (nebo parametr). Podívejte se do dokumentace k atributu.
