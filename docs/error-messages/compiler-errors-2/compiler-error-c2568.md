# <a name="compiler-error-c2568"></a>C2568 chyby kompilátoru
'identifier1': nelze vyřešit přetížení funkce  
  
 Kompilátor nemůže určit, který přetížený funkce k volání. Skutečné parametry předaný funkci musíte vysílat tak, aby odpovídaly formální parametry pro jednu přetížené funkce, ale žádná shoda jeden je jednoznačně lepší, než všechny ostatní.
