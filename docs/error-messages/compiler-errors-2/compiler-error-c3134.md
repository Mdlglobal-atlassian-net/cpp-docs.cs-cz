# <a name="compiler-error-c3134"></a>C3134 chyby kompilátoru
'Hodnota': hodnota argumentu atribut 'atribut' nemá platný typ "typ"  
  
 Chyba syntaxe byla zjištěna, když hodnota byl přiřazen do atribut.
