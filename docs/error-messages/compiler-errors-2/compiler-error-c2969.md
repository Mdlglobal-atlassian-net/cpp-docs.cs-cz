# <a name="compiler-error-c2969"></a>Chyba kompilátoru C2969

Chyba syntaxe: 'symbol': byl očekáván definice členské funkce bude končit '}'

Definice šablony členské funkce má neodpovídající uzavírací závorkou.

Následující ukázka generuje C2969:

```
// C2969.cpp
// compile with: /c
class A {
   int i;
public:
   A(int i) {}
};

A anA(1);

class B {
   A a;
   B() : a(anA);   // C2969
   // try the following line instead
   // B() : a(anA) {}
};
```
