# <a name="compiler-error-c2969"></a>C2969 chyby kompilátoru
Chyba syntaxe: 'symbol': očekávaný definice členské funkce na končit '}'  
  
 Definice šablony členské funkce má levá složená závorka neodpovídající uzavírací.  
  
 Následující ukázka generuje C2969:  
  
```  
// C2969.cpp  
// compile with: /c  
class A {  
   int i;  
public:  
   A(int i) {}  
};  
  
A anA(1);  
  
class B {  
   A a;  
   B() : a(anA);   // C2969  
   // try the following line instead  
   // B() : a(anA) {}  
};  
```
