# <a name="compiler-error-c3360"></a>Chyba kompilátoru C3360

'řetězec': Nelze vytvořit název

Hodnota, která byla předána [uuid](../../windows/uuid-cpp-attributes.md) atribut nebyl platný.

Následující ukázka generuje C3360:

```
// C3360.cpp
[ uuid("1") ]
// try this line instead
// [ uuid("12341234-1234-1234-1234-123412341234") ]
struct A   // C3360
{
};

int main()
{
}
```
