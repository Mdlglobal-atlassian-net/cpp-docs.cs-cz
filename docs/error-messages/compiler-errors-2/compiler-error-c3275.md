# <a name="compiler-error-c3275"></a>Chyba kompilátoru C3275

'člen výčtu': nelze použít tento symbol bez kvalifikátoru

Při použití spravovaného kódu a výčty nejmíň dva obsahovat identifikátor se stejným názvem, musí explicitně kvalifikovat odkazy na identifikátor.

C3275 dosažitelný pouze pomocí možnosti kompilátoru zastaralé **oldSyntax**.
