# <a name="compiler-error-c3275"></a>C3275 chyby kompilátoru
"člen výčtu": nemůžete použít tento symbol bez kvalifikátoru  
  
Při použití spravovaného kódu a obsahovat minimálně dva výčty identifikátor se stejným názvem, musíte před explicitně odkazů na identifikátor.  
  
C3275 je dostupný, pomocí možnosti zastaralé kompilátoru pouze **/clr:oldSyntax**.
