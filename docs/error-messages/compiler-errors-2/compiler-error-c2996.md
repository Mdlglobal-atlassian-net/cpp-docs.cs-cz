# <a name="compiler-error-c2996"></a>Compiler Error C2996

'function': rekurzivní definice šablony funkcí

Definice funkce se pokusí vytvořit instanci jeho kořenová třída bez vizuálního vzhledu. Nejsou povoleny rekurzivní instance šablon.
