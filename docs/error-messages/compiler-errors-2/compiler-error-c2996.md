# <a name="compiler-error-c2996"></a>C2996 chyby kompilátoru
'function': definice šablony rekurzivní funkce  
  
 Definice funkce se pokusí vytvořit instanci jeho kořenová třída podle šablony. Rekurzivní šablony konkretizací nejsou povoleny.
