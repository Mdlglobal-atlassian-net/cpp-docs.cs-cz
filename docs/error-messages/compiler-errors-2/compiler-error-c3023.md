# <a name="compiler-error-c3023"></a>Chyba kompilátoru C3023

'value': v argumentu klauzule OpenMP "klauzule" se našel neočekávaný token

Hodnoty předané do klauzule nebyly platné.

Následující ukázka generuje C3023:

```
// C3023.cpp
// compile with: /openmp /link vcomps.lib
#include <stdio.h>
#include "omp.h"

int main() {
   int i;

   #pragma omp parallel for schedule(dynamic 10)   // C3023
   for (i = 0; i < 10; ++i) ;

   #pragma omp parallel for schedule(dynamic;10)   // C3023
   for (i = 0; i < 10; ++i) ;

   // OK
   #pragma omp parallel for schedule(dynamic, 10)
   for (i = 0; i < 10; ++i)
   ;
}
```
