# <a name="compiler-error-c3187"></a>Chyba kompilátoru C3187

> "&#95;&#95;func&#95;&#95;": je dostupné jenom v těle funkce

Předdefinované makro není k dispozici mimo tělo funkce.

Chcete-li opravit tuto chybu, přesuňte – makro uvnitř těla funkce.
