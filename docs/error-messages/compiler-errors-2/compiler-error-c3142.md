# <a name="compiler-error-c3142"></a>Chyba kompilátoru C3142

'%{Property_Name/': Nelze převzít adresu proměnné vlastnost

Adresa vlastnost není k dispozici pro vývojáře.

Následující ukázka generuje C3142:

```
// C3142_2.cpp
// compile with: /clr
using namespace System;
ref class CSize {
private:
   property int Size {
      int get();
   }
};

int main() {
    &CSize::Size; // C3142
}
```
