# <a name="compiler-error-c2896"></a>Chyba kompilátoru C2896

'function1': nelze použít jako argument funkce šablony "function2"

Šablony funkcí nemůže být argument pro jinou šablonu funkce.

Následující ukázka generuje C2896:

```
// C2896.cpp
template<class T1, class T2> void f1(void(*)(T1, T2));
template<class T1, class T2> void f2(T1, T2);

int main() {
   f1(f2);   // C2896
}
```

C2896 může dojít také při použití obecných typů:

```
// C2896b.cpp
// compile with: /clr
generic<class T1> void gf1(T1){}
generic<class T1> void gf2(T1){}

int main() {
   gf1(gf2);   // C2896
   gf1(1);   // OK
}
```
