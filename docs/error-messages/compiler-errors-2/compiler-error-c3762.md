# <a name="compiler-error-c3762"></a>Chyba kompilátoru C3762

Nepovedlo se zpracovat atribut Attribute.

Uživatelsky definovaný atribut, který dědí z `System.Security.Permissions.SecurityAttribute` slouží k definování atributu zabezpečení. Takový atribut nelze použít ve stejném sestavení, kde je definován.
