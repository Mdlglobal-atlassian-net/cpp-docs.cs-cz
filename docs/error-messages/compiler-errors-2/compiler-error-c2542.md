# <a name="compiler-error-c2542"></a>Chyba kompilátoru C2542

' identifier ': objekt třídy nemá žádný konstruktor pro inicializaci

Neexistuje žádný konstruktor se seznamem parametrů, který by odpovídal inicializaci.

Tato chyba může být způsobena nesprávnými parametry při inicializaci.
