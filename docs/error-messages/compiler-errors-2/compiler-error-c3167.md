# <a name="compiler-error-c3167"></a>Chyba kompilátoru C3167

Nepovedlo se inicializovat .NET Framework: Ujistěte se, že je nainstalovaná.

.NET Framework není v tomto počítači nainstalován. Nainstalujte .NET Framework.
