# <a name="compiler-error-c3212"></a>Chyba kompilátoru C3212

'specializace': explicitní specializace členu šablony musí být členem explicitní specializace

Explicitní specializace byl chybně vytvořen.

Následující ukázka generuje C3212:

```
// C3212.cpp
// compile with: /LD
template <class T>
struct S {
   template <class T1>
   struct S1;
};

template <class T>   // C3212
template <>
struct S<T>::S1<int> {};

/*
// try the following instead
template <>
template <>
struct S<int>::S1<int> {};
*/

/*
// or, the following
template <>
struct S<int> {
   template <class T1>
   struct S1;
};

template <>
struct S<int>::S1<int> {
};
*/
```
