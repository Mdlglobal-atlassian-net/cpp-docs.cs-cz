# <a name="compiler-error-c3506"></a>Chyba kompilátoru C3506

pro LIBID ID není zaregistrovaná žádná knihovna typů (TypeLib).

Knihovna TypeLib nebyla správně zaregistrována. K registraci knihovny typů použijte Regsvr32. exe.
