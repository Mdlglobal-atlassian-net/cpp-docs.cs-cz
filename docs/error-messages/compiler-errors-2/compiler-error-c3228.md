# <a name="compiler-error-c3228"></a>Chyba kompilátoru C3228

' function ': argument obecného typu pro parametr ' param ' nemůže být ' type ', musí se jednat o typ ValueType nebo popisovač

Jako argument obecného typu byl předán nesprávný typ.

Následující ukázka generuje C3228:

```cpp
// C3228.cpp
// compile with: /clr
class A {};

value class B {};

generic <class T>
void Test() {}

ref class C {
public:
   generic <class T>
   static void f() {}
};

int main() {
   C::f<A>();   // C3228
   C::f<B>();   // OK

   Test<C>();   // C3228
   Test<C ^>();   // OK
}
```
