# <a name="compiler-error-c2654"></a>Chyba kompilátoru C2654

' identifier ': pokus o přístup k členu mimo členskou funkci

V deklaraci je k dispozici člen. K datům členů lze přistupovat pouze v členských funkcích.

Tato chyba může být způsobena při pokusu o inicializaci proměnných v deklaraci. Pro tento účel použijte konstruktor.
