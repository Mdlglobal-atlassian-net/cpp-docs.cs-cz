# <a name="compiler-error-c2667"></a>C2667 chyby kompilátoru
'function': žádná číslo přetížení nemá nejlepší převod  
  
 Volání přetížené funkce nejednoznačný a nelze ho přeložit.  
  
 Převod vyžadovanou pro shodu ve volání funkce na jednu z přetížené funkce skutečného parametry musí být striktně lepší, než převody vyžadovanou všech dalších přetížených funkcí.  
  
 Částečné řazení šablon funkcí v najdete v článku znalostní báze Knowledge Base Q240869 Další informace.
