# <a name="compiler-error-c2634"></a>C2634 chyby kompilátoru
"& class::member': ukazatel na odkaz na člena je neplatný  
  
 Ukazatele na člena, odkaz je deklarován.  
  
 Následující ukázka generuje C2634:  
  
```  
// C2634.cpp  
int mem;  
struct S {  
   S() : rf(mem) { }  
   int &rf;  
};  
int (S::*pdm) = &S::rf;   // C2634  
```
