# <a name="compiler-error-c3812"></a>C3812 chyby kompilátoru
'vlastnost' musí být první token v deklarace vlastnosti  
  
 Pokud deklarace vlastnosti, `__property` – klíčové slovo musí být první token na řádek.  
  
 C3812 je dostupný, pomocí možnosti zastaralé kompilátoru pouze **/clr:oldSyntax**.
