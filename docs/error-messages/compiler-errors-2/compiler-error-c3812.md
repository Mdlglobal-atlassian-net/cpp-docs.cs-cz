# <a name="compiler-error-c3812"></a>Chyba kompilátoru C3812

'property' musí být prvním tokenem v deklaraci vlastnosti

Při deklaraci vlastnosti, `__property` – klíčové slovo musí být prvním tokenem v řádku.

C3812 dosažitelný pouze pomocí možnosti kompilátoru zastaralé **oldSyntax**.
