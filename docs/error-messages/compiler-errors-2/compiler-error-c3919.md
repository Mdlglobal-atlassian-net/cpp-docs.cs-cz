# <a name="compiler-error-c3919"></a>Chyba kompilátoru C3919

'event_method': funkce musí mít typ 'type'

Metoda přístupového objektu události nebyl deklarován správně.

Další informace o událostech najdete v tématu [události](../../extensions/event-cpp-component-extensions.md).

Následující ukázka generuje C3919:

```
// C3919.cpp
// compile with: /clr /c
using namespace System;
delegate void D(String^);
ref class R {
   event D^ e {
      int add(int);   // C3919
      int remove(int);   // C3919

      void add(D^);   // OK
      void remove(D^);   // OK
   }
};
```
