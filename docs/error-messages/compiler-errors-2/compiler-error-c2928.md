# <a name="compiler-error-c2928"></a>Chyba kompilátoru C2928

explicitní vytváření instancí; identifikátor není funkcí nebo statickým datovým členem třídy Template-Class.

Nemůžete explicitně vytvořit instanci člena `class`, který není funkcí ani `static` proměnnou.
