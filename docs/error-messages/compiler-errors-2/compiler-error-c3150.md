'element': 'atribut' lze použít pouze ke třídě, rozhraní, pole nebo ukazatele  
  
`__gc`můžete použít pouze na třídu, rozhraní nebo pole.  
  
 C3150 je dostupný, pomocí možnosti zastaralé kompilátoru pouze **/clr:oldSyntax**.
