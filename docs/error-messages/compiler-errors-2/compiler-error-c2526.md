# <a name="compiler-error-c2526"></a>Chyba kompilátoru C2526

' identifier1 ': funkce propojení C nemůže vrátit C++ třídu ' identifier2 '.

Funkce definovaná s propojením jazyka C nemůže vracet uživatelsky definovaný typ.
