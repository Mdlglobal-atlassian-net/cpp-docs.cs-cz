# <a name="compiler-error-c2531"></a>C2531 chyby kompilátoru
"identifikátor": odkaz na trochu neplatné pole  
  
 Odkazy na bitová pole nejsou povoleny.  
  
 Následující ukázka generuje C2531:  
  
```  
// C2531.cpp  
// compile with: /c  
class P {  
   int &b1 : 10;   // C2531  
   int b2 : 10;   // OK  
};  
```
