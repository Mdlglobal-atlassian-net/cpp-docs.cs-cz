# <a name="compiler-error-c3135"></a>Chyba kompilátoru C3135

' Property ': vlastnost nemůže mít typ const nebo volatile

Klíčová slova [const](../../cpp/const-cpp.md) a [volatile](../../cpp/volatile-cpp.md) nejsou u vlastností povolena.
