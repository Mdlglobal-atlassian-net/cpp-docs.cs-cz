# <a name="compiler-error-c3135"></a>Chyba kompilátoru C3135

'property': vlastnost nemůže mít 'const' nebo zadejte typu volatile.

[Const](../../cpp/const-cpp.md) a [volatile](../../cpp/volatile-cpp.md) klíčová slova nejsou povolené u vlastností.
