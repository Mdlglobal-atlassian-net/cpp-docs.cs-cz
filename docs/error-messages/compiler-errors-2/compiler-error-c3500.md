# <a name="compiler-error-c3500"></a>Chyba kompilátoru C3500

neplatné ProgID ' ProgID '

Byl zadán neplatný identifikátor ProgID s příkazem `#import`. Zkontrolujte registr systému Windows a ujistěte se, že zadáváte platný identifikátor ProgID.
