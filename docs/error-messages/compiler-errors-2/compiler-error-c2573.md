# <a name="compiler-error-c2573"></a>Chyba kompilátoru C2573

' class ': nelze odstranit ukazatele na objekty tohoto typu; Třída nemá přetížení bez umístění pro operátor delete.

Ve třídě chybí operátor delete bez umístění.
