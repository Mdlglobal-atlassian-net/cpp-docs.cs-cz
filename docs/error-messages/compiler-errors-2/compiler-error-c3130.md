# <a name="compiler-error-c3130"></a>C3130 chyby kompilátoru
Vnitřní chyba kompilátoru: se nepodařilo zapsat blok vloženého kódu do souboru PDB  
  
 K této chybě dojde, pokud kompilátor se nepodařilo zapsat bloku vloženého kódu do souboru pdb. Nejčastější příčinou selhání je nedostatek místa na disku.
