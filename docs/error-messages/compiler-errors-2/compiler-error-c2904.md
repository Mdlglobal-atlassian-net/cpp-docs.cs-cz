# <a name="compiler-error-c2904"></a>C2904 chyby kompilátoru
"identifikátor": název se již používá pro šablonu v aktuálním oboru  
  
 Zkontrolujte kód pro duplicitní názvy.  
  
 Následující ukázka generuje C2904:  
  
```  
// C2904.cpp  
// compile with: /c  
void X();  // X is declared as a function  
template<class T> class X{};  // C2904  
```
