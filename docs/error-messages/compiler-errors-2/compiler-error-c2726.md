# <a name="compiler-error-c2726"></a>Chyba kompilátoru C2726

'gcnew' slouží jenom k vytvoření objektu s spravované nebo typ WinRT

Nelze vytvořit instanci nativního typu v haldě uvolňování paměti.

Následující ukázka generuje C2726 a ukazuje, jak ho opravit:

```
// C2726.cpp
// compile with: /clr
using namespace System;
class U {};
ref class V {};
value class W {};

int main() {
   U* pU = gcnew U;    // C2726
   U* pU2 = new U;   // OK
   V^ p2 = gcnew V;   // OK
   W p3;   // OK

}
```
