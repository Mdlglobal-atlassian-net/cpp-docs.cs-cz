# <a name="compiler-error-c2636"></a>Chyba kompilátoru C2636

'identifier': ukazatel na odkazový člen je neplatný

Ukazatel na odkazový člen je deklarovaný.

Následující ukázka generuje C2636:

```
// C2636.cpp
struct S {};
int main() {
   int &S::*prs;   // C2636
   int S::*prs1;   // OK
   int *S::*prs2;   // OK
}
```
