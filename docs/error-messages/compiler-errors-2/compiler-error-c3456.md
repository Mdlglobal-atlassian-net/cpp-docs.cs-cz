# <a name="compiler-error-c3456"></a>Chyba kompilátoru C3456

[source_annotation_attribute] není povolené pro deklaraci spravovaného nebo WinRT Class.

source_annotation_attribute slouží k definování vlastních atributů, které mají být použity nástrojem analýza kódu.

Tyto vlastní atributy jsou užitečné pouze v případě, že existuje odpovídající modul plug-in, který pochopení atributu.
