# <a name="compiler-error-c3010"></a>Chyba kompilátoru C3010

"štítek": přejít mimo strukturovaný blok OpenMP není povolený.

Kód nelze přejít do nebo z něj blok OpenMP.

Následující ukázka generuje C3010:

```
// C3010.c
// compile with: /openmp
int main() {
   #pragma omp parallel
   {
      #pragma omp parallel
      {
         goto lbl3;
      }
   }
   lbl3:;   // C3010
}
```
