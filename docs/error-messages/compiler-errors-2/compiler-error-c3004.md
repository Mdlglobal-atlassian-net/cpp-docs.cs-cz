# <a name="compiler-error-c3004"></a>Chyba kompilátoru C3004

'klauzule': není platný "direktiva" direktivy OpenMP – klauzule

OpenMP – klauzule byl použit v direktivě, pro který není povolen.

Následující ukázka generuje C3004:

```
// C3004.c
// compile with: /openmp
int main()
{
   int x, y, z;

   // Shared clause not allowed for 'single' directive.
   #pragma omp single shared(x, y)   // C3004

   x = y;
}
```
