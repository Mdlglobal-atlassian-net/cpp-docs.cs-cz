# <a name="compiler-error-c2881"></a>Chyba kompilátoru C2881

' Namespace1 ': už se používá jako alias pro ' Namespace2 '.

Pro dva obory názvů nelze použít stejný název jako alias.

Následující ukázka generuje C2881:

```cpp
// C2881.cpp
// compile with: /c
namespace A {
   int k;
}

namespace B {
   int i;
}

namespace C = A;
namespace C = B;   // C2881 C is already an alias for A
```
