# <a name="compiler-error-c3761"></a>Chyba kompilátoru C3761

' function ': ' retval ' se může vyskytovat pouze v posledním argumentu funkce

Atribut [retval](../../windows/retval.md) byl použit u argumentu funkce, který nebyl posledním argumentem v seznamu.

Následující ukázka generuje C3761:

```cpp
// C3761.cpp
#define _ATL_ATTRIBUTES 1
#include <atlbase.h>
#include <atlcom.h>

[ module(name=test) ];

[dispinterface]
__interface I
{
   [id(1)] HRESULT func([out, retval] int* i, [in] int j);
   // try the following line instead
   // [id(1)] HRESULT func([in] int i, [out, retval] int* j);
};

[coclass]
struct C : I {   // C3761
   HRESULT func(int* i, int j)
   // try the following line instead
   // HRESULT func(int j, int* i)
   {
      return S_OK;
   }
};

int main()
{
}
```
