# <a name="compiler-error-c3501"></a>Chyba kompilátoru C3501

pro ProgID ' ProgID ' není registrována žádná knihovna TypeLib

ID třídy pro daný identifikátor ProgID nemá přidruženou knihovnu typů. Proto nemůžete předat tento identifikátor ProgID do příkazu `#import`.
