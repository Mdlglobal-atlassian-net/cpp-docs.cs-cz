# <a name="compiler-error-c2529"></a>C2529 chyby kompilátoru
"název": na odkaz je neplatný  
  
 Tato chyba může být stanovena pomocí syntaxe ukazatele a deklarace odkaz na ukazatel.  
  
 Následující ukázka generuje C2529:  
  
```  
// C2529.cpp  
// compile with: /c  
int i;  
int &ri = i;  
int &(&rri) = ri;   // C2529  
```
