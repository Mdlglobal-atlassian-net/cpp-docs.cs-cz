# <a name="compiler-error-c3013"></a>Chyba kompilátoru C3013

"klauzule!: klauzule se může vyskytovat jenom jednou na"direktiva"OpenMP – direktiva

Klauzule zobrazovaly dvakrát pro stejné direktivu. Odstraňte jeden výskyt klauzule.

Následující ukázka generuje C3013:

```
// C3013.cpp
// compile with: /openmp
int main() {
   int a, b, c, x, y, z;

   #pragma omp parallel shared(a,b,c) private(x)

   #pragma omp for nowait private(x) nowait   // C3013
   // The previous line generates C3013, with two nowait clauses
   // try the following line instead:
   // #pragma omp for nowait private(x)
   for (a = 0 ; a < 10 ; ++a) {
   }
}
```
