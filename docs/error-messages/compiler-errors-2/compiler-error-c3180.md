# <a name="compiler-error-c3180"></a>C3180 chyby kompilátoru
"název typu": název překračuje limit metadata znaků 'limit.  
  
 Kompilátor zkrácen název spravovaného typu v metadatech. Zkrácení bude nepoužitelnost typ s `#using` – direktiva (nebo ekvivalent v jiném jazyce).  
  
 Název typu omezení se vztahuje na všechny kvalifikaci oboru názvů.
