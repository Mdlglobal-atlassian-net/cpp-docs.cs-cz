# <a name="compiler-error-c2876"></a>C2876 chyby kompilátoru
'class::symbol': Ne všechny přetížení, které jsou dostupné  
  
 Všechny přetížené forms funkce v základní třídě musí být přístupný pro odvozené třídy.  
  
 Následující ukázka generuje C2876:  
  
```  
// C2876.cpp  
// compile with: /c  
class A {  
public:     
   double a(double);  
private:  
   int a(int);  
};  
  
class B : public A {  
   using A::a;   // C2876 one overload is private in base class  
};  
```
