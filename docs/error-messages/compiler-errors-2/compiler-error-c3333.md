# <a name="compiler-error-c3333"></a>C3333 chyby kompilátoru
'knihovny typů': nelze #import poškozený typ knihovny  
  
 Knihovnu typů určenou ve `#import` příkaz nečitelný kompilátorem. Můžete buď obnovit knihovny typů, pokud je to možné, nebo požádat svého dodavatele nové knihovny typů. Můžete použít OLE prohlížeč, dodává s Visual C++, chcete-li zobrazit typ soubor knihovny a zjistěte, co je věci s ním.
