# <a name="compiler-error-c3189"></a>Chyba kompilátoru C3189

"typeid\<zadejte abstraktním deklarátorem >': Tato syntaxe už není podporovaná, použijte:: typeid místo

Zastaralé formy [typeid](../../windows/typeid-cpp-component-extensions.md) byl použit, použít nový formulář.

Následující ukázka generuje C3189:

```
// C3189.cpp
// compile with: /clr
int main() {
   System::Type^ t  = typeid<System::Object>;   // C3189
   System::Type^ t2  = System::Object::typeid;   // OK
}
```
