# <a name="compiler-error-c3860"></a>Chyba kompilátoru C3860

Typ seznamu argumentů za názvem typu třídy musí být parametry seznamu v pořadí používaném v seznamu parametrů typu

Rozvrhy generic nebo šablony seznam argumentů byl chybně vytvořen.

Následující ukázka generuje C3860:

```
// C3860.cpp
// compile with: /LD
template <class T1, class T2>
struct A {
   void f();
};

template <class T2, class T1>
void A<T1, T2>::f() {}   // C3860
```

Možná řešení:

```
// C3860b.cpp
// compile with: /c
template <class T1, class T2>
struct A {
   void f();
};

template <class T2, class T1>
void A<T2, T1>::f() {}
```

C3860 může dojít také při použití obecných typů:

```
// C3860c.cpp
// compile with: /clr
generic<class T,class U>
ref struct GC {
   void f();
};

generic<class T, class U>
void GC<T,T>::f() {}   // C3860
```

Možná řešení:

```
// C3860d.cpp
// compile with: /clr /c
generic<class T,class U>
ref struct GC {
   void f();
};

generic<class T, class U>
void GC<T,U>::f() {}
```
