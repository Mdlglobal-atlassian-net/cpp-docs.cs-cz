# <a name="compiler-error-c3396"></a>Chyba kompilátoru C3396

' Attribute ': v ' namespace ' se nenašel vlastní atribut.

C3396 označuje, že nepoužíváte aktuální modul CLR (Common Language Runtime).  Přeinstalujte modul CLR, aby se vyřešil.
