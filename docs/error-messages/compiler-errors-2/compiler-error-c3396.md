# <a name="compiler-error-c3396"></a>C3396 chyby kompilátoru
'atribut': vlastní atribut nebyl nalezen v atribut namespace.  
  
 C3396 označuje, že nepoužíváte aktuální modulu CLR.  Přeinstalujte vaší CLR vyřešit.
