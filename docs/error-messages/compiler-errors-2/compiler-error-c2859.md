# <a name="compiler-error-c2859"></a>C2859 chyby kompilátoru
Název souboru není typ souboru, který byl použit v okamžiku vytvoření této předkompilovaných hlaviček, znovu vytvořte předkompilovaných hlaviček.  
  
 Projekt databáze a soubory předkompilovaných hlaviček musí být vytvořeny současně zajistit konzistentní informace. Znovu sestavte projekt znovu vytvořit předkompilovaných hlaviček.  
  
 Informace o předkompilovaných hlaviček v tématu [/Y (předkompilované hlavičky)](../../build/reference/y-precompiled-headers.md).
