# <a name="compiler-error-c2843"></a>Chyba kompilátoru C2843

'member': nejde adresovat nestatický datový člen nebo metodu spravované nebo typu WinRT

Instance je potřebné k provedení adresu nestatické datové členy spravované nebo WinRT třídu nebo rozhraní.

Následující ukázka generuje C2843 a ukazuje, jak ho opravit:

```
// C2843_2.cpp
// compile with: /clr
public ref class C {
public:
   int m_i;
};

ref struct MyStruct {
   static void sf() {}
   void f() {}
};

int main() {
   MyStruct ^ps = gcnew MyStruct;
   void (__clrcall MyStruct::*F1)() = & MyStruct::f;   // C2843
   void (__clrcall MyStruct::*F2)() = & ps->f;   // C2843
   void (__clrcall MyStruct::*F3)();   // C2843

   void (__clrcall *F5)() = MyStruct::sf;   // OK
   void (__clrcall *F6)() = & ps->sf;   // OK

   interior_ptr<int> i = &C::m_i; // C2843
   C ^x = gcnew C();
   interior_ptr<int> ii = &x->m_i;
}
```
