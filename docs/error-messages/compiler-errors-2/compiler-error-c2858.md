# <a name="compiler-error-c2858"></a>Chyba kompilátoru C2858

možnost příkazového řádku '/Yc (/Fdfilename) ' není konzistentní s předkompilovanou hlavičkou, která používala '/Fdfilename '

Databáze programu určená možností použít předkompilovanou hlavičku ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) není ta, která je určena předchozí možností vytvoření předkompilované hlavičky ([/YC](../../build/reference/yc-create-precompiled-header-file.md)).
