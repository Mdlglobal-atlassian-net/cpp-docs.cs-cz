# <a name="compiler-error-c2858"></a>C2858 chyby kompilátoru
možnost příkazového řádku ' /Yc (nebo Fdfilename) se konzistentní s předkompilovaných hlaviček, který používá se nebo Fdfilename.  
  
 Do programu databáze určené parametrem použití předkompilovaných hlaviček ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) možnost není určenému předchozí vytvořit předkompilovaných hlaviček ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) možnost.
