# <a name="compiler-error-c2858"></a>C2858 chyby kompilátoru

možnost příkazového řádku "/Yc (/ Fdfilename)" konzistentní s předkompilovanou hlavičkou, která používá "/ Fdfilename.

Použití předkompilovaných hlaviček určené databázi programu ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) možnost není určeno předchozí vytváření předkompilované hlavičky ([/Yc](../../build/reference/yc-create-precompiled-header-file.md)) možnost.
