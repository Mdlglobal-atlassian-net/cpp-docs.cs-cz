# <a name="compiler-error-c2839"></a>Chyba kompilátoru C2839

Neplatný typ vrácené vlastnosti pro přetížený operátor->

Operátor `->` musí vracet třídu, strukturu nebo sjednocení nebo odkaz na jednu.
