# <a name="compiler-error-c2906"></a>Chyba kompilátoru C2906

'specializace': explicitní specializace vyžaduje "<> šablony.

Je nutné použít novou syntaxi pro explicitní specializace šablon.

Následující ukázka generuje C2906:

```
// C2906.cpp
// compile with: /c
template<class T> class X{};   // primary template
class X<int> { }   // C2906
template<> class X<int> { };   // new syntax
```
