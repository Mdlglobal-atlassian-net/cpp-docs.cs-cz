# <a name="compiler-error-c3232"></a>C3232 chyby kompilátoru
'param': Parametr obecného typu nelze použít kvalifikovaný název  
  
 Parametr obecného typu nebyl použit správně.  
  
 Následující ukázka generuje C3232:  
  
```  
// C3232.cpp  
// compile with: /clr  
generic <class T>  
ref class C {  
   typename T::TYPE t;   // C3232  
};  
```
