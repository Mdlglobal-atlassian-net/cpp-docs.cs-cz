# <a name="compiler-error-c2873"></a>Chyba kompilátoru C2873

' symbol ': symbol nemůže být použit v deklaracích using-

V direktivě `using` chybí klíčové slovo [Namespace](../../cpp/namespaces-cpp.md) . To způsobí, že kompilátor interpretuje kód jako [deklaraci using](../../cpp/using-declaration.md) místo [direktivy using](../../cpp/namespaces-cpp.md#using_directives).
