# <a name="compiler-error-c2873"></a>C2873 chyby kompilátoru
'symbol': symbol nelze použít v pomocí deklarace  
  
 A `using` chybí směrnice [obor názvů](../../cpp/namespaces-cpp.md) – klíčové slovo. To způsobí, že kompilátor nesprávně interpretovat jako kód [pomocí deklarace](../../cpp/using-declaration.md) ne [using – direktiva](../../cpp/namespaces-cpp.md#using_directives).
