# <a name="compiler-error-c2934"></a>Chyba kompilátoru C2934

' class ': typ-class-ID se předefinovalo jako vnořená ' Item ' identifikátoru '

Jako vnořenou položku nelze použít obecnou třídu nebo třídu šablony.
