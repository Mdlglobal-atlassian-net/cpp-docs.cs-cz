# <a name="compiler-warning-level-1-c4811"></a>Upozornění kompilátoru (úroveň 1) C4811

Hodnota direktivy pragma shody (forScope, show) = = Value

Toto upozornění se vydá, když použijete možnost **Zobrazit** direktivu pragma [shody](../../preprocessor/conform.md) . *hodnota* je aktuální hodnota shody.
