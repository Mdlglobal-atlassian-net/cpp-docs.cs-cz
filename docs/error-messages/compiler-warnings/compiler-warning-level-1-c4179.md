# <a name="compiler-warning-level-1-c4179"></a>C4179 kompilátoru upozornění (úroveň 1)

> ' / / *': analyzovat jako '/' a ' /\*': záměně se standardní ' / /' komentáře  
  
__//\*__ je nesprávné komentář oddělovač. Použití __//__ nebo __/ \*__ místo.
