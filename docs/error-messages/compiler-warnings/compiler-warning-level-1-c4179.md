# <a name="compiler-warning-level-1-c4179"></a>Upozornění kompilátoru (úroveň 1) C4179

> '//* ': analyzováno jako '/' a '/\*': záměna se standardními komentáři '/'

__//\*__ je nesprávný oddělovač komentářů. Místo toho použijte __//__ nebo __/\*__ .
