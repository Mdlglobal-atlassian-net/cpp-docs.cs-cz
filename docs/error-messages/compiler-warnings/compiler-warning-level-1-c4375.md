# <a name="compiler-warning-level-1-c4375"></a>C4375 kompilátoru upozornění (úroveň 1)
Metoda neveřejný 'method2' nepřepisuje 'method1.  
  
 Typ, který implementuje jiný typ definované metodu přepsat, ale přepsání nebyla veřejné. Metoda proto nemůže přepsat metodu základního typu.
