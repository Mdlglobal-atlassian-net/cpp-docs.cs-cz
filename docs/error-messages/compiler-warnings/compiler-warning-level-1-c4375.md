# <a name="compiler-warning-level-1-c4375"></a>Upozornění kompilátoru (úroveň 1) C4375

neveřejná metoda "Method2" nepřepisuje "– metoda1"

Typ, který implementuje jiný typ, definoval metodu přepsání, ale přepsání není veřejné. Proto metoda nepřepisuje metodu základního typu.
