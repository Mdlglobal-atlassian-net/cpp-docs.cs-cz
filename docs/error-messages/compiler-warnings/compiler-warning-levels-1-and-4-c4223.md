# <a name="compiler-warning-levels-1-and-4-c4223"></a>Upozornění kompilátoru (úrovně 1 a 4) C4223

používá se nestandardní rozšíření: pole, které není typu lvalue, bylo převedeno na ukazatel.

V jazyce Standard C nelze převést pole, které není lvalue, na ukazatel. S výchozími rozšířeními Microsoft ([/ze](../../build/reference/za-ze-disable-language-extensions.md)) můžete.
