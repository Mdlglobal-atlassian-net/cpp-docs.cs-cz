# <a name="compiler-warning-level-1-c4031"></a>Kompilátor upozornění (úroveň 1) C4031

druhý seznam formálních parametrů delší než první seznam.

Funkce deklarováno s různými formální parametry. Kompilátor používá formální parametry zadané v první deklaraci.
