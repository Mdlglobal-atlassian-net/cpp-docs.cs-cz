# <a name="compiler-warning-level-1-c4113"></a>Upozornění kompilátoru (úroveň 1) C4113

' identifier1 ' se v seznamech parametrů liší od ' identifier2 '

Ukazatel na funkci je přiřazen k jinému ukazateli funkce, ale seznamy formálních parametrů těchto funkcí nesouhlasí. Přiřazení je zkompilováno beze změny.
