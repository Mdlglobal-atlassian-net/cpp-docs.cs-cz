# <a name="compiler-warning-level-1-c4113"></a>Kompilátor upozornění (úroveň 1) C4113

"identifier1" se liší v seznamech parametrů z "identifier2.

Ukazatel na funkci je přiřazena k jinému ukazateli na funkci, ale seznam formálních parametrů funkce vzájemně nesouhlasí. Přiřazení je kompilován bez úprav.
