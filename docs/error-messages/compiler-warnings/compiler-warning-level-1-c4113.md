# <a name="compiler-warning-level-1-c4113"></a>C4113 kompilátoru upozornění (úroveň 1)
"identifier1" se liší v seznamy parametrů z 'identifier2.  
  
 Ukazatel na funkci je přiřazen k jiné ukazatel na funkci, ale seznamy formální parametr funkce si neodpovídají. Přiřazení je kompilovat bez úprav.
