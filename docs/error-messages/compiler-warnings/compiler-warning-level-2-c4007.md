# <a name="compiler-warning-level-2-c4007"></a>C4007 kompilátoru upozornění (úroveň 2)
'function': musí být 'atributu.  
  
 Povinný atribut pro funkci není výslovně uvedeny. Například funkce **hlavní** musí mít `__cdecl` atribut. Kompilátor vynutí atribut.
