# <a name="compiler-warning-level-4-c4206"></a>Upozornění kompilátoru (úroveň 4) C4206

**používá se nestandardní rozšíření: Překladová jednotka je prázdná.**

Soubor byl po předběžném zpracování prázdný.

Toto rozšíření může zabránit přenosům kódu do jiných kompilátorů. Vygeneruje chybu v rámci kompatibility ANSI ([/za](../../build/reference/za-ze-disable-language-extensions.md)) a vztahuje se pouze na zdrojový kód jazyka C.
