# <a name="compiler-warning-level-1-c4025"></a>C4025 kompilátoru upozornění (úroveň 1)
"číslo": na základě ukazatel předaný funkci s proměnné argumenty: Parametr číslo  
  
 Předávání na základě ukazatel na funkci s proměnné argumenty způsobí, že má ukazatel na budou normalizovány s vést k neočekávaným výsledkům. Základní ukazatelé na funkce pomocí proměnné argumenty nepředávejte.
