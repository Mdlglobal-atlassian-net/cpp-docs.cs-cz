# <a name="compiler-warning-level-1-c4025"></a>Upozornění kompilátoru (úroveň 1) C4025

' Number ': ukazatel založený na funkci předaný funkci s argumenty proměnné: číslo parametru

Předání ukazatele na základě funkce s proměnnými argumenty způsobí, že bude ukazatel normalizován, s nepředvídatelnými výsledky. Nedávejte ukazatelům na funkce s proměnnými argumenty.
