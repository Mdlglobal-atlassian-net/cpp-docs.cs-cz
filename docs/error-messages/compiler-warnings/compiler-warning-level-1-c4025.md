# <a name="compiler-warning-level-1-c4025"></a>Kompilátor upozornění (úroveň 1) C4025

'number': ukazatel based předaný funkci s argumenty proměnné: Parametr číslo

Předání ukazatele na funkci s proměnnými argumenty způsobí, že ukazatel normalizovány s nepředvídatelnými výsledky. Nepředávejte ukazatelé na funkce s proměnnými argumenty.
