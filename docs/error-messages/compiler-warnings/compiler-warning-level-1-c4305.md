 Identifikátor je převést na menší typu, což vede ke ztrátě informací.  
  
 Pokud jste použili Průvodce bodu připojení ATL, najdete v článku znalostní báze:  
  
-   Q250847: Průvodce bodu připojení ATL generovaného kódu událost s VARIANT Argument dává C4305 upozornění
