# <a name="compiler-warning-level-1-c4344"></a>C4344 kompilátoru upozornění (úroveň 1)
Změna v chování: použijte explicitní šablony argumenty výsledků volání 'function'.  
  
 Volání funkce pomocí explicitní šablony argumenty volá jinou funkci, než by tomu bylo, kdyby nebyl zadán explicitní argumenty
