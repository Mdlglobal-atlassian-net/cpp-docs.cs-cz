# <a name="compiler-warning-level-1-c4344"></a>Kompilátor upozornění (úroveň 1) C4344

Změna chování: použití výsledků šablony explicitní argumenty ve volání funkce 'function'

Volání funkce použitím explicitní argumenty šablony volá jinou funkci, než by tomu bylo, kdyby nebyl zadán explicitní argumenty
