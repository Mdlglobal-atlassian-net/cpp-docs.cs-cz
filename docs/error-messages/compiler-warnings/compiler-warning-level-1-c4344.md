# <a name="compiler-warning-level-1-c4344"></a>Upozornění kompilátoru (úroveň 1) C4344

Změna chování: použití explicitních argumentů šablony má za následek volání funkce Function.

Volání funkce pomocí explicitních argumentů šablony volá jinou funkci, než kdyby byla zadána explicitní argumenty.
