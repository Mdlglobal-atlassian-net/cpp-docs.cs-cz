# <a name="compiler-warning-level-1-c4034"></a>C4034 kompilátoru upozornění (úroveň 1)
sizeof – vrátí hodnotu 0  
  
 `sizeof` Operátor se použije pro operand velikosti nula (prázdnou strukturou, union, třída nebo výčtového typu nebo typ `void`).
