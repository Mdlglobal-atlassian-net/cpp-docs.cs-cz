# <a name="compiler-warning-level-1-c4034"></a>Kompilátor upozornění (úroveň 1) C4034

sizeof vrací 0

`sizeof` Operátor je použít pro operand o velikosti nula (prázdná struktura, sjednocení, třídy nebo výčtového typu nebo typu `void`).
