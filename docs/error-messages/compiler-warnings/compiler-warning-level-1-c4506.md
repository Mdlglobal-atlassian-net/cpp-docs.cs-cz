# <a name="compiler-warning-level-1-c4506"></a>Upozornění kompilátoru (úroveň 1) C4506

žádná definice pro vloženou funkci Function

Daná funkce byla deklarována a označena pro vkládání, ale nebyla definována.

Kompilátor nevloží funkci.

Ujistěte se, že externí funkce, které mají být vloženy, jsou deklarovány pomocí klíčového slova `extern`.
