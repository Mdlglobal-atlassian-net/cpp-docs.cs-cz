# <a name="compiler-warning-level-1-c4506"></a>Kompilátor upozornění (úroveň 1) C4506

definice pro vložené funkce 'function'

Dané funkce byla deklarována a označen pro vkládání ale nebyl definován.

Kompilátor nebyla vložená funkce.

Ujistěte se, že externí funkce se nedá vložit. jsou deklarovány pomocí `extern` – klíčové slovo.
