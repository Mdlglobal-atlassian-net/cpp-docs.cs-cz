# <a name="compiler-warning-level-1-c4506"></a>C4506 kompilátoru upozornění (úroveň 1)
definice pro vložená funkce 'function'.  
  
 Danou funkci byl deklarován a označeny vložené však nebyl definován.  
  
 Kompilátor nebyla vnořené funkce.  
  
 Ujistěte se, že jsou externí funkce jako vložená deklarovat s `extern` – klíčové slovo.
