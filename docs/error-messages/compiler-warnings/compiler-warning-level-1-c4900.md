# <a name="compiler-warning-level-1-c4900"></a>Upozornění kompilátoru (úroveň 1) C4900

Neshoda mezijazyku mezi "tool1" verze "version1" a "Tool2" verze "Version2"

Převodní jazyk používaný v *tool1* a *Tool2* se neshodoval. Ověřte, jestli je nainstalovaná nejnovější verze každého nástroje.
