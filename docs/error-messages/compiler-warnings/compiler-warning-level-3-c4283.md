# <a name="compiler-warning-level-3-c4283"></a>Kompilátor upozornění (úroveň 3) C4283

a až po typ 'type'

Tato pokračování upozornění [C4281](../../error-messages/compiler-warnings/compiler-warning-level-3-c4281.md) ukazuje, že **operator ->** zavolá sama sebe prostřednictvím `type`.
