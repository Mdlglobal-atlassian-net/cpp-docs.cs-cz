# <a name="compiler-warning-level-1-c4374"></a>C4374 kompilátoru upozornění (úroveň 1)
'function1': nesmí být implementována metoda rozhraní nevirtuálních metodou "funkce2"  
  
 Kompilátor se očekává nalezení [virtuální](../../cpp/virtual-specifier.md) – klíčové slovo v definici metody.  
  
 Následující ukázka generuje C4374:  
  
```  
// C4374.cpp  
// compile with: /clr /W1 /c /WX  
public interface class I {  
   void f();  
};  
  
public ref struct B {  
   void f() {  
      System::Console::WriteLine("B::f()");  
   }  
};  
  
public ref struct C {  
   virtual void f() {  
      System::Console::WriteLine("C::f()");  
   }  
};  
  
public ref struct D : B, I {};   // C4374  
public ref struct E : C, I {};   // OK  
```
