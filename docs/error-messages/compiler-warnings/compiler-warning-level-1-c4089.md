# <a name="compiler-warning-level-1-c4089"></a>Upozornění kompilátoru (úroveň 1) C4089

' function ': různé typy ve skutečném parametru ' Number ', formální parametr ' Number '

Odpovídající formální a skutečné parametry mají různé typy. Skutečný parametr se předává beze změny. Funkce přetypování skutečný parametr na typ zadaný v definici funkce.
