# <a name="compiler-warning-level-3-c4013"></a>Upozornění kompilátoru (úroveň 3) C4013

funkce undefined; Předpokládá se extern vracející int.

Kompilátor zaznamenal volání nedefinované funkce.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Opravu provedete kontrolou následujících možných příčin.

1. Nesprávný pravopis názvu funkce

1. Externí funkce nejsou prototypované jako `extern`
