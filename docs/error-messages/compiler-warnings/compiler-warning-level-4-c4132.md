# <a name="compiler-warning-level-4-c4132"></a>Kompilátor upozornění (úroveň 4) C4132

'object': je třeba inicializovat objekt const

Konstanty není inicializován. Hodnota symbol s **const** atributu nelze změnit po vlastní definici, by měly mít inicializační hodnotu.
