# <a name="compiler-warning-level-1-c4182"></a>Upozornění kompilátoru (úroveň 1) C4182

\#zahrnout úroveň vnoření je "Number" hluboká; možné nekonečné rekurze

Kompilátor nemá dostatek místa na haldě z důvodu počtu vnořených souborů include. Soubor zahrnutí je vnořený, když je zahrnutý z jiného vloženého souboru.

Tato zpráva je informativní a předá chybu [C1076](../../error-messages/compiler-errors-1/fatal-error-c1076.md).
