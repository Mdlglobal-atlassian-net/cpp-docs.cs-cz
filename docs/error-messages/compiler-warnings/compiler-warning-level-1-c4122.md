# <a name="compiler-warning-level-1-c4122"></a>Upozornění kompilátoru (úroveň 1) C4122

' function ': alloc_text platí pouze pro funkce s propojením jazyka C

Direktiva pragma **alloc_text** se vztahuje pouze na funkce deklarované s **extern c**. Nedá se použít s externími C++ funkcemi.

Direktiva pragma je ignorována.
