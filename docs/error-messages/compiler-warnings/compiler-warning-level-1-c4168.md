# <a name="compiler-warning-level-1-c4168"></a>Upozornění kompilátoru (úroveň 1) C4168

limit kompilátoru: nedostatek typů ladicího programu, odstraňte databázi aplikace a znovu sestavte.

Soubor databáze programu je nutné znovu sestavit tak, aby vyhovoval všem typům v programu.
