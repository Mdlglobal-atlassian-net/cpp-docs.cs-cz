# <a name="compiler-warning-level-1-c4264"></a>Kompilátor upozornění (úroveň 1) C4264

'virtual_function': přepsání není k dispozici pro virtuální členskou funkci ze základní třídy 'class'; funkce je skrytá.

C4264 generováno vždy po [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).

Toto upozornění je vypnuto ve výchozím nastavení. Zobrazit [kompilátoru upozornění, že je vypnuto ve výchozím nastavení](../../preprocessor/compiler-warnings-that-are-off-by-default.md) Další informace.
