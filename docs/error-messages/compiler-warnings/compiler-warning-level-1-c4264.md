# <a name="compiler-warning-level-1-c4264"></a>Upozornění kompilátoru (úroveň 1) C4264

' virtual_function ': není k dispozici žádné přepsání pro virtuální členskou funkci ze základní třídy ' class '; funkce je skrytá.

C4264 se vždy generuje za [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).

Toto upozornění je ve výchozím nastavení vypnuté. Další informace najdete v tématu [Upozornění kompilátoru, která jsou ve výchozím nastavení vypnutá](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
