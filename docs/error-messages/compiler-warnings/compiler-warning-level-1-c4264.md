# <a name="compiler-warning-level-1-c4264"></a>C4264 kompilátoru upozornění (úroveň 1)
'virtual_function': přepsání není k dispozici pro člena virtuální funkci z základní 'class'.; funkce je skrytá.  
  
 C4264 vždy po vygenerování [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).  
  
 Toto upozornění je ve výchozím nastavení vypnutý. V tématu [kompilátoru upozornění, že jsou vypnout ve výchozím nastavení](../../preprocessor/compiler-warnings-that-are-off-by-default.md) Další informace.
