# <a name="compiler-warning-level-1-c4022"></a>Upozornění kompilátoru (úroveň 1) C4022

' function ': Neshoda ukazatelů pro skutečný parametr ' Number '

Typ ukazatele vlastního parametru se liší od typu ukazatele odpovídajícího formálního parametru. Skutečný parametr se předává beze změny.
