# <a name="compiler-warning-level-1-c4153"></a>Upozornění kompilátoru (úroveň 1) C4153

převod ukazatele funkce/dat ve výrazu

Ukazatel na funkci je převeden na nebo z ukazatele na data. Tento převod je povolený v rozšíření Microsoft Extensions (/ze), ale ne v ANSI C.
