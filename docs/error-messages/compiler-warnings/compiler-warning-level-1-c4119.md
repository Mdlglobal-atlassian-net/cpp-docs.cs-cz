# <a name="compiler-warning-level-1-c4119"></a>Upozornění kompilátoru (úroveň 1) C4119

jsou zadané různé základy ' Base1 – ' a ' base2 – '.

Dva ukazatelé na bázi jsou nekompatibilní, protože mají různé základy. Kompilátor nemůže mezi nimi převádět.
