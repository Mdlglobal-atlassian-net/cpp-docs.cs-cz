# <a name="compiler-warning-level-1-c4119"></a>Kompilátor upozornění (úroveň 1) C4119

base1 různých základů "–" a base2 "–" zadaný

Dvě základní ukazatele nejsou kompatibilní, protože mají jiný základních tříd. Kompilátor nelze převést mezi nimi.
