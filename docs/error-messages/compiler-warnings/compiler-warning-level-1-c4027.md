# <a name="compiler-warning-level-1-c4027"></a>Upozornění kompilátoru (úroveň 1) C4027

funkce deklarovaná bez seznamu formálních parametrů

Deklarace funkce nemá žádné formální parametry, ale v definici funkce nebo ve skutečných parametrech volání existují formální parametry. Následná volání této funkce předpokládají, že funkce přebírá skutečné parametry typů nalezené v definici funkce nebo volání.
