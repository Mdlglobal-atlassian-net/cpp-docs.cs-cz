# <a name="compiler-warning-level-1-c4821"></a>C4821 kompilátoru upozornění (úroveň 1)  
  
> Nelze určit typ kódování Unicode, uložte soubor s podpisem (BOM)  
  
Kompilátor nelze určit typ kódování souboru. Chcete-li vyřešit toto upozornění, uložte soubor s značku pořadí bajtů. V tématu [Správa souborů s kódování](/sql/ssms/solution/manage-files-with-encoding) Další informace.
