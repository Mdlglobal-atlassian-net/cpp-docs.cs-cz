# <a name="compiler-warning-level-1-c4821"></a>Kompilátor upozornění (úroveň 1) C4821

> Nepovedlo se zjistit typ kódování Unicode, uložte prosím soubor s podpisem (BOM)

Kompilátor nemůže určit typ kódování souboru. Pokud chcete vyřešit toto upozornění, uložte soubor s značku pořadí bajtů. Zobrazit [Správa souborů s kódováním](/sql/ssms/solution/manage-files-with-encoding) Další informace.
