# <a name="compiler-warning-level-1-c4821"></a>Upozornění kompilátoru (úroveň 1) C4821

> Nejde určit typ kódování Unicode, uložte prosím soubor s signaturou (BOM).

Kompilátor nemohl určit typ kódování pro soubor. Chcete-li vyřešit toto upozornění, uložte soubor s označením pořadí bajtů. Další informace najdete v tématu [Správa souborů s kódováním](/sql/ssms/solution/manage-files-with-encoding) .
