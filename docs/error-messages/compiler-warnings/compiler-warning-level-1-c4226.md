# <a name="compiler-warning-level-1-c4226"></a>C4226 kompilátoru upozornění (úroveň 1)

> nestandardní rozšíření používané: '*– klíčové slovo*' je zastaralý klíčové slovo

Aktuální verze Visual C++ nepoužívá toto klíčové slovo.

Toto upozornění je automaticky povýšen na chybu.
