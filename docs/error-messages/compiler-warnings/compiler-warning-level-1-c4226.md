# <a name="compiler-warning-level-1-c4226"></a>Kompilátor upozornění (úroveň 1) C4226

> používá se nestandardní rozšíření: '*– klíčové slovo*"je zastaralé klíčové slovo

Aktuální verze aplikace Visual C++ nepoužívá toto klíčové slovo.

Toto upozornění je automaticky povýšen na chybu.
