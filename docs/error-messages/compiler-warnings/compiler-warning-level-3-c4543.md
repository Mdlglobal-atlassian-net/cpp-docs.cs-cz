# <a name="compiler-warning-level-3-c4543"></a>Kompilátor upozornění (úroveň 3) C4543

Vložený text se potlačil na základě atributu no_injected_text.

[No_injected_text –](../../windows/no-injected-text.md) atribut zobrazovaly ve zdrojovém kódu, což znamená, že kompilátor zabrání atributy vkládání kódu.

Toto upozornění se vás upozorní, že atributy nebude možné vložit kód.
