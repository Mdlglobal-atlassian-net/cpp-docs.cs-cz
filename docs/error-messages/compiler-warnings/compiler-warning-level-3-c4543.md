# <a name="compiler-warning-level-3-c4543"></a>Upozornění kompilátoru (úroveň 3) C4543

Vložený text potlačený atributem no_injected_text

Atribut [No_injected_text](../../windows/no-injected-text.md) se objevil ve zdrojovém kódu, což znamená, že kompilátor nebude bránit v vkládání kódu.

Toto upozornění vás upozorní na to, že atributy nebudou moci vložit kód.
