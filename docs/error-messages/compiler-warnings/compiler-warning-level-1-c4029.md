# <a name="compiler-warning-level-1-c4029"></a>Kompilátor upozornění (úroveň 1) C4029

deklarovaný seznam formálních parametrů liší od definice

Typy formálních parametrů v deklaraci funkce nesouhlasí s těmi v definici funkce. Kompilátor používá seznam parametrů z definice.
