# <a name="compiler-warning-level-1-c4029"></a>C4029 kompilátoru upozornění (úroveň 1)
deklarovaný formální parametr seznamu liší od definice  
  
 Formální parametr typy v deklaraci funkce nejsou shodná s těmi, která v definici funkce. Kompilátor používá seznam parametrů z definice.
