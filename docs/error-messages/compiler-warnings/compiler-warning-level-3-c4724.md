# <a name="compiler-warning-level-3-c4724"></a>Upozornění kompilátoru (úroveň 3) C4724

potenciální mod do 0

Druhý operand ve zbývající operaci byl v době kompilace vyhodnocen jako nula a poskytuje nedefinované výsledky.
