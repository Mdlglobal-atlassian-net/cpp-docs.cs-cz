# <a name="compiler-warning-level-1-c4175"></a>Upozornění kompilátoru (úroveň 1) C4175

komponenta \#pragma (v prohlížeči): na příkazovém řádku musí být nejdřív zadané informace o prohlížeči.

Chcete-li použít direktivu pragma [komponenty](../../preprocessor/component.md) , je nutné vygenerovat informace o procházení během kompilace ([/fr](../../build/reference/fr-fr-create-dot-sbr-file.md)).
