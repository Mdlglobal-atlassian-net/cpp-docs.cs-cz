# <a name="compiler-warning-level-1-c4402"></a>C4402 kompilátoru upozornění (úroveň 1)
musíte použít PTR – operátor  
  
 Typ se používá na operand bez PTR operátor při odkazující na nebo přetypování na typ v kódu vnořeného sestavení.  
  
 Kompilátor předpokládá operátor PTR.
