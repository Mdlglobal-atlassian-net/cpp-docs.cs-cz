# <a name="compiler-warning-level-1-c4402"></a>Kompilátor upozornění (úroveň 1) C4402

musí používat operátor PTR

Typ se používá s operandem bez operátor PTR při odkazování na nebo přetypování na typ v vložený kód sestavení.

Kompilátor předpokládá, operátor PTR.
