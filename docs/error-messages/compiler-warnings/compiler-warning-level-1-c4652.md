# <a name="compiler-warning-level-1-c4652"></a>Upozornění kompilátoru (úroveň 1) C4652

možnost kompilátoru Option je nekonzistentní s předkompilovanou hlavičkou. aktuální možnost příkazového řádku přepíše, která je definována v předkompilované hlavičce.

Zadaná možnost příkazového řádku se liší od, která byla zadána při vytvoření předkompilované hlavičky (. pch). Byla použita možnost zadaná v aktuálním příkazovém řádku.

Toto upozornění se může vyhnout tím, že znovu generuje předkompilovanou hlavičku s daným parametrem příkazového řádku.
