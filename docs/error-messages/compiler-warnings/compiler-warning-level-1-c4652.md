# <a name="compiler-warning-level-1-c4652"></a>C4652 kompilátoru upozornění (úroveň 1)
možnost kompilátoru "možnost" konzistentní s předkompilovaných hlaviček; Aktuální možnost příkazového řádku se přepíše která definována v předkompilovaných hlaviček  
  
 Danou možnost příkazového řádku se liší od zadané při vytváření předkompilovaných hlaviček (.pch). Možnost zadán v příkazovém řádku aktuální byl použit.  
  
 Toto upozornění můžete zabránit znovu vygenerovat předkompilovaných hlaviček s danou možností příkazového řádku.
