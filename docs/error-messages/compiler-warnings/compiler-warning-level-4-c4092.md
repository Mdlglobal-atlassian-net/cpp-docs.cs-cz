# <a name="compiler-warning-level-4-c4092"></a>Kompilátor upozornění (úroveň 4) C4092

sizeof vrací "unsigned long.

Operand `sizeof` operátor byla velmi velká, takže `sizeof` vrátila bez znaménka **dlouhé**. Toto upozornění se zobrazí pod rozšíření společnosti Microsoft ([/Ze](../../build/reference/za-ze-disable-language-extensions.md)). V části kompatibility ANSI (/Za) výsledek zkrácen místo.
