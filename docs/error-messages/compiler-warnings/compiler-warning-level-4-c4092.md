# <a name="compiler-warning-level-4-c4092"></a>Upozornění kompilátoru (úroveň 4) C4092

sizeof vrátí "unsigned long"

Operand operátoru `sizeof` byl velmi velký, takže `sizeof` vrátil **dlouhý**unsigned. Toto upozornění se objevuje v rámci rozšíření společnosti Microsoft ([/ze](../../build/reference/za-ze-disable-language-extensions.md)). V části Kompatibilita s ANSI (/za) se výsledek zkrátí.
