# <a name="compiler-warning-level-1-c4711"></a>Kompilátor upozornění (úroveň 1) C4711

Funkce 'function' vybraných pro vložené rozšíření

Kompilátor provádí vkládání na danou funkci, i když nebyl označen pro vkládání.

C4711 je povolená, pokud [/ob2](../../build/reference/ob-inline-function-expansion.md) určena.

Vkládání se provádí na základě vlastního uvážení kompilátoru. Toto upozornění je informační.

Toto upozornění je vypnuto ve výchozím nastavení. Pokud chcete povolit upozornění, použijte [varování #pragma](../../preprocessor/warning.md). Zobrazit [kompilátoru upozornění, že je vypnuto ve výchozím nastavení](../../preprocessor/compiler-warnings-that-are-off-by-default.md) Další informace.
