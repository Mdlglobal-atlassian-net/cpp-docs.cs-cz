# <a name="compiler-warning-level-1-c4711"></a>C4711 kompilátoru upozornění (úroveň 1)
funkce vybrané pro rozšíření vložené funkce  
  
 Kompilátor provést vložené na danou funkci, přestože nebyl označen pro vložené.  
  
 C4711 je povolena, pokud [/Ob2](../../build/reference/ob-inline-function-expansion.md) je zadán.  
  
 Vložené provádí uvážení kompilátoru. Toto upozornění je informační.  
  
 Toto upozornění je ve výchozím nastavení vypnutý. Povolit upozornění, použijte [#pragma – upozornění](../../preprocessor/warning.md). V tématu [kompilátoru upozornění, že jsou vypnout ve výchozím nastavení](../../preprocessor/compiler-warnings-that-are-off-by-default.md) Další informace.
