# <a name="compiler-warning-level-3-c4133"></a>Upozornění kompilátoru (úroveň 3) C4133

' type ': Nekompatibilní typy-z ' typ1 ' na ' typ2 '

Toto upozornění může být způsobeno tím, že se pokusí odečíst dva ukazatele různých typů.

Chcete-li se tomuto upozornění vyhnout, zadejte odpovídající přetypování.
