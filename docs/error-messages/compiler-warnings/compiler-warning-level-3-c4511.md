# <a name="compiler-warning-level-3-c4511"></a>Kompilátor upozornění (úroveň 3) C4511

'class': kopírovací konstruktor nelze generovat.

Kompilátor nelze vygenerovat výchozí konstruktor copy třídy; Základní třída může mít soukromý kopírovací konstruktor.
