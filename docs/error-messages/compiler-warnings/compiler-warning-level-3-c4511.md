# <a name="compiler-warning-level-3-c4511"></a>C4511 kompilátoru upozornění (úroveň 3)
'class': Nepodařilo se vygenerovat kopírovací konstruktor  
  
 Kompilátor nelze generovat výchozí kopie – konstruktor pro třídu; Základní třída může mít privátní konstruktor copy.
