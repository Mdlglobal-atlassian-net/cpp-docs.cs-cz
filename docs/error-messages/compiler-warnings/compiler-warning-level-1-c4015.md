# <a name="compiler-warning-level-1-c4015"></a>Kompilátor upozornění (úroveň 1) C4015

'identifier': typ bitového pole musí být integrálního typu

Bitové pole není deklarovaný jako typ integer. Kompilátor předpokládá základní typ bitového pole s bude bez znaménka. Bitová pole musí být deklarovány jako typy celých čísel bez znaménka.
