# <a name="compiler-warning-level-1-c4015"></a>C4015 kompilátoru upozornění (úroveň 1)
"identifikátor": typ bit pole musí být celočíselné  
  
 Bitová pole není deklarován jako typ celé číslo. Kompilátor předpokládá základní typ bit pole, které chcete být bez znaménka. Bitová pole musí být deklarován jako typy celé číslo bez znaménka.
