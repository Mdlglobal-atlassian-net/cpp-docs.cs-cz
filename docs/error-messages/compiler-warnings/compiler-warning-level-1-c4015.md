# <a name="compiler-warning-level-1-c4015"></a>Upozornění kompilátoru (úroveň 1) C4015

' identifier ': typ bitového pole musí být integrální

Bitové pole není deklarováno jako typ Integer. Kompilátor předpokládá základní typ bitového pole, které má být bez znaménka. Bitová pole musí být deklarována jako typy unsigned integer.
