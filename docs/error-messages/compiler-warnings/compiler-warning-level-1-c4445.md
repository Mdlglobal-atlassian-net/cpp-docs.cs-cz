# <a name="compiler-warning-level-1-c4445"></a>Upozornění kompilátoru (úroveň 1) C4445

' function ': virtuální metoda v WinRT nebo spravovaném typu nemůže být privátní.

Je-li virtuální funkce soukromá, nelze k ní přivodit odvozený typ. Chcete-li tuto chybu opravit, změňte přístupnost virtuální členské funkce na Protected nebo Public.
