# <a name="compiler-warning-level-1-c4088"></a>Upozornění kompilátoru (úroveň 1) C4088

' function ': Neshoda ukazatelů ve skutečném parametru ' Number ', formální parametr ' Number '

Odpovídající formální a aktuální parametry mají jinou úroveň dereference. Skutečný parametr se předává beze změny. Volaná funkce interpretuje svou hodnotu jako ukazatel.
