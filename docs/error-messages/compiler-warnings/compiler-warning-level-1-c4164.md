# <a name="compiler-warning-level-1-c4164"></a>Upozornění kompilátoru (úroveň 1) C4164

' identifier ': vnitřní funkce není deklarována

Zadaná vnitřní funkce není deklarována. Možná budete muset #include příslušný hlavičkový soubor.
