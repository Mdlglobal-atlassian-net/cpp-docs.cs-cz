# <a name="compiler-warning-level-1-c4116"></a>Upozornění kompilátoru (úroveň 1) C4116

Nepojmenovaná definice typu v závorkách

Struktura, sjednocení nebo Výčtový typ bez názvu jsou definovány ve výrazu kulatého závorky. Definice typu nemá smysl.

Ve volání funkce jazyka C má definice globální rozsah. Ve volání C++ funkce má definice stejný obor jako volání funkce.
