# <a name="compiler-warning-level-1-c4473"></a>Kompilátor upozornění (úroveň 1) C4473

> "*funkce*": nepředal není dostatečný počet argumentů pro zástupné symboly formátu řetězce a očekávají, že jejich parametrů *číslo* počet variadických argumentů, ale *číslo* zadaný je tento počet chybějící variadický argument *index* je vyžadován formátovacím řetězcem "*format_string*' Tento argument se používá ve specifikátoru převodu. Tento argument se používá jako šířka pole. Tento argument se používá jako přesnost. Tento argument se používá jako velikost vyrovnávací paměti

Kompilátor zjistil nesoulad mezi počet argumentů, které jsou nutné k uspokojení zástupné texty v řetězci formátu a počet argumentů zadána. Správné použití printf a scanf rodiny variadické funkce vyžaduje, abyste zadali tolik argumentů stanovené ve formátovacím řetězci. Některé symboly vyžaduje další argumenty, chcete-li určit šířku, přesnost nebo velikost vyrovnávací paměti, stejně jako argument pro obsah. Neshoda obecně znamená, že je chyba v kódu.

Informace pro argumenty přidružené k zástupné symboly printf řady formát řetězce najdete v tématu [syntaxe specifikace formátu: funkce printf a wprintf](../../c-runtime-library/format-specification-syntax-printf-and-wprintf-functions.md). Najdete v dokumentaci pro informace specifické pro funkce *funkce*.

Toto upozornění je nového v sadě Visual Studio 2015.

## <a name="example"></a>Příklad

Tyto ukázky zobrazit dva způsoby, jak formátovací řetězec může být následek upozornění C4473 neshoda jeho argumentů a také ukazují, jak opravit problémy.

```cpp
// c4473p.cpp
// compile with: cl /c /W4 c4473p.cpp
#include <stdio.h>

void print_func(int a, float f)
{
    printf_s("This takes %*f arguments.\n", f); // C4473, missing * width argument
    printf_s("This takes %*f arguments.\n", a, f); // fix: supply width argument
}
```

V první chybovou zprávu kompilátor zjistí, že argument nebyl nalezen, ale nemůže zjistit, zda je chybí argument šířku nebo obsahu, takže předpokládá, že zadaný argument je první z nich, šířku, a druhý je chybí argument , pro specifikátor převodu typu. Musí zkontrolovat řetězec formátu k určení skutečné chybí argument.

```cpp
// c4473s.cpp
// compile with: cl /c /W4 c4473s.cpp
#include <stdio.h>

void scan_func(int a, float f)
{
    char name[20];

    scanf_s("%s", &name);                // C4473, missing buffer size argument
    scanf_s("%s", &name, sizeof(name));  // fix: supply buffer size argument
}
```

V tomto příkladu **scanf_s** vyžaduje pro každý zástupného symbolu, jeden slouží k poskytování adresu k zápisu do a druhý k zadání velikosti první dva argumenty. V dokumentaci pro každou funkci knihovny vysvětlení povinnými argumenty.
