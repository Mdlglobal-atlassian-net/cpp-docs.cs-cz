# <a name="compiler-warning-level-1-c4098"></a>Upozornění kompilátoru (úroveň 1) C4098

' function ': funkce void vrací hodnotu

Funkce deklarovaná pomocí návratového typu [void](../../cpp/void-cpp.md) má příkaz `return`, který vrací hodnotu. Kompilátor předpokládá, že funkce vrátí hodnotu typu `int`.
