# <a name="compiler-warning-level-1-c4098"></a>Kompilátor upozornění (úroveň 1) C4098

'function': funkce void vrací hodnotu

Funkce deklarovaná s návratovým typem [void](../../cpp/void-cpp.md) má `return` příkazu, který vrací hodnotu. Kompilátor předpokládá, funkce vrátí hodnotu typu `int`.
