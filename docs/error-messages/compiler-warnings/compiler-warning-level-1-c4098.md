# <a name="compiler-warning-level-1-c4098"></a>C4098 kompilátoru upozornění (úroveň 1)
'function': void funkce návrat hodnoty  
  
 Funkce deklarovat s návratovým typem [void](../../cpp/void-cpp.md) má `return` příkaz, který vrátí hodnotu. Kompilátor předpokládá, funkce vrátí hodnotu typu `int`.
