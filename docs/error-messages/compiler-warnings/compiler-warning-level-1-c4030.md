# <a name="compiler-warning-level-1-c4030"></a>Kompilátor upozornění (úroveň 1) C4030

**první seznam formálních parametrů delší než druhý seznam.**

Funkce byla znovu deklarována s různými formální parametry. Kompilátor používá formální parametry zadané v první deklaraci.
