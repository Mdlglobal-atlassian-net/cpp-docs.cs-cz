# <a name="compiler-warning-level-1-c4030"></a>Upozornění kompilátoru (úroveň 1) C4030

**první seznam formálních parametrů je delší než druhý seznam.**

Funkce byla znovu deklarována s různými formálními parametry. Kompilátor používá formální parametry uvedené v první deklaraci.
