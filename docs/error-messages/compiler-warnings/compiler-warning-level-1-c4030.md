# <a name="compiler-warning-level-1-c4030"></a>C4030 kompilátoru upozornění (úroveň 1)
**první formální parametr seznamu delší než druhý seznamu**  
  
 Funkce byla znovu deklarována s odlišnými formální parametry. Kompilátor používá formální parametry v prvním deklaraci.
