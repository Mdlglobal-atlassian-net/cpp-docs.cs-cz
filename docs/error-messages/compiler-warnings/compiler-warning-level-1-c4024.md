# <a name="compiler-warning-level-1-c4024"></a>Upozornění kompilátoru (úroveň 1) C4024

' function ': různé typy pro formální a skutečný parametr ' Number '

Odpovídající formální a skutečné parametry mají různé typy. Kompilátor předá samotný parametr bez změny. Funkce přijímání převede typ parametru na očekávaný typ.
