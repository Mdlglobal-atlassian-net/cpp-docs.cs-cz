# <a name="compiler-warning-level-1-c4166"></a>Upozornění kompilátoru (úroveň 1) C4166

**Neplatná konvence volání pro konstruktor/destruktor**

Konstruktory a destruktory nemohou mít jiné konvence volání než výchozí pro platformu (kromě případů, kdy výslovně zadáte **__clrcall**).
