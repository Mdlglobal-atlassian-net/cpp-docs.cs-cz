# <a name="compiler-warning-level-1-c4166"></a>C4166 kompilátoru upozornění (úroveň 1)
**Neplatné konvenci volání pro konstruktor nebo – destruktor**  
  
 Konstruktory a destruktory nemůže mít konvence jiné než výchozí volání pro platformu (s výjimkou když explicitně zadáte **__clrcall**).
