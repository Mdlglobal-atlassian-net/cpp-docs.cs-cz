# <a name="compiler-warning-level-1-c4166"></a>Kompilátor upozornění (úroveň 1) C4166

**Neplatná konvence volání pro konstruktor/destruktor**

Konstruktory a destruktory nelze mít, než je výchozí konvencí volání v prostředí pro platformu (s výjimkou případů, kdy je explicitně zadat **__clrcall**).
