# <a name="compiler-warning-level-1-c4403"></a>Upozornění kompilátoru (úroveň 1) C4403

Neplatný operátor PTR

Operátor PTR se v vloženém kódu assembleru používá nevhodně.

Kompilátor ignoroval operátor PTR.
