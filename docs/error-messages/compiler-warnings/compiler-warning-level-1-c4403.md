# <a name="compiler-warning-level-1-c4403"></a>Kompilátor upozornění (úroveň 1) C4403

Neplatný operátor PTR

PTR – operátor se používá nevhodně v vloženého kódu assembleru.

Kompilátor ignoruje operátor PTR.
