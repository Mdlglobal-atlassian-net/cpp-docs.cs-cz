# <a name="compiler-warning-level-1-c4403"></a>C4403 kompilátoru upozornění (úroveň 1)
Neplatný PTR – operátor  
  
 Operátor PTR se nesprávně používá v vloženého assembleru kódu.  
  
 Kompilátor ignoruje operátor PTR.
