# <a name="compiler-warning-level-3-c4023"></a>Upozornění kompilátoru (úroveň 3) C4023

' symbol ': ukazatel based předaný do neprototypované funkci: číslo parametru

Předání ukazatele na neprototypované funkci způsobí, že bude ukazatel normalizován, s nepředvídatelnými výsledky.

Toto upozornění může být opraveno, pokud používáte funkce prototypu, které jsou předány na základě ukazatelů.
