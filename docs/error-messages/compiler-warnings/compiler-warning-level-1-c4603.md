# <a name="compiler-warning-level-1-c4603"></a>Upozornění kompilátoru (úroveň 1) C4603

\<> identifikátoru: makro není definované nebo se definice liší od použití předkompilované hlavičky.

Makro určené zástupným znakem *identifikátoru* je buď odlišné, nebo již není definováno po použití hlavičky předkompilace.
