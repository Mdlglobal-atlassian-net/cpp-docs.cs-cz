# <a name="compiler-warning-level-1-c4120"></a>C4120 kompilátoru upozornění (úroveň 1)
na základě unbased neshoda  
  
 Kompilátor nemůže převést mezi dvěma ukazatele, protože jeden na základě a druhý není.
