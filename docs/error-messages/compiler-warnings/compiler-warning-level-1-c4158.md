# <a name="compiler-warning-level-1-c4158"></a>Upozornění kompilátoru (úroveň 1) C4158

> Předpokládá se #pragma pointers_to_members (full_generality, dědičnost).

**#Pragma pointers_to_members (** *jedna* &#124; *vícenásobná* &#124; *virtuální* **)** byla vydána bez doprovodné **#pragma pointers_to_members (full_generality)** .
