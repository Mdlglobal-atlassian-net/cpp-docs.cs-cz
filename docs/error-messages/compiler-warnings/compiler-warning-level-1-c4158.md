# <a name="compiler-warning-level-1-c4158"></a>C4158 kompilátoru upozornění (úroveň 1)
za předpokladu, že pointers_to_members – #pragma (full_generality, dědičnost)  
  
 A **pointers_to_members – #pragma (***jedním &#124; několika &#124; virtuální***)** byl vydán bez doprovázejících **#pragma pointers_to_ Members(full_generality)**.
