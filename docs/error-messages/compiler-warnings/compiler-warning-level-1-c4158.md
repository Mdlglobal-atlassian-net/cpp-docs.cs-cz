# <a name="compiler-warning-level-1-c4158"></a>Kompilátor upozornění (úroveň 1) C4158

> za předpokladu, že #pragma pointers_to_members (full_generality, dědičnost)

A **#pragma pointers_to_members (** *jeden* &#124; *více* &#124; *virtuální* **)** byl vydán bez je v doprovodné **#pragma pointers_to_members(full_generality)**.
