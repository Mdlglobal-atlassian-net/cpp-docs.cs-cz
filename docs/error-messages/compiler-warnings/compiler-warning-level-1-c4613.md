# <a name="compiler-warning-level-1-c4613"></a>Kompilátor upozornění (úroveň 1) C4613

'segmentu': nelze změnit třídu segmentu

Pokusili jste se vytvořit segment s názvem třídy, segment, který používá kompilátor. Nebyla vytvořena žádná nová třída segmentu.
