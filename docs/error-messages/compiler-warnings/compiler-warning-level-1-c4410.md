# <a name="compiler-warning-level-1-c4410"></a>C4410 kompilátoru upozornění (úroveň 1)
Neplatná velikost operand  
  
 Jeden z operandy na pokyn měl nesprávnou velikost. Nejmenší velikost právní pro operand byl použit.
