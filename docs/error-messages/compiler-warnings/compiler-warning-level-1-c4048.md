# <a name="compiler-warning-level-1-c4048"></a>Upozornění kompilátoru (úroveň 1) C4048

různé deklarované podscripty pole: ' identifier1 ' a ' identifier2 '

Výraz zahrnuje ukazatele na pole jiné velikosti. Ukazatele jsou používány bez převodu.

Toto upozornění může být opraveno, pokud explicitně přetypujte pole na stejný nebo ekvivalentní typ.
