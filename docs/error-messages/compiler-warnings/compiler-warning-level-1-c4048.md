# <a name="compiler-warning-level-1-c4048"></a>Kompilátor upozornění (úroveň 1) C4048

různé deklarované subscripty pole: "identifier1" a "identifier2.

Zahrnuje výraz ukazatele na pole s jinou velikostí. Bez převodu se používají ukazatele.

Toto upozornění může opravit, pokud explicitně přetypován na stejné nebo ekvivalentní typ pole.
