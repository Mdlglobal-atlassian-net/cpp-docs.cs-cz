# <a name="compiler-warning-level-1-c4048"></a>C4048 kompilátoru upozornění (úroveň 1)
různé deklarovaný dolní indexy pole: 'identifier1' a 'identifier2.  
  
 Výraz zahrnuje ukazatele na pole jinou velikost. Následující ukazatele používají bez převodu.  
  
 Toto upozornění může být stanovena, pokud explicitně přetypovat na stejné nebo ekvivalentní typ pole.
