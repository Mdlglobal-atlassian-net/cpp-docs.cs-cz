# <a name="compiler-warning-level-1-c4340"></a>C4340 kompilátoru upozornění (úroveň 1)
'Hodnota': hodnota zabalené z kladné na zápornou hodnotu.  
  
 `enum` Hodnota je větší než největší `enum` kladnou hodnotu zabalené kolem na zápornou hodnotu.
