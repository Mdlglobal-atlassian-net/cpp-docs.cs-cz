# <a name="compiler-warning-level-1-c4340"></a>Upozornění kompilátoru (úroveň 1) C4340

' value ': hodnota zabalené z kladné na zápornou hodnotu

Hodnota `enum` je větší než největší `enum` kladná hodnota, která je zabalená kolem záporné hodnoty.
