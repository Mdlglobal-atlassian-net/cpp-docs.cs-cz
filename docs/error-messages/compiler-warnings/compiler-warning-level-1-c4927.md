# <a name="compiler-warning-level-1-c4927"></a>Kompilátor upozornění (úroveň 1) C4927

Neplatný převod; implicitně použit více než jeden uživatelsky definovaný převod.

Více než jeden uživatelsky definovaný převod implicitně platí pro jednu hodnotu--kompilátor nenalezl explicitní převod, ale našli převod, který se používá.

Následující ukázka generuje C4927:

```
// C4927.cpp
// compile with: /W1
struct B
{
   operator int ()
   {
      return 0;
   }
};

struct A
{
   A(int i)
   {
   }

   /*
   // uncomment this constructor to resolve
   A(B b)
   {
   }
   */
};

A f1( B& b)
{
   return A(b);
}

B& f2( B& b)
{
   return b;
}

A f()
{
   B b;
   return A(b);   // ok
   return f1(b);  // ok
   return b;      // C4927
   return B(b);   // C4927
   return f2(b);  // C4927
}

int main()
{
   B b;
   A a = b;
   A a2(b);
}
```
