# <a name="compiler-warning-level-1-c4631"></a>Upozornění kompilátoru (úroveň 1) C4631

Služba MSXML nebo XPath není k dispozici, komentáře k dokumentu XML nebudou zpracovány. reason

Vaše instalace modulu CLR (Common Language Runtime) neobsahovala potřebné soubory pro podporu zpracování komentáře k dokumentu. Přeinstalujte modul CLR (Common Language Runtime).
