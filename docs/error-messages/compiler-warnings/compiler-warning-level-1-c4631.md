# <a name="compiler-warning-level-1-c4631"></a>C4631 kompilátoru upozornění (úroveň 1)
MSXML nebo není k dispozici, XPath dokumentu XML, který komentáře nebude zpracováno. Důvod  
  
 Běžné instalace modulu runtime jazyka neměl potřebné soubory pro podporu komentáře doc zpracování. Znovu nainstalujte modul common language runtime.
