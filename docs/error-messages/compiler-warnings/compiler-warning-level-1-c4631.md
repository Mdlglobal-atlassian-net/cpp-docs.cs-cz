# <a name="compiler-warning-level-1-c4631"></a>Kompilátor upozornění (úroveň 1) C4631

MSXML nebo XPath není k dispozici, dokument XML, který nezpracují se komentáře. Důvod

Běžné instalace modulu runtime jazyka nemá potřebné soubory pro podporu zpracování komentář. Znovu nainstalujte modul common language runtime.
