# <a name="compiler-warning-level-1-c4045"></a>Upozornění kompilátoru (úroveň 1) C4045

' Array ': přetečení hranic pole

Pole má příliš mnoho inicializátorů. Nadbytečné Inicializátory se ignorují.

Ujistěte se, že prvky pole a Inicializátory odpovídají velikosti a množství.
