# <a name="visual-c-in-visual-studio"></a>Visual C++ v sadě Visual Studio

Microsoft Visual C++, obvykle zkrátí tak, aby Visual C++ nebo MSVC, je název pro C++, C a jazyk sestavení nástroje pro vývoj a knihovny, které jsou k dispozici jako součást Visual Studio v systému Windows. Tyto nástroje a knihovny umožňují vytvářet aplikace pro univerzální platformu Windows (UWP), nativní počítače a serverové aplikace systému Windows, knihovny napříč platformami a aplikace, které běží na Windows, Linux, Android, iOS, a také spravované aplikace a knihovny, které používají .NET Framework. Můžete zapsat nic z jednoduché konzolové aplikace do nejvíce sofistikované a složité aplikace pro Windows desktop z ovladačů zařízení a součásti operačního systému k hrám a platformy pro mobilní zařízení a z nejmenší zařízení IoT k Visual C++ více serverů s vysokým výkonem v cloudu Azure.
