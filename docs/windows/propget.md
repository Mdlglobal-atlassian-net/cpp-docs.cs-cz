# <a name="propget"></a>propget

## <a name="syntax"></a>Syntaxe  
  
```

```

 **Propget –** atribut C++ má stejné funkce jako [propget –](http://msdn.microsoft.com/library/windows/desktop/aa367145) MIDL atribut.
