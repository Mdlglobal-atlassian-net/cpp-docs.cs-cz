# <a name="resources-for-creating-a-game-using-directx"></a>Prostředky pro vytvoření hry s použitím rozhraní DirectX
Nejlepší hry pro stolní počítače vytvoříte s technologiemi Visual C++ a DirectX.  
  
## <a name="resources"></a>Prostředky  
 Jazyk Visual C++ je zahrnut ve všech edicích sady Visual Studio. Informace o sadě Visual Studio najdete v tématu [Visual Studio](http://www.visualstudio.com/) webu.  
  
 Pro [!INCLUDE[win81](../misc/includes/win81_md.md)], DirectX je součástí sady Windows SDK. Nemusíte stahovat žádné další položky. Další informace najdete v tématu [kde je sada DirectX SDK?](http://msdn.microsoft.com/library/windows/desktop/ee663275.aspx).  
  
 Chcete-li vytvořit DirectX aplikace pro Windows 7, Windows Vista nebo Windows XP, můžete stáhnout samostatné [DirectX SDK](http://www.microsoft.com/download/details.aspx?displaylang=en&id=6812).  
  
 [Grafiky DirectX a herní](http://msdn.microsoft.com/library/windows/desktop/ee663274\(v=vs.85\).aspx) informace a ukázky k vám pomůže začít pracovat s DirectX programování.  
  
 [Win32 – aplikace Windows](../windows/windows-desktop-applications-cpp.md) obsahuje články, které popisují, jak vytvořit základní aplikaci plochy Windows.  
  
 [Vývoj her](http://msdn.microsoft.com/library/windows/apps/hh452744.aspx) Popisuje vývoj her pro univerzální platformu Windows, ale platí také pro stolní hry mnoho konceptů.
