 Pokud atribut předchází Visual C++ element, který není v oboru atributu, blok atribut je zpracovaná jako komentář.

|---------------|-----------------|

|[Atributy třídy](../windows/class-attributes.md)|Platí to – klíčové slovo jazyka C++.|  
|[Atributy metody](../windows/method-attributes.md)|Platí pro metody ve třídě, třída typu coclass nebo rozhraní.|
