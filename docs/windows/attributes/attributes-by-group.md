# <a name="attributes-by-group"></a>Atributy podle skupin

C++ Atributy jsou uspořádány do následujících funkčních skupin.

|Atribut|Popis|
|---------------|-----------------|
|[COM – atributy](com-attributes.md)|Vložení kódu pro podporu řady oblastí vývoje modelu COM a .NET Framework vývoje společného jazykového modulu runtime.|
|[IDL – atributy](idl-attributes.md)|Povolte změnu souboru. idl ze souboru zdrojového kódu bez použití průvodce a bez znalosti struktury a syntaxe daného souboru.|
|[Atributy příjemce technologie OLE DB](ole-db-consumer-attributes.md)|Vložení kódu založeného na [šablonách příjemce OLE DB](../../data/oledb/ole-db-consumer-templates-reference.md)– pro vytvoření pracovního OLE DBho příjemce, který provádí úkoly, jako je například otevírání tabulek, spouštění příkazů a přístup k datům.|
|[Atributy kompilátoru](compiler-attributes.md)|Dodává se kompilátorem C++ Microsoftu.|

Informace o tom, jak definovat vlastní atributy pro aplikaci .NET Framework, naleznete v tématu [uživatelsky definované atributy](../../extensions/user-defined-attributes-cpp-component-extensions.md).
