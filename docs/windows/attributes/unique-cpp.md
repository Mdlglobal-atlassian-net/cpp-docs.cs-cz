# <a name="unique-c"></a>unique (C++)

Určuje jedinečný ukazatel.

## <a name="syntax"></a>Syntaxe

```cpp
[unique]
```

## <a name="remarks"></a>Poznámky

**Jedinečný** C++ atribut má stejné funkce jako [jedinečný](/windows/win32/Midl/unique) atribut MIDL.

## <a name="example"></a>Příklad

V ukázkovém příkladu použijte **jedinečný** [parametr](ref-cpp.md) .
