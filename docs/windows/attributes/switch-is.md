# <a name="switch_is"></a>switch_is

Určuje výraz nebo identifikátor fungující jako Discriminant sjednocení, který vybírá člena sjednocení.

## <a name="syntax"></a>Syntaxe

```cpp
[switch_is]
```

## <a name="remarks"></a>Poznámky

Atribut **switch_is** C++ má stejné funkce jako atribut [switch_is](/windows/win32/Midl/switch-is) MIDL.
