# <a name="array-attributes"></a>Atributy pole

Následující atributy se vztahují na pole nebo `SAFEARRAY`s.

|Atribut|Popis|
|---------------|-----------------|
|[library_block](library-block.md)|Umístí konstrukci uvnitř bloku knihovny souboru IDL.|
|[satype](satype.md)|Určuje datový typ `SAFEARRAY` struktury.|
|[string](string-cpp.md)|Označuje, že jednorozměrné pole **char**, **wchar_t**, `byte` (nebo obdobných) pole nebo ukazatel na takové pole musí být považované za řetězec.|
