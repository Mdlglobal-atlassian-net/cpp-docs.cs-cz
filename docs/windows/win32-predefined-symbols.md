# <a name="win32-predefined-symbols"></a>Předdefinované symboly systému Win32

Tyto symboly jsou definovány v hlavičkových souborech Win32 a podporují standardní funkce a akce aplikace systému Windows. Tyto symboly se používají hlavně u běžných prvků uživatelského rozhraní. Při práci s ovládacími prvky v editorech prostředků se tyto symboly zobrazí v [okno Vlastnosti](/visualstudio/ide/reference/properties-window) přidružených k běžným ovládacím prvkům. Například pokud má být na panelu nástrojů zobrazena ikona aplikace, ikona bude přidružena k symbolu IDI_SMALL v okně **vlastností** .

|||
|-|-|
|IDABORT|nad Dialogové okno, tlačítko přerušit|
|IDC_STATIC|nad Statický text v dialogovém okně|
|IDCANCEL|nad Dialogové okno, tlačítko Storno|
|IDD_ABOUTBOX|dialogový Dialogové okno o produktu|
|IDI_PROJECTNAME|ikona Ikona aktuálního projektu|
|IDI_SMALL|ikona Nízká ikona aktuálního projektu|
|IDIGNORE|nad Při použití s tlačítkem ignorovat na dialogových oknech|
|IDM_ABOUT|(položka nabídky) Používá se s help... O produktu...|
|IDM_EXIT|(položka nabídky) Použito se souborem... Ukončit...|
|IDNO|nad Dialogové okno, žádné tlačítko|
|IDOK|nad Dialogové okno, tlačítko OK|
|IDRETRY|nad Dialogové okno, tlačítko Opakovat|
|IDS_APP_TITLE|řetezce Název aktuální aplikace|
|IDYES|nad Dialogové okno, tlačítko Ano|

## <a name="requirements"></a>Požadavky

Win32
