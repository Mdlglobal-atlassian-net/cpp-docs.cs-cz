# <a name="finding-binary-data"></a>Hledání binárních dat
Můžete hledat řetězce ASCII nebo hexadecimální bajty. Například pokud chcete najít "text Hello", můžete hledat buď text "Hello" nebo pro "48 65 6C 6 C 6 f" (hexadecimální ekvivalentní).  
  
### <a name="to-find-binary-data"></a>Najít binární data  
  
1.  Z **upravit** nabídky, klikněte na tlačítko [najít](/visualstudio/ide/reference/find-command).  
  
2.  V **najít** pole, v rozevíracím seznamu vyberte předchozí hledaný řetězec nebo zadejte dat, které chcete najít.  
  
3.  Vyberte některé z **najít** možnosti.  
  
4.  Klikněte na tlačítko **najít další**.
