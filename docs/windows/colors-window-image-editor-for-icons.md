# <a name="colors-window-image-editor-for-icons"></a>Okno Barvy (editor obrázků pro ikony)
Okno barvy má dvě části:  
  
1.  Barevná paleta, což je pole barev vzorků, které představují barev, které můžete použít. Můžete kliknout na ukázky vybrat barvy popředí a na pozadí při použití nástroje grafiky.  
  
2.  Indikátor barvy, který se zobrazuje barvy popředí a na pozadí a selektory obrazovky a inverzní barvy.  
  
     ![Barvy – okno](../windows/media/vccolorswindow.gif "vcColorsWindow")  
Okno Barvy  
  
> [!NOTE]
>  **Obrazovky barva** a **inverzní barvy** nástroje jsou dostupné pouze pro ikony a kurzory.  
  
 Okno barvy s můžete použít [panelu nástrojů editoru obrázků](../windows/toolbar-image-editor-for-icons.md).
