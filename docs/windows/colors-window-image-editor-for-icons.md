# <a name="colors-window-image-editor-for-icons"></a>Okno Barvy (editor obrázků pro ikony)

**Barvy** okno má dvě části:

1. **Barvy palety**, což je pole vzorky barev, které představují barvy, které můžete použít. Můžete kliknout na ukázky a vyberte barvu popředí a pozadí, při použití nástroje grafiky.

2. **Indikátor barvy**, který zobrazuje barvy popředí a pozadí a selektory obrazovky a inverzní barvy.

   ![Okno barvy](../windows/media/vccolorswindow.gif "vcColorsWindow")<br/>
   Okno Barvy

> [!NOTE]
> **Barvu obrazovky** a **inverzní barvy** nástroje jsou dostupné jenom pro ikony a kurzory.

Můžete použít **barvy** okna [panelu nástrojů editoru obrázků](../windows/toolbar-image-editor-for-icons.md).
