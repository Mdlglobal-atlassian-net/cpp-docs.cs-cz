# <a name="custom-image-dialog-box-image-editor-for-icons"></a>Dialogové okno Vlastní obrázek (editor obrázků pro ikony)
Umožňuje vytvořit novou bitovou kopii s vlastní velikost a počet barev.
 **Šířka**  
 Místo pro zadání šířku vlastního obrázku v pixelech (1-512 limit 2048).  
  
 **Výška**  
 Místo pro zadání výšku pro vlastní obrázku v pixelech (1-512 limit 2048).  
  
 **Barvy**  
 Místo abyste si zvolili počet barev pro vlastní obrázek: 2, 16 nebo 256.
