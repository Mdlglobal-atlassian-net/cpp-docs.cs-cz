# <a name="setting-the-size-of-the-combo-box-and-its-drop-down-list"></a>Nastavení velikosti pole se seznamem a jeho rozevíracího seznamu
Pole se seznamem může velikost, při přidání do dialogových oken. Můžete také zadat velikost pole rozevíracího seznamu.  
  
### <a name="to-size-a-combo-box"></a>Velikost pole se seznamem  
  
1.  Ve vašem dialogovém okně vyberte ovládacího prvku pole se seznamem.  
  
     Na začátku pouze vlevo a vpravo úchyty jsou aktivní.  
  
2.  Úchyty slouží k nastavení šířka pole se seznamem.  
  
 Můžete také nastavit velikost svislé části rozevíracím seznamem.  
  
#### <a name="to-set-the-size-of-the-combo-box-drop-down-list"></a>Chcete-li nastavit velikost se seznamem se pole rozevíracího seznamu  
  
1.  Klikněte na tlačítko se šipkou rozevíracího seznamu na pravé straně pole se seznamem.  
  
     ![Šipka na pole se seznamem v projektu knihovny MFC](../mfc/media/vccomboboxarrow.gif "vcComboBoxArrow")  
  
     Osnova řízení změn zobrazíte velikost pole se seznamem s oblasti rozevíracího seznamu rozšířené.  
  
2.  Nižší úchyt použijte ke změně počáteční velikost oblasti rozevíracího seznamu.  
  
     ![Pole se seznamem & č. 45; nastavení velikosti pole v projektu knihovny MFC](../mfc/media/vccomboboxsizing.gif "vcComboBoxSizing")  
  
3.  Klikněte na šipku rozevíracího seznamu zavřete části rozevíracího seznamu pole se seznamem.  
  
 Informace o přidávání zdrojů do spravovaných projekty, najdete v tématu [prostředků v aplikacích plochy](https://msdn.microsoft.com/library/f45fce5x.aspx) v *rozhraní .NET Framework – příručka vývojáře.* Informace na ručně přidejte soubory prostředků na spravované projekty, přístup k prostředkům, zobrazení statické prostředky a přiřazení k vlastnosti řetězce prostředků najdete v tématu [vytváření souborů prostředků pro aplikace plochy](https://msdn.microsoft.com/library/xbx3z216.aspx). Informace o globalizace a lokalizace prostředků do spravovaných aplikací najdete v tématu [Globalizing a lokalizace aplikací .NET Framework](https://msdn.microsoft.com/library/h6270d0z.aspx).
