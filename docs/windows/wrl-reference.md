# <a name="wrl-reference"></a>Referenční dokumentace ke knihovně WRL

Tato část obsahuje referenční informace pro Windows Runtime C++ šablony knihovny (WRL).
  
> [!NOTE]
> Knihovna šablon C++ Windows Runtime definuje funkci, která podporuje infrastrukturu rozhraní knihovny šablon jazyka C++ Windows Runtime a není určena pro použití přímo v kódu. Tato funkce je uvedeno v této dokumentaci.
  
## <a name="in-this-section"></a>V tomto oddílu

[Microsoft::WRL – obor názvů](../windows/microsoft-wrl-namespace.md)<br/>
Definuje základní typy, které tvoří knihovna šablon C++ Windows Runtime.
  
[Microsoft::WRL::Wrappers – obor názvů](../windows/microsoft-wrl-wrappers-namespace.md)<br/>
Definuje typy prostředků pořízení je inicializace (RAII) obálky, které zjednodušují správu životnosti objektů, řetězce a obslužné rutiny.
  
[Microsoft::WRL::Wrappers::HandleTraits – obor názvů](../windows/microsoft-wrl-wrappers-handletraits-namespace.md)<br/>
Popisuje charakteristiky běžné typy prostředků pro popisovač.
  
[Windows::Foundation – obor názvů](../windows/windows-foundation-namespace.md)<br/>
Umožňuje základní funkce Windows Runtime, jako je vytvoření objektu a objektu pro vytváření.
