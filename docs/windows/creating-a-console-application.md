# <a name="creating-a-console-application"></a>Vytvoření konzolové aplikace

Nejjednodušší způsob, jak vytvořit konzolovou aplikaci Win32 je použít **Průvodce aplikací Win32**.

### <a name="to-create-a-win32-console-application"></a>K vytvoření konzolové aplikace Win32

1. Postupujte podle pokynů v tématu nápovědy [vytvoření projektu pomocí Průvodce aplikací jazyka Visual C++](../ide/creating-desktop-projects-by-using-application-wizards.md).

2. V **nový projekt** dialogu **projekt konzoly Win32** v **šablony** podokně otevřete průvodce.

3. Definujte vaši [nastavení aplikace](../windows/application-settings-win-32-project-wizard.md) pomocí [Průvodce aplikací Win32](../windows/win32-application-wizard.md).

   > [!NOTE]
   > Přeskočení tohoto kroku zachová průvodce výchozí nastavení.

4. Klikněte na tlačítko **Dokončit** zavřete průvodce a nově vytvořený projekt se otevře v **Průzkumníka řešení**.
