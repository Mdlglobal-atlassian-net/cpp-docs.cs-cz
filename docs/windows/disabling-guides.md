# <a name="disabling-guides"></a>Zákaz vodítek
Speciální klávesy můžete použít ve spojení s myší zakázat přichycení účinek příručky. Pomocí klávesy ALT zakáže přichycení důsledky tohoto průvodce vybrali. Přesunutí průvodce pomocí klávesy SHIFT zabrání zaseknutá ovládací prvky přesunutím podle vodítka.  
  
### <a name="to-disable-the-snapping-effect-of-the-guides"></a>Chcete-li zakázat přichycení účinek příručky  
  
1.  Přetáhněte je ovládací prvek **ALT** klíč.  
  
### <a name="to-move-guides-without-moving-the-snapped-controls"></a>Chcete-li přesunout příručky bez přesouvání zaseknutá ovládacích prvků  
  
1.  Přetáhněte je v průvodci **SHIFT** klíč.  
  
### <a name="to-turn-off-the-guides"></a>Chcete-li vypnout vodítka  
  
1.  Z **formátu** nabídce zvolte **nastavení průvodce**.  
  
2.  V [dialogové okno nastavení průvodce](../windows/guide-settings-dialog-box.md)v části **vodítka rozložení**, vyberte **žádné**.  
  
    > [!NOTE]
    >  Můžete také dvakrát klikněte na panelu pravítka pro přístup **nastavení průvodce** dialogové okno.  
  
 \- nebo –  
  
-   Na **formátu** nabídky, klikněte na tlačítko **Přepnout vodítka**.
