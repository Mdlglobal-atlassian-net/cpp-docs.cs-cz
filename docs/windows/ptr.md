# <a name="ptr"></a>ptr
Ukazatel se označí jako úplné ukazatel.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
[ptr]  
  
```  
  
## <a name="remarks"></a>Poznámky  
 **Ptr** atribut C++ má stejné funkce jako [ptr](http://msdn.microsoft.com/library/windows/desktop/aa367149) MIDL atribut.  
  
## <a name="example"></a>Příklad  
 Podívejte se na příklad pro [defaultvalue](../windows/defaultvalue.md) pro ukázkové použití **ptr**.
