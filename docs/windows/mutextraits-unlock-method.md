# <a name="mutextraitsunlock-method"></a>MutexTraits::Unlock – metoda

## <a name="syntax"></a>Syntaxe  
  
```  
inline static void Unlock(  
   _In_ Type h  
);  
```  
  
### <a name="parameters"></a>Parametry

 Popisovač objektu mutex.
