# <a name="unique-c"></a>unique (C++)

## <a name="syntax"></a>Syntaxe


**Jedinečný** C++ atribut má stejné funkce jako [jedinečný](http://msdn.microsoft.com/library/windows/desktop/aa367294) atribut MIDL.
