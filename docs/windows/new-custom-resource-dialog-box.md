# <a name="new-custom-resource-dialog-box"></a>Dialogové okno Nový vlastní prostředek
Umožňuje vytvořit nový vlastní prostředek.  
  
 **Typ prostředku**  
 Poskytuje textové pole pro zadání názvu vlastní typ prostředku. Visual C++ Automatická oprava název při ukončení **nový vlastní prostředek** dialogové okno.  
  
## <a name="remarks"></a>Poznámky  
 Nezadávejte název prostředků nebo ID, které je rezervován [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]. Vyhrazené názvy jsou DESIGNINFO, HWB, a TEXTINCLUDE a vyhrazené ID je 255.
