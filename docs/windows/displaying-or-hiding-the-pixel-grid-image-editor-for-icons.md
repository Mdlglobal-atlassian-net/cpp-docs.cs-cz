# <a name="displaying-or-hiding-the-pixel-grid-image-editor-for-icons"></a>Zobrazení nebo skrytí mřížky pixelů (editor obrázků pro ikony)
Pro všechny podoken Editor obrázků s faktoru zvětšení 4 nebo vyšší můžete zobrazit mřížky, které vymezuje jednotlivých pixelů v bitové kopii.  
  
> [!TIP]
>  Popisy tlačítek Zobrazit při přesunutí ukazatele nad tlačítka panelu nástrojů. Tyto tipy můžete identifikovat funkce každé tlačítko.  
  
### <a name="to-display-or-hide-the-pixel-grid"></a>K zobrazení nebo skrytí mřížky pixelů  
  
1.  Na **bitové kopie** nabídky, klikněte na tlačítko **nastavení mřížky**.  
  
2.  Vyberte **mřížky pixelů** zaškrtněte políčko Zobrazit mřížky, nebo zrušte zaškrtnutí políčka skrytí mřížky.  
  
 Informace o přidávání zdrojů do spravovaných projekty, najdete v tématu [prostředků v aplikacích plochy](https://msdn.microsoft.com/library/f45fce5x.aspx) v *rozhraní .NET Framework – příručka vývojáře.* Informace na ručně přidejte soubory prostředků na spravované projekty, přístup k prostředkům, zobrazení statické prostředky a přiřazení k vlastnosti řetězce prostředků najdete v tématu [vytváření souborů prostředků pro aplikace plochy](https://msdn.microsoft.com/library/xbx3z216.aspx).
