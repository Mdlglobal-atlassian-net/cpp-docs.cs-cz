# <a name="open-ltdevicegt-image-dialog-box-c-image-editor-for-icons"></a>Otevřít &lt;zařízení&gt; dialogového okna obrázku zařízení (Editor obrázků pro ikony) (C++)

Použijte toto dialogové okno otevřete obrázcích zařízení v projektech C++. Zobrazí seznam stávajících zařízení imagí v aktuální prostředek (bitové kopie, které jsou součástí aktuální prostředek).

### <a name="current-images"></a>Aktuálních Imagí

Obsahuje seznam imagí, které jsou zahrnuté v prostředku. Výběr typu image, kterou chcete otevřít.
