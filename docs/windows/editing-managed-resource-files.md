# <a name="editing-managed-resource-files"></a>Úprava spravovaných zdrojových souborů
Spravovaných zdrojových souborů (RESX) jsou soubory formátu XML. Když přidáte do projektu ze souboru spravovaných prostředků **přidat novou položku** dialogové okno, **spravované prostředky Editor** otevře ve výchozím nastavení.  
  
 Další informace najdete v tématu [prostředky v. Formát souboru RESX](http://msdn.microsoft.com/en-us/0c476133-87e4-47e8-b0ef-4b88f4ef3dc5).
