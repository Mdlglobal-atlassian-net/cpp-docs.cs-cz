# <a name="text-tool-dialog-box-image-editor-for-icons"></a>Dialogové okno Textový nástroj (editor obrázků pro ikony)
Použití **Text nástroj** dialogové okno Přidat text do kurzoru, rastrový obrázek nebo ikonu prostředků.  
  
 Chcete-li získat přístup k tohoto dialogového okna, otevřete [Editor obrázků](../windows/window-panes-image-editor-for-icons.md). Vyberte **nástroje** z **Image** nabídce a potom vyberte **Text nástroj** příkaz.  
  
 **Tlačítko písma**  
 Otevře se [dialogové okno písmo textového nástroje](../windows/text-tool-font-dialog-box-image-editor-for-icons.md), ve kterém můžete změnit písmo, styl nebo velikost písma v bodech kurzoru. Změny se použijí na text zobrazovaný v části textu.  
  
 **Textová oblast**  
 Zobrazí text, který se zobrazí v rámci prostředku. Tato oblast je původně prázdné.  
  
> [!NOTE]
>  Pokud **průhledné pozadí** není nastaven, pouze text bude umístěna do bitové kopie. Pokud **neprůhledné pozadí** nastavena ohraničující obdélník, naplní se [barva pozadí](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md), se umístí za text. Další informace najdete v tématu [výběr neprůhledných a průhledná pozadí](../windows/choosing-a-transparent-or-opaque-background-image-editor-for-icons.md).  
  
 Můžete klikněte pravým tlačítkem na **Text nástroj** dialogové okno pro přístup k výchozí místní nabídky, který obsahuje seznam standardních příkazů systému Windows.
