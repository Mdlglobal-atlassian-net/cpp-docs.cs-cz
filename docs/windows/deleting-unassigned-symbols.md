# <a name="deleting-unassigned-symbols"></a>Odstranění nepřiřazených symbolů
### <a name="to-delete-an-unassigned-unused-symbol"></a>Chcete-li odstranit symbol nepřiřazené (nepoužívané)  
  
1.  V [symboly prostředků – dialogové okno](../windows/resource-symbols-dialog-box.md), vyberte symbol, který chcete odstranit a klikněte na tlačítko **odstranit**.  
  
    > [!NOTE]
    >  Před odstraněním nepoužívané symbol v souboru prostředků, zkontrolujte, zda že se nepoužívá jako jinde v aplikaci ani soubory prostředků, které jsou zahrnuty v době kompilace.  
  

  
 **Požadavky**  
  
 Win32
