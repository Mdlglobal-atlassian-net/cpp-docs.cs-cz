Provede operaci vydání COM v rozhraní přidružené k této ComPtr.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
void InternalRelease();  
```  
  
## <a name="remarks"></a>Poznámky  
 Tato metoda je chráněný.
