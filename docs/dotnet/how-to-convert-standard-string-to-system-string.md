# <a name="how-to-convert-standard-string-to-systemstring"></a>Postupy: Převod standardního řetězce na typ System::String

Toto téma ukazuje, jak převést C++ řetězec standardní knihovny ([\<řetězec >](../standard-library/string.md)) na <xref:System.String>.

## <a name="example"></a>Příklad

```cpp
// convert_standard_string_to_system_string.cpp
// compile with: /clr
#include <string>
#include <iostream>
using namespace System;
using namespace std;

int main() {
   string str = "test";
   cout << str << endl;
   String^ str2 = gcnew String(str.c_str());
   Console::WriteLine(str2);

   // alternatively
   String^ str3 = gcnew String(str.c_str());
   Console::WriteLine(str3);
}
```

```Output
test
test
test
```
