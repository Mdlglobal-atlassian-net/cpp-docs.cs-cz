# <a name="net-programming-with-ccli-visual-c"></a>.NET – programování s C + +/ CLI (Visual C++)
Následující témata popisují, jak Visual C++, můžete vybrat platformě .NET.  
  
 Ve výchozím nastavení CLR – projekty vytvořené pomocí sady Visual Studio 2015 cílové rozhraní .NET Framework 4.5.2. Cíl rozhraní .NET Framework 4.6, když vytvoříte nový projekt v **nový projekt** dialogové okno, změňte cílový framework v rozevírací nabídce v horní polovině dialogového okna. Chcete-li změnit cílový framework pro existujícího projektu, Zavřít projekt, upravte soubor projektu (VCXPROJ) a změňte hodnotu verzi rozhraní Target Framework na 4.6. Při příštím otevření projektu, nastavení vstoupí v platnost.
