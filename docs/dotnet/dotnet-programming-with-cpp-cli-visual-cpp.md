# <a name="net-programming-with-ccli-visual-c"></a>.NET – programování s C + +/ CLI (Visual C++)

Toto téma popisuje, jak může Visual C++ zaměřit na platformě .NET.

Ve výchozím nastavení CLR projekty vytvořené pomocí sady Visual Studio 2015 cíleny na rozhraní .NET Framework 4.5.2. Pro cílové rozhraní .NET Framework 4.6 při vytváření nového projektu v **nový projekt** dialogu změnit cílovou architekturu v rozevírací nabídce nahoře uprostřed dialogového okna. Chcete-li změnit cílový rámec pro existující projekt, Zavřít projekt, upravte soubor projektu (.vcxproj) a změňte hodnotu cílovou architekturu na verzi 4.6. Při příštím otevření projektu, nastavení se projeví.
