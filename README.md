# <a name="visual-studio-documentation-for-microsoft-c"></a>Dokumentace k sadě Visual Studio pro Microsoft C++

Vítáme vás! Toto úložiště obsahuje zdrojové soubory pro technickou dokumentaci k pracovnímu postupu Microsoft C++ (MSVC nebo Visual C++). Články jsou publikovány v [dokumentaci jazyka C++ v sadě Visual Studio na webu dokumentace](https://docs.microsoft.com/cpp).

Dokumentace pro Visual Basic a Visual C# se nachází v samostatném úložišti na [https://github.com/dotnet/core-docs](https://github.com/dotnet/core-docs)portálu a dokumentace k sadě Visual Studio se nachází v úložišti, které se [https://github.com/Microsoft/visualstudio-docs](https://github.com/Microsoft/visualstudio-docs)nachází na adrese.

## <a name="contributing-to-the-documentation"></a>Přispívání do dokumentace

Uvítáme vaše příspěvky, abychom nám pomohli vylepšit MSVC docs. Komplexní průvodce pro přispívání do docs.microsoft.com najdete v tématu [Přehled příručky pro přispěvatele Microsoft docs](https://docs.microsoft.com/contribute). Podrobnosti o tom, jak přispět k dokumentaci k MSVC, najdete v [Průvodci přispívajícím](CONTRIBUTING.md).

Některé oblasti funkcí MSVC mají vlastní složky v tomto úložišti, `standard-library` jako jsou například články ve standardní knihovně jazyka c++, `ide` pro články specifické pro jazyk c++ na interaktivním vývojovém prostředí (IDE) sady Visual Studio a tak dále. `/media` Podsložka v každé složce obsahuje soubory obrázků pro články. [Průvodce přispívání](CONTRIBUTING.md) obsahuje další informace.
